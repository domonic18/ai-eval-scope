一、建构性原理提炼

以下5条原理从"电场强度"概念的完整建构过程中提炼，可迁移到其他学习情境中。点击每条原理查看详情。

1 从冲突情境中发现问题 驱动型
有效的建构始于一个让人困惑的问题——日常经验与已有认知的冲突产生了建构新知识的驱动力

示例体现：从"脱毛衣的噼啪声"出发，引出"看不见的力怎么传递的"这个核心问题

其他情境："为什么铁屑在磁铁周围会排列成特定图案？"

⚠ 注意事项：问题要具体可探究，避免过于宏大

2 用思想实验进行推理 方法型
当无法直接观察时，通过设计理想化情境推演逻辑关系

示例体现：引入试探电荷q，想象把它放在带电体Q周围不同位置，测量F随位置的变化

其他情境：伽利略斜坡实验推理、爱因斯坦追光思想实验

⚠ 注意事项：假设要明确，推理要符合逻辑

3 用比值定义法建立物理量 核心型
当物理量A与B成正比时，比值A/B可以定义一个新物理量，反映系统的某种固有属性

示例体现：F/q定义为电场强度E，与试探电荷无关，只反映电场本身性质

其他情境：密度（m/V）、速度（s/t）、电阻（U/I）

⚠ 注意事项：确认比值确实与试探量无关，且在不同情境下具有物理意义

4 用多种方式表达同一概念 表达型
完整的理解需要数学公式（精确）、图示（直观）和文字描述（解释力）相互印证

示例体现：E=F/q（公式）+ 电场线（图示）+ 文字描述三种方式

其他情境：函数的解析式+图像+表格、化学键的模型+电子式+描述

⚠ 注意事项：不同表达方式要一致，说明各方式的适用范围

5 通过极端情况检验理解 验证型
将概念放到极端条件下检验自洽性——出现矛盾说明理解还有漏洞

示例体现：q→0的极限、r→∞时E→0、点电荷附近E→∞

其他情境：牛顿第二定律在v→c时的失效、理想气体在高压下的偏离

⚠ 注意事项：极端检验要符合物理意义，不引入逻辑矛盾

二、建构性原理迁移应用

📝 练习1：建构"电势"概念

情境：电场中不同位置电荷受力不同，从能量角度看，电荷在不同位置具有不同的势能

建构任务：类比电场强度的建构过程，建构"电势"概念

建议使用原理：原理2（思想实验推理）+ 原理3（比值定义法）

脚手架提示：想想密度怎么从质量和体积中提取的；从"单位电荷的电势能"切入

表达：用公式（φ=Ep/q）、图示（等势面）和文字三种方式表达

📝 练习2：建构"电容器电容"概念

情境：两个导体板间加上电压，板上储存电荷；不同电容器在相同电压下储存的电荷量不同

建构任务：建构"电容"的物理概念和定量表达式

建议使用原理：原理3（比值定义法）+ 原理5（极端情况检验）

脚手架提示：找出影响电容的因素；用Q/U的比值试试看

表达：公式C=Q/U + 影响因素的实验结论 + 不同电容器的特点

  5. 完成两个迁移练习后，你觉得哪条原理最有用？哪条还需要多加练习？
