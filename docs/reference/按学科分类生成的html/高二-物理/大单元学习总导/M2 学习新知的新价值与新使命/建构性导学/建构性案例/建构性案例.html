案例一🧠 画出你的"电场大脑地图" 个人理解▼

情境：你已从示例中理解了电场强度概念。现在用一张图展示"电场知识是怎么长出来的"。

建构任务

制作一张个人化的"电场认知概念图"，展示电荷→电场强度→电场线→电势→电势差之间的逻辑关系。

建议使用原理

原理4 多方式表达——用概念图表达知识的结构

建构步骤
  1. 列出所有相关概念
  2. 确定概念之间的逻辑关系（包含、因果、递进）
  3. 用箭头和连接词标注关系
  4. 检查完整性
  5. 美化表达
🧩 脚手架：概念图模板、关键词连接词列表（"导致""由……定义""与……成正比"）
成果表达

A4纸大小的概念图，至少包含8个核心概念，用分级布局展示知识层次。

✅ 自评量规：[ ]包含至少8个静电场核心概念 [ ]箭头标注了概念间的关系 [ ]有清晰的分层结构
案例二🏭 解释静电除尘器的工作原理 生活应用▼

情境：工厂烟囱排放的废气含大量粉尘，静电除尘器利用电场让粉尘带电并吸附到电极上。

建构任务

基于电场知识建构对静电除尘器工作过程的理解方案。

建议使用原理

原理1 从冲突发现问题 + 原理4 多方式表达

建构步骤
  1. 分析粉尘在电场中会受到什么力
  2. 画出除尘器的结构简图
  3. 标注电场方向和粉尘运动轨迹
  4. 用文字描述完整工作过程
  5. 检查科学准确性
🧩 脚手架：教材中静电除尘的原理图、电场力分析模板
成果表达

静电除尘器工作原理示意图（含文字解释）。

✅ 自评量规：[ ]标注了电场方向 [ ]解释了粉尘如何带电 [ ]画出了粉尘的受力/运动轨迹
案例三📝 "电场到底是什么？"科学小短文 学科表达▼

情境：你被邀请向全班做3分钟"科学短讲"，主题是"电场到底是什么"。

建构任务

撰写一篇300-500字的科学解释短文，用类比+递进式逻辑说明"电场是一种物质"。

建议使用原理

原理2 思想实验推理 + 原理4 多方式表达

建构步骤
  1. 确定核心类比（如"磁场用铁屑显示、电场用试探电荷显示"）
  2. 按"现象→问题→推理→结论"的逻辑组织
  3. 加入1-2个形象比喻（如"看不见的风"）
  4. 检查语言是否通俗
  5. 试读一遍
🧩 脚手架：科学写作模板、常用类比库、简短报告结构要点
成果表达

300-500字科学短文，含至少一个类比。

✅ 自评量规：[ ]没有直接抄教材定义 [ ]用了至少一个生动的类比 [ ]逻辑链条完整（现象→问题→解释）
案例四🧤 设计教室防静电方案 问题解决▼

情境：冬天教室空气干燥，同学之间经常"电击"——开门被电、脱外套噼啪响、触摸屏异常。

建构任务

设计一份教室静电防护方案，包含原因分析和可行的解决方案。

建议使用原理

原理1 发现问题 + 原理5 极端检验

建构步骤
  1. 调查校园静电现象的具体表现
  2. 分析静电产生的物理原理（摩擦起电→电荷积累→放电）
  3. 提出3种以上解决方案
  4. 评估各方案的可行性和效果
  5. 形成书面方案
🧩 脚手架：问题-原因-方案分析表、静电防护知识卡片
成果表达

一页A4防静电方案书，含问题描述、原因分析和不少于3条建议。

✅ 自评量规：[ ]分析了静电产生的物理机制 [ ]建议方案有科学依据 [ ]方案考虑了成本可行性
案例五🧬 人体中的电场——跨学科信息图 跨学科▼

情境：人体细胞膜两侧存在约-70mV的电位差（膜电位），神经信号传递就是电场变化在神经元上的传播。

建构任务

制作一张"人体中的电场"跨学科信息图，整合物理（电场知识）和生物（神经传导）相关知识。

建议使用原理

原理2 思想实验推理 + 原理4 多方式表达

建构步骤
  1. 查阅神经信号传递的基本机制
  2. 找出其中涉及电场/电势差的关键环节
  3. 建立"膜电位变化→电场变化→信号传播"的因果链
  4. 用信息图形式呈现
🧩 脚手架：教科书中膜电位数据、神经传导示意图、跨学科信息图模板
成果表达

A4信息图，含至少3个学科关键术语和1个过程流程图。

✅ 自评量规：[ ]正确使用了膜电位、去极化等生物术语 [ ]标注了电场变化的方向和过程 [ ]信息图布局清晰易读
案例六🔋 假如静电可以被高效储存 未来畅想▼

情境：如果有一天我们可以像装水一样"装电"，利用静电储能技术（超级电容器），城市能源格局会发生什么变化？

建构任务

写一份未来畅想方案，基于电容知识，设想超高效静电储能对社会的影响。

建议使用原理

原理5 极端情况检验——把电容器的储能效率推到极限

建构步骤
  1. 回顾电容器储能的基本原理（E=½CU²）
  2. 设想储能密度提升100倍
  3. 分析对交通（电动车续航）、家庭（储能）、电网（调峰）的影响
  4. 思考可能的技术瓶颈
  5. 形成方案
🧩 脚手架：超级电容器研究进展资料、储能技术对比表
成果表达

500字以内未来畅想方案，含基于科学原理的合理外推。

✅ 自评量规：[ ]基于电容储能的基本原理 [ ]外推有合理假设 [ ]分析了至少3个社会领域的影响
案例七⚙️ 设计一个静电驱动的小装置 综合创新▼

情境：你想用静电原理做一个有趣的小装置——让物体"隔空"运动、分离混合物或做一个小马达。

建构任务

独立设计一个利用电场/静电原理工作的简易装置原型方案。

建议使用原理

全部5条原理 综合运用

建构步骤

  3. 画出装置结构草图

  6. 预期效果

设计方案书（含原理说明、结构图、操作步骤和预期效果）。
