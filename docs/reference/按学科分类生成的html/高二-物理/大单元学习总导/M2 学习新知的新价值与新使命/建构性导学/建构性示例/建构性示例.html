一、真实建构情境

冬天脱毛衣时听到的 "噼啪"声、用塑料梳子梳头后头发飘起的现象——这些都在提示你：电荷之间存在看不见的作用力。

冲突点：这些现象说明带电体之间有相互作用，但我们看不见这种作用——"力是怎么传过去的？"

核心问题：电荷之间的力究竟是怎么传递的？它们之间有没有某种"东西"？

二、建构目标

你将在本示例中建构的核心成果：

⚡ 一个可以定量描述电场强弱的物理量——电场强度

你将学会：从实验现象出发 → 提出科学问题 → 设计思想实验 → 形成物理概念。

三、建构过程示范（7步）

点击每一步展开详细推理内容 👇

1 观察与提问

从摩擦起电现象出发：用塑料梳子梳头发后，梳子能吸引小纸片；冬天脱毛衣时听到噼啪声、看到火星。

记录现象：带电体之间存在相互作用（吸引或排斥）。

提出问题："电荷之间的作用力如何描述？它们之间是否隔着空无一物的空间？"

2 寻找证据

回顾已学知识：库仑定律 —— 两个点电荷之间的力为 F = kQq / r²

关键认识：这个定律描述的是两个点电荷之间的相互作用力。但如果只有一个带电体 Q 呢？它周围是否存在某种东西？

3 思维实验：引入试探电荷

思想实验：如果只有一个带电体 Q，我怎样知道它周围"有什么"？

办法：引入一个很小的试探电荷 q，将它放在 Q 周围的各个位置。

结果：试探电荷 q 在不同位置受到的力大小和方向都不同——说明 Q 周围的空间确实存在某种"东西"，它在不同位置强弱不同。

4 从力到场：比值定义

试探电荷 q 受力 F 同时取决于 Q（产生者）和 q（被作用者）。如何只描述 Q 的性质？

关键发现：F/q 这个比值，与 q 无关！ 只取决于 Q 本身。

E = F / q

这就是电场强度的定义：单位试探电荷所受的电场力。

5 完善概念：方向与叠加

方向性：正试探电荷所受力的方向规定为电场方向。

叠加性：多个电荷产生的电场，在空间某点的总场强等于各电荷在该点产生场强的矢量和。

6 表达成果

用两种方式表达电场强度：

① 数学语言：E = F/q（定义式，适用于任何电场）

② 图像语言：电场线——从正电荷发出、终止于负电荷的曲线，其疏密表示场强大小，切线方向表示场强方向。

7 反思验证

检查这个定义在不同情境下是否自洽：

✔ 点电荷周围：E = kQ/r²（从库仑定律推导）→ 与定义式一致

✔ 匀强电场中：在平行板之间，E 处处大小相等、方向相同 → 符合直觉

✔ 任意电荷分布：场的叠加原理成立 → 自洽
