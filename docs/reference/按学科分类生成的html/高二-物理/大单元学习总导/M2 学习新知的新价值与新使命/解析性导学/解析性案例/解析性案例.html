生活情境 第1阶 · 脱毛衣的"火花"与摸门把手的"触电" ▶

案例材料：冬天脱毛衣时听到"噼啪"声，黑暗中能看到小火花。你用手靠近门把手前，有时会突然被电一下。

解析任务：解释脱毛衣时的"火花"和摸门把手的"触电"分别是什么物理现象？二者本质是否相同？

🔬 解析路径

1识别问题两种静电现象的本质归因
2确定对象毛衣纤维、人体、门把手
3拆解要素摩擦起电、电荷积累、电势差、放电
4梳理关系摩擦→电荷转移→人体带电→与地之间有电势差→击穿空气
5解释原因毛衣与化纤衣物摩擦使人体积累净电荷，与接地导体间形成高电势差，空气被电离
6寻找证据湿度越低现象越明显；金属门把手比木质更容易触发
7迁移判断油罐车尾部拖铁链——导走积累的静电荷

🏷️ 原理调用

电场强度（击穿场强≈3×10⁶ V/m）电势差静电感应

两根不同材料摩擦→电子从一物转移到另一物→人体成为带电体→人体与大地间形成电场→场强超过空气击穿阈值→空气电离发光/发声/放电。二者本质相同，都是静电放电现象。
校园情境 第2阶 · 通草球的"吸引→接触→弹开"四阶段 ▶

案例材料：物理课上老师用起电机使金属球带电，把轻质通草球用细线悬挂靠近金属球。通草球先被吸引过去接触金属球，然后又迅速弹开，不再靠近。

解析任务：解释通草球的"吸引→接触→弹开→不再靠近"全过程。

🔬 解析路径

1识别问题通草球经历四个阶段的受力和运动原因
2确定对象金属球、通草球、细线
3拆解要素电荷、电场力、接触起电、同性相斥
4梳理关系金属球带电→周围存在电场→通草球极化后被吸引→接触时电荷重新分配→带上同种电荷→被排斥
5解释原因通草球是中性的，在电场中极化产生感应电荷而被吸引；接触后分得同种电荷，电场力变为斥力
6寻找证据弹开后通草球稳定在某个偏离竖直方向的角度
7迁移判断静电复印机中感光鼓带电→吸附碳粉→转印到纸张

🏷️ 原理调用

电场电场力静电感应接触起电电荷守恒

未接触时通草球在非均匀电场中极化，等效偶极子受电场力指向场强更大的方向→接触后电荷中和并重新分配，通草球获得与金属球同种电荷→二者之间电场力为斥力→通草球偏离竖直方向。
家庭情境 第3阶 · CRT 电视屏幕为何"吸灰"？ ▶

案例材料：老式 CRT 电视机和电脑显示器屏幕使用时会吸附灰尘。用手靠近屏幕表面时汗毛会被吸引竖起，用湿润的抹布擦屏幕时灰尘很难一次性擦干净，反而越擦越花。

解析任务：分析屏幕吸附灰尘的原因，并解释为什么用湿抹布擦静电灰尘效果差。

🔬 解析路径

1识别问题屏幕带电与除尘困难的原因
2确定对象屏幕玻璃表面、灰尘颗粒、抹布、水
3拆解要素电子束轰击、表面电荷积累、电场、极化
4梳理关系电子枪轰击荧光屏→表面积累负电荷→形成电场→灰尘被极化吸附
5解释原因屏幕带电后对轻小物体有吸引力；湿抹布擦时水填充电场薄弱区域但不导走全部电荷
6寻找证据擦完后屏幕局部仍有静电吸力；用验电器靠近可见指针偏转
7迁移判断半导体车间防静电措施的本质也是消除表面积累电荷

🏷️ 原理调用

静电场带电体吸引轻小物体极化电势

CRT 屏幕电子束使表面带电→带电表面周围存在静电场→灰尘中分子/原子被极化→感应电荷与屏幕电荷间有引力→水是极性分子，湿擦时改变局部电场但水不导电（纯水）→电荷未导走，水分蒸发后静电依然存在。
学科核心 第4阶 · 三小球多次接触的电荷分配规律 ▶

案例材料：两个完全相同的金属小球 A 和 B，A 带电 +Q，B 不带电。让 A 和 B 接触后分开，再给一个不带电的小球 C，用 C 先后与 A 和 B 接触。

解析任务：求最终 A、B、C 三个小球各带多少电荷？若 A 开始带 -Q，结果如何？找出规律。

🔬 解析路径

分步推导：

A 与 B 接触 → 各得 Q/2
C 与 A 接触 → 各得 Q/4
C 与 B 接触 → 各得 (Q/2 + Q/4)/2 = 3Q/8
最终：A = Q/4, B = 3Q/8, C = 3Q/8

规律：相同导体接触→总电荷按接触球的数量均分。若初始为 -Q，符号反向但数值规律一致。

🏷️ 原理调用

电荷守恒定律电荷均分原理对称性分析

若四个球多次接触，电荷分配可以用等比数列描述。这种"对称性+守恒"的分析方法同样适用于动量守恒中的碰撞问题、热平衡中的温度分布问题等场景。
跨学科 第5阶 · 电鳐的生物电—物理建模 ▶

案例材料：电鳐体内有数千个"电板"串联组成的发电器官，每个电板可产生约 0.15V 的电势差。电鳐放电电压可达 600V，电流约 1A，放电持续时间约 2ms。

解析任务：估算电鳐需要多少个电板串联？估算一次放电释放的总能量。比较电鳐放电原理与电容器放电的异同。

🔬 解析路径

串联总电压 = 单板电压 × 数量 → 数量 = 600 / 0.15 = 4000 个
能量 W = UIt = 600 × 1 × 0.002 = 1.2 J

生物学家解剖发现电鳐发电器官由约 4000 个电板单元构成，与物理估算吻合！电鳐通过神经信号同步控制所有电板同时放电，实现瞬时高电压输出。

与电容器放电的异同：都是"电荷存储→快速释放"，但电容器存储的是静电能，电板存储的是化学能转化来的电势能。

🏷️ 原理调用

电势差串联电容储能 E=½CU²电能 W=UIt

社会议题 第6阶 · 工业区粉尘管道静电隐患与接地方案 ▶

案例材料：某工业区夜晚听到"滋滋"声，金属屋顶在干燥天气下偶尔出现小火花。调查发现是除尘排气管排放含大量粉尘的废气，排放管道未良好接地。

解析任务：分析排气管放电的原因，设计减少风险的技术方案，评估方案必要性。

🔬 解析路径

1识别问题工业粉尘管道静电积累与安全风险
2拆解要素摩擦起电（粉尘与管壁）、电荷积累、尖端放电
3解释原因粉尘非导体，摩擦产生的电荷无法快速导走，在管壁积累达到击穿阈值
4设计方案① 可靠接地——管道与大地相连；② 增大管道内壁导电涂层；③ 增加空气湿度；④ 安装静电消除器

🏷️ 原理调用

尖端放电电场强度接地静电屏蔽

当管道浮空电势可达数千伏至数万伏，一旦放电火花能量超过粉尘最低点燃能量即引发爆炸。起火事故统计中粉尘静电是常见诱因，干燥天气事故率显著升高。因此接地方案非常必要。
综合迁移 第7阶 · 静电扬声器——薄膜在电场中的振动 ▶

案例材料：一张极薄的导电薄膜（质量约0.1g）悬在两个固定金属板之间。当在薄膜与金属板之间施加变化的电压 U=500sin(ωt)V 时，薄膜振动发声。金属板间距 2mm。

解析任务：画出薄膜的受力分析图，推导薄膜所受电场力与电压的关系，说明这种结构如何实现精确控制薄膜振动。

🔬 解析路径

两端施加 U(t) → 板间场强 E = U/d → 薄膜上下表面感应异号电荷
电场力 F ∝ U² → 电压包含直流偏置和交流分量 → F中产生与交流信号线性相关的分量
→ 薄膜按音频信号振动 → 还原声音

静电扬声器（ESL）的频响曲线平坦度优于动圈扬声器，因薄膜质量极小，瞬态响应极好。同样的原理广泛应用于 MEMS 麦克风、加速度计等微机电系统。

🏷️ 原理调用

匀强电场 E=U/d电容 C=ε₀S/d电场力 F=qE静电力与电压平方关系受迫振动
