场景一：静电场实验室

在沉浸式3D实验室中观察并操作点电荷、匀强电场，直观理解电场线与等势面。

👁️ 观察任务

  • 观察正点电荷周围的电场线分布——从正电荷发出的蓝色辐射线
  • 切换为负点电荷，对比电场线方向的变化（指向负电荷的红色线条）
  • 添加第二个电荷形成电偶极子，观察电场线从正到负的弯曲路径
  • 开启"等势面"叠加模式，观察等势面与电场线始终垂直

🕹️ 操作任务

  • 用VR手柄拖拽电荷在空间中移动，观察电场线与等势面实时变形
  • 在匀强电场区域释放试探电荷，观察匀加速直线运动轨迹
  • 调整板间电压旋钮，观察试探电荷加速度的变化
  • 使用"场强计"工具读取任意位置的电场强度矢量

📝 记录任务

场景二：高压带电作业训练场

在模拟高压环境中体验静电平衡、尖端放电、静电屏蔽等概念。

👁️ 观察任务

  • 观察铁塔尖端（避雷针）周围密集的红色电场线——尖端附近场强极大
  • 观察高压输电线下方等势面分布：地面附近波浪状起伏
  • 进入法拉第笼，观察内部电场线全部消失——静电屏蔽

🕹️ 操作任务

  • 手持金属杆缓慢靠近高压带电体，观察尖端电晕放电（紫色光晕）
  • 将接地金属板放入电场中，观察电场线垂直终止于板面
  • 启动"闪电模拟"：调整云层电荷量，观察闪电路径

📝 记录任务

思考题：

  • 为什么高压设备表面要尽量做成光滑的球面或圆角？
    （结合尖端放电原理：尖端处场强极大，易发生电晕放电，造成能量损失和安全风险）
  • 静电屏蔽给我们的生活带来了哪些实际应用？
    （例如：电子设备的金属外壳屏蔽干扰、高压输电线的屏蔽线）

场景三：静电除尘工程师

扮演一名环保工程师，通过调整静电除尘器参数优化除尘效率。

👁️ 观察任务

  • 进入虚拟ESP内部，观察电晕电极周围强电场使气体分子电离
  • 观察带电粉尘颗粒在电场力下向集尘极板漂移的过程

🕹️ 操作任务

  • 调整电晕电压（30kV–80kV），观察击穿警告线与效率的关系
  • 改变极板间距，观察场强分布——太窄易击穿，太宽效率下降
  • 找到使出口排放达标的最优电压与间距组合
