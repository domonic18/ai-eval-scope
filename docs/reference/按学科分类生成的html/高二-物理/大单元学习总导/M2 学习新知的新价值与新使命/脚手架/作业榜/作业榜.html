📝 任务B-1：概念再现卡

  • 电容（C）——电容器的作用和电容的定义
自我检查清单：
  • 每个概念的定义能用自己的话表述，不依赖背诵

🔍 任务C-1：电场诊断报告

情境：一个带电量为+2×10⁻⁶ C的点电荷，固定在真空中。在距离它0.3 m处放置一个电量为+1×10⁻⁶ C的试探电荷。

  1. 计算试探电荷所受的库仑力大小和方向
  2. 计算该点的电场强度大小和方向
  3. 如果将试探电荷换成−1×10⁻⁶ C的电荷，上述两个结果分别变成多少？
  4. 写出你的诊断结论："电场强度由谁决定？电场力由谁决定？"
评分要点（rubric）：
  • ✅ 库仑定律公式使用正确（2分）
  • ✅ 电场强度定义式E=F/q使用正确（2分）
  • ✅ 方向描述正确（1分）
  • ✅ 换试探电荷后能判断：E不变，F大小不变但方向相反（2分）
  • ✅ 结论清晰：电场强度由电场本身决定，电场力由电场和试探电荷共同决定（3分）

总分10分，7分以上为通过

📊 适合你如果：概念基本掌握，想在真实情境中应用和深化。

📐 任务B-1：等势面分析报告

情境：匀强电场 E = 200 N/C，方向水平向右。A在B正左方0.1 m处，C在B正下方0.1 m处。

  1. 计算Uₐ_b，判断A、B哪点电势高
  2. 计算U_b_c
  3. 画出示意图：电场线方向、三点位置、等势面（至少三条）
  4. 电子从A→B，电场力做正功还是负功？电势能增加还是减少？
  5. 拓展思考：闪电电压高达数亿伏，用匀强电场模型估算云层与地面之间的电场强度
维度优秀（3分）合格（2分）需改进（1分）
计算全部正确，单位规范基本正确，少量笔误方向或符号错误较多
图示电场线、等势面标注清晰完整主要元素画了但不够完整缺失关键元素
分析推理能量转化分析准确，逻辑链完整分析正确但表述不够清晰逻辑链有断裂
拓展思考有自己的估算思路，数量级合理只套用公式无从下手或数量级严重偏离

总分12分，8分以上为通过

⚡ 适合你如果：概念和计算都已扎实，想挑战综合性、创造性的任务。

🏗️ 任务A-1：静电屏蔽装置设计方案

背景：精密电子设备需要屏蔽外部电场干扰。请你设计一个"简易静电屏蔽装置"。

  1. 原理分析（30%）——用静电平衡原理解释导体壳如何屏蔽外部电场
  2. 设计方案（40%）——材料选择、形状设计、孔洞问题、接地判断
  3. 安全提示与反思（20%）——实际操作注意事项和设计局限性
  4. 拓展连接（10%）——列举至少一个生活中的静电屏蔽实例
维度优秀（4分）合格（2-3分）需改进（0-1分）
原理分析静电平衡原理解释准确，图示完整标注清晰原理基本正确但图示不够完整概念错误或缺失关键环节
设计方案材料、形状、孔洞、接地等有具体分析有设计思路但不够具体方案笼统缺乏细节
安全与反思有实际考量，体现科学思维提到安全但不够具体缺少此部分
拓展连接实例与原理逻辑对应清晰举出了实例但分析较浅实例不准确或无拓展

总分16分，10分以上为通过
