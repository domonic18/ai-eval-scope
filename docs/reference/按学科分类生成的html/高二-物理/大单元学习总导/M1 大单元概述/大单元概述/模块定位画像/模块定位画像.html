📖 使用指南

  • 学习路线选择：根据你的基础和目标，在 M2~M20 中选择适配的模块组合
  • 路径可切换：学习过程中可根据掌握情况，随时切换到其他路径
  • 跨模块连接：学习 M5（艺术思维）时可回顾 M8（科学思维）中的电场线概念
  • 输出导向：最终 M20 的学术总结应整合所有模块的学习成果
