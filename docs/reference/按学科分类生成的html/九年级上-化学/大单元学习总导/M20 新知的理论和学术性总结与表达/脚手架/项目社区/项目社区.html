📋 短周期项目 · 1-3天
⚡ 短周期

化学知识卡片制作

⏱ 1-3天 · 个人完成

将「走进化学世界」核心概念制作成一套知识卡片，方便随时翻阅和复习。

项目目标：制作一套知识卡片，覆盖第1章核心概念。

阶段任务：

  • 选择8-10个核心概念（化学变化、物理变化、化学性质、物理性质、质量守恒等）
  • 为每个概念设计卡片（正面：概念名称；背面：定义+关键特征+例子）
  • 检查卡片内容的准确性和表达的规范性

成果形式：一套纸质或电子知识卡片

✅ 评价标准
  • 覆盖至少8个核心概念
  • 定义准确，使用规范的化学语言
  • 每个概念配1-2个生活例子

📊 中周期项目 · 1-2周
🔥 中周期

生活中的化学变化调查报告

⏱ 10天 · 个人或小组（2-3人）

调查并整理日常生活中5-8个化学变化实例，用化学语言进行系统分析和学术表达。

项目目标：发现身边的化学，学会用化学视角观察世界。

阶段任务：

  • 第1-2天：确定调查范围和方向（厨房、卫生间、花园、街道……）
  • 第3-4天：实地调查，拍照记录，收集现象和数据
  • 第5-6天：分析每个现象，判断变化类型，寻找背后的化学原理
  • 第7-8天：撰写调查报告，使用规范的化学术语和学术表达
  • 第9-10天：修改完善，制作展示材料

成果形式：图文结合的调查报告

✅ 评价标准
  • 至少包含5个化学变化实例
  • 每个实例都有准确的现象描述和科学的化学解释
  • 报告结构完整，使用了规范的化学语言
  • 有学生的个人见解和反思

🎪 长周期项目 · 1个月+
🌱 长周期

化学新知学术展览

⏱ 4周 · 小组合作（3-4人）

策划并完成一场「化学新知学术展览」，向全校师生展示你从「走进化学世界」中学到的新知识。

项目目标：综合运用所学知识，以展览形式进行学术表达。

阶段任务：

  • 第1周：确定展览主题和展示框架，分工协作
  • 第2周：收集素材，包括核心概念图和化学变化案例
  • 第3周：制作展板内容（图文并茂，学术语言表达）
  • 第4周：排练讲解词，布置展区，开幕展示

成果形式：

  • 2-3张学术展板（含核心概念、实验案例、化学变化图解）
  • 一份展览宣传册
  • 3分钟口头讲解
✅ 评价标准
  • 展板内容覆盖第1章核心知识
  • 学术表达准确规范，语言面向听众
  • 视觉效果清晰美观
  • 讲解自然流畅，能回答观众提问
