📌 议题1（基础）：物理变化和化学变化的界限真的那么清晰吗？

讨论引导：你在生活中是否遇到过"既像物理变化又像化学变化"的现象？比如：电灯发光发热是物理变化还是化学变化？蜡烛燃烧时蜡熔化是物理变化，为什么？请用教材中的判断标准来分析。

📌 议题2（进阶）：拉瓦锡的实验设计高明在哪里？

讨论引导：如果让你来设计一个实验证明"物质燃烧后质量不变"，你会怎么做？拉瓦锡的密封装置比你设计的方案好在哪里？

📌 议题3（挑战）：没有标准的化学语言，科学会怎样？

讨论引导：在拉瓦锡之前，化学家们各用各的术语，互相难以交流。想象一下，如果化学术语没有统一标准，今天的化学课堂会是什么样？
