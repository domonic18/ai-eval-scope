👁️ 任务一：化学实验室虚拟漫游

目标：在VR环境中"走进"一个标准的化学实验室，熟悉实验器材的摆放位置和使用场景。

观察要点：

🧪

实验台布局

观察实验台的区域划分，了解各项功能区的分布

🔬

常见仪器

试管、酒精灯、烧杯、量筒等的外观和存放位置

🛡️

安全设施

灭火器、洗眼器、通风橱的位置和使用方法

🧴

试剂存放

化学试剂的分类存放方式和标签规范

📝 记录要求：画出你观察到的实验室布局简图，标注至少 8种 仪器的位置。

🔬 任务二：微观粒子世界

目标：通过VR进入物质的微观世界，观察分子、原子的运动。

观察要点：

  • 不同状态下粒子的排列方式（固态、液态、气态）
  • 粒子间距离的变化
  • 粒子的运动方式（振动、移动、自由运动）

📝 记录要求：画出三种状态下粒子的排列示意图，标注你的发现。

🖐️ 虚拟实验操作：碳酸氢铵加热分解

目标：在VR环境中完成"碳酸氢铵加热分解"虚拟实验。

操作步骤（点击标记完成 👇）：

  • 选择并安装实验仪器
  • 取用碳酸氢铵
  • 加热操作
  • 观察并收集生成的气体
  • 检验生成物

📝 记录要求：记录每一步的操作要点和观察到的现象。
