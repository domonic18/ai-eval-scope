案例一 · 生活情境

🥩 铁钉生锈是化学变化吗？

🏠 生活化学 🔧 全面观察法 + 证据链构建
📝 案例材料：你发现一枚放在院子里的铁钉，表面出现了红褐色的锈斑。和钉子原有的银白色表面相比，生锈的部分变得松软、粗糙。爸爸说"这是铁被氧化了"，妈妈说"这就是铁烂了"。
🎯 解析任务：请分析铁钉生锈的过程属于哪种变化类型？你的判断依据是什么？

🔗 解析路径：识别现象 → 确定分析对象 → 拆解要素 → 寻找证据 → 判断结论

项目变化前变化后新物质生成？结论

案例二 · 校园情境

🔥 酒精灯燃烧的现象分析

🏫 课堂实验 🔧 多维度分析 + 排除法思维
📝 案例材料：化学课上，老师点燃了酒精灯。你观察到：灯芯上的酒精先变成蒸汽，然后蒸汽燃烧产生蓝色火焰。同时，烧杯底部出现了黑色的炭黑。玻璃棒伸到火焰中取出后，发现玻璃棒上也有黑色物质。
🎯 解析任务：请分析酒精燃烧的过程中，哪些环节属于物理变化，哪些属于化学变化？炭黑的出现说明了什么？

🔗 解析路径：观察现象 → 区分变化类型 → 分析原因 → 得出结论

现象变化类型判断依据
酒精受热变成蒸汽物理变化状态改变，无新物质生成
蒸汽燃烧产生蓝色火焰化学变化生成CO₂和H₂O
烧杯底出现黑色炭黑化学变化不完全燃烧生成碳单质
玻璃棒上附着的黑色物质化学变化同样为碳单质（新物质）

💡 炭黑的启示：酒精不完全燃烧会生成碳，说明燃烧的条件（氧气量）会影响产物。

案例三 · 家庭情境

🧪 厨房里的"生气"现象

🍳 生活化学 🔧 全面观察法 + 结论规范化表达
📝 案例材料：你在厨房中发现妈妈用小苏打（碳酸氢钠）和白醋混合去除了水壶中的水垢。混合时产生了大量气泡，水垢逐渐消失了。妈妈告诉你，这叫"酸碱中和"。
🎯 解析任务：请分析这个过程中发生了什么类型的化学反应？产生气泡和去除水垢分别对应什么化学过程？

🔗 解析路径：识别现象 → 分析反应类型 → 解释原理

现象反应物生成物变化类型解释
产生大量气泡碳酸氢钠+醋酸CO₂气体化学变化酸与碳酸盐反应生成CO₂
水垢消失水垢+醋酸可溶性醋酸盐化学变化酸碱中和反应

💡 规范化表达：碳酸氢钠（NaHCO₃）与醋酸（CH₃COOH）反应，生成醋酸钠（CH₃COONa）、二氧化碳（CO₂）和水（H₂O）。

案例四 · 学科核心情境

⚛️ 物理变化 vs 化学变化的本质对比

📚 核心概念 🔧 证据链构建 + 多维度分析
📝 案例材料：以下3组实验：
实验A：水加热沸腾变成水蒸气，冷凝后变回水。
实验B：镁条点燃发出耀眼白光，生成白色粉末（氧化镁）。
实验C：食盐溶在水中，蒸发后食盐重新出现。
🎯 解析任务：分析三组实验的变化类型，从"分子/原子"层面解释物理变化和化学变化的本质区别。

🔗 解析路径：识别实验 → 判断变化类型 → 分析分子层面本质 → 总结规律

实验变化类型宏观证据微观本质
A. 水→水蒸气→水物理变化状态改变，无新物质分子间距离改变，分子种类不变
B. 镁条燃烧化学变化生成白色粉末（MgO）原子重新组合，分子种类改变
C. 食盐溶解再蒸发物理变化食盐重新出现，无新物质离子分散再聚集，分子种类不变

💡 本质区别：化学变化中分子被破坏，原子重新组合成新分子；物理变化中分子本身不变。

案例五 · 跨学科情境

🌬️ 生物呼吸中的化学变化

🧬 化学×生物 🔧 多维度分析 + 结论规范化表达
📝 案例材料：人在呼吸时，吸入的是含氧气的空气，呼出的是含二氧化碳的气体。生物老师告诉你，这叫"细胞呼吸"——葡萄糖在细胞内与氧气反应，释放能量的过程。
🎯 解析任务：从化学变化的角度分析细胞呼吸：它符合化学变化的特征吗？反应物和生成物分别是什么？能量如何转化？

🔗 解析路径：列出反应物和生成物 → 判断是否为新物质 → 分析能量变化

反应物反应条件生成物变化类型能量变化
葡萄糖（C₆H₁₂O₆）+ 氧气（O₂）酶催化CO₂ + H₂O + 能量化学变化化学能→热能（维持体温）+ ATP（供细胞使用）

💡 跨学科联系：化学反应式 C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 能量，这说明生物体的生命活动本质上也是化学反应过程。

案例六 · 社会议题情境

🔋 废旧电池为什么不能随便扔？

🌍 环境化学 🔧 证据链构建 + 全面观察法
📝 案例材料：小区垃圾分类宣传栏写着"废旧电池属于有害垃圾"。电池外壳印着"含汞、镉、铅等重金属"。环保专家说，一节电池可以污染60万升水。
🎯 解析任务：分析重金属如何进入环境？涉及什么化学变化？为什么重金属对环境有害？

🔗 解析路径：分析电池构成 → 追踪重金属的"去向" → 分析化学变化 → 判断环境影响

原因过程化学变化环境影响
电池外壳腐蚀破裂重金属（Hg、Cd、Pb）泄漏到土壤和水中金属被氧化为离子（Hg²⁺、Cd²⁺、Pb²⁺）重金属离子被生物吸收，沿食物链富集
垃圾填埋或焚烧重金属进入地下水或大气部分转化为毒性更强的化合物污染水源，危害人体健康

💡 化学视角：重金属离子的毒性在于它们能与人体内的蛋白质结合，使之失去正常功能——这是化学变化在环境问题中的体现。

案例七 · 综合迁移情境

🔄 从实验现象到理论总结

🎯 综合运用 🔧 综合运用全部解析性原理
📝 案例材料：化学课上你做了以下实验：
实验一：将铜丝在酒精灯上加热，表面变黑。
实验二：将变黑的铜丝放到氢气中加热，铜丝恢复原来的颜色。
🎯 解析任务：分析两次加热分别发生了什么化学变化？写出文字表达式。从"得氧"和"失氧"角度发现规律。

🔗 解析路径：观察现象 → 分析反应物和生成物 → 寻找规律 → 总结原理

实验阶段现象反应物生成物变化类型规律发现
加热铜丝表面变黑Cu + O₂CuO化学变化铜获得氧 → 被氧化
黑铜丝+H₂加热恢复铜色CuO + H₂Cu + H₂O化学变化氧化铜失去氧 → 被还原

💡 核心规律：物质"得氧"的过程是氧化反应，物质"失氧"的过程是还原反应。氧化和还原是同时发生的——这就是"氧化还原反应"的本质。
