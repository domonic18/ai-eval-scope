建构性案例：七阶适配案例导学

M20 新知的理论和学术性总结与表达 · 九年级化学

以下 7 个案例由浅入深，帮助你运用建构性原理，在不同情境中主动建构对化学新知的系统化理解和学术化表达。

💡 点击每个案例展开详情。每个案例都包含：建构情境 → 建构任务 → 原理调用提示 → 建构步骤 → 脚手架推荐 → 成果表达 → 自评量规

1

我的"化学变化"概念图

▼

📍 真实建构情境：你已经学完"走进化学世界"第 1 章，头脑中有很多零散的知识点——物理变化、化学变化、物理性质、化学性质、实验仪器……现在请你把这些知识点编织成一张概念图。

🎯 建构任务：用一张概念图展示"化学变化"这个核心概念与其他概念之间的关系。

🔬 原理调用提示 二分归类法——先判断哪些概念属于"变化"，哪些属于"性质"。

📋 建构步骤：

  1. 列出所有学过的概念
  2. 找出核心概念
  3. 分类归纳
  4. 画出关系
  5. 标注连接词
🔧 脚手架推荐：使用 XMind 思维导图软件，或手绘在 A4 纸上

📤 成果表达：A4 纸大小的概念图，包含至少 10 个概念，用箭头和连接词标明关系。

✅ 自评量规：
是否包含"化学变化"核心概念
是否区分了变化和性质
是否有清晰的层级结构
2

家庭化学变化调查报告

▼

📍 真实建构情境：你的厨房就是一个化学实验室！做饭时发生的变化，哪些是物理变化，哪些是化学变化？

🎯 建构任务：调查你家厨房中 3 个化学变化实例和 3 个物理变化实例，写出调查报告。

🔬 原理调用提示 现象驱动法——仔细观察每个变化的现象。

📋 建构步骤：

  1. 确定调查对象
  2. 观察并记录现象
  3. 判断变化类型
  4. 写出报告
🔧 脚手架推荐：观察记录表模板（现象、变化前、变化后、变化类型）

📤 成果表达：图文结合的调查报告，每个变化配一张照片或示意图。

✅ 自评量规：
是否包含了 6 个实例
每个实例是否都有现象描述
变化类型判断是否正确
3

实验室安全操作指南

▼

📍 真实建构情境：新同学第一次进化学实验室，不知道如何安全操作。

🎯 建构任务：设计一份"化学实验室安全操作指南"海报。

🔬 原理调用提示 模型表达法——将安全规范可视化表达出来。

📋 建构步骤：

  1. 列出常见危险操作
  2. 总结安全规则
  3. 设计海报布局
  4. 制作成品
🔧 脚手架推荐：查阅教材中的实验室安全规则

📤 成果表达：A3 大小的海报，包含至少 8 条安全规则和对应的图示。

✅ 自评量规：
是否涵盖了教材中的所有安全规则
是否有图示辅助理解
语言是否清晰易懂
4

用化学语言描述实验

▼

📍 真实建构情境：你在实验室做了一个"加热碳酸氢铵"的实验，现在需要用规范的化学语言描述这个过程。

🎯 建构任务：写一份"加热碳酸氢铵"的实验描述报告。

🔬 原理调用提示 学术语言转化法——将观察到的事实转化为化学语言。

📋 建构步骤：

  1. 记录实验现象
  2. 确定变化类型
  3. 用化学语言描述
  4. 总结规律
🔧 脚手架推荐：实验报告模板（目的、现象、化学方程式、结论）

📤 成果表达：包含实验目的、实验现象、化学方程式、结论的结构化报告。

✅ 自评量规：
实验现象描述是否完整
是否使用了化学术语
结论是否基于证据
5

跨学科视角——燃烧的三要素

▼

📍 真实建构情境：消防员说"燃烧需要三个条件"，物理学老师说"燃烧释放能量"，化学老师说"燃烧是氧化反应"。

🎯 建构任务：建构一个跨学科视角的"燃烧综合解释模型"。

🔬 原理调用提示 证据链建构法——整合来自不同学科的证据。

📋 建构步骤：

  1. 收集各学科对燃烧的解释
  2. 找出共同点
  3. 建构综合模型
  4. 展示
🔧 脚手架推荐：三栏对比分析表（化学视角 | 物理视角 | 生活视角）

📤 成果表达：一幅包含化学、物理、生活视角的"燃烧综合解释图"。

✅ 自评量规：
是否包含了 3 个学科视角
各视角是否有证据支持
是否整合为一个连贯的模型
6

环境问题中的化学原理

▼

📍 真实建构情境：新闻报道说"某地区酸雨严重，导致了建筑腐蚀和土壤酸化"。

🎯 建构任务：建构一个"酸雨形成的化学过程解释模型"。

🔬 原理调用提示 模型表达法 + 证据链建构法

📋 建构步骤：

  1. 分析新闻中的现象
  2. 找出化学原因
  3. 建构形成过程
  4. 提出建议
🔧 脚手架推荐：原因—过程—结果分析框架

📤 成果表达：一份"酸雨的化学过程"说明图，包含反应过程和对环境的影响。

✅ 自评量规：
是否解释了酸雨的形成原因
是否用化学语言表达
是否提出了可行的缓解建议
7

化学新知学术展览

▼

📍 真实建构情境：学校要举办"化学新知学术展览"，你的任务是设计一个展板，向全校师生展示"走进化学世界"这一章的核心新知识。

🎯 建构任务：设计一份"化学新知学术展览展板"。

🔬 原理调用提示 综合运用所有五条建构性原理

📋 建构步骤：

  1. 梳理全章知识点
  2. 确定展示主题
  3. 设计展板结构
  4. 制作展板
  5. 准备讲解词
🔧 脚手架推荐：参考科学展览的展板设计范例（标题+核心概念图+关键实验+学术总结）

📤 成果表达：一张展板（含标题、核心概念图、关键实验、学术总结）+ 3 分钟讲解词。

✅ 自评量规：
是否覆盖了第 1 章核心内容
展板是否有清晰的逻辑结构
讲解词是否面向听众
是否体现了学术化表达
