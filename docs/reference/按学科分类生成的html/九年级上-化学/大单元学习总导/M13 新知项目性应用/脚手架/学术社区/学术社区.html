家校连接

支持与反馈

  • 家庭化学角：在家找3-5个涉及化学变化的物品（发酵面团、铜器生锈……），拍照分享到班级群
  • 家长互动：邀请家长分享他们工作和生活中的化学知识
  • 学习反馈机制：同学互评 + 家长反馈 + 自我反思
