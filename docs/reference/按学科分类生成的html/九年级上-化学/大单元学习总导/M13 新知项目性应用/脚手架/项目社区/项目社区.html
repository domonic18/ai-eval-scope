📌 什么是项目库？

项目库以真实问题为驱动，是长期学习的"引擎"。通过"做项目"在真实情境中深化化学理解，从短周期微项目到长周期综合项目，逐步提升探究能力。

🔍 微项目 1-3天
🔥 专题项目 1-2周
🗺️ 综合项目 1月+

🔍 项目1：家庭化学侦探

▼

在家中找出3-5处正在发生或已经发生的化学变化，记录并分析。

⏱️ 1-3天 📄 A4侦探报告

📅 阶段任务

  • 第1天：搜查家中不同区域（厨房、卫生间、阳台），拍照记录可疑现象
  • 第2天：查阅教材或网络资料，判断每个现象的变化类型
  • 第3天：制作"家庭化学侦探报告"（图文并茂）

📋 评价标准

  • 现象真实——找到生活中真实发生的化学变化
  • 判断准确——正确区分物理变化和化学变化
  • 记录完整——有图片/描述 + 判断依据
  • 表达清晰——报告结构合理，逻辑清楚

🔥 项目2：探究蜡烛燃烧的秘密

▼

按照科学探究的完整步骤，系统研究蜡烛燃烧过程中的变化。

⏱️ 1-2周 📄 科学探究报告

📅 阶段任务

  • 第1-2天：观察并记录蜡烛燃烧的所有可观察现象（火焰颜色、蜡熔化、黑烟等）
  • 第3-5天：设计实验验证"蜡烛燃烧是否有新物质生成"（用干冷烧杯检验水蒸气，用澄清石灰水检验CO₂）
  • 第6-7天：整理实验记录，完成分析，形成探究报告初稿
  • 第8-10天：准备展示材料（海报/PPT），在班级中分享

📋 评价标准

  • 科学探究步骤完整（观察→提问→假设→实验→分析→结论）
  • 实验设计合理（有控制变量、有检验方法）
  • 结论有证据支持（数据完整，分析符合逻辑）
  • 展示表达清晰（PPT/海报结构好，语言流利）

🗺️ 项目3：身边的化学——校园物质变化地图

▼

绘制一份校园化学变化/物理变化分布地图，用项目统整所学知识。

⏱️ 1个月以上 🗺️ 校园变化地图 + 说明书

📅 阶段任务

  • 第1周：规划校园区域（教学楼、操场、食堂、花园等），设计观察记录表
  • 第2周：分区域实地观察和记录，拍照取样
  • 第3周：分析数据，绘制地图初稿（标注每个区域的变化类型和说明）
  • 第4周：完善地图，撰写变化说明书，准备展示

📋 评价标准

  • 覆盖范围广——涵盖校园多个区域
  • 分类准确——化学变化/物理变化判断正确
  • 表达美观——地图可视化效果好
  • 有原创发现——找到了别人可能忽略的变化
