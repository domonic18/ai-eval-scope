法老的金字塔与黄金"不朽"

古埃及法老的黄金面具历经数千年仍光亮如新。
❓ 核心问题：为什么黄金不会生锈，而铁会？

决策路径

  1. 古人选择黄金作为陪葬品
  2. 基于对金属性质的观察经验
  3. 金属的稳定性决定了用途
💡 可迁移启示：金属的化学性质（是否容易与其他物质反应）决定了它的用途和价值。
挑战 · 系统

地球氧气的来源

约24亿年前，地球大气氧气含量突然上升（大氧化事件）。
❓ 核心问题：氧气从哪来？化学变化如何改变了整个地球？

决策路径

  1. 蓝藻的光合作用：CO₂ + H₂O → 有机物 + O₂
  2. 持续了数亿年
  3. 彻底改变了地球大气
💡 可迁移启示：缓慢的化学变化在长时间尺度上可以产生巨大影响。
