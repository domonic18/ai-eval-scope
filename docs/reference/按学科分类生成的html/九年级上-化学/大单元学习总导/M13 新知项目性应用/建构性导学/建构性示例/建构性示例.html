建构性示例 — 蜡烛燃烧的化学秘密

化学 · 九年级上 · 第1章 走进化学世界 · M13 新知项目性应用

一、建构性示例：蜡烛燃烧是化学变化吗？

本示例围绕一个生活中最常见的化学现象——蜡烛燃烧展开，完整演示"从观察到建构"的学习路径。你将看到如何从一个看似简单的问题出发，通过系统性观察、证据收集、假设验证和模型化表达，最终建构出"判断化学变化的方法"这一核心理解。

功能定位 新知项目性应用 · 第1章核心概念建构

二、真实建构情境

晚饭后，家里突然停电了。你点亮了一支蜡烛，微弱的火光在黑暗中摇曳。你静静地看着蜡烛燃烧——火焰跳动着，烛身一点点变短，蜡油顺着烛身流下，在烛台边缘凝结成白色的蜡滴。你伸手靠近火焰，感受到一股灼热；用玻璃杯罩住火焰，不一会儿火就熄灭了，杯壁上出现了一层水雾……

你忍不住思考：蜡烛燃烧时，它还在吗？烛身变短是消失了，还是变成了别的东西？这究竟是一种物理变化，还是化学变化？

💡 激发建构需要： 分辨"物理变化"和"化学变化"是学习化学的第一把钥匙。你将从蜡烛燃烧这个熟悉的现象开始，亲手建构这一区分标准。

三、建构目标

通过本示例，你将建构以下核心理解与能力：

  • 概念目标：能区分物理变化与化学变化的本质特征，掌握判断化学变化的关键依据——是否有新物质生成。
  • 方法目标：掌握科学探究的基本路径：观察→提问→假设→验证→结论→表达。
  • 思维目标：形成基于证据推理的科学思维方式，不凭感觉下结论。
  • 表达目标：能用概念图、表格等形式结构化呈现自己的理解。

四、建构任务说明

  • 成果名称：《物理变化与化学变化区分指南》——包含判断标准、关键特征、实例对比和概念图。
  • 用途和对象：帮助同学快速掌握物理变化与化学变化的区分方法，用于第1章的学习自测。
  • 形式和最低标准：A4纸单页（电子版或手绘均可），必须包含：①两种变化的定义对比 ②至少各3个实例 ③一张概念关系图 ④判断关键词（"是否有新物质生成"）。

五、建构过程示范（7步）

点击每行查看该步骤的详细示范。

步骤做什么怎么做生成什么简短示范
第1步 观察情境，发现问题 从情境中识别关键现象 问题记录 "蜡烛燃烧时烛身变短、蜡油熔化、火焰发光发热——这些现象中，哪些表明有新物质生成？"
示范：你不只是看"蜡烛在燃烧"，而是有目的地观察：①火焰颜色分层（外焰橙黄、内焰淡蓝）；②烛身不断变短；③杯壁上出现水珠；④凑近闻有特殊气味。把这些现象逐一记下来，标出"可能是变化的证据"。
第2步 收集证据，调用经验 查阅教材/回忆已有知识 证据清单 "回忆物理变化（冰融化、纸撕碎）的特征是形状/状态改变但物质不变；查阅教材发现化学变化的特征是生成新物质。"
示范：打开教材第1章，找到物理变化和化学变化的定义。列一个对比清单：物理变化→无新物质生成（冰→水）；化学变化→有新物质生成（木柴燃烧→灰烬+气体）。回忆生活经验：冰化成水还是水，但食物腐败后变酸变臭——产生了新物质。将蜡烛燃烧的现象对照这两类变化进行初步归类。
第3步 提出假设，设计思路 基于证据形成初步解释 建构思路图 "假设：蜡烛燃烧涉及化学变化，因为产生了水（杯壁水珠）和二氧化碳（能使石灰水变浑浊）。"
示范：你的假设是"蜡烛燃烧是化学变化"——因为观察到水珠（新物质）和气味（新物质）。如果只是熔化和汽化（物理变化），蜡烛应该还能变回原状，但实际上不能。画出思路图：蜡烛+氧气 → 水+二氧化碳+其他产物？列出还需要验证的点。
第4步 验证调整，完善方案 对照证据检验假设 修正记录 "用干燥烧杯罩在火焰上方→出现水珠（验证生成水）；用澄清石灰水润湿烧杯壁→变浑浊（验证生成二氧化碳）。"
示范：①用干燥冷烧杯罩在蜡烛火焰上方10秒→杯壁出现水雾（水蒸气冷凝）→证明蜡烛燃烧生成了水。②在烧杯内壁涂一层澄清石灰水再罩上→石灰水变浑浊→证明蜡烛燃烧生成了二氧化碳。③修正记录：蜡烛+氧气点燃→水+二氧化碳+……（确认是化学变化）
第5步 形成成果，清晰表达 组织语言/图示 完整成果 "绘制概念图：中央为'物质变化'，分出'物理变化'（无新物质）和'化学变化'（有新物质），分别列出3-4个实例。"
示范：用A4纸绘制概念图——
①中心：物质变化
②左分支：物理变化（特点：状态/形状改变，无新物质；实例：冰融化、水蒸发、玻璃破碎、铁熔化）
③右分支：化学变化（特点：有新物质生成；实例：蜡烛燃烧、铁钉生锈、食物腐败、木柴燃烧）
④底部标注判断方法："是否有新物质生成？"
第6步 反思过程，提炼方法 回顾建构路径 方法笔记 "整个建构过程遵循：真实情境→观察现象→调用已有知识→提出假设→实验验证→得出结论→可视化表达。"
示范：写一段方法笔记——"我今天建构知识的过程不是背课本，而是：从停电点蜡烛的真实经历出发→观察到水珠和气味→回想起物理变化的定义→提出蜡烛燃烧可能有化学变化的假设→用烧杯和石灰水验证→得出结论→画出概念图。这个过程可以用到其他化学知识的学习中。"
第7步 拓展联系，迁移应用 思考同类问题 迁移记录 "除了蜡烛燃烧，还有哪些日常现象可以用'是否有新物质生成'来判断？铁钉生锈、食物变质、酒精挥发……"
示范：列一个迁移清单——①铁钉生锈：铁锈是新物质→化学变化；②酒精挥发：酒精从液态变气态但没有新物质→物理变化；③纸张燃烧：变成灰烬和气体→化学变化；④食盐溶解：食盐仍存在于水中可析出→物理变化。这个判断框架可以应用到所有后续化学学习中。

七、建构成果样例

成果名称：《化学变化特征概念图》——物理变化 vs 化学变化对比指南

成果评价要求：

  • 科学性：概念定义准确，实例分类正确，无知识错误
  • 完整性：包含两种变化的定义、特征、判断方法、不少于6个实例
  • 美观性：布局清晰，配色协调，文字工整或打印清晰

推荐表达格式：概念图 + 对比表格。中央为"物质变化"，左右分支展开两类变化，底部标注判断流程。

完成建议：

  • 建议先用草稿纸梳理所有素材，再誊写到成稿上
  • 实例可以从教材和生活经验中各选一半，增强实用性

八、成果优化建议

  • 深化区分：在概念图中增加"容易混淆的实例"专区，如"酒精挥发（物理）vs 酒精燃烧（化学）"的对比，强化辨析能力。
  • 增加应用层：在概念图底部添加一个"判断三步法"操作框：①问"有新的物质产生吗？"②如有→化学变化，如无→物理变化 ③用实验验证判断。
  • 美化提示：用两种对比色区分两类变化（如蓝色→物理、橙色→化学），并在实例旁加简笔画或图标，增强记忆效果。
