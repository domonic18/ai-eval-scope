📘 M13 新知项目性应用

第1章 · 走进化学世界 · 项目实践阶段
🔬 化学 · 九年级上

🎯 本模块学习什么

核心主题：将"走进化学世界"学到的新知——物理变化与化学变化、化学性质与物理性质、科学探究方法——应用到真实项目情境中。

核心任务：通过解析他人的探究案例和自主建构自己的探究方案，深化对化学基本概念的理解。

内容范围：第1章 走进化学世界的核心概念 + 项目化应用

💡 本模块为什么重要

  • 化学不只是一堆概念和公式，更是一种认识和改造世界的方法
  • 让你亲身体验"学以致用"——把刚学到的化学知识立刻用到真实情境中
  • 帮助你建立"学化学有什么用"的切身感受，激发后续学习动力
  • 培养科学探究的基本素养，为今后的实验操作打下基础

🔗 本模块与大单元主题的关系

大单元"走进化学世界"的核心任务是：认识化学、了解化学的研究方法、建立对化学的基本理解。

本模块是"走进化学世界"的应用实践阶段——在前置模块理解基本概念的基础上，通过项目化应用将知识转化为能力，体现"从认识到实践"的学习闭环。
