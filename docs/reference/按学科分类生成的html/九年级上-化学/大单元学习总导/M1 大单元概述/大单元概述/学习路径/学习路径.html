1 基础进入路径

🎯 适合对象 初次接触化学，对"物质""变化"等基本概念不太清楚的同学 📌 学习重点 掌握物理变化与化学变化的区分标准；学会基本的观察与记录方法 📋 建议模块
M1 概述 M2 价值 M3 感知 M4 概念 M8 科学思维 M12 书面应用 M16 反思 M19 系统思维 M20 总结
📦 成果要求 完成单元核心概念图 + 3个生活化学现象的观察记录 🛠️ 支持工具 概念图模板、现象记录表格、AI学伴（随时提问） 💬 自我提醒 "化学是从现象开始的，不要怕问简单的问题"

2 标准发展路径

🎯 适合对象 能跟上正常学习进度，希望在理解概念的基础上尝试应用和表达的同学 📌 学习重点 完整理解核心概念→建立学科思维→尝试书面和简单项目应用 📋 建议模块
M1 概述 M2 价值 M3 感知 M4 概念 M5 多学科 M6 多学科 M8 科学思维 M9 科学思维 M10 科学思维 M12 书面应用 M13 项目 M16 反思 M17 反思 M19 系统思维 M20 总结
📦 成果要求 核心概念图 + 实验报告 + 简单项目方案 + 学习反思 🤝 合作方式 2~3人小组合作：一人记录现象，一人分析原理，一人负责表达呈现 💬 自我提醒 "每一个概念都要找到生活中的例子，每一个实验都要写观察记录"

3 高阶挑战路径

🎯 适合对象 基础较好，愿意挑战跨学科项目、创新表达和深度研究的学生 📌 学习重点 完成全模块学习，深入探索多学科思维（艺术/技术/数学/哲学/STEAM） 📋 建议模块
全部20个模块 重点：M5~M11 多元思维 重点：M14~M15 STEAM与三创
📦 成果要求 完整的大单元学习成果展示包（概念图 + 项目报告 + 创造作品 + 反思总结） 🏆 挑战任务 选择一个化学与生活/社会/未来的真实课题，完成一次迷你研究项目 💬 自我提醒 "不要满足于'会做'，要问自己'能不能更好'"

4 个体学习建议

看看你属于哪种类型，选择最适合你的路径。路径是灵活的，你可以在不同模块切换不同路线。

📖
概念薄弱型
🌱 基础路径
重点花时间在M3和M4，多用生活中的例子帮助理解；概念不懂时，及时用AI学伴追问
🎨
表达困难型
🌿 标准路径
尝试用非文字方式表达（画图、口头讲给同学听、录短视频）；多参与小组讨论
🔧
实践兴趣型
🌳 高阶路径
重点投入M13~M15（项目／STEAM／三创应用），把想法动手做出来
🔬
研究挑战型
🌳 高阶路径
主动关注M16~M20中的反思和总结部分，尝试提出自己的研究问题
🤝
合作成长型
🌿 标准或高阶
主动承担小组中的组织和协调角色；在合作中学习，在反馈中进步
🤖
数智工具探索型
🔄 所有路径
充分利用AI学伴、智能诊断、在线实验等数智工具；你的工具感就是你的优势
💡 提示：这些分类只是参考，你可以在不同模块切换不同路径——概念薄弱的可以在M4选基础，在实践部分选高阶。路径是灵活的，选择权在你。
