案例一：燃烧的蜡烛

📌 案例材料：观察一根燃烧的蜡烛。蜡烛变短、蜡油熔化、火焰上方有黑烟。这些现象中，哪些是物理变化？哪些是化学变化？

🔍 解析任务：区分蜡烛燃烧过程中的物理变化和化学变化

⚙️ 建议原理：变化类型优先判断法

🔄 解析路径：识别现象 → 分类 → 查找证据 → 得出结论

蜡烛变短化学变化 + 物理变化部分蜡燃烧生成CO₂和H₂O（化学），部分熔化流走（物理）
蜡油熔化物理变化蜡由固态变液态，没有新物质生成
火焰上方有黑烟化学变化生成碳颗粒（新物质），是蜡不完全燃烧的结果

案例三：老旧水管为什么漏水

📌 案例材料：老旧小区的铁质水管出现漏水，管壁变薄，有红褐色物质附着。

🔍 解析任务：分析水管漏水的原因，判断铁管变薄和红褐色物质生成属于什么变化

⚙️ 建议原理：反应物—条件—产物分析法

🔄 解析路径：观察 → 分析反应物和条件 → 判断变化类型 → 提出预防建议

📊 分析记录表

变化现象反应物条件产物变化类型
管壁变薄铁潮湿空气铁离子进入溶液化学变化
红褐色物质铁、氧气、水常温、潮湿铁锈（Fe₂O₃·xH₂O）化学变化

💡 预防建议：

  • 给水管刷防锈漆（隔绝氧气和水）
  • 保持管道干燥
  • 定期检查，及时更换

铁 + 氧气 + 水 → 铁锈（Fe₂O₃·xH₂O）。铁锈疏松多孔，不能保护内部铁材，会持续向內腐蚀。

案例四：水的"变身"之旅

📌 案例材料：水在自然界中不断循环——液态水蒸发成水蒸气、水蒸气冷凝成云、水结成冰、水在电解条件下分解成氢气和氧气。

🔍 解析任务：分析上述水的变化过程，区分哪些是物理变化、哪些是化学变化

⚙️ 建议原理：变化类型优先判断法 + 科学判断 vs 直觉判断区分法

🔄 解析路径：列出变化序列 → 逐项分析 → 分类 → 总结规律

🌐 跨学科情境

案例五：植物的光合作用

📌 案例材料：绿色植物在阳光下吸收CO₂和H₂O，合成有机物（C₆H₁₂O₆）并释放O₂。这是一系列化学变化的总和。

🔍 解析任务：从化学角度分析光合作用过程中的变化类型，解释为什么这是一个典型的化学变化

⚙️ 建议原理：反应物—条件—产物分析法

🔄 解析路径：确定反应物 → 分析条件 → 确定产物 → 判断变化类型 → 解释意义

📊 分析记录表

反应物条件产物变化类型判断判断依据
CO₂ + H₂O光照、叶绿素C₆H₁₂O₆ + O₂化学变化有新物质生成

📝 光合作物用化学方程式表示：6CO₂ + 6H₂O →光、叶绿素 C₆H₁₂O₆ + 6O₂

💡 CO₂和H₂O是反应物，C₆H₁₂O₆（葡萄糖）和O₂是产物——有新物质生成，所以是化学变化。

🌍 社会议题情境

案例六：食品包装中的"双吸剂"

📌 案例材料：许多食品包装里有一小包"双吸剂"，主要成分是铁粉。说明书说它可以同时吸收氧气和水分，防止食品变质。

🔍 解析任务：分析双吸剂的工作原理，判断其"吸收氧气和水分"过程中发生了什么变化

⚙️ 建议原理：反应物—条件—产物分析法 + 科学判断 vs 直觉判断区分法

🔄 解析路径：观察 → 提出推测 → 查找资料 → 验证 → 得出结论

📊 分析记录表

成分作用原理变化类型证据来源
铁粉吸收氧气铁与氧气反应生成铁锈化学变化铁锈是不同于铁的新物质
铁粉+其他吸收水分部分成分吸水（物理）+铁与水反应（化学）两者皆有需具体分析

铁粉与氧气和水反应生成铁锈（化学变化），消耗了包装内的氧气和水分，从而防止食品氧化变质。这就是为什么叫"双吸剂"——同时吸收氧气和水分。

🔑 核心原理：利用铁生锈的化学变化，消耗包装内的O₂和H₂O，创造一个低氧、干燥的保存环境。

案例七：我的"化学侦探"报告

📌 案例材料：有同学说"把铁钉放入硫酸铜溶液中，铁钉表面会覆盖一层红色物质"。请你亲自观察并分析这个现象。

🔍 解析任务：综合运用所学原理，完整分析铁钉与硫酸铜溶液反应的全部变化

⚙️ 建议原理：全部三条原理综合运用

🔄 解析路径：观察 → 假设 → 实验验证 → 分析 → 结论 → 表达

📊 分析记录表

初始状态观察到的变化变化类型判断化学原理结论
银白色铁钉 + 蓝色硫酸铜溶液铁钉表面出现红色物质，溶液蓝色变浅化学变化铁 + 硫酸铜 → 硫酸亚铁 + 铜生成了新物质——红色的铜单质

化学反应方程式：Fe + CuSO₄ → FeSO₄ + Cu

这属于置换反应，也是湿法冶铜的原理——铁把铜从硫酸铜溶液中置换出来。

✅ 这是一个化学变化：生成了红色的铜单质（新物质），和原来的铁完全不同。

📜 历史小知识：我国西汉时期就有"曾青得铁则化为铜"的记载，就是利用这个反应。
