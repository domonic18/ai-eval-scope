📖 从示例到原理

学习化学变化，光看案例是不够的。解析性原理就是从具体的案例中提炼出的通用方法——掌握了它们，你就能把今天学到的分析能力用到任何新场景中。

🍲 案例："铁锅煮苹果汤"
· 现象：苹果汤煮着煮着变成了黑褐色
· 疑问：这汤还能喝吗？是"营养流失"还是发生了化学变化？
· 分析：铁锅中的铁离子 + 苹果中的鞣酸 → 鞣酸铁（黑褐色）
· 启示：从具体案例→提炼通用方法，这就是解析性原理的价值

以下 3 条解析性原理 就是从这个案例中提炼出来的分析工具，点击每一条深入了解它们如何帮助你分析生活中的化学变化。

1 变化类型优先判断法 ▼
📌 说明

遇到物质变化现象时，先判断是物理变化还是化学变化

🍲 在示例中的体现

从"汤变黑"入手，先判断是否生成新物质

🔧 如何使用

观察 → 问"有新物质生成吗？"→ 找证据

🔄 可迁移到

所有生活化学现象的分析判断

⚠️ 使用提醒

有些变化同时包含物理和化学变化，需要分别分析

2 反应物—条件—产物分析法 ▼
📌 说明

把化学变化拆解为"什么物质（反应物）+ 什么条件 → 产生了什么（产物）"

🍲 在示例中的体现

铁 + 鞣酸 + 高温/酸性条件 → 鞣酸铁

🔧 如何使用

列出所有参与物质 → 分析反应条件 → 推断产物

🔄 可迁移到

化学方程式的分析、实验现象的解释

⚠️ 使用提醒

产物需要证据支持，不能凭空猜测

3 科学判断 vs 直觉判断区分法 ▼
📌 说明

区分"这是我觉得的"和"这是证据证明的"

🍲 在示例中的体现

妈妈的"营养流失"是直觉判断，化学分析是科学判断

🔧 如何使用

先列出直觉判断，再查找科学证据验证

🔄 可迁移到

日常决策、广告判断、新闻分析

⚠️ 使用提醒

科学判断也不等于绝对真理，需要持续检验

1 紫甘蓝汁变色之谜 ▼
🧪 情境

紫甘蓝汁分别滴入白醋和小苏打水中，颜色变得不同

🎯 解析任务

分析颜色变化的原因，判断是物理变化还是化学变化

💡 建议原理

变化类型优先判断法 + 反应物—条件—产物分析法

🔎 分析提示

紫甘蓝汁是一种酸碱指示剂——颜色变化可能与pH有关，但这本身就是化学反应吗？

✍️ 表达方式

用图表 + 文字说明你的分析过程
