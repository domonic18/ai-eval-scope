🎯 建构目标

通过一个完整的化学探究案例，示范如何从日常现象出发，运用"观察→提问→假设→验证→建模"的科学方法，建构关于化学变化本质的理解模型。

核心问题：如何区分物理变化和化学变化？化学变化的本质是什么？

🔬 真实建构情境

🪙 情境：爷爷的老铁钉

你在整理老家的工具箱时，发现一个锈迹斑斑的铁钉盒。同放在一个盒子里的塑料纽扣和新买的不锈钢螺丝却完好无损。

"为什么铁钉会生锈，而其他东西没有变化？" 这个看似简单的问题，隐藏着化学变化的核心秘密。

📝 建构成果：化学变化判断模型

成果名称：化学变化本质概念模型图

用途：帮助自己区分物理变化和化学变化

形式：概念模型图（含判断维度）

🧪 化学变化判断模型

① 看新物质
是否生成新物质？
→
② 看能量
有无能量变化？
→
③ 看可逆性
能否简单复原？
→
④ 看分子
分子种类变了吗？

四个维度综合判断，满足①＋④即为化学变化

👣 建构过程示范（7步法）

点击每一步标记完成，追踪你的建构进度

1

观察现象

铁钉表面出现红褐色物质，部分铁钉变脆、有凹陷。塑料纽扣、不锈钢螺丝无变化。

2

提出问题

"铁钉上的红褐色物质是什么？""为什么只有铁发生了变化？""铁生锈需要什么条件？"

3

收集证据

查阅教材：铁锈主要成分是Fe₂O₃·xH₂O。回忆实验：铁在潮湿空气中更易生锈。

4

建立解释

铁与空气中的氧气和水发生了化学反应，生成了新物质——铁锈。这是一个化学变化。

5

验证修正

设计对照实验：铁钉在干燥空气中 + 铁钉在水中 + 铁钉在潮湿空气中，观察哪组生锈。

6

形成模型

将理解可视化：化学变化 = 有新物质生成 + 分子种类改变 + 伴随能量变化。

7

表达分享

用清晰的语言描述建构成果："铁生锈是铁与氧气和水发生的氧化反应，产生了不同于铁的新物质铁锈。"
