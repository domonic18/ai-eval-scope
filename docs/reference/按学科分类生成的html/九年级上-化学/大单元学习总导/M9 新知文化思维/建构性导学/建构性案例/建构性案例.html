案例一：我的化学观察日志

情境：你每天都会接触到各种化学变化——铁门生锈、煤气燃烧、食物变质。选择一个你最熟悉的日常变化，用化学的眼光重新观察它。

建构任务：建构一份"化学变化观察日志"，记录现象、提出问题、作出假说。

建议原理：现象驱动提问法

建构步骤：观察 → 记录 → 提问 → 假设 → 验证计划

✅ 自评检查清单

  • ☐ 我记录的现象包含时间、地点、变化前和变化后的状态
  • ☐ 我至少提出了 2 个具体的化学问题
  • ☐ 我对其中一个问题作出了可检验的假设
  • ☐ 我设计了简单的验证方案
  • ☐ 我用完整的句子写下了观察日志

案例二：家庭安全用气指南

情境：新闻报道中提到燃气泄漏事故。天然气的主要成分甲烷在什么条件下会燃烧/爆炸？

建构任务：建构一份"家庭燃气安全使用化学原理指南"。

建议原理：证据分层法模型迭代法

建构步骤：收集信息 → 分析条件 → 建构原理 → 形成指南

✅ 自评检查清单

  • ☐ 我查到了甲烷燃烧的化学方程式
  • ☐ 我说明了燃烧的三个条件
  • ☐ 我列出了泄漏时的正确处理步骤
  • ☐ 我的指南能让家人看懂
  • ☐ 我标注了需要紧急求助的情况

案例三：实验室事故预防方案

情境：化学实验室中，酒精灯使用不当可能引发火灾。你如何让低年级同学安全使用酒精灯？

建议原理：多维验证法

建构步骤：分析风险 → 建构安全操作流程 → 设计验证

✅ 自评检查清单

  • ☐ 我列出了至少 3 个酒精灯使用的风险点
  • ☐ 我绘制了清晰的操作流程图
  • ☐ 我为每个风险点设计了预防措施
  • ☐ 我设计了验证操作是否安全的方法
  • ☐ 安全提示卡图文并茂、一目了然
