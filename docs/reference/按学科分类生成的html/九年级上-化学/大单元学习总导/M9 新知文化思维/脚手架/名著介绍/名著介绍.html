📚 化学名著

中国经典

《天工开物》 · 宋应星

明末 · 三卷18章 · 选读片段

核心问题：古代中国有哪些化工生产技术？

技术来源于对物质变化规律的观察和掌握。涵盖食品、纺织、冶铸、造纸、制糖等。

📖 查看关键片段导读 ↓

石灰烧制：将石灰石（碳酸钙）高温煅烧，分解为生石灰（氧化钙）和二氧化碳。古人虽不知化学式，却已熟练掌握了这一化学变化。

陶瓷烧制：黏土在高温下发生脱水、氧化、玻璃化等多重化学变化，最终成为坚硬致密的陶瓷。这一过程至今仍是材料化学的研究课题。

获取：人民教育出版社出版 / 微信读书

外国经典

《科学的旅程》 · 雷·斯潘根贝格

科学史 · 基础—进阶

核心问题：科学是如何一步步发展成今天这个样子的？

科学不是真理的堆积，而是不断修正的探索过程。

📖 查看关键片段导读 ↓

第5章"化学的诞生"：从炼金术到现代化学的转变——炼金术士在寻找"点石成金"的过程中积累了丰富的物质变化知识，波义耳将化学确立为一门独立的科学，拉瓦锡用定量实验推翻了燃素说。化学的每一步进步，都是对旧观念的"革命"。

获取：各大书店 / 电子书平台

当代科普

《元素的盛宴》 · 山姆·基恩

趣味科普 · 基础

核心问题：元素周期表中的每个元素有什么有趣故事？

元素背后的人、事、物构成了化学的精彩世界。

📖 查看关键片段导读 ↓

第1章"战争中的元素"：火药的发明改变了世界格局——硫磺、木炭和硝石按比例混合，点燃时发生剧烈氧化还原反应，瞬间产生大量气体推动弹丸。火药中的化学看似简单，却蕴含着氧化剂、还原剂、燃烧反应的核心原理。

获取：中信出版社 / 得到APP有声书
