🌱 基础层：生活案例

生活为什么铁制品会生锈？

▼
📌 背景

一把放在户外的铁锁，三年后锈迹斑斑。

⚡ 核心问题

铁为什么会"变"成铁锈？铁锈和铁还是同一种物质吗？

🛤️ 决策路径

方案A 涂油防锈 → 方案B 镀锌 → 方案C 保持干燥 → 对比分析

✅ 结果

隔绝水和氧气可以防止生锈——这是一个化学变化。

💡 可迁移启示：所有"防止化学变化"的策略，核心都是"移除反应条件"。

🔥 进阶层：行业 / 历史案例

历史杜邦公司的"特氟龙"发明

▼
📌 背景

1938年，杜邦公司科学家罗伊·普朗克特在研究制冷剂时，意外发现一种不粘物质——聚四氟乙烯（特氟龙）。

⚡ 核心问题

这种物质几乎不与任何物质发生化学变化——这有什么价值？

🛤️ 决策路径

从军工（原子弹项目密封剂）→ 厨具（不粘锅）→ 纺织（防水面料）

✅ 结果

特氟龙的化学惰性使其在多个领域大放异彩。

💡 可迁移启示：有时候"不变化"本身也是一种有价值的"性质"。

🚀 挑战层：跨领域案例

前沿光催化自洁玻璃

▼
📌 背景

东京工业大学的科学家发现，二氧化钛在紫外线下可以分解污渍。

⚡ 核心问题

如何让玻璃自己保持清洁？这涉及哪些化学变化？

🛤️ 决策路径

发现现象 → 解释机理 → 优化材料 → 产业化

✅ 结果

光催化自洁玻璃已广泛应用于建筑幕墙。

💡 可迁移启示：利用化学变化的"连锁反应"（光能→化学能→分解污渍）可以实现自我维护。
