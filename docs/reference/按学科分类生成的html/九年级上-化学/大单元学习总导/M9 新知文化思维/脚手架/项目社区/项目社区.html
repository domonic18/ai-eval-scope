🏠 家庭化学变化寻宝

⏱ 1–3 天 · 微项目
🎯 项目目标

在家庭环境中找到至少 5 种化学变化，用照片/视频 + 文字记录

📋 阶段任务
  1. 浏览家中各区域（厨房、卫生间、阳台等）
  2. 记录发现的变化现象
  3. 判断是否属于化学变化并说明理由
  4. 整理成"家庭化学变化地图"
📦 成果形式

图文记录表（含照片、变化描述、判断依据）

⭐ 评价标准
🎯
判断准确率
📝
记录完整性
🔬
理由科学性
🎨
呈现美观性
🎉 太棒了！从厨房开始探索吧——铁锅生锈、小苏打+醋冒泡都是经典的化学变化！

📖 生活防化学变化指南

⏱ 1–2 周 · 专题项目
🎯 项目目标

针对生活中 3 种不受欢迎的化学变化（如铁生锈、食物腐败、塑料老化），研究预防方法，制作实用指南

📋 阶段任务
  1. 选择 3 种"不受欢迎的化学变化"
  2. 研究它们发生的条件
  3. 查找或设计预防方案
  4. 制作成图文并茂的指南
📦 成果形式

"防化学变化指南"（小册子 / 海报 / PPT）

⭐ 评价标准
🔬
科学性
💡
实用性
🎨
美观性
📚
完整性
🎉 好选择！铁生锈是最经典的案例——从"隔绝氧气和水"入手去设计方案吧！

🔬 未来材料设计师

⏱ 1月+ · 综合项目
🎯 项目目标

基于化学变化原理，设计一种具有特定功能的新型材料概念方案

📋 阶段任务
  1. 调研：了解当前材料科学的 3 个前沿方向
  2. 选题：确定你想"设计"的材料类型
  3. 设计：画出材料结构概念图和制备流程
  4. 论证：说明这种材料的工作原理（涉及什么化学变化）
  5. 展示：制作展板或演示文稿
📦 成果形式

材料设计方案 + 展示海报

⭐ 评价标准
💡
创新性
🔬
科学性
✅
可行性
🎤
展示表现力
🎉 挑战自我！可以从"自修复材料"或"环保降解塑料"入手，这些是材料科学的热门方向！
