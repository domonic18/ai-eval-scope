实验模板

🧪 实验设计模板

用途：系统设计化学实验方案

模板结构：实验目的 → 假设 → 材料 → 步骤 → 数据记录 → 结论

实验目的：探究铁生锈的条件
假设：铁生锈需要水和空气
材料：铁钉×3、试管×3、水、植物油、干燥剂
步骤：①试管1：铁钉+水（半浸）②试管2：铁钉+干燥剂 ③试管3：铁钉+煮沸后水（去空气），加油封
记录：每天观察并拍照
结论：铁生锈需要水和氧气同时存在
