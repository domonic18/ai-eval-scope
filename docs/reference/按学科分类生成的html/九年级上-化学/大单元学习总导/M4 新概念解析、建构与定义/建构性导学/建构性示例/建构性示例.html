建构性示例：从蜡烛燃烧看"新物质如何诞生"

建构化学变化的概念 · 九年级化学 · 模块 M4

🧪 真实建构情境

你在生日厨房中看到蜡烛的三个现象：熔化、燃烧冒黑烟、蜡烛变短。

这些现象一样吗？为什么有的可逆、有的不可逆？你需要一个框架来区分"只是看起来变了"和"真的变成了别的东西"。

🎯 建构目标  你将建构出——一个能清晰区分物理变化和化学变化的判断框架

📋 建构任务说明

  • 成果名称：你的"变化类型鉴别器"概念框架
  • 用途：可通用到生活中任何"东西变了"的场景
  • 最低标准：包含区分依据 + 至少 3 个自己的例子

🔄 建构过程示范（7步）

👇 点击每一步展开详细内容

1 观察
▼
蜡烛燃烧时你看到了哪些现象？
熔化→烛泪流下、发光发热、冒黑烟、蜡烛变短。
先别判断，全部写下来。
2 提问
▼
这些现象归为一类还是多类？依据是什么？
提示：熔化后的烛蜡冷却可以变回固体，但烧掉的蜡烛还能变回原样吗？
3 经验调用
▼
回忆以前见过的"变了"的例子：
❄️ 冰块融化 → 冰变成水
🔩 铁生锈 → 银白金属变成红褐色铁锈
🔥 纸燃烧 → 纸张变成灰烬
4 证据收集
▼
对比这些现象——哪些生成了新物质？哪些没有？
冰融化成水：还是 H₂O，无新物质
铁生锈：生成 Fe₂O₃·xH₂O，新物质
纸燃烧：生成 CO₂ 和灰烬，新物质
5 概念生成
▼
定义两个核心概念：
物理变化：没有新物质生成的变化（形态变，本质不变）
化学变化：有新物质生成的变化（本质变了）
📌 核心区分依据：是否有新物质生成
6 验证
▼
用框架重新判断蜡烛燃烧的全部现象：
✅ 蜡烛熔化 → 无新物质生成 → 物理变化
✅ 蜡烛燃烧冒黑烟 → 新物质（炭黑、CO₂） → 化学变化
✅ 蜡烛变短 → 两种变化同时发生
7 表达
▼
用你自己的语言写出判断框架：
"我观察到______，这说明______（有/没有）新物质生成，所以属于______变化。"
📝 尝试填入你自己的例子，完成这个句式。
