🏗️ 建构性案例

化学变化·九年级上 — 七阶递进建构练习，在真实情境中迁移运用

个人理解型

📝 我的"变化日记"

📋 情境

回顾你一天中遇到的5个"东西变了"的场景。

🎯 建构任务

建构一张"一天中的变化地图"，标注每个变化是物理还是化学变化。

⚙️ 建议调用原理

现象分拆法 — 把每个变化场景单独拆开分析；框架建构法 — 用"有无新物质生成"标准分类。

1 / 7
生活应用型

🏠 家用物品的"身份卡"

📋 情境

选择3种厨房物品（如：食盐、食醋、小苏打），查资料了解它们的性质。

🎯 建构任务

为每种物品建构"物理性质和化学性质身份卡"。

⚙️ 建议调用原理

本质提取法 — 区分物理性质和化学性质；自我验证法 — 用已学判断标准检验分类。

2 / 7
问题解决型

🧪 实验室安全"决策树"

📋 情境

化学实验室中，有同学不小心打翻了酒精灯。

🎯 建构任务

建构一个"实验事故响应决策树"。

⚙️ 建议调用原理

框架建构法 — 设计"如果…那么…"的决策分支；经验对比法 — 对比不同事故的应对差异。

3 / 7
学科表达型

🔬 化学变化的"可视化图解"

📋 情境

教材中提到了镁条燃烧、铁与硫酸铜反应等变化。

🎯 建构任务

建构"化学变化微观解释图"，用粒子模型解释变化本质。

⚙️ 建议调用原理

本质提取法 — 从宏观现象提炼微观本质；框架建构法 — 建立"分子→原子"解释框架。

4 / 7
跨学科整合型

🍞 厨房里的"科学说明书"

📋 情境

发酵面团——用化学（产生二氧化碳）和生物（酵母菌）两个视角解释。

🎯 建构任务

建构一份"馒头发酵的跨学科科学说明书"。

⚙️ 建议调用原理

现象分拆法 — 将发酵分成化学和生物两个维度；经验对比法 — 对比化学变化与生物过程。

5 / 7
社会议题型

♻️ 社区垃圾分类的"化学视角"

📋 情境

社区推广垃圾分类，但很多人不清楚"有害垃圾"为什么有害。

🎯 建构任务

建构一份"从化学性质看垃圾处理"的社区宣传方案。

⚙️ 建议调用原理

本质提取法 — 揭示有害垃圾的化学成分和危害机理；框架建构法 — 建立"垃圾类别→处理方式"对应框架。

6 / 7
综合创新型

🧠 我的"化学变化预测器"

📋 情境

给你3种未知物质（描述其外观和已知性质），预测它们混合后可能发生的变化。

🎯 建构任务

建构一个"物质反应预测模型"。

⚙️ 建议调用原理

全部5条原理综合运用：现象分拆法、经验对比法、本质提取法、框架建构法、自我验证法。
