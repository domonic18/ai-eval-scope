🔍 现象分拆法

来源：蜡烛燃烧被拆分为熔化、燃烧、变短三个子现象。

适用：任何需要分析复杂变化或过程的场景。

⚠️ 注意：拆分后不要丢失整体联系。
原理 2

🔄 经验对比法

调用自身已有经验与新现象对比，通过异同识别本质。

👆 翻面查看更多

🔄 经验对比法

来源：将蜡烛燃烧与已知的冰融化、铁生锈对比。

适用：学习新概念时，先对比已知概念。

⚠️ 注意：旧经验可能不准确，需要验证。
原理 3

🎯 本质提取法

透过现象找本质——从"看起来怎样"到"实际上发生了什么"。

👆 翻面查看更多

🎯 本质提取法

来源：从"冒黑烟"到"有新物质碳生成"的本质提取。

适用：需要穿透表象、建立概念定义时。

⚠️ 注意：本质判断需要证据支撑，不能仅凭感觉。
原理 4

🏗️ 框架建构法

将新概念组织成可复用的判断框架（如鉴别器）。

👆 翻面查看更多

🏗️ 框架建构法

来源：将"有无新物质生成"作为判断标准，形成两个分类。

适用：需要区分两个易混淆概念时。

⚠️ 注意：框架要简洁，分类要覆盖全部可能。
原理 5

✅ 自我验证法

用建构好的框架去检验新案例，在应用中修正完善。

👆 翻面查看更多

✅ 自我验证法

来源：用变化判断框架重新验证蜡烛现象，再扩展到其他现象。

适用：任何新建构的概念或模型。

⚠️ 注意：反例是最有价值的修正信号。

🏃 建构性原理迁移应用

🔪 厨房里的"变化侦探"

情境：妈妈做饭时，你观察到：水烧开变成水蒸气、鸡蛋煮熟变硬、白糖溶化、白糖加热变焦。

建构任务：用上述原理分拆这些现象，建构新的"变化分类图"。

建议使用原理：现象分拆法 + 本质提取法 + 框架建构法

🌿 自然界的"物质变身"

情境：秋天树叶变黄落下、苹果切开变褐、铁门生锈。

建构任务：建构一个"自然界变化类型图谱"，区分物理与化学变化。

建议使用原理：经验对比法 + 框架建构法 + 自我验证法
