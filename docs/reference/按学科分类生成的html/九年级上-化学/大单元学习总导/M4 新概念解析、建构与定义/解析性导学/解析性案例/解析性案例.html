🏠 生活情境

🥚 案例1：厨房里的"变化"大考验

案例材料：妈妈在厨房做了三件事——煮鸡蛋（蛋黄变硬凝固）、切黄瓜（黄瓜片变小）、烤面包（表面变焦黄）。判断这三个变化各属于什么类型，列出你的证据链。
前后对比法 证据层级法

🔎 解析：

煮鸡蛋 — 蛋清/蛋黄由液态→固态，蛋白质变性，有新物质生成 → 化学变化

切黄瓜 — 只是形状变小，物质本身还是黄瓜 → 物理变化

烤面包 — 表面焦黄，糖类和蛋白质发生美拉德反应，有新物质生成 → 化学变化

证据链：前后对比+核心证据（有无新物质）

🔬 校园情境

🧪 案例2：化学实验室的"神秘液体"

案例材料：小明在实验室发现一瓶无色液体，标签脱落。他做了以下实验：①闻起来有酒精味 ②加热后液体消失 ③点燃时能燃烧。解析任务：①、②、③分别属于对什么的分析（性质还是变化）？液体消失和燃烧分别是什么变化？
操作定义法 证据层级法

🔎 解析：

① 闻味道 → 分析的是物理性质（气味）

② 加热消失 → 物理变化（液体蒸发为气体，物质仍为乙醇）

③ 点燃燃烧 → 化学变化（乙醇+氧气→CO₂+H₂O，有新物质生成），同时体现化学性质（可燃性）

🏘️ 社区情境

🔩 案例3：小区里的"金属腐蚀"调查

案例材料：你观察到三个场景——①铁门生锈（红褐色粉末）②铝窗框表面发暗 ③铜水龙头变绿。三个案例的共同点和不同点是什么？哪些属于化学变化？判断依据是什么？
前后对比法 反例检验法

🔎 解析：

三种现象均属于金属被空气中的O₂/H₂O/CO₂等腐蚀，均生成新物质 → 都是化学变化。

共同点：都是金属与空气发生化学反应

不同点：产物不同（Fe₂O₃·xH₂O红褐色、Al₂O₃致密膜、Cu₂(OH)₂CO₃铜绿）

反例检验："有颜色变化就是化学变化"→ 不成立（如颜料溶解变色是物理变化）

🧬 学科核心情境

🔬 案例4："变化的微观解释"

案例材料：水和冰的分子排列图（物理变化：分子间距变化）和碳燃烧生成二氧化碳的分子变化图（化学变化：原子重新组合）。从分子/原子的微观层面，解释物理变化和化学变化的本质区别。
操作定义法 前后对比法

🔎 解析：

物理变化：分子种类不变，只是分子间的距离或排列方式改变（如H₂O分子在水和冰中都是H₂O）。

化学变化：分子分裂成原子，原子重新组合成新分子（如C + O₂ → CO₂，碳原子和氧原子重新排列）。

本质区别：化学变化中分子种类改变，物理变化中分子种类不变。

🧊 跨学科情境

🥩 案例5：冰冻食品的"科学解冻"

案例材料：从冰箱取出冻肉，用三种方式解冻——①室温自然解冻 ②冷水浸泡 ③微波炉解冻。记录温度变化和肉质变化。解冻过程中发生了什么变化？微波加热是物理变化还是化学变化？为什么？
证据层级法 反例检验法

🔎 解析：

解冻过程：冰→水（状态变化），肉本身成分不变 → 物理变化

微波加热：微波使水分子振动产热，水分子结构未变 → 物理变化（虽然产热了，但没有新物质生成）

反例检验："因为产热了所以是化学变化"对吗？→ 搓手也产热但无新物质生成，反例证伪。

🌍 社会议题情境

🌡️ 案例6：全球变暖中的"物理vs化学"

案例材料：全球变暖涉及两个重要过程——①CO₂分子吸收红外辐射（温室效应）②化石燃料燃烧释放CO₂。解析任务：①和②分别属于物理变化还是化学变化？全球变暖本身是"变化"还是"现象"？
操作定义法 证据层级法

🔎 解析：

① CO₂吸收红外辐射 → 分子振动能级跃迁，CO₂分子种类未变 → 物理变化

② 化石燃料燃烧 → CₓHᵧ + O₂ → CO₂ + H₂O，有新物质生成 → 化学变化

全球变暖本身 → 是现象（温度升高的宏观表现），不是"变化"的概念类型。

🧪 案例7：未知物质的"身份鉴定"

案例材料：你收到3种白色粉末（可能是食盐、小苏打、淀粉）。提供工具：水、白醋、碘酒、pH试纸。设计鉴别方案——区分哪些是物理性质、哪些是化学性质；哪些操作涉及物理变化、哪些涉及化学变化。
前后对比法 证据层级法 反例检验法 操作定义法

🔎 解析：

鉴别方案设计：

  • 溶于水观察 — 物理变化（溶解），食盐全溶、小苏打部分溶、淀粉不溶
  • 加碘酒 — 化学变化（淀粉变蓝，新物质生成）→ 鉴别淀粉
  • 加白醋 — 化学变化（小苏打产生CO₂气泡，新物质生成）→ 鉴别小苏打
  • pH试纸 — 测的是化学性质（酸碱性），属物理操作+化学性质的体现
