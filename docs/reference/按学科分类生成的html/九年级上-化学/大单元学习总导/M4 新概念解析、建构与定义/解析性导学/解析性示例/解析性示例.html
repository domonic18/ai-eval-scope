✨ 解析性示例：镁条燃烧的秘密

M4 新概念解析、建构与定义 — 跟着专家思路学习如何分析化学变化

1. 解析任务情境

🔥 情境设定：化学课上，老师点燃了一根银白色的镁条。你看到：剧烈燃烧、发出耀眼白光、放出大量热、生成白色粉末。问题：这是物理变化还是化学变化？凭什么判断？

三个感官现象（光、热、颜色变化）+ 一个物质状态变化（金属变粉末）

2. 核心问题

🤔 镁条燃烧过程中，发生了什么？如何用本模块的标准判断变化类型？

3. 关键信息提取
情境信息 说明什么 对解决问题的重要性
银白色金属 镁条的物理外观（物理性质） 初始状态的参照
剧烈燃烧、发光放热 变化过程中伴随的能量现象 化学变化的常见特征（非充分条件）
生成白色粉末 有新物质生成 判断化学变化的关键证据
反应前是镁，反应后是氧化镁 物质种类发生变化 变化的本质判断
4. 专家解析过程
1

先看见什么

最直观的是"发光发热"和"颜色从银白变白"

2

判断问题属于哪一类

这是"判断变化类型"的问题，需要用"有无新物质生成"这个标准

3

调用哪些概念或方法

物理变化（无新物质生成）vs 化学变化（有新物质生成）

4

如何进行推理

白色粉末不可能是镁（银白色金属），所以一定生成了新物质 → 属于化学变化

5

怎样形成结论

镁条燃烧是化学变化，证据是生成了新物质（氧化镁白色粉末）

5. 结论表达

镁条燃烧 → 化学变化

证据链：银白色镁条 + 氧气 → 燃烧（发光发热） → 白色氧化镁粉末（新物质）

6. 方法提炼

"以后遇到类似问题，可以这样做……"

  1. 列出变化前后的物质分别是什么
  2. 对比前后物质的"身份"是否相同
  3. 如果生成新物质 → 化学变化；如果只是形状/状态变化 → 物理变化
7. 易错提醒

❌ 常见错误

"发光发热就是化学变化"——反例：电灯泡也发光发热，但那是物理变化。

❌ 常见错误

"颜色变了就是化学变化"——反例：色素溶于水变颜色属于物理变化。

✅ 正確思考

要抓住"有没有新物质生成"这个本质标准。

发光、发热、颜色变化、气体产生等现象都可能出现在物理变化中（灯泡发光发热、水蒸发、色素扩散），唯有"是否有新物质生成"在物理变化中一定不出现，在化学变化中一定出现。这是充分必要条件。

8. 自主迁移小练习

🧪 小练习

铁钉放入蓝色硫酸铜溶液中，铁钉表面出现红色物质，蓝色溶液变浅。
这是什么变化？为什么？

物理变化 — 只是颜色变化
化学变化 — 生成了新物质（红色物质）
既有物理变化又有化学变化
