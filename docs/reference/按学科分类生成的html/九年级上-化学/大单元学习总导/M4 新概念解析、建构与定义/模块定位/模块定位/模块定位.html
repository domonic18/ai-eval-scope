1 本模块学习什么？ ▼

你将学到三个核心内容：

  • 什么是物理变化和化学变化 —— 根本区分在于"有没有新物质生成"
  • 什么是物理性质和化学性质 —— 性质是物质"本来就有的特征"，需要在一定条件下才能表现出来
  • 化学实验基本操作 —— 如何安全规范地使用化学仪器
2 本模块为什么重要？ ▼

这是你进入化学世界的第一道"概念门槛"：

  • 物理变化 / 化学变化是整个化学学科最基本的分类框架
  • 物理性质 / 化学性质是描述任何物质的标准语言
  • 如果基础不牢，后面的化学反应、物质制备等内容都会混淆
💡 一句话：M4 是整栋化学大厦的"地基"模块！
3 本模块与大单元主题的关系？ ▼

"走进化学世界"大单元的主题是"化学是什么"。M4 承担的任务是：

  • 在 M1 建立了全景认知后，M4 深入到最基础的两对核心概念
  • 为大单元后续模块提供概念工具
  • M4 是整栋化学大厦的"地基"模块
M1 全景认知 → M4 核心概念 → M5 性质探究 → M6 反应类型
4 本模块与前后模块的衔接？ ▼
  • 前接模块：M1 大单元概述 —— 你已了解化学的总体面貌
  • 后接模块：后续模块将基于本模块的概念框架，深入探究具体物质的性质和反应
  • 内部顺序：先完成"解析性导学"（理解概念），再完成"建构性导学"（内化概念）
5 本模块预期学习成果？ ▼

完成本模块后，你将能：

  • ✓ 独立判断 5 个生活常见现象分别属于物理变化还是化学变化
  • ✓ 用自己的话给"物理性质"和"化学性质"下定义，并各举 2 个例子
  • ✓ 用概念图或表格展示变化与性质的完整知识体系
  • ✓ 掌握酒精灯、试管、量筒等基本仪器的使用方法
🧭 M4 是你走进化学世界的第一道门——把地基打牢，后面的学习会越来越轻松！
