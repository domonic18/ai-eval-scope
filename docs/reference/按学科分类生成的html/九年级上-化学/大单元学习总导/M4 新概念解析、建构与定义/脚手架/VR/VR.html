🔍 VR 观察任务

  • ✓ 在 VR 中观察水从液态变为气态（沸腾）的微观粒子运动变化
  • ✓ 观察铁在潮湿空气中逐渐生锈的宏观与微观同步展示
  • ✓ 对比观察蜡烛熔化（物理变化）和蜡烛燃烧（化学变化）的粒子层面差异

💡 观察提示：关注粒子之间的间距、排列方式和种类是否改变

✋ VR 操作任务

  • ✓ 在 VR 实验室中安全完成镁条燃烧实验操作
  • ✓ 通过拖拽粒子模型，模拟化学变化中原子重新组合的过程
  • ✓ 在虚拟环境中测试不同物质的物理性质和化学性质

⚡ 操作注意：镁条燃烧发出耀眼白光，请在 VR 中佩戴好防护镜

  • ✓ 画出你在 VR 中看到的粒子排列变化示意图
