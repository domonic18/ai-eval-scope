基 再现与理解

  • 列举5个生活中常见的物理变化和5个化学变化，并说明你的判断依据
  • 完成概念填空：物理变化是指没有生成其他物质的变化，化学变化是指生成了其他物质的变化
  • 对照教材，填写常见化学仪器的名称和用途匹配表

分 分层任务（A / B / C 选一题）

A层 · 基础巩固给出一组现象描述（如"玻璃破碎""纸张燃烧"），分类并说明理由
💡 提示：注意区分现象和本质。玻璃破碎只是形状变化，没有新物质生成；纸张燃烧产生了新物质（如二氧化碳、水蒸气）。
B层 · 应用分析给出一段生活场景描述（如"铁锅生锈""食物冷藏"），分析其中的变化类型和性质
💡 提示：铁锅生锈是化学变化（铁与空气中的氧气、水反应生成铁锈），食物冷藏是物理变化（温度降低，没有新物质生成）。
C层 · 综合探究设计一个家庭小实验，验证"食盐溶解在水中是物理变化还是化学变化"
💡 提示：可以将食盐水加热蒸发，观察是否有白色固体析出。如果析出的固体就是食盐本身，说明溶解过程没有产生新物质，属于物理变化。
