基础案例

🔩 为什么铁锅会生锈？

背景：厨房里的铁锅使用一段时间后，表面出现红褐色锈迹。

❓ 核心问题：生锈属于什么变化？如何防止？
👆 查看分析
方案 A：涂油隔绝空气 → 利用油脂阻隔氧气
方案 B：保持干燥 → 无水环境下铁难生锈
方案 C：使用不锈钢锅 → 合金耐腐蚀
💡 可迁移启示：化学变化的控制可以改善生活质量——理解了生锈的原理，就能找到防锈的办法。
进阶案例

🚢 泰坦尼克号的钢铁

背景：泰坦尼克号船体钢材在冰冷海水中发生脆性断裂。

❓ 钢材的物理性质（韧性）在低温下如何变化？这属于物理变化还是化学变化？
👆 查看分析
决策路径：材料工程师需要根据使用环境的温度，选择在该温度下仍保持足够韧性的钢材。
低温使钢材从韧性变为脆性——这是物理性质的改变，但没有新物质生成，属于物理变化。
💡 可迁移启示：物质的性质（物理性质）决定了它的适用范围。走进化学世界不只是学概念，更是理解为什么有的材料"不合适"。
挑战案例

🔋 电池的发明——伏打电堆

背景：1800 年伏打发现不同金属通过电解质连接能产生持续电流。

❓ 电池工作时发生了化学变化还是物理变化？
👆 查看分析
答案：化学变化——化学能转化为电能，生成了新物质。

古代人的疑惑：古人认为电是"流体"，伏打通过控制变量实验证明电流来自化学反应，而非金属本身"带电"。
💡 可迁移启示：一种现象（产生电流）可能来自多种原因，需要通过控制变量实验来判断本质。
