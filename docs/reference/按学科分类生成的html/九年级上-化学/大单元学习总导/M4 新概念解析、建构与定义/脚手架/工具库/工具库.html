双栏对比表

比较分析

系统对比两个易混淆概念，如物理变化 vs 化学变化

使用步骤：
  1. 列出对比维度（定义、本质、特征、例子）
  2. 分别为两个概念填写
  3. 找出关键差异，标记在表格下方
对比维度物理变化化学变化
定义无新物质生成有新物质生成
本质分子不变分子改变
例子水结冰铁生锈
