🥣 第1阶 · 生活情境 ★☆☆ 基础
家里的食盐为什么"受潮结块"？——分析潮解是物理变化还是化学变化

📌 案例材料

夏天，厨房里的食盐（主要成分NaCl）放置在空气中一段时间后，原本干燥的盐粒变得潮湿，甚至结块。妈妈问你："这盐还能吃吗？它是变质了吗？"

🎯 解析任务

分析食盐潮解的过程，判断它属于物理变化还是化学变化，并说明判断依据。

🔍 解析路径

识别问题→确定对象（食盐、水蒸气）→拆解要素（外观变化、成分是否改变）→梳理过程→寻找证据→得出结论

📖 原理调用

① 物理变化与化学变化的区分原理（有无新物质生成）
② 潮解的定义：物质吸收空气中的水分，表面溶解

实验室的镁条燃烧发出耀眼白光——分析这个变化的过程和产物

📌 案例材料

化学实验课上，老师用坩埚钳夹住一根银白色镁条，在酒精灯上点燃。镁条剧烈燃烧，发出耀眼的白光，放出大量热。燃烧后镁条消失，留下白色粉末。

🎯 解析任务

分析镁条燃烧过程中发生了什么变化，判断变化类型，写出化学方程式。

📖 原理调用

① 燃烧的定义（剧烈的发光放热的化学变化）
② 化学变化的判断依据（有新物质生成）
③ 化学方程式书写原理

🧪 第3阶 · 家庭/社区情境 ★★☆ 进阶
为什么用"小苏打+醋"可以疏通厨房下水道？

📌 案例材料

厨房下水道堵塞了，妈妈倒了一些小苏打（碳酸氢钠NaHCO₃）和白醋（含醋酸CH₃COOH），管道里立刻冒出大量气泡，过了一会儿管道就通了。

🎯 解析任务

分析小苏打与醋混合产生气泡的原因，判断产生的气体是什么，说明为什么气泡能帮助疏通管道。

📖 原理调用

① 酸与碳酸盐的反应原理（产生CO₂）
② 气体生成对管道内压力变化的影响

⚗️ 第4阶 · 学科核心情境 ★★☆ 进阶
分析铁丝在氧气中燃烧——为什么集气瓶底要放水？

📌 案例材料

将细铁丝盘成螺旋状，一端系一根火柴，点燃火柴后伸入盛有氧气的集气瓶中。铁丝剧烈燃烧，火星四射，生成黑色固体。集气瓶底事先放了一些水或细沙。

🎯 解析任务

分析铁丝燃烧的现象变化，解释"瓶底放水"的目的，判断变化类型。

参考方向：铁丝燃烧：3Fe + 2O₂ 点燃 Fe₃O₄（四氧化三铁，黑色）。火星四射说明反应剧烈，熔化后的高温生成物溅落。瓶底放水是为了防止高温熔融物炸裂集气瓶底。属于化学变化（新物质Fe₃O₄生成）。
🧫 第5阶 · 跨学科情境 ★★★ 挑战
夏天剩饭容易"馊"掉——从化学和生物两个角度分析

📌 案例材料

夏天的米饭放在室温下，第二天就发出酸馊味，不能再吃了。而放在冰箱里的米饭可以保存好几天。

🎯 解析任务

从化学角度（物质变化、化学反应速率）和生物角度（微生物繁殖、酶的作用）分别分析米饭馊掉的原因，并说明温度在其中起什么作用。

化学视角：米饭中的淀粉在微生物分泌的酶催化下水解为糖类，糖类进一步发酵产生有机酸（醋酸、乳酸）和醇类 → 产生酸馊味。属于化学变化。
生物视角：微生物（细菌、霉菌）在温暖环境中快速繁殖，分解米饭中的有机物。
温度作用：温度影响化学反应速率和微生物繁殖速度。高温（夏天）加速，低温（冰箱）减慢。
综合结论：馊饭是微生物引发的复杂化学变化过程。
🌧️ 第6阶 · 社会议题情境 ★★★ 挑战
分析"酸雨"的形成过程和危害——涉及哪些化学变化？

📌 案例材料

酸雨是指pH值小于5.6的酸性降水。化石燃料燃烧产生的SO₂和NO₃等气体排放到大气中，经过一系列化学反应转化为硫酸和硝酸，随雨水降落。

🎯 解析任务

分析酸雨形成过程中涉及的化学变化，写出SO₂转化为硫酸的化学方程式，说明酸雨对环境的主要危害。

化学变化过程：
① S + O₂ 点燃 SO₂（含硫燃料燃烧）
② 2SO₂ + O₂ ⇌ 2SO₃（在催化剂/光照下）
③ SO₃ + H₂O → H₂SO₄（形成硫酸）
危害：腐蚀建筑物（大理石CaCO₃ + H₂SO₄ → CaSO₄ + CO₂↑ + H₂O）、酸化土壤和湖泊、损害植物、腐蚀金属制品。
防治：使用清洁能源、烟气脱硫、推广新能源汽车。
🧪 第7阶 · 综合迁移情境 ★★★ 挑战
给你一瓶未知白色粉末，设计一套分析方案判断它是什么

📌 案例材料

你得到一瓶未知的白色粉末，标签已脱落。已知它可能是：食盐(NaCl)、小苏打(NaHCO₃)、蔗糖(C₁₂H₂₂O₁₁)、面粉（主要成分淀粉）中的一种。

🎯 解析任务

设计一套完整的化学分析方案，利用你已学过的化学知识和方法，逐步确定该白色粉末的成分。要求写出每一步的检验方法、预期现象和判断依据。

参考方案：
第一步（物理方法）：观察颜色、状态，闻气味——排除明显异味物质。
第二步（溶解性测试）：取少量样品溶于水——蔗糖易溶、小苏打可溶、面粉难溶形成悬浊液。
第三步（加白醋测试）：取样品加白醋——小苏打产生CO₂气泡，其他不反应。
第四步（碘液测试）：取样品加碘液——面粉变蓝（淀粉遇碘变蓝）。
第五步（不可尝药品！）：严禁在实验室尝药品！用专用试剂检测Cl⁻（如AgNO₃溶液）
综合判断：通过排除法和确定性检验，逐步缩小范围，确定未知粉末成分。
