分层作业 · 知识内化与表达 · 走进化学世界

🎯 任务定位

通过分层作业实现知识的 "内化与表达"。你可以根据自己的学习水平选择适合的层次，逐步巩固对"走进化学世界"核心内容的理解。

三层作业结构

🌱 基础层：再现与理解

全体同学必做，巩固基础知识

1. 概念辨析
判断下列现象是物理变化还是化学变化，并说明你的判断依据：
  1. 冰融化成水
  2. 蜡烛燃烧
  3. 铁棒被磁铁吸引
  4. 食物腐烂
提示：判断关键是有无 新物质生成。
① 冰→水：无新物质（物理变化）
② 蜡烛燃烧：有新物质 CO₂ 和水（化学变化）
③ 铁棒被磁铁吸引：无新物质（物理变化）
④ 食物腐烂：有新物质生成（化学变化）
2. 知识填空
化学变化的基本特征是 ______。
物理变化和化学变化的本质区别是 ______。
答案：化学变化的基本特征是 有新物质生成。本质区别是 是否有新物质生成。
3. 观察描述
描述你身边的一个化学变化和一个物理变化，每种变化至少写 3 个观察特征。

🌿 分层层（三选一）

选择最适合你的方案完成

A方案（图文型）
画一幅 "物理变化 vs 化学变化" 的海报，至少包含 3 组对比案例，每组用图文结合的方式展示变化前、变化后的状态及判断依据。
B方案（写作型）
写一段 "化学侦探" 故事，找出故事中出现的所有化学变化并分析。要求：至少包含 3 个化学变化，每个变化说明判断依据。
C方案（实践型）
在家里做一个小实验（如醋+小苏打 🧪），记录完整观察过程并分类。
步骤：准备材料 → 记录实验前状态 → 操作 → 记录实验后状态 → 判断变化类型 → 说明理由

🌟 开放层：创造性表达

任务
用你最喜欢的形式（视频 / 漫画 / 流程图 / 小论文）回答这个问题：

要求至少运用本模块学到的 3 个化学概念。
