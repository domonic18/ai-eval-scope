🏗️ 项目社区：用化学改造世界

📌 任务定位：以真实问题为驱动的长期学习机制，通过项目化学习综合运用本模块所学。选择感兴趣的项目开始吧！

🌱 短周期（1-3天）：微项目

短周期 · 1-3天 🏠 家中探索

项目名称：我家中的化学变化地图

🎯 项目目标 找到家中至少 5处 正在发生或可能发生的化学变化

📋 阶段任务

  • 在家中各区域（厨房、卫生间、阳台）做观察
  • 拍照或记录现象
  • 判断变化类型并说明依据
🔍 观察 📸 记录 📝 判断

📦 成果形式

一张 A4纸手绘地图，标注各位置的化学变化类型

📊 评价标准

★ ★ ★ ★ ★
自评标准：正确性 ⭐ | 数量≥5处 ⭐ | 表达清晰度 ⭐
💡 观察小贴士：厨房（铁锅生锈、切开的苹果变色、小苏打+醋反应）→ 卫生间（肥皂和硬水反应、漂白剂氧化）→ 阳台（植物光合作用、橡胶老化）

📐 中周期（1-2周）：专题项目

中周期 · 1-2周 🧪 化学链条

项目名称：铁的"一生"——从铁矿石到铁锈

🎯 项目目标 完整追踪铁从开采到锈蚀的化学变化链条

📋 阶段任务

  • 研究铁矿石 → 生铁 → 熟铁 → 铁制品 → 铁锈的化学变化过程
  • 记录每个环节的变化类型和化学原理
  • 设计一套减缓铁制品生锈的方案
📖 研究 📝 记录 🛡️ 防锈方案

📦 成果形式

信息图表（含文字说明）+ 防锈方案书

📊 评价标准

科学性 · 完整性 · 创意性

📎 参考：铁矿石种类（磁铁矿Fe₃O₄、赤铁矿Fe₂O₃） 📎 参考：高炉炼铁原理 📎 参考：电化学腐蚀原理

🌟 长周期（1月+）：综合项目

长周期 · 1月+ 🔬 校园调查

项目名称：校园化学安全侦探

🎯 项目目标 调查校园中化学品使用和管理情况，提出改进建议

📋 阶段任务

  • 调查实验室、生物园、清洁间等处的化学品存放情况
  • 分析每种化学品可能发生的化学变化和风险
  • 制作"校园化学安全手册"
🔎 调查 ⚠️ 风险评估 📘 制作手册

📦 成果形式

调查报告 + 安全手册（可打印分发）

📋 学校实验室安全守则 📋 MSDS化学品安全数据库 📋 危险化学品管理条例

📝 项目反思

完成项目后，问自己：

  • 我通过这个项目学到了什么新的化学知识？
  • 在项目中遇到了什么困难？是如何解决的？
  • 我的成果是否可以改进？如何改进？
  • 我是否可以用同样的方法研究其他物质（如铜、铝）的"一生"？
