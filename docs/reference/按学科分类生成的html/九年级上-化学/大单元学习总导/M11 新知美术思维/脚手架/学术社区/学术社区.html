📌 讨论议题 1（结构化）

"有人说'铁生锈'是化学变化，'铁被磁铁吸引'是物理变化。
你能用证据说服一个不同意的人吗？"

思考路径：想一想：铁生锈后产生了什么新物质？（红棕色的铁锈——主要成分 Fe₂O₃）。铁被磁铁吸引后，铁还是铁，没有新物质生成。区别就在"是否有新物质产生"。

📌 讨论议题 2

"你能想出 3 个看起来像是化学变化、实际上是物理变化的例子吗？"

参考示例：
① 玻璃破碎——看起来"变了"，但玻璃还是玻璃（SiO₂）
② 冰块融化——固体变液体，但 H₂O 还是 H₂O
③ 食盐溶于水——盐"不见了"，但蒸干水后盐又出现了
