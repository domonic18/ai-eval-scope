⏱️ 播放至 6:00 — 总结段时暂停

👆 点击展开思考提示

想一想：视频中的判断标准是否覆盖了所有情况？
💡 有些变化可能同时包含物理和化学变化，比如点燃蜡烛。

✍️ 输出任务

用你的语言写一段"什么是化学变化"的说明（50-100字）

参考范例：

化学变化是指物质的分子结构发生改变，生成新物质的过程。与物理变化不同，化学变化中原子重新组合，旧化学键断裂，新化学键形成。判断标准是是否有新物质生成——比如铁生锈（生成Fe₂O₃）是化学变化，而铁熔化（仍为Fe）是物理变化。

✍️ 输出任务

写一段 100字 的反思——化学家"设计分子"和美术家"构图创作"有什么相似之处？

示例观点：

化学家设计分子如同美术家构图创作：美术家选择色彩和形状来传达情感，化学家选择原子和键合方式来创造功能。美术家考虑构图的平衡与美感，化学家考虑分子的稳定性与活性。两者都遵循"结构决定功能"的原理，都在创造前所未有的"作品"——只不过一个在画布上，一个在分子世界里。
