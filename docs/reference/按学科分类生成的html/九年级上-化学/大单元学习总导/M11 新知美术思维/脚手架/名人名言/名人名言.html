安东尼·拉瓦锡

奠基人物 · 1743–1794
"物质燃烧后真的'消失'了吗？"
👆 点击翻转查看思维路径

🧠 思维突破

关键问题
物质燃烧后真的"消失"了吗？
突破点
不再接受"燃素说"的权威解释，而是用精密天平做 定量实验——称量燃烧前后所有物质的重量。
关键证据
金属在密闭容器中加热，总质量不变——说明燃烧不是"失去"什么，而是 "结合"了空气中的某种成分（氧气）。
对当下的启示
敢于用 量化数据 挑战直觉和权威。
🧪

罗伯特·波义耳

跨界人物 · 1627–1691
"怎么确定一种物质是'元素'还是'混合物'？"
👆 点击翻转查看思维路径

🧠 思维突破

关键问题
怎么确定一种物质是"元素"还是"混合物"？
突破点
把化学从炼金术的迷雾中拉出来，提出 "元素是不可再分解的简单物质"。
关键实验
用石蕊试液检验酸碱性——开创了 指示剂 的先河。
对当下的启示
分类思维 是科学的基础——先把东西分清楚，才能进一步研究。
🌿

屠呦呦

当代人物 · 1930–
"如何从传统草药中找到抗疟有效成分？"
👆 点击翻转查看思维路径

🧠 思维突破

关键问题
如何从传统草药中找到抗疟有效成分？
突破点
不限于水煮法，改用 乙醚低温提取——保留了青蒿的有效成分（青蒿素）。
成就
2015 年获得 诺贝尔生理学或医学奖，成为首位获科学类诺贝尔奖的中国本土科学家。
对当下的启示
解决问题需要 突破思维定式，敢于尝试非常规方法。
