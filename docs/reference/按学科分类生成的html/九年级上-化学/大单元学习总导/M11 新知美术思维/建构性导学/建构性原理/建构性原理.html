🧩 五大建构性原理
原理 1 ▾

观察锚定法

任何建构从"我看到了什么"开始

说明：面对陌生物质或现象时，先不做判断，先做描述。用五感 + 工具做完整记录：颜色、气味、状态、温度变化等。

适用情境：面对陌生物质/现象时，先不做判断，先做描述。

原理 2 ▾

证据驱动法

每一步判断都需要证据支持

说明：你的每一步判断都需要有证据支持——"我是因为看到了 ___，才得出了 ___ 的结论"。

适用情境：需要做出分类或判断时。

原理 3 ▾

分类建模法

把相似现象归成组，用对比建构概念

说明：把相似的现象归成组，用"两组对比"的方式来建构概念。比如物理变化 vs 化学变化，通过对比差异来加深理解。

适用情境：区分容易混淆的概念时。

原理 4 ▾

自定文法

先用你自己的语言说清楚理解

说明：在学到规范概念之前，先用自己的语言说清楚你的理解。用自己的话复述，才是真正的内化。

适用情境：初次接触新概念时。

原理 5 ▾

检验修正法

随时准备用新证据调整原有理解

说明：建构出的理解需要接受新证据的检验，随时准备调整。科学知识本身就是不断被修正的。

适用情境：学到更多新知识后回顾原有理解时。

🧪 迁移练习

📝 练习 1：面对一种新的化学物质（铁锈）

  • 用"观察锚定法"描述铁锈的颜色、质地、来源
  • 用"证据驱动法"判断铁锈还是铁吗？需要什么实验验证？

📝 练习 2：面对生活中一个化学现象（苹果切开发黄）

  • 用"分类建模法"判断是物理变化还是化学变化
  • 用"自定文法"写出你的解释
