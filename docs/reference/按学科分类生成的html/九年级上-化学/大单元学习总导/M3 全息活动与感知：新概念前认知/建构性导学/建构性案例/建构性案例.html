个人理解型
案例 1 / 7

我的"化学变化"概念图

把"什么是化学变化"用自己的方式整理出来。

情境：学完蜡烛燃烧后，需要将"化学变化"的概念系统整理。

建构任务：绘制"化学变化"核心概念图——定义、判断依据、与物理变化的区别、3个生活例子。

原理调用：📊 可视化模型法

建构步骤：整理概念 → 确定层级 → 建立关系 → 加入实例 → 检查完整性

📎 成果：化学变化核心概念图（手绘或电子版）

生活应用型
案例 2 / 7

家庭物品变化观察报告

判断5种物品变化是化学变化还是物理变化。

情境：你找到5种物品：苹果变褐、铁锅生锈、冰块融化、酒精挥发、纸张燃烧。

建构任务：判断哪些是化学/物理变化，说明判断依据。

原理调用：🔍 现象驱动法 + 🔗 证据链建构

建构步骤：列出物品 → 观察变化 → 判断有无新物质生成 → 分类 → 制作对照表

📎 成果：变化分类报告（对照表+判断依据）

案例 3 / 7

校园实验安全指南

设计化学实验安全操作要点指南。

情境：为新同学制作"化学实验安全指南"海报。

建构任务：设计包含"5个化学变化实验的安全操作要点"的指南。

原理调用：🔄 检验修正循环

建构步骤：确定实验清单 → 分析风险点 → 制定措施 → 设计版面 → 互检修订

📎 成果：化学实验安全指南海报

案例 4 / 7

实验现象"翻译官"

用规范化学语言描述"镁条燃烧"实验。

情境：做了"镁条燃烧"实验，观察到耀眼白光、白色粉末生成。

建构任务：规范实验报告——目的、步骤、现象、结论。

原理调用：💬 表达即理解 + 🔗 证据链建构

建构步骤：回忆实验 → 关键词记录现象 → 专业术语描述 → 现象→结论对应 → 撰写报告

📎 成果：镁条燃烧实验的规范报告

跨学科整合型
案例 5 / 7

跨学科"变化"主题研究

化学、物理、生物中的"变化"对比。

情境："变化"不仅发生在化学中——水循环（物理）、铁生锈（化学）、树木生长（生物）。

建构任务：制作"不同学科中的变化"对比海报。

原理调用：📊 可视化模型法 + 🔍 现象驱动法

建构步骤：列出各学科案例 → 分析特征 → 找联系与区别 → 设计对比框架 → 呈现成果

📎 成果：跨学科"变化"对比海报

案例 6 / 7

社区环保"化学侦探"

设计水质污染化学检测方案。

情境：小区河水颜色异常，怀疑有工厂排放污染物。

建构任务：设计"用化学方法检测水质污染"的简易方案（方案设计，不需实际检测）。

原理调用：🔗 证据链建构 + 🔄 检验修正循环

建构步骤：确定检测目标 → 设计检测方法 → 预测现象 → 分析可靠性 → 撰写方案

📎 成果：水质污染检测方案（含指标、方法、预期现象、判断标准）

案例 7 / 7

"化学变化"互动展品设计

设计让参观者亲身体验的互动展品方案。

建构任务：设计互动展品方案——让参观者亲身体验化学变化。

原理调用：🔍 现象驱动法 🔗 证据链建构 📊 可视化模型法 🔄 检验修正循环 💬 表达即理解

建构步骤：确定主题 → 设计互动 → 规划材料 → 评估安全 → 设计展板 → 检查科学性
