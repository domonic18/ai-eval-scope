🕯️ 建构性示例

从"观察蜡烛燃烧"到"我的化学变化解释模型" · 九年级上
🎯 建构目标与情境

🔥 核心矛盾

你亲眼看到蜡烛"消失了"——但科学告诉你：物质并没有消失，只是变成了其他物质。

你点起一根蜡烛，看到火焰跳动、蜡油流淌、烛芯变黑。你真的"看见"它们了吗？蜡烛燃烧时到底发生了什么？燃烧后的物质去了哪里？

📌 建构目标

建构你对"化学变化"的第一个科学解释模型：用证据说明蜡烛燃烧前后物质的变化，形成"化学变化中物质转化但不消失"的初步理解。

成果名称：蜡烛燃烧的化学变化解释模型（图文结合）

对象：向同桌或家长解释"蜡烛燃烧时到底发生了什么"

形式：A4纸大小的手绘示意图 + 文字说明

最低标准：包含3个观察证据、2个推理链条、1个核心结论

📊 建构成果样例

成果名称：蜡烛燃烧的化学变化解释图

蜡烛 + 氧气 → 水 + 二氧化碳 + 能量

评价要求：

  • 科学性：至少正确体现"反应物→生成物"的转化关系
  • 完整性：包含观察证据、推理过程、核心结论三部分
  • 美观性：图文结合，使用箭头清晰标注变化方向

✨ 成果优化建议

  • 检查模型是否能回答"物质是否消失了"这个问题
  • 加入"能量转化"的视角：燃烧放出的热量从哪里来？
  • 把你的模型和同学对比，看看谁的解释更完整
