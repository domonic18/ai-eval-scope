🔬 实验记录模板
规范记录每一次化学实验的完整过程
进阶 ▼

模板结构：实验名称 → 实验目的 → 实验器材 → 实验步骤 → 实验现象 → 分析与结论

示例（蜡烛燃烧实验）：

【实验现象】火焰上方冷烧杯内壁出现水雾 → 说明燃烧生成了水
【实验现象】澄清石灰水变浑浊 → 说明燃烧生成了二氧化碳
【分析与结论】蜡烛燃烧是化学变化，生成了水和二氧化碳

适用场景：所有实验课、家庭小实验
