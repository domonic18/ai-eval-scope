📜 蔡伦改进造纸术
东汉时期，蔡伦想要制造一种廉价、轻便的书写材料。
❓ 如何将树皮、麻头等植物纤维转化为可书写的纸张？

方案B 捣碎加水浸泡 点击查看结果 →
🟡 结果：有了一些纤维，但不成形
方案C 碱煮+捣碎+抄纸+晾干 点击查看结果 →
🟢 结果：造出了质地均匀的纸张

🌍 从"白色污染"到"绿色塑料"
传统塑料（如塑料袋）在自然环境中几百年不分解，造成严重的环境污染。
❓ 如何设计一种既有塑料的便利性，又能在使用后通过化学变化自然降解的材料？
方案A 开发可降解塑料（PLA） 点击查看结果 →
🟡 结果：可降解，但成本高，需特定降解条件
方案B 回收利用现有塑料 点击查看结果 →
🟡 结果：需分类和特殊工艺，回收率低
方案C 微生物"吃"塑料 点击查看结果 →
🟢 结果：有前景但技术尚未成熟
💡可迁移启示：理解化学变化的本质（物质转化）是解决环境问题的关键。

• 如果只选方案A，10年后可降解塑料成本降低但需完善回收体系，50年后可能替代50%传统塑料
