💬 讨论议题

基础
分享你发现的一个"生活中容易误判的化学变化"
点击展开思考方向
思考提示：比如"玻璃打碎"是物理变化还是化学变化？还有哪些日常现象容易让人误判？
进阶
你的"化学变化解释模型"和同学的有什么不同？谁的更有说服力？
点击展开思考方向
思考提示：关注模型的要素：是否包含证据？推理是否完整？是否考虑了反例？
挑战
"化学变化中一定有发光发热吗？"——找反例来论证
点击展开思考方向
思考提示：如铁生锈会发热但发光不明显；有些化学变化吸热进行。你能找到更多反例吗？
