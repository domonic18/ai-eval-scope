🌱 基础层：再现与理解

所有同学必做，巩固核心概念

作业 A1：写出化学变化和物理变化的定义，各举 3 个生活中的例子。
作业 A2：判断下列变化属于化学变化还是物理变化，并说明理由：
  • ① 湿衣服晾干
  • ② 食物腐败
  • ③ 玻璃杯摔碎
  • ④ 木柴燃烧
  • ⑤ 铁水铸成铁锅
作业 A3：填写"蜡烛燃烧"实验的观察记录表（参考课堂提供的空表模板）。
记忆口诀：化学变化"新物质"诞生，物理变化"外貌"改变。
区分关键：有没有 新物质 产生。

🌿 分层作业（三选一）

根据自己的水平选择适合的挑战

基础 作业 B1：选择一个化学变化和一个物理变化，用图文结合的方式对比呈现它们的区别。
进阶 作业 B2：设计一个简单的家庭实验，证明"物质在化学变化中不会消失"，写出实验步骤并预测现象。
挑战 作业 B3：查阅资料，找出一个"人们曾经误以为是物理变化，后来发现是化学变化"的案例，写一篇200字分析短文。
📌 选定后请在作业开头标注所选编号（如 B1）

🌸 开放层：创造性表达（五选一）

用你喜欢的方式展示你对化学变化的理解

写作 作业 C1：以"一个化学原子的自述"为主题，写一篇微型科幻故事，描述它经历一次化学变化的感受。
图示 作业 C2：绘制一张"化学变化侦探手册"信息图，列出"判断化学变化的5个线索"。
视频 作业 C3：拍摄一段3分钟短视频，用家里的物品演示一个化学变化并解释原理。
演讲 作业 C4：准备一段2分钟的"化学变化到底是什么"讲解稿，用比喻帮助小学生理解。
设计 作业 C5：设计一个"化学变化变身卡"游戏——每张卡片正面写一种变化，背面写"是/不是化学变化"及其理由。
🎯 选一个最能激发你创意的任务！
