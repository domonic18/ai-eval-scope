"错误的事实可以靠正确的方法修正，但错误的方法即使面对正确的事实也得不出正确的结论。"
——弗朗西斯·培根

📖 经典著作导读

《天工开物》 中国 宋应星 · 1637年
❓ 古代中国人是如何开发利用自然物质的？
目录：乃粒 → 彰施 → 陶埏 → 杀青 → 五金

核心观点：书中记录了古代中国人在染色、制陶、冶金等过程中对"化学变化"的无意识应用。

"凡石灰，经火焚炼为用。"——古人利用高温让石灰石发生分解反应制取生石灰。

适读人群：基础层适合阅读精选片段和译文。

📕 中华书局版 📱 微信读书 📱 豆瓣阅读（免费）
《怀疑的化学家》 外国经典 波义耳 · 1661年
❓ 化学是一门独立的科学，还是炼丹术的附属品？
对话体 · 六次对话讨论"什么是化学研究的对象"

核心观点：波义耳提出"化学元素"概念，认为化学应有自己的研究对象和方法，不再是炼金术的仆人。

"我把元素理解为某些原始的、简单的、完全未被混合的物质……"

适读人群：进阶层（有文言/英文阅读基础）；基础层可读导读版。

📕 北京大学出版社 🌐 Project Gutenberg（免费英文版）
《元素的盛宴》 当代 山姆·基恩 · 2011年
❓ 元素周期表中的每一个元素背后有什么故事？
以故事串联科学史，每章讲一个元素背后的传奇

核心观点：化学不是枯燥的元素符号，而是一部充满传奇的探险史。

第3章 "铅中毒的罗马贵族"——古代人如何在无意识中经历了化学变化导致健康问题。

适读人群：基础→进阶都适合。
