🔍 本模块学习什么

用你的眼睛、鼻子、手和大脑，全方位"感知"化学变化到底是什么。

  • 观察真实的化学现象（蜡烛燃烧等）
  • 学习"专家是怎么分析这些现象的"
  • 自己动手建构对"化学变化"的解释模型
  • 在7个不同情境中反复练习分析能力

💡 为什么重要

"化学变化"是整个化学学科的核心概念。

不理解化学变化，后面学的一切（元素、分子、化学反应方程式）都会像建在沙子上的城堡。

本模块让你先拥有对化学变化的感性体验，再理解它的科学含义。
