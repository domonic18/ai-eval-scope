1 早餐桌上的化学 生活情境 ▼

📋 案例情境

早上你倒了一杯牛奶，喝了一口后放在桌上。晚上回家发现牛奶结块了，散发出酸味。冰箱里的同一盒牛奶却没有变化。

🎯 解析任务

判断牛奶变酸是不是化学变化，并分析"温度"在这个变化中扮演了什么角色。

🧩 解析路径

识别现象 → 确定解析对象（牛奶酸败过程）→ 拆解要素（温度、微生物、时间）→ 判断是否有新物质生成 → 寻找证据

⚙️ 原理调用

核心判断准则法（有没有新物质生成？）+ 多重证据交叉验证（结块、气味变化是两个独立证据）

💡 参考思路

牛奶变酸涉及乳酸菌将乳糖转化为乳酸——有新物质生成。温度影响了微生物的活性。可以对比冰箱内外牛奶的不同结果来分析。

2 校园垃圾桶的"怪味" 校园情境 ▼

📋 案例情境

学校食堂的厨余垃圾桶在夏天总会散发出难闻的气味，而纸类垃圾桶几乎没有气味。

🎯 解析任务

分析"气味"的来源——这些气味物质是垃圾中本来就有的，还是后来通过化学变化新产生的？

🧩 解析路径

建立"有气味的垃圾 vs 无气味的垃圾"对比 → 分析垃圾成分 → 推理气味产生条件 → 判断变化类型

⚙️ 原理调用

系统观察法（全面观察不同垃圾桶）+ 推理链构建法（气味来源的推理链）

💡 参考思路

厨余垃圾中的有机物在微生物作用下发生分解（化学变化），产生了氨气、硫化氢等有气味的新物质。

3 妈妈的银手镯变黑了 家庭情境 ▼

📋 案例情境

妈妈一直戴的一只银手镯最近变黑了。她怀疑是质量问题，但店员说"银和空气中的某些物质发生了反应"。

🎯 解析任务

判断银手镯变黑是物理变化还是化学变化，并找出"凶手"是什么物质。

🧩 解析路径

识别变化（颜色变化）→ 收集信息（银的性质、空气成分）→ 分析可能的反应 → 寻找证据 → 形成判断

⚙️ 原理调用

核心判断准则法 + 概念检验法

💡 参考思路

银变黑是因为银与空气中的硫化氢反应生成黑色硫化银——有新物质生成，所以是化学变化。

4 水的"变身"三部曲 学科核心 ▼

📋 案例情境

将冰块放入烧杯加热：冰→水→水蒸气。将一张白纸点燃：纸→灰烬+烟。

🎯 解析任务

为什么水和纸的变化看似类似，但一个是物理变化一个是化学变化？请用核心标准详细对比分析。

🧩 解析路径

对比两种变化 → 关注"物质种类是否改变" → 分析变化过程中的产物 → 总结本质区别

⚙️ 原理调用

核心判断准则法 + 推理链构建法

💡 参考思路

水的三态变化中物质种类（都是H₂O）没有变；纸燃烧生成了新物质。核心区别在于分子种类是否改变。

5 光合作用——地球的"空气净化器" 跨学科 ▼

📋 案例情境

植物在阳光照射下，吸收二氧化碳和水，产生氧气和葡萄糖。这个过程被称为"光合作用"。

🎯 解析任务

从化学变化的角度分析光合作用——为什么它既是化学变化，又是地球上最重要的化学反应之一？

🧩 解析路径

确定变化发生场所（叶绿体）→ 识别反应物和生成物 → 分析能量变化 → 评估意义

⚙️ 原理调用

系统观察法 + 概念检验法（用质量守恒检验原子种类和数量是否相等）

💡 参考思路

CO₂在植物体内变成了O₂和营养物质——这是典型的化学变化。光合作用将无机物转化为有机物，是生命的基础。

6 空气污染中的"化学" 社会情境 ▼

📋 案例情境

某城市PM2.5超标，主要污染物是SO₂和氮氧化物。这些污染物进一步发生化学变化生成酸雨。

🎯 解析任务

分析"从工厂排放的气体到酸雨"的过程中发生了哪些化学变化，画出变化链。

🧩 解析路径

确定污染源 → 追踪化学转化路径 → 分析最终产物 → 评估影响

⚙️ 原理调用

多重证据交叉验证 + 推理链构建法

💡 参考思路

SO₂ + H₂O + O₂ → H₂SO₄（硫酸），这是一个完整的化学变化链。可以从酸雨对建筑/土壤的影响寻找证据。
