1 本模块学习什么？

核心内容：本模块是"走进化学世界"大单元的第一个跨学科应用模块。你将在已经学到的化学知识基础上，融合科学（S）、技术（T）、工程（E）、艺术（A）和数学（M）五个领域的思维和方法，完成一个"化学变化可视化"的建构任务。

  • 范围：基于化学第1章已学知识（化学变化与物理变化、常见仪器与基本操作）
  • 方式：案例跟学 + 独立建构
  • 产出：可视化作品 + 建构日志
2 本模块为什么重要？

化学不只是背方程式——它是理解世界的一种思维方式。本模块的独特价值在于：

  • 打破学科边界：你第一次被要求"不只是做化学题"，而是综合运用多学科工具
  • 从输入到输出：不再是"老师讲、你听"，而是你自己建构知识、产出作品
  • 真实世界问题：现实中的科学家和工程师每天都在跨学科协作解决问题
3 本模块与大单元主题的关系？

本模块是"第1章 走进化学世界"的重要组成部分。

  • 第1章让你认识了化学（什么是化学变化、化学实验室是什么样）
  • 本模块让你使用化学——用STEAM的方式去"做"化学
  • 它完美体现了"走进化学世界"的核心理念：化学就在你身边，你可以用它来创造
4 本模块与前后模块的衔接？

前面模块打下的基础：

  • 化学变化的概念和判断方法
  • 常见化学仪器的识别和使用
  • 基本的实验观察和记录能力

为后续学习做的准备：

  • 建立"化学变化中原子重新组合"的微观意识，为第2章"空气与氧气"中的化学反应微观解释做铺垫
  • 培养跨学科分析能力，为后续更复杂的STEAM项目积累经验
5 本模块预期学习成果？

学完本模块后，你能拿出以下成果：

  • 📦 核心作品：一份"化学变化过程可视化模型/海报/方案"
  • 📝 学习记录：一份建构日志（记录你的每一步思考和决策）
  • 🎤 表达输出：能用口头语言向他人清晰介绍你的作品和建构过程

成果形式要求：可视、可展示、可解释。可以是纸质海报、翻页模型、简易动画脚本等。
