学习目标
▼
知识理解：掌握化学变化的基本特征（新物质生成），能识别化学变化与物理变化的区别，理解"原子重新组合"这一核心概念。
能力形成：形成跨学科综合应用能力——能将化学知识与科学探究、技术工具、工程思维、艺术表达和数学方法相结合，解决一个真实问题。
思维发展：从"单一学科思维"升级为"系统建构思维"——在观察化学现象的基础上，学会从多个学科角度分析问题、设计方案、产出成果。
表达产出：完成一件可视化的STEAM融合作品（如翻页海报、模型、动画脚本），并能用口头和书面语言清晰解释你的建构过程。
🚀
高精尖价值与未来意义
▼

STEAM思维不是课本上的概念——它是现代科技创新的底层操作系统。

  • 新材料研发：科学家用化学原理 + 计算机模拟（T/M）设计新型催化剂，实现更高效的能源转化
  • 环境监测：工程师将化学传感器 + 数据分析（S/T/M）结合，实时检测空气和水质
  • 科普可视化：用化学知识 + 艺术设计（A）让抽象的微观世界"被看见"，这就是你在这个模块要尝试的方向

未来，无论你从事什么职业——跨学科解决问题的能力才是最有竞争力的核心素养。

这个模块就是你的第一次STEAM实战演练。

📖
推荐学习方式与方法
▼

本模块不是"听讲"模式，而是做中学。推荐以下方法：

  • 观察法：先从生活中真实的现象入手（铁钉生锈、酸碱变色），做好观察记录
  • 比较法：对比变化前后的差异，对比不同设计方案的效果
  • 建模法：用示意图、海报、模型等方式将微观过程"可视化"
  • 实验探究法：在安全条件下动手做小实验，获取第一手数据
  • 反思迭代法：完成初稿后，用评价量表自检并改进
💡 具体操作建议：准备一本"建构日志本"，每完成一个环节记录一步。先完整跟学建构性示例，再独立完成建构任务。

关键人物、事件与成果
▼
  • 🧪 安托万·拉瓦锡 (1743-1794) — "现代化学之父"。他以精确的实验和定量思维推翻了燃素说，确立了化学变化中的质量守恒定律。他的研究方法本身就是"科学+数学"的典范。
  • 🖼️ 大卫·A·古德塞尔 — 科学可视化先驱。他开创了用计算机图形学展示蛋白质结构和分子运动的方法，让"看不见的分子世界"变得可见，是"科学(S)+技术(T)+艺术(A)"的典范。
  • 🏭 乔治·华盛顿·卡弗 (1864-1943) — 化学家兼发明家。他从花生、红薯等农产品中提取出300多种衍生产品，完美示范了"化学+工程+应用"的STEAM思维。

🔍 拓展阅读提示：搜索"科学可视化竞赛""分子模型 3D打印"可以看到更多跨学科结合的精彩案例。
