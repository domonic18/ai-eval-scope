基础层 — 理解与识别

是什么？怎么样？—— 围绕化学核心知识的识别性问题

# 对比

化学变化和物理变化的本质区别是什么？请各举出 3 个生活中的例子。

💡 想想"有没有新物质产生"这个判据。

# 开放

实验室中如何区分一瓶蒸馏水和一瓶食盐水？请设计至少两种方法。

💡 可以从物理性质和化学性质两个角度思考。

# 元认知

当你看到一个物质变化时，你大脑中第一个跳出来的"判断标准"是什么？你是怎么确信它正确的？

💡 追问自己的思维过程——这比答案本身更重要。

📘

进阶层 — 分析与应用

为什么？怎么用？—— 需要运用分析能力和知识迁移的问题

# 对比

为什么铁制品在潮湿空气中容易生锈，而在干燥空气中不易生锈？你能设计一个实验来验证你的猜想吗？

💡 铁生锈需要同时接触氧气和水。

# 反事实

如果把一块金属钠（Na）投入水中会发生剧烈反应。如果不小心发生了这种情况，你觉得应该怎么处理？如果反应已经发生，产物是什么？

💡 钠是活泼金属，2Na + 2H₂O → 2NaOH + H₂↑。

# 开放

家中水壶里的水垢（主要成分CaCO₃）用什么方法去除最有效？请从化学角度分析你的方案。

💡 想想厨房里有什么物质能跟碳酸盐反应？

📕

挑战层 — 创造与批判

如果……会怎样？你怎么看？—— 需要批判性思维和创造性思考

# 反事实

如果没有化学变化，我们的世界会是什么样子？请从至少 3 个方面（能源、材料、生命）展开描述。

💡 想想你周围的一切——有多少是化学变化的产物？

# 开放

"塑料是20世纪最伟大的发明，也是最糟糕的发明。"你同意这个观点吗？用化学知识支持你的立场。

💡 从塑料的化学性质（稳定性、可塑性、难降解性）展开分析。

# 元认知

回顾你今天的学习——哪一个问题让你思考最久？你是如何调整思路最终得出结论的？这个调整对你有什么启发？

💡 反思自己的思维过程，就是最好的学习方法。
