🧪 物质的变化与性质——化学科普动画

来源：国家中小学智慧教育平台
🎯 观看任务
观看前明确：你的任务是在视频中找到并记录 3个化学变化现象 和 3个物理变化现象，并用一句话说明你判断的依据。
❓ 关键问题（看完回答）
1. 视频中是如何区分化学变化和物理变化的？判据是否一致？
2. 有没有看似像化学变化其实是物理变化的例子？为什么容易混淆？
3. 如果让你向一个小学五年级的同学解释这两个概念，你会怎么说？
⏸️ 暂停思考点
⏱️ 02:15 — 蜡烛燃烧思考：蜡烛融化是物理变化还是化学变化？蜡烛燃烧呢？两者同时发生时，如何分类？ ⏱️ 04:30 — 铁生锈思考：铁生锈需要哪些条件？（后来你会在第2章学到铁生锈的条件——现在可以先猜一猜）

📝 输出任务

完成一张"变化分类表"：列出你在视频中看到的6个现象（3化+3物），标注判断理由。

🏭 化学在工业生产中的应用——纪录片片段

来源：《我们需要化学》系列纪录片
🎯 观看任务
观看过程中记录：纪录片中提到了 至少3个不同的工业生产环节，每个环节运用了哪些学科知识（化学 + 物理 + 可能涉及的数学、工程等）？
❓ 关键问题（看完回答）
1. 纪录片中哪个环节让你印象最深刻？为什么？
2. 如果没有化学，这些工业过程会有什么替代方案？
3. 这些工业过程中，哪些步骤可能产生环境污染？如何处理？
⏸️ 暂停思考点
⏱️ 08:20 — 合成氨工厂思考：合成氨反应 N₂ + 3H₂ ⇌ 2NH₃ 是一个可逆反应。工业上为什么要用高压？如果不用高压会怎样？ ⏱️ 12:45 — 石油裂解思考：石油裂解是在高温下将大分子烃打断成小分子。这个过程遵循什么规律？小分子有什么用途？

📝 输出任务

绘制一张"工业生产中的STEAM交叉图"：选一个你感兴趣的工业生产过程，分析其中涉及了哪些学科的知识，画出交叉关系图。

辩论/前沿型 · 挑战

🌿 绿色化学：新材料如何改变未来——TED演讲

来源：TED演讲 / 一席
🎯 观看任务
带着批判性思维去听：演讲者提出了什么观点？他用什么证据支撑？你有没有发现漏洞或不同的视角？
❓ 关键问题（看完回答）
1. 演讲者关于绿色化学的核心主张是什么？你同意吗？
2. 视频中提到了哪些"新材料的可能性"？它们的化学原理可能是什么？
3. 提出一个质疑：如果这些新材料大规模推广，可能会带来什么新的问题？
⏸️ 暂停思考点
⏱️ 05:30 — 可降解塑料的化学原理思考：可降解塑料真的能完全降解吗？降解产物对环境友好吗？请查资料验证演讲者的说法。 ⏱️ 09:00 — 碳捕集材料思考：演讲者说"这种材料可以吸收空气中90%的CO₂"——你认为这个数据在实验室和在实际大规模应用中的效果会一样吗？原因是什么？

📝 输出任务

写一篇 200 字左右的"观点评论"：你对演讲者的核心主张持什么态度？支持/反对/部分同意？清楚地说明你的理由。
