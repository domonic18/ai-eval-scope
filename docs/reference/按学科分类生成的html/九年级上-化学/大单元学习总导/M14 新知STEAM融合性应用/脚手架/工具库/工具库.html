进阶

概念图绘制

用节点 + 连线 + 标签表示概念之间的因果或逻辑关系。

  • 列出核心概念（不超过 8 个）
  • 按"一般→具体"排列层级
  • 用箭头连接概念，在连线上写关系词（"导致""包含""例如"）
    • 示例：燃烧概念图：
    可燃物 →（需要）→ 氧气 →（达到）→ 着火点 →（产生）→ 新物质 + 能量

    AI 辅助学习

    用 AI 查资料、整理思路、生成练习。不是代替思考，而是加速探索。

    使用原则：① 先自己思考 → ② 向 AI 提问 → ③ 批判地吸收 → ④ 记录你的判断

    示例提问：
    "请列出化学变化和物理变化的 5 个区别，并用生活中的例子说明。""请帮我梳理燃烧三要素之间的关系，用概念图的形式。"

    演示工具 — PPT / Canva

    将你的研究成果用视觉化方式呈现给他人。

    Excel / WPS 数据图表

    将实验数据转化为可视化图表，发现趋势和规律。

    图表类型适用场景示例

    饼图占比分析空气中各成分体积分数

    ② 选中数据 → 插入图表
    ③ 选择图表类型
    ④ 添加标题和轴标签

    高阶

    图表深度分析方法

    不只是"看图说话"，而是从数据中挖掘规律、异常和假设。

  • 描述：你看到了什么？（趋势、极值、异常点）
  • 解释：为什么会这样？（联系化学原理）
  • 预测：如果继续会怎样？（外推趋势）
  • 质疑：数据可靠吗？有干扰因素吗？

    进阶

    虚拟实验平台推荐

    无法到实验室时，用数字模拟完成实验探究。

  • PhET 互动模拟（科罗拉多大学）— 免费，涵盖化学、物理多个主题
  • NOBOOK 虚拟实验 — 国内平台，支持初高中化学实验
  • 中央电化教育馆虚拟实验 — 国家平台资源
    • 先用虚拟实验熟悉操作流程，再进实验室操作。虚拟实验可以大胆尝试"如果加多一倍试剂会怎样"——试错成本为零。

    💡 工具使用建议

    1. 接到任务时先问自己：我需要"想清楚"（思维工具）还是"做出来"（数字工具）？

    2. 组合使用效果更好：先画思维导图理清思路 → 用 Excel 整理数据 → 用 Canva 呈现结论。

    3. 选工具不贪多：基础工具熟练后再挑战进阶和高阶。
