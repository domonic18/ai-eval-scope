第1章 走进化学世界 · 基础检测

▾
  • 填空 物理变化与化学变化的本质区别是什么？列举3个生活中的化学变化实例。
  • 选择 下列物质的性质中，属于化学性质的是？A. 颜色 B. 密度 C. 可燃性 D. 熔点
  • 匹配 将下列仪器与其用途连线：试管 / 酒精灯 / 量筒 / 托盘天平
  • 简答 描述蜡烛燃烧实验中的现象，并区分哪些是物理变化、哪些是化学变化。
🚀 分层任务 · 选择你的挑战 三选一

A层 · 巩固强化

▾
  • 绘制"物质的变化与性质"思维导图，涵盖概念、分类和实例。
  • 完成教材课后习题中关于实验基本操作的练习，记录易错点。
  • 设计一张"实验室安全守则"知识卡片，列出5条关键规则并说明原因。

B层 · STEAM应用

▾
  • 设计一个简易净水装置（利用过滤、吸附等原理），画出工程图并标注材料。
  • 撰写一篇300字短文：化学如何帮助解决某一环境问题（如水污染/空气污染）。
  • 制作一份"化学与烹饪"海报，解释发面、腌制、煎炸等过程中蕴含的化学原理。

C层 · 深度挑战

▾
  • 选择一个生活场景（如灭火、食品保鲜、金属防锈），分析其中涉及的全部化学原理，撰写分析报告。
  • 设计一个原创STEAM融合实验方案，包含：问题提出 → 跨学科分析 → 实验设计 → 预期结果。
  • 录制一段3-5分钟的"化学小课堂"视频，讲解一个化学概念并配以实验演示或动画示意。
