🧼 为什么小苏打能去污？

背景：厨房中的油污、茶渍、锅底焦黑——小苏打（NaHCO₃）是厨房里的"万用清洁剂"。
核心问题：小苏打为什么能去除油污？它比洗洁精好在哪？
化学原理：小苏打溶于水呈弱碱性（pH ≈ 8.4），能与油脂发生皂化反应，将不溶于水的油脂转化为可溶于水的脂肪酸盐（肥皂）和甘油。
决策路径：
  • 方案A：小苏打干擦 —— 简单快捷
  • 方案B：小苏打+白醋 —— 升级版
  • 方案C：强力化学去污剂 —— 高效但有代价
✅ 成功
小苏打干擦对轻度油污效果极佳，安全无毒，成本低。
⚠️ 视情况而定
气泡辅助对顽固污渍效果更好，但白醋可能损伤某些表面（如大理石）。
结果：小苏打已成为全球公认的绿色清洁方案——经济、安全、环保。
✨ 可迁移启示：一种物质的"隐藏性质"（弱碱性）可以解决看似不相关的问题。下次遇到清洁难题，先想想：有没有常见的物质可以利用它的化学性质？

🏭 工业制碱法的演变 — 从索尔维法到侯氏制碱法

背景：纯碱（Na₂CO₃）是玻璃、造纸、肥皂等工业的基础原料。19世纪欧洲垄断索尔维法技术，中国长期依赖进口。
核心问题：如何在一个没有索尔维法专利的国家，自主研制出纯碱生产工艺？
决策路径：
  • 方案A：侯德榜自主创新路线
  • 方案B：继续进口依赖
  • 方案C：简单仿制
✅ 成功
侯氏制碱法（联合制碱法）于1943年成功，将制碱与合成氨联合，同时生产纯碱和化肥NH₄Cl，成为世界制碱技术的里程碑。
❌ 失败
进口依赖导致新中国初期纯碱极度短缺，"洋碱"价格高企，民族工业举步维艰。
✨ 可迁移启示：技术突破的路径不只有"从头发明"——消化吸收再创新是一种高效的策略。化学工艺中的循环利用思想（减少浪费、提高效率）也是可持续发展的重要思路。

🧪 站在1940年的十字路口：

索尔维法的核心反应是：NaCl + NH₃ + CO₂ + H₂O → NaHCO₃↓ + NH₄Cl。副产物NH₄Cl之前被当作废液排放。如果你面临一样的技术困境，你会从哪个方向寻找突破？写下你的思考。

🌱 碳中和中的化学 — 从CO₂捕集到转化利用

背景：全球CO₂年排放量超过350亿吨。人类需要同时做到：减少排放 + 捕集已经排放的CO₂ + 将CO₂转化为有用物质。
核心问题：捕集的CO₂能做什么？化学转化是否是碳中和的真正出路？
决策路径：
  • 方案A：CO₂→甲醇 — 化学转化
  • 方案B：CCS地质封存
  • 方案C：生物固碳
✅ 前沿突破
CO₂加氢制甲醇被视为"液态阳光"——用可再生能源制氢，再用氢将CO₂转化为液体燃料。碳原子从"废物"变成了"资源"。
⚠️ 有限可行
CCS封存每年只能处理不到1亿吨CO₂（不到排放量的0.3%），且长期安全性未完全验证。
✨ 可迁移启示：最难的化学问题往往不是"能不能反应"，而是"能不能经济、环保、大规模地反应"。真实的化学创新，需要同时考虑效率、成本和环境——这正是STEAM的核心精神。

🧪 模拟决策：

你是某市环保局长。市里一座大型热电厂的CO₂排放问题需要处理。你有三个方案：① 投资CO₂转化化工厂（10亿，5年回收）② 建CCS封存设施（6亿，但每年运营成本高）③ 补贴电厂转型可再生能源（15亿，需要10年）。请做出你的选择并说明理由。
