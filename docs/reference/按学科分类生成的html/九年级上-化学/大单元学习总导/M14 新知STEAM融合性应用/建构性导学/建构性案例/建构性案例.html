案例1 🧑 个人理解型 · 制作"水的化学"理解卡片

📖 真实建构情境
你在化学课上学了"水是由氢元素和氧元素组成的化合物"，但你发现同桌仍然不太理解"为什么水是化合物而非混合物"。
🎯 建构任务
用A5大小的卡片，以可视化方式解释"为什么水是化合物"和"水由什么组成"，让同桌一看就懂。
🔧 原理调用提示
用"可视化表达法"把抽象的分子结构画出来；用"证据锚定法"引用水的电解实验作为证据。
📋 建构步骤
①明确核心信息：水是化合物（不同元素组成的纯净物）
②收集证据：水的电解实验（产生H₂和O₂）
③设计可视化方案：分子结构图+电解实验示意图
④制作卡片并加入简短解释文字
⑤请同桌阅读并反馈，优化表达
🧰 脚手架推荐
手绘+文字；或者用PPT/Canva制作电子版
📦 成果表达
名称："水的化学秘密"理解卡片；格式：A5大小，图文并茂
✅ 自评量规
✓ 准确解释了水是化合物而非混合物的理由
✓ 包含了水的电解实验证据
✓ 可视化设计清晰，非化学专业的同学也能看懂
✓ 卡片内容精炼，无冗余信息
✓ 经过至少1次反馈并优化

案例2 🏠 生活应用型 · 自制简易净水装置

📖 真实建构情境
你发现家里自来水有时会有漂白粉的味道，想给家里设计一个简单实用的净水方案。你手头有：空矿泉水瓶、纱布、活性炭、细沙、小石子、棉花。
🎯 建构任务
利用上述材料设计并制作一个简易净水装置，能用化学原理解释每一层材料的作用。
🔧 原理调用提示
用"迭代改进思维"先做第一版→测试→改进；用"可视化表达法"画出示意图。
📋 建构步骤
①查阅每种材料的过滤/吸附原理（化学视角）
②设计装置结构图（分层方案）
③动手制作第一版装置
④用浑浊水测试效果，记录问题
⑤迭代优化（调整材料顺序/比例）
🧰 脚手架推荐
参考：活性炭→吸附异味和色素；细沙→过滤细小颗粒；小石子→过滤大颗粒；纱布和棉花→防止材料流失
📦 成果表达
名称：家庭简易净水装置（实物+说明图）；格式：实物照片+化学原理解释说明
✅ 自评量规
✓ 装置能有效去除水中的可见杂质
✓ 能用化学原理解释每种材料的作用
✓ 装置有结构示意图
✓ 记录了至少1次测试和改进过程
✓ 装置使用了至少3种不同材料

案例3 🏫 校园解决型 · 化学实验室安全改进方案

📖 真实建构情境
学校化学实验室的安全设施较为陈旧——通风橱排风不够快，废液缸经常满溢，灭火器位置不显眼。化学老师请你帮忙做一个改进方案。
🎯 建构任务
调查实验室安全隐患，设计一套包含"硬件改进+流程优化"的实验室安全方案。
🔧 原理调用提示
用"从现象到问题法"从安全隐患中提炼关键问题；用"多学科视角切换术"化学（危化品）+ 美术（标识设计）+ 工程（通风方案）。
📋 建构步骤
①调查实验室现状，列出安全隐患清单
②区分"紧急可解决"和"需长期改进"的问题
③针对每个问题提出2种以上解决方案
④设计安全标识和应急流程图（视觉化）
⑤整合为完整的方案报告
🧰 脚手架推荐
校园安全记录表、实验室平面图（可手绘）、安全标识参考图集
📦 成果表达
名称：实验室安全改进方案报告；格式：A3海报或PPT，含危险清单+改进措施+效果预估
✅ 自评量规
✓ 已识别至少5个安全隐患
✓ 每个问题都提出了可行的改进措施
✓ 方案包含至少2个学科视角
✓ 方案包含视觉化元素（安全标识/流程图）
✓ 方案具有可操作性，能真正落地

案例4 📚 学科表达型 · "化学反应"跨媒介表达

📖 真实建构情境
化学老师在课堂上说"化学反应的本质是旧化学键断裂、新化学键形成"。你觉得这个概念太抽象了，想用一种更有创意的方式表达出来。
🎯 建构任务
选择一种跨学科媒介（连环画/短视频脚本/短剧/音乐/诗歌/动画分镜），表达"化学反应的本质"这一核心概念。
🔧 原理调用提示
用"可视化表达法"把断裂和形成的过程视觉化；用"迭代改进思维"先做草稿，请同学看是否理解，再优化。
📋 建构步骤
①列出要表达的核心知识点（化学键、断裂、形成）
②选择1种媒介形式，确定表达方式
③创作第一版本（草稿/分镜/初稿）
④请至少2位同学看后说出"你理解到了什么"
⑤根据反馈优化表达方式
🧰 脚手架推荐
可以查阅"化学键"的科普动画视频作为灵感参考
📦 成果表达
名称："化学键之舞"跨媒介作品；格式：根据选择的媒介形式而定
✅ 自评量规
✓ 准确表达了"旧键断裂、新键形成"的核心概念
✓ 选择了跨学科的媒介形式
✓ 作品有创意，不是简单复制教材定义
✓ 至少收集了2位同学的反馈
✓ 作品经过了至少1次迭代优化

案例5 🔬 跨学科整合型 · 光合作用与化学反应的联结

📖 真实建构情境
生物课上你学到了光合作用的反应式：6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂。化学老师也说这是一个化学反应。但你发现很多同学把这当成"生物的东西"来背，而没有看到里面的化学本质。
🎯 建构任务
制作一份"光合作用×化学"整合分析图，从化学视角重新解读光合作用——包括反应物和生成物的化学性质、能量转化、元素守恒等。
🔧 原理调用提示
用"多学科视角切换术"切换化学与生物视角；用"可视化表达法"将反应关系图表化。
📋 建构步骤
①列出光合作用的化学要素（反应物、生成物、条件）
②从化学角度分析每条要素（元素组成、化学键变化）
③从生物角度补充意义（能量转换、物质循环）
④设计整合分析图，体现两个学科的联结
⑤撰写简要说明文字
🧰 脚手架推荐
元素守恒示意图、化学键变化对比表；可参考概念图（Concept Map）的形式
📦 成果表达
名称：光合作用·化学深度解读图；格式：A4图文整合报告
✅ 自评量规
✓ 准确写出了光合作用的化学方程式
✓ 从化学和生物两个学科进行了分析
✓ 体现了元素守恒的思想
✓ 用可视化方式表达了两学科的联结
✓ 内容没有知识性错误

案例6 🌍 社会/未来型 · 社区塑料垃圾可循环方案

📖 真实建构情境
你所在的社区每天产生大量塑料垃圾（矿泉水瓶、塑料袋、餐盒等），虽然分类了但最终还是被焚烧或填埋。社区居委会想征集一个"塑料垃圾回收再利用"的创意方案。
🎯 建构任务
设计一个从"收集—分类—处理—再生"全流程的塑料循环方案，需包含化学原理说明和可行性评估。
🔧 原理调用提示
用"从现象到问题法"提炼塑料污染的核心问题；用"系统分析法"（来自解析性原理）做全流程设计；用"可视化表达法"呈现循环流程图。
📋 建构步骤
①调查社区塑料垃圾的种类和数量
②了解塑料的化学组成（聚乙烯/聚丙烯等）
③调研现有的塑料回收技术（物理回收/化学回收）
④设计适合本社区的全流程方案
⑤评估方案的成本和可行性
🧰 脚手架推荐
可以搜索"塑料回收标志编号"了解不同类型塑料的回收方式
📦 成果表达
名称：社区塑料循环方案书；格式：图文报告（含流程图+可行性分析）
✅ 自评量规
✓ 涵盖了收集、分类、处理、再生全流程
✓ 包含塑料的化学组成说明
✓ 方案具有社区实际可操作性
✓ 包含可视化流程图
✓ 有可行性分析（成本/效果/困难）

案例7 🚀 综合创新型 · 自由STEAM创作

📖 真实建构情境
到目前为止，你已经用建构性原理完成了6个不同方向的案例。现在，你可以自由选择自己最感兴趣的方向，完成一个完整的STEAM融合创作。
🎯 建构任务
自选一个与你生活相关的问题或主题，运用至少3条建构性原理，完成一个包含化学知识和至少另一个学科的STEAM融合创作。
🔧 原理调用提示
所有5条建构性原理都可用。特别推荐组合使用"从现象到问题法"→"多学科视角切换术"→"可视化表达法"→"迭代改进思维"，形成完整的工作流。
📋 建构步骤（参考）
①选定主题：列出3个候选，选择最感兴趣的一个
②提出问题：用"从现象到问题法"明确核心问题
③跨学科视角：用"多学科视角切换术"拓展思路
④设计创作方案：确定成果形式和内容
⑤制作初版方案并迭代优化
⑥完成作品并做创作说明
🧰 脚手架推荐
回顾前6个案例的结构和格式，选择你最擅长的方式
📦 成果表达
名称：自定；要求：含化学+至少1个其他学科，有完整的建构过程说明
✅ 自评量规
✓ 主题与生活相关，有实际意义
✓ 融合了化学和至少1个其他学科
✓ 运用了至少3条建构性原理
✓ 经过了至少2次迭代改进
✓ 有创作说明，能解释建构过程
