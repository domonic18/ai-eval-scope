案例1：为什么苹果切开后会变褐？ 生活情境

案例材料：小明切了一个苹果准备做水果沙拉，5分钟后发现苹果切面变成了褐色。他记得柠檬汁可以防止变色，但家里没有柠檬。

解析任务：分析苹果变褐的原因，并设计一个用家中常见材料防止变色的方案。

建议原理：因果链追踪法要素拆解法

解析路径提示：识别核心问题（酶促褐变）→ 拆解要素（苹果中的酶+氧气+酚类物质）→ 思考如何阻断反应链。家中可用食醋或盐水替代柠檬汁（酸性环境抑制酶活性）。

分析记录：建议用表格记录：观察到的现象 → 涉及的化学原理 → 可采取的措施 → 预期效果。

案例2：实验室通风橱的使用 校园情境

案例材料：化学实验室的通风橱在使用时有三个开窗高度档位。有同学认为开得越大越好，有同学认为调到最小挡才安全。

解析任务：分析通风橱的工作原理，确定不同实验应选择哪一档开窗高度，并说明理由。

建议原理：因果链追踪法视角切换法

解析路径提示：通风橱靠负压排气——开窗太大降低风速（安全风险），开窗太小操作不便。需平衡安全与操作需求：挥发剧烈实验用小开口，一般操作用中档。

分析记录：建议从"安全"和"操作便利"两个视角对比分析三档的利弊。

案例3：家庭除垢剂的科学 社区情境

案例材料：妈妈发现水壶底部积了厚厚一层白色水垢。邻居推荐用白醋除垢，也有朋友说用小苏打更有效。

解析任务：分析水垢的化学成分，判断白醋和小苏打哪个更有效，并用化学方程式解释原因。

建议原理：对比分析法要素拆解法

解析路径提示：水垢主要是CaCO₃和Mg(OH)₂。白醋（醋酸）与之反应生成可溶性盐，小苏打（弱碱）无法有效分解碳酸钙。醋酸除垢：CaCO₃+2CH₃COOH→(CH₃COO)₂Ca+H₂O+CO₂↑。

分析记录：从反应类型、反应速度、安全性、成本四个维度对比两种方法。

案例4：金属活动性顺序的验证 学科核心

案例材料：教材给出金属活动性顺序：K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au。实验室提供了铁钉、铜丝、硫酸铜溶液和稀盐酸。

解析任务：设计2个以上实验验证铁和铜在活动性顺序中的相对位置，预测现象并分析原理。

建议原理：要素拆解法证据分级法

解析路径提示：思路一：铁钉+硫酸铜溶液→铁表面出现红色铜（Fe+CuSO₄→FeSO₄+Cu）。思路二：铁+稀盐酸→产生气泡（Fe+2HCl→FeCl₂+H₂↑）；铜+稀盐酸→无现象。两个实验互为证据。

分析记录：填写实验目的→操作步骤→预测现象→实际现象→结论→证据等级。

案例5：光合作用与化学 跨学科

案例材料：植物光合作用产生葡萄糖（C₆H₁₂O₆），化学方程式：6CO₂+6H₂O→光照/叶绿素→C₆H₁₂O₆+6O₂。

解析任务：从化学视角分析光合作用的反应类型、能量转化和物质变化，并探讨如何通过调控条件提高光合作用效率。

建议原理：要素拆解法视角切换法

解析路径提示：化学视角看反应类型（氧化还原+吸热）→ 生物视角看酶的作用 → 环境视角看CO₂浓度的影响 → 工程视角看温室调控。

分析记录：从S（化学方程式分析）、E（温室设计）、M（效率计算）三维度填写。

案例6：塑料的"可降解"争议 社会议题

案例材料："可降解塑料"被宣传为白色污染的解决方案。但研究发现很多可降解塑料在自然环境中需要数十年才能降解，且降解产物可能含微塑料。

解析任务：区分不同类型"可降解塑料"的降解机制和条件，从多角度评价是否应全面推广可降解塑料。

建议原理：证据分级法视角切换法

解析路径提示：区分生物降解塑料（需特定菌群+温湿度）和光降解塑料（需紫外线）。区分不同主体的立场：生产商、消费者、环保部门、科学家。查阅标准：GB/T 20197-2006。

分析记录：事实与观点分开列；为每条证据标注等级（A/B/C）；按角色列出各方的核心关切。

案例7：新能源汽车的"锂"程碑 综合应用

案例材料：锂电池电动车快速发展，但锂矿开采（南美盐湖/澳大利亚矿山）存在生态影响。同时废旧锂电池的回收处理面临技术难题——传统回收法效率低且污染大。

解析任务：综合运用多条解析性原理，从化学原理、环境影响、技术路线、经济成本四个角度分析锂电池"全生命周期"的可持续性。

建议原理：全部5条原理

解析路径提示：要素拆解法分解S/T/E/A/M维度→因果链追踪法分析开矿→电池→回收的环境影响→证据分级法评估不同回收技术的优劣→对比分析法比较锂电池vs其他电池→视角切换法从不同角色（矿工、车企、环保组织）看问题。

分析记录：用综合表格完成"全生命周期分析"，覆盖开采→制造→使用→回收4个阶段。
