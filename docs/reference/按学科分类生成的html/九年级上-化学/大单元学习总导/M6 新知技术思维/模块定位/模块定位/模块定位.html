📖 一、本模块学习什么 ▶

本模块是"走进化学世界"的收尾模块——新知技术思维。你将站在更高的视角回望整章内容，系统梳理：

  • 化学变化与物理变化
  • 化学性质与物理性质
  • 科学探究的方法与步骤

同时了解化学学科对人类文明发展的深远影响，认识化学前沿技术如何改变我们的生活，并尝试用科学思维框架去分析身边的化学问题。

❓ 二、本模块为什么重要 ▶

如果把化学学习比作建造一座大厦，本模块就是蓝图和望远镜。

前面的课程帮你认识了"砖块"（基本概念），本模块则带你站上高处，看清这些砖块如何拼成宏伟建筑，以及建筑之外还有怎样的风景。它帮助你建立对化学学科的整体认知格局——这是从"学化学"到"用化学看世界"的关键一跃。

🔗 三、与大单元主题的关系 ▶

本单元"走进化学世界"的核心任务是建立化学学科的基本认知框架：

M1-M5 具体概念：变化、性质、科学探究过程
M6 整合升华：从"是什么"走向"为什么"和"还能怎样"

完成从知识积累到思维方式的跃迁。

🧩 四、与前后模块的衔接 ▶
← 前接 M1-M5 变化与性质区分、科学探究步骤、实验基本操作
后接 后续章节 → 空气、氧气、物质构成的奥秘……提供方法论支撑

培养的"宏观—微观—符号"三重表征思维和科学探究素养，将持续为后续学习输出方法论支持。

🏆 五、预期学习成果 ▶

完成本模块后，对照检查你是否达成：

用思维导图独立梳理整章核心概念及其逻辑关系
运用"变→因→规→用"框架分析1-2个生活中的化学现象
口述或书面阐述化学学科对人类文明的两项重要贡献
对化学前沿技术产生兴趣，并提出至少1个有价值的追问
