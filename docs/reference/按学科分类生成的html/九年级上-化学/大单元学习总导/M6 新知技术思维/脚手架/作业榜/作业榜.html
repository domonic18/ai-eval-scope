📝 A 级 · 应用（适合基础较弱的同学）

参照课本"蜡烛燃烧实验"的描述方式，写出"镁条在空气中燃烧"的完整观察记录（现象、产物、反应类型）。

镁条燃烧发出耀眼白光，放出大量热，生成白色固体（氧化镁）。反应类型：化合反应。现象描述要点：发光、放热、产物颜色状态。

📊 B 级 · 分析（适合大部分同学）

制作一张"物理变化 vs 化学变化"双气泡对比图，至少列出 5 项对比维度，附 100 字以内总结。

维度：定义、特征、实例、能量变化、分子变化。总结：物理变化分子不变、无新物质；化学变化分子改变、有新物质生成，常伴随能量变化。
