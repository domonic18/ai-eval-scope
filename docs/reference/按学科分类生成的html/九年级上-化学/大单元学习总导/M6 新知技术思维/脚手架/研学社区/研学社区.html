生活案例

为什么铁制品会生锈？

随处可见的铁锈背后藏着化学原理
核心问题：铁生锈需要什么条件？能否防止？
决策路径：观察现象 → 提出假设（水/空气/温度）→ 设计对照实验（铁钉在不同环境）→ 记录 → 结论
结论：铁生锈是铁与氧气、水共同作用的结果；涂油、镀层可防锈。
可迁移启示：生活中防锈措施的原理（如自行车链条涂油）；化学变化受多种条件影响。

拉瓦锡与氧气的发现

18世纪 · 化学革命的开端
核心问题：燃烧的本质是什么？是否存在"燃素"？
决策路径：质疑燃素说 → 设计汞+空气加热实验 → 测量密闭容器质量变化 → 发现氧气 → 建立氧化说
结果：推翻燃素说，化学革命的开端。
可迁移启示：科学进步来自对既有理论的质疑与实验验证；质量守恒是化学反应的基石。

联合制碱法——侯德榜的化学智慧

20世纪初 · 打破国际垄断
核心问题：如何用国产原料高效制碱？

结果：中国人自主掌握制碱技术，打破国际垄断。

跨领域案例

臭氧层空洞——从发现到全球行动

1970年代至今 · 环境化学与全球治理
核心问题：臭氧层为何被破坏？人类如何应对？
决策路径：监测数据异常 → 提出CFCs破坏假说 → 化学机理研究（Cl原子催化分解臭氧）→ 实验室模拟验证 → 国际谈判 → 蒙特利尔议定书 → 替代品研发
结果：全球禁用CFCs，臭氧层逐步修复。
可迁移启示：化学与环境政策密切相关；科学数据推动全球治理；科技进步需要配套监管机制。
