奠基人物 拉瓦锡 Antoine Lavoisier · 1743–1794

"现代化学之父"——引入定量方法，否定燃素说，提出氧化说

关键问题燃烧的本质是什么？物质变化前后质量守恒吗？
思维突破引入精密天平，用数据而非直觉判断；否定"燃素说"提出"氧化说"
失败/争议因税务官身份在法国大革命中被处决；氧气发现存在优先权争议
当下启示科学进步需要定量思维和实验验证精神
跨界人物 法拉第 Michael Faraday · 1791–1867

从装订工到科学巨匠——发现电解定律，开创电化学

关键问题电与化学之间是否存在联系？
思维突破发现电解定律，开创电化学；从物理跨到化学，证明自然力可以相互转化
失败/争议仅有小学学历，被皇家学会多次拒绝；早年研究因资金不足夭折
当下启示跨学科思维是重大发现的源泉；出身不决定上限
当代人物 屠呦呦 1930–

诺贝尔奖得主——从古籍中提取青蒿素，拯救数百万生命

关键问题如何从中草药中发现治疗疟疾的有效成分？
思维突破从古籍《肘后备急方》获得灵感，用乙醚低温萃取法提取青蒿素
失败/争议研究历经 190 次失败；青蒿素发现归属曾引发争议
当下启示现代化学要与传统智慧结合；失败是科学探索的常态
