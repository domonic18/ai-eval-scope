💧 校园水环境调查与化学分析

运用化学方法调查校园中不同水源的水质，形成调查报告。

🎯 目标运用化学方法调查校园水源水质，形成报告
📋 阶段
  • 第1-3天：确定采样点（饮用水/直饮水/雨水/池塘水），设计方案
