🧑‍🤝‍🧑 同伴连接 · 讨论与协作

选择一个议题，组内讨论后全班分享。

A 思辨讨论

"如果法拉第生在今天，他会成为科学家还是被应试教育埋没？"

🔍 讨论线索：法拉第仅有小学学历，靠自学和实验精神成为科学巨匠。今天的教育体系能发现和培养这样的"偏才"吗？
B 实验复盘

"拉瓦锡用定量实验推翻燃素说——我们今天能用什么方法来检验一个常识观点？"

🔍 讨论线索：想想生活中哪些"大家都这么说"的观点可能是错的？你能设计一个小实验检验它吗？
C 未来构想

"AI 能取代化学家做实验吗？化学家的独特价值在哪里？"

🔍 讨论线索：AI 可以分析数据、预测反应，但提出"好问题"、设计"新颖实验路径"——这些需要人类的创造力。

🏠 家校连接 · 支持与反馈

课前 和孩子一起完成"化学版你画我猜"——根据现象猜物理/化学变化
课中 孩子分享"我今天印象最深的化学实验现象"，家长听完签名确认
课后 家长协助拍摄"家庭小实验"（蜡烛燃烧、鸡蛋沉浮等）短视频上传班级群
