水壶中的水垢

🏠 生活情境
📖 情境你的家用烧水壶底部积累了一层白色硬壳状物质（水垢）。妈妈倒了一些白醋进去煮沸，水垢消失了，还冒出了小气泡。
🎯 任务分析"白醋去除水垢"的原理，写出化学方程式
🔗 解析路径现象锚定 → 类型归类（酸与碳酸盐反应）→ 概念调用 → 证据链推理 → 结论
📊 原理调用现象锚定 · 类型归类 · 概念调用 · 证据链推理
📋 分析框架观察现象 → 判断变化类型 → 调用化学概念 → 推理过程 → 结论与方程式
水垢主要成分是CaCO₃（碳酸钙），与醋酸反应：CaCO₃ + 2CH₃COOH → Ca(CH₃COO)₂ + H₂O + CO₂↑。生成可溶性醋酸钙、水和二氧化碳气体（气泡的来源）。
2

镁条燃烧

🏫 校园情境
📖 情境化学课上，老师点燃了一根镁条，镁条发出耀眼白光，产生大量白烟，最后留下白色粉末。
🎯 任务分析镁条燃烧过程中发生了什么，判断物理还是化学变化，写反应方程式
🔗 解析路径现象锚定（光→烟→白色粉末）→ 类型归类（燃烧/化合反应）→ 概念调用（镁、氧气、氧化物）→ 证据链 → 结论
📊 原理调用全部4条原理
📋 分析框架反应前物质 → 反应中现象 → 反应后物质 → 变化类型 → 新物质判断
镁与氧气反应生成氧化镁：2Mg + O₂ →点燃 2MgO。白色粉末即MgO（氧化镁）。耀眼光芒是剧烈的放热反应表现。白烟是固体MgO小颗粒分散在空气中。属于化学变化（生成了新物质）。
3

自检碘盐含碘量

👨‍👩‍👧‍👦 家庭社区
📖 情境爷爷听说加碘盐中的碘容易流失，让你帮忙检验家里的碘盐是否还含碘。你查到：碘盐中的碘以碘酸钾(KIO₃)形式存在，遇淀粉碘化钾试液会变蓝。
🎯 任务设计一个简单的检验方案，分析其中的化学原理
🔗 解析路径目标锚定 → 条件分析 → 原理调用 → 方案设计 → 结论预测
📊 原理调用现象锚定 · 概念调用 · 证据链推理
📋 分析框架检验目标 → 检验原理 → 所需材料 → 预期现象 → 结论判断标准
KIO₃在酸性条件下与I⁻反应生成I₂，I₂使淀粉变蓝。方案：碘盐+食醋+KI溶液→用淀粉试纸检测。若变蓝则含碘，不变蓝则碘已流失。

探究氧气的性质

🧪 学科核心
📖 情境实验室用高锰酸钾(KMnO₄)加热制取氧气，将带火星木条伸入集气瓶，木条复燃。再将铁丝在氧气中点燃，铁丝剧烈燃烧、火星四射、生成黑色固体。
🎯 任务分析两个实验的化学原理：①高锰酸钾制氧的分解反应 ②铁丝在氧气中燃烧的反应
🔗 解析路径物质识别 → 反应类型判断 → 方程式书写 → 现象与产物对应 → 推理总结
📊 原理调用全部4条（重点：类型归类、证据链推理）
📋 分析框架实验名称 → 反应物 → 反应条件 → 现象描述 → 产物判断 → 化学方程式
① 2KMnO₄ →△ K₂MnO₄ + MnO₂ + O₂↑（分解反应）
② 3Fe + 2O₂ →点燃 Fe₃O₄（化合反应，黑色固体为四氧化三铁）
木条复燃证明是氧气，铁丝火星四射证明氧气支持剧烈燃烧。

pH值与植物生长

🔗 跨学科
📖 情境学校生物园绣球花今年开蓝色的花，去年开粉色花。生物老师解释：绣球花颜色与土壤pH值有关（酸性→蓝色，碱性→粉色）。
🎯 任务①设计pH试纸检测土壤酸碱性的方案 ②分析土壤pH变化原因 ③提出调整花色的化学方法
🔗 解析路径跨学科连接（化学+生物）→ 问题分解 → 概念调用（pH、酸碱指示剂）→ 推理 → 方案
📊 原理调用现象锚定 · 类型归类 · 概念调用
📋 分析框架现象 → 涉及学科 → 化学原理 → 实验检验方案 → 可能解释
取土壤浸出液，用pH试纸测pH。去年浇水偏碱性→花呈粉；今年雨水多→pH下降→花变蓝。石灰水可调碱，硫磺粉可调酸。这是化学与生物学的典型交叉点。

塑料降解的时间之谜

🌍 社会未来
📖 情境普通塑料自然降解需数百年，"可降解塑料"（如PLA聚乳酸）宣称堆肥条件下几个月分解。你手中的外卖餐盒就是可降解的。
🎯 任务①分析普通塑料难降解的化学原因 ②分析PLA的降解原理 ③评价"可降解"的真实含义
🔗 解析路径问题界定 → 化学结构分析（聚合物→单体）→ 条件对比 → 原理推理 → 价值判断
📊 原理调用全部4条（重点：概念调用、证据链推理）
📋 分析框架材料类型 → 化学结构 → 降解条件 → 降解产物 → 对环境的影响
普通塑料C-C键稳定，微生物难分解。PLA由乳酸单体聚合，在堆肥条件（高温+微生物）下水解为乳酸，再降解为CO₂和H₂O。但家庭条件下难以达到降解要求——"可降解"不等于"随便扔"。

从"野火"到"肥料"

🌲 综合迁移
📖 情境森林大火后过火区域长出的植物异常茂盛。科学家分析：大火燃烧产生的灰烬含有钾、钙、镁的碳酸盐和磷酸盐，改变了土壤的化学成分。
🎯 任务①分析木材燃烧的化学变化类型和产物 ②分析灰烬矿物质形成 ③分析矿物质如何改善肥力 ④推理完整逻辑链
🔗 解析路径多步综合解析：燃烧化学→灰分转化→矿物质→土壤改良→生态恢复。每环用5步法。
📊 原理调用全部4条，尤其证据链推理（多步链条）
📋 分析框架环节序号 → 现象/事实 → 化学原理 → 推理 → 环节结论 → 对整体的贡献
木材燃烧（有机物→CO₂+H₂O+灰分）→ 灰分含K₂CO₃、CaO等 → 溶于水呈碱性 → 中和酸性土壤 → 释放矿物质 → 植物可吸收利用。这是"化学在自然界循环中"的典型体现。
