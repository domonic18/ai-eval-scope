名著介绍

第1章 走进化学世界 · M2 学习新知的新价值与新使命

名言 · 与大师对话

点击名言展开思考引导：

"我不期望我的观点立刻被接受，因为长期形成的思维习惯不会轻易让位给新思想。" — 拉瓦锡
思考：当你学到的新知识和已有认知冲突时，你选择相信谁？拉瓦锡的这句话提醒我们：科学进步从来不是一帆风顺的。
"生活中没有什么可怕的事物，只有需要理解的事物。" — 居里夫人
思考：这句名言和"学习新知的新价值与新使命"有什么关系？居里夫人告诉我们：恐惧源于未知，而学习是消除恐惧的唯一途径。
"科学不仅与灵性相容，而且是深度灵性的源泉。" — 卡尔·萨根
思考：学习化学的"灵性"在哪里？仅仅是算方程式吗？萨根告诉我们，当你知道一滴水中有多少原子、一个反应如何发生时，你就在经历科学之美。

名著 · 三本必读

《天工开物》— 宋应星（明代）

▼
核心问题

中国古代工匠如何利用化学原理制造陶瓷、冶炼金属、酿造美酒？

核心观点

化学不是西方独有的，中国古人早已在实践中运用了复杂的化学变化。

关键片段

"陶埏"一章中关于瓷釉烧制的描述，涉及氧化还原反应的基本原理。

适读建议 基础

获取：中华书局简体版 / 微信读书电子版

《化学简史》— J.R.帕廷顿

▼
核心问题

化学是如何从炼金术发展到现代科学的？

核心观点

化学的进步是"一个错误修正另一个错误"的螺旋上升过程。

关键片段

拉瓦锡推翻燃素说的章节——看一位科学家如何用实验击败千年偏见。

适读建议 进阶

获取：商务印书馆译本 / 有声书平台

《元素的盛宴》— 山姆·基恩

▼
核心问题

元素周期表中的每一个元素背后都有怎样的故事？

核心观点

化学元素不仅存在于实验室，它们塑造了人类历史。

关键片段

关于"铅与罗马帝国衰亡"的章节——重金属如何影响一个帝国的命运？

适读建议 基础—进阶

获取：中信出版社译本 / 微信读书 / 解读音频
