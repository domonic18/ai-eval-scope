🧰 工具库

化学学习需要好工具。从思维方法到数字资源，助你高效建构新知。

基础

📊 双栏对比表

快速对比两个相似概念，找出关键区别。

  1. 画两栏，分别标注两个概念
  2. 列出各自特征（各3条以上）
  3. 对比找出核心区别

示例：左栏"物理变化"→形状改变、无新物质、可逆；右栏"化学变化"→有新物质、不可逆、伴随能量变化

概念辨析物质分类

🧠 思维导图

建立知识网络，可视化概念间的关系。

  3. 二级分支细化
  4. 关联连线标注关系

高阶

🔗 TOC 思维树

分析复杂问题，从现象追溯根本原因。

当前现象 → 中间原因 → 根本原因。适用于实验失败原因分析、复杂现象解释。

示例：酒精灯加热实验现象异常 → 温度不够高（中间原因1）→ 酒精量不足 / 灯芯过短（根本原因）

🤖 AI 对话工具

用 DeepSeek、通义千问等验证化学理解。

  1. 写下自己的理解
  2. 向AI提问"我的理解对吗？"
  3. 对比AI反馈，找出差异
  4. 修正完善自己的认知

建构完物理/化学变化概念后，请AI给出3个新例子检验自己。例如："蜡烛熔化"是什么变化？

进阶

🔬 分子3D可视化

用 MolView、Chem3D 查看分子立体结构，理解化学变化中分子的变化。

  1. 输入化学式（如 H₂O）
  2. 旋转观察三维结构
  3. 截图记录关键结构
  4. 标注原子和化学键
高阶

🎮 PhET 互动模拟

科罗拉多大学出品，在虚拟环境中做化学实验。

  1. 选择模拟模块
  2. 调整参数（温度、浓度等）
  3. 记录实验现象
  4. 分析变化规律

🧫 实验室器材认识

常见器材的名称、用途和使用注意事项：

试管 — 少量反应酒精灯 — 加热 烧杯 — 盛液量筒 — 量体积

安全提醒：酒精灯要用外焰加热，用完用灯帽盖灭，不可吹灭。
