第1章 走进化学世界 · M2 学习新知的新价值与新使命

同伴连接 · 讨论与协作

点击议题展开详细说明：

1 你在生活中有没有发现过有趣的化学变化？

分享给小组同学，看看谁能给出最合理的解释。可以从厨房（醋+小苏打）、户外（铁钉生锈）、浴室（水垢与醋反应）等场景寻找素材。

2 "如果没有化学，世界会怎样？"

每人从不同角度（衣、食、住、行）讨论，汇总成一份小组报告。例如：没有化学就没有塑料、没有药物、没有手机屏幕……

3 你在建构或解析过程中遇到了什么困难？

请同伴帮你分析。互相分享学习中的困惑，集体智慧往往能解决个人难以突破的瓶颈。

角色分工（点击选择你的角色）：

👁️ 观察员 负责记录讨论要点
❓ 质疑者 专门提出不同意见
📋 归纳者 负责总结小组共识
🎤 表达者 负责汇报成果

家校连接 · 支持与反馈

🧪 家庭小实验 邀请家长一起做"醋泡鸡蛋"或"紫甘蓝指示剂"实验，记录化学变化现象。
📝 家长反馈表 家长评价你在家中的科学探索表现——观察是否仔细、是否主动查阅资料等。

社会连接 · 真实世界

👩‍⚕️ 专家连线 采访身边与化学相关的人（医生、药房工作者、环境检测人员等），了解化学在他们工作中的实际应用。
🏭 机构探访 虚拟参观自来水厂、污水处理厂或制药企业，观察化学原理的实际应用。
💬 线上社区 加入"化学爱好者"线上论坛，参与"生活中的化学"话题讨论。

输出机制

展示形式：小组讨论记录 / 采访报告 / 家庭实验照片 + 化学现象分析
