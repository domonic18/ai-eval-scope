从厨房到实验室：发现蜡烛燃烧的秘密

🧪 真实建构情境：从厨房到实验室

周末你帮妈妈做饭，点燃蜡烛时注意到——蜡烛在燃烧，蜡油流下来，火焰跳动。你突然想起化学课上老师说的："化学变化的本质是有新物质生成。"

你环顾厨房：小苏打和白醋混合冒出气泡、铁锅用久了生锈、冰块在碗里融化、水烧开变成蒸汽、苹果切开后变色……这些变化中，哪些产生了新物质？哪些只是形态变了？

💡 核心问题：如何区分物理变化和化学变化？
🔑 判断标准：变化前后是否有 新物质 生成。

🎯 建构目标

通过本示例，你将建构：

  • 一种理解：物理变化 vs 化学变化的本质区别——是否产生新物质
  • 一种方法：观察→提问→找证据→建构解释→表达的化学探究思维
  • 一个成果：物质变化分类概念图 + 化学变化观察笔记

建构任务

  • 成果名称：化学变化观察笔记 / 物质变化分类模型
  • 形式：图文结合的观察记录 + 概念图
  • 最低标准：至少识别 3种化学变化 和 3种物理变化，用自己的语言描述区别

📋 建构过程示范（7步法）

1 观察现象
点燃蜡烛、混合小苏打和白醋、观察铁钉生锈。看颜色、闻气味（安全范围内），记录变化。
2 提出问题
"蜡烛燃烧产生了什么？小苏打和白醋冒出的气泡是什么？铁锈和铁是同一种物质吗？"
3 收集证据
查阅教材P6-10，对比物理变化和化学变化的特征。化学变化常伴随发光、放热、变色、产生气体或沉淀。
4 建构解释
用自己的话总结："物理变化只是形状、状态改变，没有新物质；化学变化产生了全新的物质。"
5 建构模型
用概念图/双栏表格表达两种变化的区别与联系。
6 表达交流
用文字描述："我观察到蜡烛燃烧产生了二氧化碳和水蒸气（新物质），所以是化学变化；冰块融化成水只是形态改变，所以是物理变化。"
7 反思改进
对照教材修正：蜡烛燃烧时蜡油融化是物理变化，燃烧产生新物质是化学变化——一个过程中可能同时包含两种变化！
