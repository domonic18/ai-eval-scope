📍 模块定位

M18 新知学习收获的幸福与福祉性思维与反思 · 九年级化学

📌 一句话定位

M18 是第一章"走进化学世界"的收尾模块——它不讲授新的化学知识点，而是引导你完成一次 视角转换：从"学化学知识"转向"感受学化学的意义"。

一、本模块学什么

本模块围绕两核心理念展开，点击下方卡片深入了解：

😊

新知学习的幸福

点击展开 →

回顾你刚接触化学这两个多月的旅程，那些让你恍然大悟的瞬间、第一次成功完成实验操作的成就感、理解一个抽象概念时的快乐——这些都是真实的学习情感，值得被觉察、记录和珍惜。
🌱

福祉性思维

点击展开 →

一种看待化学的新视角——学习化学不只是为了考试得分，更是为了获得一种"用化学让生活变得更好"的思维方式。它要求你在学习每一个知识点时，多问一句："这个知识能用来做什么？它能解决什么问题？"

二、为什么重要

❌

只会解题不会思考

点击展开 →

遇到真实世界的化学问题（比如如何选择适合的消毒液、怎么判断水质是否安全）会完全无措。只会解题的人，离开试卷就失去了方向。
✅

怀有福祉性思维

点击展开 →

即使暂时不具备某个具体知识，也会主动去查、去问、去实验——因为他知道化学答案的背后关乎人的安全和福祉。这门课的目标正是培养这样的人。

从"学会化学"到"用化学思考生活"，中间缺的正是这一步。这门课的目标不只是培养"化学考试的高手"，更是培养 用化学眼光观察世界、用化学方法解决生活问题的人。M18 就是这个目标的落地环节。
