三本名著深度导读
天
《天工开物》 中国著作
作者：宋应星（明朝）
核心问题：古人如何运用化学知识改变生活？
核心观点
明朝宋应星通过实地考察，记录了古代中国的农业、手工业技术，其中"陶埏""杀青""五金"等章节涉及原始的化学工艺（制陶、造纸、冶金）。
关键片段
建议阅读"陶埏"篇——看古人如何用黏土烧制成陶瓷。
适读人群
适合对历史+化学交叉感兴趣的同学。
获取路径
微信读书搜索《天工开物（白话全译）》
元
《元素的盛宴》 外国经典
作者：凯恩·西蒙（美国）
核心问题：化学元素周期表上的元素，背后有什么"人"的故事？
核心观点
美国作家凯恩·西蒙用一种"讲故事"的方式讲述元素周期表。
关键片段
建议阅读"磷的故事"章节——17世纪的炼金术士为了"炼制黄金"，意外从尿液中提取出磷。
适读人群
适合所有同学，尤其推荐给"觉得化学枯燥"的同学。
获取路径
当当/京东购买 / 图书馆借阅
寂
《寂静的春天》 跨学科 / 当代作品
作者：蕾切尔·卡逊（美国）
核心问题：化学产品（如农药DDT）给环境带来了什么影响？
核心观点
美国海洋生物学家蕾切尔·卡逊揭示：化学品的滥用会通过食物链传递和富集。
关键片段
建议阅读第一章"明天的寓言"——作者用诗意的语言描绘了一个被农药"杀死"的春天。
适读人群
适合对环境、社会议题感兴趣的同学。
获取路径
图书馆（经典常销）/ 有声书App

输出挑战：写短文
