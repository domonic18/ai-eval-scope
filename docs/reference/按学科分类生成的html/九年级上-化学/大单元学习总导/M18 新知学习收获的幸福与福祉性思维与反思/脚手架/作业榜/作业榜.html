作业榜

作业不是终点，而是你成长的"路标"。本节点提供三种层次的练习选择，你可以根据自己的节奏和目标，选择适合的挑战路径。

第1层：基础层 — 再现与理解 全体必做

目标：巩固课堂所学，确保核心概念过关。

形式题目示例
写作用你自己的话解释：什么是物理变化？什么是化学变化？各举2个生活中的例子。
图示画出"蜡烛燃烧"的流程图，标注出哪些是物理变化、哪些是化学变化。
问答完成10道选择题（小程序/纸质），涉及：化学研究内容、变化分类、仪器识别。
角色扮演脚本写一段2分钟的"化学变化 vs 物理变化"对话脚本，A同学扮演物理变化，B同学扮演化学变化。
写作提示：请围绕"物理变化与化学变化的区别"展开，举例需源于生活，如冰融化成水（物理变化）、铁锅生锈（化学变化）。字数建议：200~400字。
图示提示：蜡烛燃烧包含多个阶段——烛芯熔化（物理变化）、石蜡燃烧（化学变化）、蜡油凝固（物理变化）。可用流程图或图文结合方式呈现。
问答提示：10道选择题涵盖：①化学的研究对象；②物理变化与化学变化的判断；③常见仪器名称与用途。完成后可对照答案自查。
脚本提示：2分钟对话，建议结构：自我介绍 → 各自陈述特点 → 生活中的例子 → 互相提问 → 总结。可以有幽默元素。

第2层：分层任务 — 差异化挑战 三选一

挑战难度 ★★

实验复盘者

选择一个课堂实验（如"对蜡烛及其燃烧的探究"），写成一份图文并茂的实验报告。要求：含目的、步骤、现象、结论，并标注出你的"观察失误时刻"——当时看错了什么，后来怎么纠正的。

挑战难度 ★★★

生活侦探

在家中找到3个与化学有关的物品/现象（如铁锈、小苏打冒泡、水垢等），拍照记录。分析每个现象是物理变化还是化学变化，写出你的分析依据。可制作成"生活中的化学"小海报或短视频。

挑战难度 ★★★★

概念联结者

选择第1章中的任意3个核心概念，用思维导图将它们联系起来。要求：每个概念配1个生活实例，连线处标注关系说明（如"因果关系""递进关系""对比关系"）。形式不限（手绘/幕布/XMind均可）。

第3层：开放层 — 创造性表达 自选加分

你的选择无限——以下仅为启发：

形式描述
写作写一篇800字的科幻微小说，主题为"假如世界上没有化学变化"。
视频拍摄3分钟的"化学第一课"趣味讲解视频，风格自定（脱口秀/动画/实拍）。
演讲准备5分钟课堂演讲"我身边的化学"，面向班级展示。
图示创作一幅"化学世界地图"——用图画+关键词展现第1章的知识全景。
手工制作一套"化学变化识别卡"（扑克牌大小，每张写一个变化及其分类）。
写作提示：科幻微小说需要设定一个合理的世界观——没有化学变化的世界是什么样的？食物不会变质，但也无法被消化；铁不会生锈，但也无法冶炼。试着辩证思考。
视频提示：3分钟"化学第一课"可用趣味实验片段、动画讲解或脱口秀形式呈现。建议准备脚本，确保包含至少3个化学知识点。
演讲提示：5分钟演讲"我身边的化学"，可从厨房、卫生间、阳台等场景切入，用生活中的化学现象引起共鸣。建议配PPT或实物展示。
图示提示："化学世界地图"可以是思维导图、手绘地图或信息图。中心为"第1章 化学使世界更美好"，分支涵盖物质变化、化学性质、实验仪器等。
手工提示："化学变化识别卡"每张卡片包含：变化名称、化学/物理属性、生活实例、示意图。建议制作8~12张，可正面印变化名称、反面印分类和实例。
