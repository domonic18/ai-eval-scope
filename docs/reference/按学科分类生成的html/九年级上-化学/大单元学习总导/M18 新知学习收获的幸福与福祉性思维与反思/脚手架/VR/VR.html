  • 使用手机 + 简易VR纸盒（Cardboard）体验360度视频
  • 使用电脑端的3D交互软件（如ChemDraw 3D、Avogadro免费版）
  • 直接观看高质量3D化学实验动画视频，完成同等的观察与记录任务
