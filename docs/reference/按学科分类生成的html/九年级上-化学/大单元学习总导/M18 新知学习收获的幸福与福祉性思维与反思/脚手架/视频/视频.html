视频1：化学变化 vs. 物理变化

📺 教材配套微课 · ⏱ 约8分钟

📋 观看任务：记录视频中出现的 5个化学变化 和 5个物理变化 示例，分别列出到笔记本上。
❓ 关键问题：
1. 判断化学变化的唯一标准是什么？
2. 发光发热一定是化学变化吗？请举例反驳。
⏸️ [02:30] 暂停！ 蜡烛燃烧既有物理变化也有化学变化，你能区分哪些是物理变化（蜡熔化）、哪些是化学变化（蜡燃烧生成CO₂和H₂O）吗？
⏸️ [05:15] 暂停！ 先自己判断"胆矾研磨"属于哪类变化，再看答案揭晓。
✍️ 输出任务：画一张"化学变化 vs. 物理变化"双气泡对比图，标出共同点和不同点。

视频：实验安全与事故处理

📺 央视《实验室安全》纪录片 / B站科普 · ⏱ 约12分钟

📋 观看任务：记录视频中出现的 3种以上实验事故 及其原因。
❓ 关键问题：
1. 为什么不能凑近容器口闻气体气味？正确的闻气体方法是什么？
2. 酒精灯洒出着火时，为什么用湿抹布盖灭而不是用水浇？（从燃烧条件分析）
⏸️ [06:20] 暂停！ 如果是你，会怎么处理浓酸溅到皮肤上的情况？先自己思考再看视频中的处理方法。
✍️ 输出任务：编写一份《实验室安全守则》（至少8条），要求每条都写清楚"为什么这么做"。

💡 例如："禁止用嘴吹灭酒精灯——因为可能引燃灯体内酒精蒸气，导致爆炸。"

视频：化学，让生活更美好还是更危险？

📺 TED演讲 / TED-Ed · ⏱ 约15分钟

📋 观看任务：记录演讲者的核心论点以及至少三个支撑论据。
❓ 关键问题：
1. 演讲者如何看待"化学"与"化学品"这两个概念的混淆？
2. 你是否同意"没有化学就没有现代医学"？用具体例子支撑你的观点。
✍️ 输出任务：写一篇200字左右的小短文，标题为"我眼中的化学"，要求呈现辩证观点，不片面否定也不盲目赞美。
