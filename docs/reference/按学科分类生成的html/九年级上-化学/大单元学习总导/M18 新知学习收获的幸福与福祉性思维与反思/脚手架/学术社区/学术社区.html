🧪 家庭实验推荐（安全，无试剂）

🧂
食盐与水的溶解
物理变化 — 观察食盐在水中逐渐消失
将一勺食盐加入一杯温水，搅拌观察。食盐"消失"了，但水变咸了——这是溶解，属于物理变化。
💨
小苏打 + 白醋
化学变化 — 产生气泡
将一小勺小苏打放入碗中，倒入少量白醋，立即产生大量气泡。产生了新物质——二氧化碳，这是化学变化！
🔩
铁钉的潮湿环境
化学变化 — 观察一周
将一枚铁钉放在潮湿的纸巾上，每天观察并记录变化。几天后你会看到铁锈——铁与水和氧气发生了化学反应。
