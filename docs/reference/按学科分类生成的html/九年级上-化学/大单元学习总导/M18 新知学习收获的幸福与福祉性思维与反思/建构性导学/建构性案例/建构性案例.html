第一阶：基础观察型 ★☆☆☆☆

情境描述：你拿到一枚铁钉和一截镁条。铁钉表面呈银灰色有光泽；镁条也呈银白色但表面有些发暗。将两者暴露在空气中三天：铁钉出现红棕色斑点，镁条表面变暗区域扩大了。

建构任务

运用六步法，完成"金属在空气中发生的变化"这一概念建构。请记录每一步的思考内容。

原理一（情境→问题）：现象对比中发现了什么差异？
原理二（问题→观察）：需要重点关注哪些具体变化？
形成"金属与空气中的某些成分发生了反应"的初步认知。

第二阶：因果推演型 ★★☆☆☆

情境描述：给你一杯水、一杯食用油和一个密封真空袋。将铁钉分别放入这三个环境中，一周后观察：水中的铁钉生锈严重，油中的铁钉无明显变化，真空袋中的铁钉几乎无变化。

建构任务

思考：铁钉生锈需要什么条件？请设计一个对比实验方案来检验你的假设。

原理三（观察→假设）：空气和水哪个才是关键？
原理四（假设→验证）：三组实验构成了怎样的对照逻辑？
建构"铁生锈需要同时接触水和氧气"的概念。

第三阶：类比迁移型 ★★★☆☆

情境描述：切开一个苹果，暴露在空气中一段时间后果肉变黄变褐；用保鲜膜包好的另一半保持原色。削皮的土豆在水里泡着就没有明显变色。

建构任务

类比铁生锈的建构过程，探究苹果变色的原因和条件。你的"假设→验证"路径如何设计？

原理五（验证→概念）：这个概念与铁生锈的概念有何异同？
迁移思考：化学中的"反应"概念是否可以跨现象统一？
建构"氧化反应"的广义概念，理解不同物质与氧气反应的表现可以完全不同。

第四阶：变量控制型 ★★★★☆

情境描述：三个不透明的密封集气瓶，分别装有空气、纯氧气和纯氮气。将点燃的蜡烛依次伸入瓶中：空气瓶→蜡烛持续燃烧，氧气瓶→燃烧更旺、发出白光，氮气瓶→立刻熄灭。

建构任务

"燃烧到底需要什么？"运用变量控制思维设计探究方案。哪些变量是相同的（控制变量）？哪些是不同的（自变量）？结论是什么（因变量）？

全六步法综合运用。特别关注"控制变量"在假设验证中的作用。
建构"燃烧需要氧气（助燃气体）"和"氮气不支持燃烧"的概念。

第五阶：跨现象建模型 ★★★★☆

情境描述：回顾前面所有情境——铁钉生锈（需要水+氧气）、苹果变色（需要接触空气）、蜡烛燃烧（需要氧气）。这三个现象看起来完全不同，但都涉及同一种物质——氧气。

建构任务

建立一个统一的"氧气参与反应"模型框架，将这三个现象纳入其中。用你自己的方式（文字描述、流程图或示意图）构建这个模型。

六步法中的"概念"步骤需要升级为"模型构建"。关注抽象和概括思维。
建构一个初步的"化学反应类型模型"，根据物质与氧气反应的方式不同进行分类。

第六阶：实验设计型 ★★★★★

情境描述：实验室有两瓶无色气体，标签脱落，已知分别是氧气和二氧化碳。不能直接闻，也不能直接尝。你需要自己设计实验来鉴别它们。

建构任务

完成完整的自主实验设计，包含：问题提出、假设、验证方案设计、预期结果判断。详细写出每一步。

六步法完整独立运用。回忆第四阶中氧气和氮气对燃烧的不同影响。
完成"鉴别氧气和二氧化碳"的完整实验方案，建构"不同气体支持燃烧程度不同"的认知。

第七阶：综合建模与反思型 ★★★★★

情境描述：站在"走进化学世界"的高度，回顾所有现象和概念：

现象/概念与什么有关属于什么类型
铁钉生锈水+氧气缓慢氧化
镁条燃烧氧气剧烈氧化（燃烧）
蜡烛燃烧氧气燃烧，有焰
苹果变色氧气酶促氧化
酒精灯分层氧气接触程度燃烧结构
气体鉴别气体性质差异物质检验
建构任务

创建一个"化学变化全息图谱"，将所有概念以你的方式组织在一起。至少包含三个维度：变化类型、条件、宏观表现。反思你的思维方式和幸福感发生了什么变化。

全部六步法+跨情境综合建模。元认知反思：自己的建构过程本身。
一张个性化的"第1章化学变化概念图谱"，以及一份"我的建构之路"反思记录。

📌 本Slot小结

在这七阶训练中，你从最基本的观察描述逐步升级到综合建模和元认知反思。每一次进阶都在用六步建构法解决更复杂的问题。建构性原理不再是纸上谈兵，而内化成了你的思维方式。

在后续学习中，无论面对什么新知识，请始终记得：

"没有建构过的知识，不是真正的知识。"

  • 我建立了跨章节的概念关联模型
  • 我反思并记录了自己的建构过程
  • 我感受到主动建构带来的思维幸福
