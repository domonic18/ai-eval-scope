观察与描述 — 收集原始信息

老师演示了以下操作：

  1. 用一根火柴梗平放在火焰中约2秒后取出
  2. 观察火柴梗上不同位置的变化

结果：火柴梗两端（接触外焰的部分）碳化变黑最明显，中间（接触焰心的部分）碳化轻微。

验证与实验 — 检验假设

为了验证"外焰温度最高"的假设，老师设计了一个简单验证实验：

  • 将一根细铁丝一端插入外焰，另一端插入焰心
  • 观察铁丝两端的红热程度和速度

结果：外焰中的铁丝很快变红，焰心中的铁丝变化缓慢。

发现规律与形成概念 — 建构知识

🔑 规律提炼
  • 酒精灯火焰分为三层：外焰、内焰（中焰）、焰心
  • 外焰温度最高，焰心温度最低
  • 外焰因氧气充足而燃烧充分，温度最高

概念形成："火焰分层" — 可燃物燃烧时，由于氧气接触程度不同，火焰会形成温度梯度不同的层次结构。
