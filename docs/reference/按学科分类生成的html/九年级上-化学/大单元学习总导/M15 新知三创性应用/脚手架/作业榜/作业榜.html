🌱 基础层 再现与理解

1
识别生活中 3 个化学变化 的例子，并说明你的判断依据是什么。
2
辨析 物理性质 与 化学性质 的异同，各举 2 个例子对比说明。
✅ 达标标准：准确区分物理与化学变化，概念理解无误即可进入分层挑战。

📈 分层层 按能力选择难度

A
A 级 根据给定实验现象，判断发生了何种变化（物理/化学），并说明理由。
B
B 级 设计简单的化学变化分类体系图（思维导图/表格均可），涵盖至少 5 种变化类型。
C
C 级 分析一个生活中常见的化学变化全过程：从反应物→条件→现象→生成物，完整描述其化学本质。
💡 建议从 A 级开始，逐级挑战。能完成 C 级的同学可以直接冲刺开放层！

🚀 开放层 创造性表达

★
自选一种化学现象，创作一份 "化学变化说明书"（图文结合）。
形式可选： ✍️ 写作 📊 图示 🎬 短视频脚本 🎤 演讲提纲
