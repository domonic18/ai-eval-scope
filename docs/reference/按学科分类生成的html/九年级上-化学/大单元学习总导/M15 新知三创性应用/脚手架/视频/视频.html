❓ 关键问题

化学变化有哪些特征？如何判断一个变化是化学变化？
化学变化特征：有新物质生成。判断依据：观察颜色变化、气体产生、沉淀生成、发光发热、温度变化等现象。
物理变化和化学变化的本质区别是什么？
本质区别：是否有新物质生成。物理变化只是形态或状态改变（如冰融化成水），化学变化产生了新物质（如铁生锈生成铁锈）。

❓ 关键问题

绿色化学的核心原则是什么？它如何改变传统化学工业？
绿色化学核心原则：减少有害物质使用、提高原子经济性、使用可再生原料。它从源头减少污染，而非末端治理。
新型材料的研发如何体现"化学创造未来"？
新型材料（如可降解塑料、碳纳米管、自修复材料）通过分子设计实现特定功能，展现了化学从"发现"到"创造"的转变。
