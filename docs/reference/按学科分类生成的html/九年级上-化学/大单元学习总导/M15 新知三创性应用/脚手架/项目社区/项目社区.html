20%

🔬 "化学变化在厨房"探究报告

中周期 1-2周
🎯 项目目标
围绕厨房中的化学变化完成完整探究
📋 阶段任务
① 选题 ② 观察 ③ 记录 ④ 分析 ⑤ 报告
📺 纪录片《食品化学》 📄 科普文《发酵的化学》
📦 成果形式
探究报告（含实验数据）+ 演示文稿
💡 选题建议
· 发酵面团中的化学变化（酵母分解糖产生CO₂）
· 小苏打去污原理探究（碱性水解）
· 食醋除水垢的化学原理（酸碱反应）
· 煎蛋过程中蛋白质的变性

🌍 "化学创造生活"创意展

长周期 1月+
🎯 项目目标
展示化学变化/性质在人类生活中的创造性应用
📋 阶段任务
① 选题 ② 资料收集 ③ 实验/观察 ④ 成果制作 ⑤ 展示
📦 成果形式（任选）
展板 / 海报 / 模型 / 短视频（3-5分钟）
💡 研究方向
· 绿色化学：可降解材料如何改变我们的生活
· 能源化学：从化石燃料到氢能
· 医药化学：阿司匹林的发现与发展
· 材料化学：从青铜器到纳米材料
📊 评价标准
科学准确性 40% + 创意表达 30% + 研究深度 20% + 呈现效果 10%

📌 项目选择指南

🌱 初次尝试 → 选择短周期项目，快速体验从观察到输出的完整流程

🚀 深入学习 → 选择中周期项目，训练实验设计和数据分析能力

⚡ 挑战自我 → 选择长周期项目，完成从选题到展示的全流程研究

💡 所有项目均可独立完成或小组合作完成
