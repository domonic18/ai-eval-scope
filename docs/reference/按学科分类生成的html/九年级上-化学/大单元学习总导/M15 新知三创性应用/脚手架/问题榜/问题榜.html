🌱 基础 · 理解与识别

Q1什么是化学变化？什么是物理变化？各举3例。识别
化学变化：有新物质生成的变化。例：铁生锈、木柴燃烧、食物腐烂。
物理变化：没有新物质生成的变化。例：冰融化成水、玻璃打碎、食盐溶于水。
Q2物理性质与化学性质的区别是什么？辨析
物理性质：不需要发生化学变化就能表现的性质（如颜色、气味、密度、熔点、沸点等）。
化学性质：在化学变化中表现出的性质（如可燃性、稳定性、氧化性等）。
Q3如何判断一个实验现象属于化学变化？判断
判断依据：是否有新物质生成。辅助现象：颜色改变、产生气体、生成沉淀、发光发热、温度变化等。注意：有这些现象不一定是化学变化（如灯泡发光发热是物理变化）。

🚀 进阶 · 分析与应用

Q4为什么说"铁生锈"是化学变化而"铁熔化"是物理变化？分析
铁生锈：铁与空气中的氧气和水反应生成氧化铁（铁锈），有新物质生成 → 化学变化。
铁熔化：铁由固态变为液态，只是状态改变，没有新物质生成 → 物理变化。
Q5蜡烛燃烧过程中涉及哪些变化？请分类说明。综合
物理变化：蜡烛受热熔化（固态→液态）。
化学变化：石蜡燃烧生成水和二氧化碳（发光发热）。
蜡烛燃烧是物理变化和化学变化同时发生的过程。
Q6如果没有化学变化，世界会变成什么样？想象
没有化学变化意味着：没有燃烧（无火）、没有呼吸作用（无法获取能量）、没有光合作用（植物无法生长）、没有消化吸收……生命活动将无法进行，世界将是静止的。化学变化是物质世界"活"的动力。

⚡ 挑战 · 创造与批判

Q7你能设计一个实验证明化学变化伴随着能量变化吗？设计
实验设计示例：
① 取少量生石灰（CaO）放入烧杯，加入适量水，用温度计测量温度变化。
② 现象：温度明显升高，说明该化学变化释放热量。
③ 结论：化学变化伴随能量变化（放热或吸热）。
Q8有人说"化学变化比物理变化更重要"，你同意吗？为什么？批判
开放性答案参考：
同意：化学变化创造了新物质，推动了材料、能源、医药等领域发展。
不同意：物理变化同样重要（如水的三态变化维持水循环）。
辩证看：两者相辅相成，不同场景下重要性不同。
Q9创造一个新情境：需要同时运用物理变化和化学变化解决的问题。创造
示例：野外生存场景——你需要净化浑浊的河水。
物理变化：静置沉淀（大颗粒下沉）、过滤（分离固体和液体）。
化学变化：加入消毒剂（氯气与水反应杀灭微生物）。
两者结合才能获得安全饮用水。
