第1阶 · 个人理解型：物质变化概念图创作

个人经验 ★ 基础 概念解释/模型图

真实建构情境：你刚学完"物理变化与化学变化"的概念，但翻书时发现两者很容易混淆。你的同桌问你："冰融化成水是什么变化？蜡烛燃烧又是什么变化？你怎么判断的？"

🎯 建构任务

创作一张"物质变化分类概念图"，用你自己的逻辑清晰地展示物理变化与化学变化的判断路径。

建议调用原理：原理③ 模型建构 + 原理④ 表达输出

📋 建构步骤

  1. 列出课本中提到的所有变化实例（熔化成水、铁生锈、木柴燃烧、酒精挥发……），至少10个
  2. 按"物理变化 / 化学变化"初步分类，标注你的判断依据
  3. 提炼两类变化的本质区别（有无新物质生成），提炼判断关键词
  4. 画出概念图——可以是树状图、Venn图或流程图，标注关键特征
  5. 用你自己的实例验证概念图（至少找3个新例子测试）
  6. 完善并工整呈现最终版本
🧰 脚手架推荐：用"是 → 否"二分法画判断流程图：有新物质生成？→ 化学变化；无新物质生成？→ 物理变化。每个分支附上2个实例。

📤 成果表达

成果名称："我的物质变化概念图"
格式：A4纸手绘或数字绘图
呈现建议：中心写"物质变化"，两大分支为"物理变化"和"化学变化"，每个分支下至少5个实例和判断依据。

自评量规：
  • 我的概念图包含物理变化和化学变化两大分支
  • 每个分支至少列出5个实例
  • 每个实例旁边标注了判断依据
  • 概念图有清晰的层次结构，别人能看懂
  • 我自己用这张图能准确判断任意新实例

    • 第2阶 · 生活应用型：厨房化学安全指南

    家庭/社区 ★★ 进阶 生活化学方案

    真实建构情境：妈妈在厨房里用白醋除水垢（水垢主要成分是碳酸钙），你注意到白醋倒下去时冒出很多气泡。你立刻联想到课本中"化学变化伴随现象"的内容。你决定给家人写一份"厨房化学安全操作指南"。

    🎯 建构任务

    基于化学变化原理，制作一份面向家人的"厨房化学安全指南"，重点解释3种常见的厨房化学变化及其安全操作要点。

    建议调用原理：原理① 情境感知 + 原理④ 表达输出

    1. 观察厨房中至少3种涉及化学变化的现象（如：小苏打+醋产生气泡、切开的苹果变褐、铁锅生锈）
    2. 查资料确认这些变化背后的化学反应原理
    3. 针对每个变化，分析可能的安全隐患（如：两种清洁剂混用可能产生有毒气体）
    4. 为每个变化编写"安全操作建议"和"应急处理方法"
    5. 将上述内容整理成一份图文并茂的"厨房化学安全卡"
    🧰 脚手架推荐：用"现象→原理→风险→建议"四栏表格整理。关键句模板："当你看到____时，说明发生了化学变化，因为____。需要注意____。"

    📤 成果表达

    成果名称："我家厨房化学安全指南"
    格式：一页A4折页或卡片
    呈现建议：用图标+短句，贴在家里厨房显眼处。

    自评量规：
  • 包含至少3种厨房化学变化及其原理说明
  • 每种变化都有安全操作建议
  • 语言通俗，家人能看懂
  • 有应急处理的提示
  • 设计美观，适合贴在厨房

    • 第3阶 · 校园问题解决型：实验室药品标识改进方案

    学校 ★★ 进阶 校园化学建议

    真实建构情境：在学校化学实验室里，你发现有些试剂瓶的标签已经模糊不清，甚至脱落了。老师告诉大家："这些未知药品不能随意丢弃或混合。"化学实验室需要一套更科学的标识系统。

    🎯 建构任务

    为学校化学实验室设计一份"药品标识改进方案"，包括标识分类标准、标签内容模板和摆放建议。

    建议调用原理：原理② 证据驱动 + 原理③ 模型建构

    1. 实地调查实验室药品柜现状，记录标识问题（模糊、缺失、不统一等）
    2. 收集证据：哪种药品的标签最容易损坏？损坏的原因是什么？
    3. 设计分类标准——按物理性质/化学性质/危险性分类
    4. 制作标签模板：药品名称、化学式、性质类别、安全图标、存放位置
    5. 形成完整的改进方案书
    🧰 脚手架推荐：参考化学危险品标识符号。用"问题→原因→解决方案"三栏分析法。

    📤 成果表达

    成果名称："实验室药品标识改进方案"
    格式：方案书（含设计图和文字说明）
    呈现建议：包含现状分析、设计方案和预期效果三部分。

    自评量规：
  • 有实验室现状调查记录
  • 分类标准有化学原理依据
  • 标签模板完整，包含必要信息
  • 方案可操作、好落地
  • 考虑了药品存放的安全要求

    • 第4阶 · 学科表达型：蜡烛燃烧深度分析报告

    学科领域 ★★★ 挑战 化学分析报告

    真实建构情境：蜡烛燃烧看似简单，但背后涉及复杂的化学变化过程。假设你是化学小研究员，需要写一份关于"蜡烛燃烧中的化学变化"的深度分析报告，呈现你对这一常见现象的化学理解深度。

    🎯 建构任务

    撰写"蜡烛燃烧化学分析报告"，从物质变化、能量变化、产物检验三个维度展开。

    建议调用原理：原理① 情境感知 + 原理② 证据驱动 + 原理④ 表达输出

    1. 仔细观察蜡烛燃烧全过程（火焰分层、烛泪、黑烟、气味……），记录至少8个现象
    2. 区分物理变化（蜡烛熔化）和化学变化（燃烧反应），标注每个现象的类型
    3. 设计产物检验方案：用干燥烧杯罩在火焰上方检验水，用涂石灰水的烧杯检验二氧化碳
    4. 分析能量变化——化学能转化为光能和热能
    5. 写出"变化方程式"（文字表达式）：石蜡+氧气 → 二氧化碳+水+能量
    6. 整合分析维度，形成完整报告
    🧰 脚手架推荐：用"现象记录表"+"分类判断"+"实验验证"三步法。报告框架：引言→现象记录→变化分析→实验验证→结论。

    📤 成果表达

    成果名称："蜡烛燃烧化学分析报告"
    格式：短报告（500字左右，含实验数据）
    呈现建议：配燃烧示意图和产物检验实验图。

    自评量规：
  • 记录了至少8个燃烧现象
  • 正确区分了物理变化和化学变化
  • 有产物检验的实验设计
  • 分析报告有逻辑结构
  • 得出了有证据支撑的结论

    • 第5阶 · 跨学科整合型：厨房"小苏打"的多维研究

    多学科交叉 ★★★ 挑战 整合方案

    真实建构情境：小苏打（碳酸氢钠）是厨房常备品——烘焙用它发面，清洁用它去污，甚至有人用它做简易灭火器。为什么同一种物质有这么多用途？这背后关联着化学、物理、生物等多学科知识。

    🎯 建构任务

    制作一份"小苏打跨学科研究档案"，从化学性质出发，辐射到3个以上学科的应用解释。

    建议调用原理：原理③ 模型建构 + 原理⑤ 反思优化

    1. 查资料确认小苏打的化学性质（受热分解、与酸反应、水溶液呈弱碱性）
    2. 将每种性质映射到实际用途（受热分解→烘焙发面；与酸反应→去污）
    3. 跨学科扩展：生物（消化中的酸碱中和）、物理（灭火器原理——产生CO₂）、环境（绿色清洁）
    4. 以"小苏打"为中心，画出跨学科知识网络图
    5. 反思：还有什么用途可以补充？你的网络图有哪些连接可以强化？
    🧰 脚手架推荐：中心辐射式思维导图：中心写"小苏打NaHCO₃"，辐射分支为"化学性质→应用领域→学科连接"。每个分支标注引用来源。

    📤 成果表达

    成果名称："小苏打跨学科研究档案"
    格式：思维导图+简短文字说明
    呈现建议：用不同颜色区分不同学科，连线上标注"如何连接"。

    自评量规：
  • 小苏打的化学性质描述准确
  • 至少覆盖3个学科
  • 每个跨学科连接有合理解释
  • 思维导图层次清晰
  • 有反思和改进空间

    • 第6阶 · 社会/未来情境型：未来城市"零锈蚀"方案

    社会/未来 ★★★★ 高阶 社会化学建议

    真实建构情境：据统计，中国每年因金属锈蚀造成的经济损失高达数千亿元。埃菲尔铁塔每隔7年就要彻底除锈重漆。假设你是"未来城市材料"规划师，请设计一份面向2060年的"零锈蚀城市"方案。

    🎯 建构任务

    撰写"未来城市零锈蚀可行性方案"，分析锈蚀原理，提出至少3种防锈策略，评估其可行性和成本。

    建议调用原理：原理① 情境感知 + 原理② 证据驱动 + 原理⑤ 反思优化

    1. 分析金属锈蚀的化学原理（铁与氧气、水反应生成铁锈）
    2. 调研现有防锈技术（涂层、镀层、合金、阴极保护等），收集优缺点
    3. 提出至少3种创新防锈方案（如：自修复涂层、超疏水材料、微生物防锈）
    4. 为每种方案做"可行性评估"（成本、效果、环保性、寿命）
    5. 撰写方案总结，提出个人推荐方案
    🧰 脚手架推荐：用"原理→方案→评估→推荐"四段结构。用SWOT分析法评估每种方案。

    📤 成果表达

    成果名称："未来城市零锈蚀方案"
    格式：提案报告（含对比评估表）
    呈现建议：用对比表格展示3种方案的各项指标得分。

    自评量规：
  • 锈蚀原理解释准确
  • 至少提出3种防锈方案
  • 每种方案有可行性评估
  • 方案有创新性和未来感
  • 整体方案逻辑自洽

    • 第7阶 · 综合创新型：设计一个化学探究项目

    综合 ★★★★ 高阶 完整创新成果

    真实建构情境：学校要举办"校园科技节"，你在化学实验室发现了几种常见物质——白醋、小苏打、铁钉、铜丝、石灰水。你需要利用这些材料，设计一个有科学价值的原创化学探究项目，在科技节上展示。

    🎯 建构任务

    设计并呈现一个完整的原创化学探究项目（从提出问题→设计方案→预期成果→展示方案）。

    建议调用原理：全部5条原理综合运用

    1. 从给定材料中发现问题（如：铁钉在不同溶液中的腐蚀速度？小苏打和白醋的比例与产气量的关系？）
    2. 提出可探究的科学问题，形成假设
    3. 设计实验方案（变量控制、步骤设计、数据记录表）
    4. 运用原理②（证据驱动）设计数据记录方式
    5. 预期成果建模——预测可能的实验结果并准备多套解释方案
    6. 设计科技节展示方案（展板、互动、讲解词）
    7. 反思优化：这个项目可能有什么局限？怎么改进？
    🧰 脚手架推荐：用"PBL项目设计模板"：驱动问题→探究目标→实验设计→数据收集→结论展示。加入"如果……会怎样？"的思维实验。

    📤 成果表达

    成果名称："校园科技节化学探究项目方案"
    格式：项目计划书+展板设计草图
    呈现建议：包含海报、实验演示视频脚本、互动提问卡。

    自评量规：
  • 提出了一个有探究价值的科学问题
  • 实验方案有变量控制设计
  • 考虑了多套可能的实验结果
  • 展示方案有互动性和吸引力
  • 体现了建构性原理的综合运用
