🧪 建构性示例

M15 新知三创性应用 · 化学 · 九年级上
🎯 你的任务：跟着一个完整的建构示例，体验"从零到一建构化学知识成果"的全过程。示例主题："物质的变化与性质的创造性表达"——以蜡烛燃烧为探究对象。你不需要记住结论，而是学会"建构"的过程本身。

一、建构情境与目标

📍 真实情境：生日派对上，你点燃了蜡烛。看着跳动的火焰、缓缓流淌的烛泪、飘散的黑烟，你突然想到——"蜡烛在燃烧的过程中到底经历了什么？哪些是物理变化，哪些是化学变化？我能用一种方式把整个过程完整地表达出来吗？"

🎯 建构目标：建构一份"蜡烛燃烧全变化图"——用图示+文字的方式，完整展示蜡烛燃烧过程中涉及的物理变化、化学变化及其关系。

📋 任务说明：

  • 成果名称：蜡烛燃烧全变化图
  • 用途：向同桌讲解"蜡烛燃烧中物理变化和化学变化的区别与联系"
  • 形式：A4纸手绘图示 + 300字以内文字说明
  • 最低标准：至少包含5个具体变化，准确分类，有判断依据

二、建构过程示范

第 1 步
🔍 观察情境，提取关键信息
做什么点燃一支蜡烛，从多个角度观察燃烧全过程。
怎么做点燃前→点燃中→燃烧稳定时→吹灭后，每个阶段记录3—5个现象。
关注：火焰颜色、分层、烛蜡状态、烟雾、气味、烛芯变化。
📝 示范观察记录：
点燃前：白色固体蜡烛，有棉线烛芯，无气味。
点燃中：火焰分三层（外焰淡蓝、内焰明亮、焰心暗），烛蜡从固体→液体→沿烛身流下。
吹灭后：有白烟升起，烛芯顶端变黑（碳化），蜡油重新凝固为固体。
第 2 步
❓ 提出问题，定位探究方向
做什么基于观察记录，提炼驱动探究的核心问题。
怎么做用"为什么""是什么""如何判断"等句式提问。把问题写在纸上，筛选出最核心的1—2个。
📝 示范问题提炼：
① 蜡烛燃烧过程中，哪些现象是物理变化？哪些是化学变化？
② 如何证明蜡烛燃烧生成了新物质？
③ 物理变化和化学变化在蜡烛燃烧过程中是独立发生的还是同时发生的？

🔑 核心问题 如何用一张图同时呈现蜡烛燃烧中的物理变化和化学变化及其相互关系？

第 3 步
📚 调用已有知识和证据
做什么回顾课本知识和已有经验，为建构模型做准备。
怎么做①区分物理变化和化学变化的概念 → ②回忆实验验证方法（检验水、CO₂） → ③整理"变化伴随现象"清单
📝 示范知识调用：
· 物理变化：无新物质生成（蜡烛熔化为液体是物理变化）
· 化学变化：有新物质生成（蜡烛燃烧生成CO₂和H₂O是化学变化）
· 检验方法：用干燥烧杯罩在火焰上方→出现水雾（水）；用澄清石灰水润湿的烧杯→变浑浊（CO₂）
第 4 步
🧩 建构初步模型/方案
做什么将零散信息整合为结构化的"蜡烛燃烧全变化图"草图。
怎么做用"时间轴+双色标注"的布局：横轴为蜡烛燃烧的时间线（点燃→燃烧→吹灭），纵轴分层展示物理变化（蓝色标注）和化学变化（红色标注）。
📝 示范模型结构：
┌─ 时间轴（从左到右）：点燃 → 燃烧稳定 → 吹灭后
├─ 上层（物理变化-蓝）：蜡熔化→蜡油流动→蜡油凝固
├─ 下层（化学变化-红）：烛芯燃烧→石蜡+氧气→CO₂+水+能量
└─ 交汇处标注"能量转换"和"伴随现象"
第 5 步
✅ 检验、修正和完善
做什么用证据检验模型是否完整、准确。
怎么做自问清单：每个变化的分类正确吗？有遗漏的变化吗？文字说明和图示匹配吗？
📝 示范检验成果：
初版问题：忘记标注"蜡烛燃烧伴随发光发热"这一重要现象 → 补充到交汇区。
检查后确认：物理变化3个、化学变化2个，分类准确，证据链完整。
第 6 步
✍️ 表达呈现
做什么将模型工整地呈现为最终成果。
怎么做绘制正式版本，配文字说明（150-200字解释核心观点）。
📝 示范呈现框架：
标题：蜡烛燃烧全变化图
核心观点："蜡烛燃烧是一个物理变化与化学变化同步发生、相互依存的复杂过程。蜡的熔化（物理变化）为燃烧提供了燃料，而燃烧（化学变化）释放的热量又加速了蜡的熔化。"
配图建议：中间画蜡烛，左侧列出物理变化链，右侧列出化学变化链，底部标注"能量转换：化学能→光能+热能"。
第 7 步
🔄 反思优化
做什么回顾整个建构过程，记录改进空间。
怎么做问自己：如果再做一次，我会在哪一步花更多时间？我的模型还能加入什么新维度？
📝 示范反思：
· 做得好的：观察记录得很细，有6个关键现象
· 可以改进的：第3步调用知识时如果能查到蜡烛的成分（石蜡是烃类混合物）会更好
· 下一步拓展：可以再加入"质量变化"维度——燃烧前后质量如何变化？为什么？
