🔥 真实建构情境

你面前有一根正在燃烧的蜡烛。你看到了什么？

👁️ 观察
火焰的颜色、形状、跳动
💧 细节
蜡泪流淌、黑烟飘起、杯壁水雾
❓ 张力点
蜡烛燃烧后"消失"了？
📋 你一定注意到了：橙黄色的火焰在跳动，烛芯根部发蓝，蜡烛在慢慢变短，融化的蜡油顺着烛身流下……但这些只是表面现象。真正的问题藏在你看不见的地方。
💧 如果把一个冷玻璃杯罩在火焰上方，杯壁上会出现水雾——水从哪儿来的？黑色烟雾是炭吗？蜡油被烧掉了还是变成了别的东西？
❓ 蜡烛越烧越短，最后"消失"了。它真的消失了吗？如果消失了，那烟是什么？如果没有消失，它变成了什么？这个问题直接指向化学的核心——化学变化中物质到底变成了什么？

🎯 建构目标

建构一个能够解释"蜡烛燃烧"过程中发生了什么的概念模型，用 "化学变化" 的视角重新理解一个日常现象。

📦 成果：蜡烛燃烧的物质转化解释模型 🎨 形式：示意图 + 文字说明 ✅ 最低标准：3种观察证据 + 1个推理链条 + 1个结论

📝 步骤1：观察并记录

点燃蜡烛，从头到尾细致观察它的变化。使用5W1H表格记录：

  • What — 火焰颜色（橙黄、内焰偏蓝）、蜡熔化、冒黑烟
  • Where — 火焰分三层（外焰、内焰、焰心）
  • When — 点燃后不同时间段的变化
  • How — 蜡烛逐渐变短，蜡油从固态变为液态

产出：一份观察记录表，至少包含5条具体观察。
