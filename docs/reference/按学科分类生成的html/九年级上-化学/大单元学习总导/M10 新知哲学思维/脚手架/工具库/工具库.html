⚙️ 变量控制分析框架

设计科学的对比实验方案
  1. 确定探究的问题
  2. 列出所有可能的影响变量
  3. 选择要改变的一个变量
  4. 保持其他变量不变
  5. 对比实验结果
探究：温度对化学反应速率的影响
改变：🌡️ 温度（热水 vs 冷水）
不变：浓度、体积、反应物种类、容器
结论：温度越高，分子运动越快，反应速率越大
