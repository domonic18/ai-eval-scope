🗳️ 讨论议题一

"判断化学变化最可靠的证据是什么？" 点击你的选择，看看反例分析。

A 颜色变化
⚠️ 反例：氧气是无色气体，液氧是淡蓝色液体——从气体到液体只是物理变化，但颜色变了。所以"颜色变化"不能作为化学变化的可靠证据。
B 产生气泡
⚠️ 反例：水烧开时也会产生大量气泡（水蒸气）——但那是物理变化。产生气泡本身不能直接证明化学变化发生。
C 有热量变化（放热/吸热）
⚠️ 反例：浓硫酸倒入水中会放出大量热——但那是稀释过程（物理变化）。灯泡通电也会发热——也是物理变化。所以热量变化≠化学变化。
D 有新物质生成（检验得出的结论）
✅ 正确答案！化学变化的本质特征就是有新物质生成。其他现象（颜色变化、气泡、放热）只能作为"线索"，最终确认需要检验新物质是否存在。例如：澄清石灰水变浑浊证明有二氧化碳生成。
