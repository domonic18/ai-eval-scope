🍳 微项目：厨房化学——解码一个日常变化

短周期 · 1—3天
项目目标：选择一个家庭厨房中的化学变化现象，运用本模块的"解析→建构"完整流程进行分析和模型建构

阶段任务

选择一个现象（如"煮蛋时蛋清凝固"、"泡打粉发面"）
分析这属于什么变化？变化的条件是什么？产物是什么？
画一张"变化过程"模型图
向家人展示并解释你的发现

成果形式

A4纸大小的 "变化解码卡"（图文结合）

资源工具

  • 观察记录表：记录现象 · 条件 · 产物
  • 证据—解释对照表：帮助从现象推断原理
  • 模型图模板：原子/分子级别的变化示意
💡 启发提示："蛋清从透明变成白色固体"——是物理变化还是化学变化？想一想"蛋清的主要成分是蛋白质"，蛋白质受热会怎样？

⚗️ 专题项目：化学史上的一次"革命"

中周期 · 1—2周
项目目标：选择拉瓦锡质量守恒实验，通过角色扮演+"书本实验重演"的方式，理解"新知"是如何被建构出来的

阶段任务

研读相关史料（参考"名人名言"和"名著介绍"脚手架）
设计并完成一个简化的"质量守恒验证实验"（电子天平+塑料瓶反应容器）
撰写一篇"我是拉瓦锡"的研究日志
做一次3分钟的口头汇报

成果形式

📝 研究日志（800字 + 实验照片）

评价标准

💡 角色扮演提示：如果你穿越到1774年成为拉瓦锡，看到"物质燃烧后质量似乎减少了"这个现象——你会怎么想？你的实验怎么设计？

🌍 综合项目：社区科普展板——"变化的科学"

长周期 · 1月+
项目目标：设计并制作一块物理展板或数字展板，用通俗易懂的方式向社区居民科普"化学变化和物理变化"

阶段任务

调研——社区居民对"化学变化"存在哪些常见误解（如"燃烧就是物质消失了"）
策划——设计展板内容结构（概念区 + 案例区 + 互动实验区）
制作——完成展板内容并配图
展示——在社区或学校展示，收集反馈

成果形式

🖼️ 物理展板（60×90cm）或数字展板（Canva可画设计）

资源工具

  • 问题库："常见化学变化认知误区"相关调研问题
  • 视频脚手架：化学变化与物理变化的对比演示视频
  • Canva模板：科普展板设计模板

评价标准

💡 调研小贴士：问问身边的人"冰块融化是不是化学变化"？你会发现很多成年人也会认为是化学变化——这就是你的展板要解答的问题！
