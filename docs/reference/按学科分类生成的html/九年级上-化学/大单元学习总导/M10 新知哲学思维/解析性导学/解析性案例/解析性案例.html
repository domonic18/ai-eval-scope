★ 生活情境

暖宝宝为什么会发热

案例材料：一片打开的暖宝宝暴露在空气中后逐渐发热，内含铁粉、活性炭、食盐等成分。请你分析它的发热原理。

解析任务：暖宝宝的发热原理是什么？这是物理变化还是化学变化？证据是什么？

原理一：分类优先原理四：证据链闭合

参考思路：铁粉在空气中会生锈——这是一个缓慢氧化过程，属于化学变化。铁与氧气和水反应生成铁锈（Fe₂O₃·xH₂O）并放热。活性炭和食盐起催化加速作用。多个证据：①温度升高（发热）②铁粉消耗（质量变化）③铁锈生成（颜色变化）。

★ 校园情境

实验台上的白色粉末是什么

案例材料：三个未贴标签的试剂瓶，分别装着白色粉末——可能是食盐（NaCl）、小苏打（NaHCO₃）、淀粉。请设计不用品尝的方案区分它们。

解析任务：设计一套实验方案，区分三种白色粉末。

原理一：分类优先原理二：假设验证原理三：对比排除

参考思路：原理一：先分类——三种白色粉末属于不同类物质（盐、碳酸盐、糖类）。
原理二：假设"小苏打遇酸产气"→加白醋验证；假设"淀粉遇碘变蓝"→加碘酒验证。
原理三：三种样品做平行对比实验——哪个加醋冒泡？哪个加碘变蓝？哪个都不反应→食盐。

★★ 家庭情境

铁锅生锈的"重灾区"在哪

案例材料：一把铁锅使用后发现——锅底中心锈蚀最严重，锅壁次之，带涂层的边缘和锅盖几乎没锈。

解析任务：分析锈蚀程度差异的原因，用对比思维解释。

原理三：对比排除原理五：反向追溯

参考思路：铁生锈需要水和氧气。原理三：对比不同区域的"水+氧"条件——锅底中心接触水最多（经常泡水）；锅壁接触水和氧中等；带涂层边缘有保护层隔绝；锅盖几乎不接触水。原理五：从"锈蚀程度"反向追溯→水和氧接触越多的区域锈蚀越严重。

★★ 学科核心

镁带燃烧的"一白一黑"

案例材料：镁带在空气中燃烧发出耀眼白光，产生白色粉末（氧化镁）。将镁带放入充满CO₂的集气瓶中点燃，镁带继续燃烧，瓶壁出现黑色颗粒。

解析任务：分析两实验中白色和黑色产物分别是什么，说明化学反应发生的依据。

原理二：假设验证原理四：证据链原理五：反向追溯

参考思路：镁是活泼金属。空气中燃烧：2Mg+O₂→2MgO（白色粉末）。CO₂中燃烧：2Mg+CO₂→2MgO+C（黑色颗粒是碳）。原理二：假设"CO₂中也能燃烧"→实验验证→确实燃烧，推翻"燃烧必须氧气"的误解。原理四：白光（放热证据）+白色/黑色产物（新物质）=完整证据链。原理五：从黑色颗粒反向追溯→碳来自CO₂分解。

★★ 跨学科

"琥珀"是化学变化还是物理变化

案例材料：几千万年前松树分泌的树脂滴落，包含了昆虫或叶片，经过漫长地质作用变成了琥珀。

解析任务：从化学角度分析"树脂变成琥珀"是否发生了化学变化？如何判断？

原理一：分类优先原理五：反向追溯

参考思路：原理一：先判断类型——树脂的主要成分是萜类有机物，变成琥珀涉及氧化聚合反应，分子结构发生了改变，因此是化学变化（尽管保留了外观形态）。原理五：从"琥珀的物理性质（硬度高、不溶于水）"反向追溯→这些性质与新鲜树脂不同→证明了化学变化的发生。注意区分"形态变化"（物理）和"分子结构变化"（化学）。

★★★ 社会情境

"甲醛检测盒"的工作原理

案例材料：家用甲醛检测盒的原理——甲醛与盒中的某种化学试剂反应，使试剂颜色变化（如从无色变蓝色）。

解析任务：分析检测方法背后的化学原理，评价它是否可靠（可能存在哪些干扰？）。

原理二：假设验证原理三：对比排除原理四：证据链

参考思路：原理二：假设"甲醛与试剂反应产生颜色变化"→验证：在已知浓度的甲醛溶液中测试→颜色深度与浓度相关。原理三：对比排除——其他挥发性气体（如酒精、醋）是否也会使试剂变色？如果也变色，则检测不特异。原理四：单一颜色变化不足以定量→需要标准比色卡和对照实验才能形成可靠证据链。

★★★ 综合迁移

"氧气的发现"科学史分析

案例材料：1774年普里斯特利加热氧化汞产生了一种气体（后来拉瓦锡命名为"氧气"），这种气体使蜡烛燃烧更旺，使小鼠存活更久。

解析任务：分析普里斯特利和拉瓦锡是如何通过"实验证据→推理分析→得出结论"的，重建他们的思维过程。

全部5条原理综合运用

参考思路：原理一（分类优先）：首先判断"使蜡烛燃烧更旺"说明这种气体不同于普通空气。原理二（假设验证）：假设"这是一种支持燃烧的特殊气体"→验证：插入带火星木条→复燃。原理三（对比排除）：比较空气、CO₂、这种"新气体"对燃烧的影响。原理四（证据链）：燃烧更旺+小鼠存活更久+木条复燃→多角度证据。原理五（反向追溯）：从"氧化汞受热产生气体"反向推断→汞与这种气体结合形成了氧化汞。注意区分"观察到什么现象"和"这些现象意味着什么"。
