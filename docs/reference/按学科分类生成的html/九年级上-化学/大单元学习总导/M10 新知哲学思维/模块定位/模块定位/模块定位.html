本模块学习什么？

不是学习某一种具体的化学知识，而是追问——"我们是怎么知道这些化学知识的？"

元认知层面
  • 🔍 解析 — 面对化学现象如何拆解、分析、寻找证据、推理判断
  • 🏗️ 建构 — 如何基于观察和实验证据，生成自己的理解、模型或解释
  • 🧰 脚手架工具 — 遇到困难时知道去哪里找答案
💡
2

本模块为什么重要？

不学本模块，化学知识只是一堆需要记忆的"结论"；学完本模块，你将明白每个结论背后的"发现方法"。

短期+长期价值

🎯 短期价值：学会分析和建构方法，后续学习更高效

🌟 长期价值：这个思维模式可以用到物理、生物甚至生活中的问题解决中

——从"学化学"到"像化学家一样思考"
