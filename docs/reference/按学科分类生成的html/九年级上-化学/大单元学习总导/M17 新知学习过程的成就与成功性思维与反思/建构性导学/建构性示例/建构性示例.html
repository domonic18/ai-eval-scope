从"铁丝在氧气中燃烧"到"我自己的化学变化解释模型"

🎯 真实建构情境

你在化学课上看到了一个经典实验：铁丝在氧气中剧烈燃烧，火星四射，生成黑色固体。老师告诉你这是"化合反应"，但你心里冒出一串问题：铁丝不是金属吗？金属也能燃烧？这和木头燃烧是一回事吗？生成的黑色固体又是什么？——这些问题是建构新理解的起点。

建构目标

建构一个"化学变化类型辨析模型"，能够用这个模型判断和解释任何一个新的化学变化属于什么类型。

📦 成果名称
化学变化类型辨析模型手绘图
🎯 用途
帮助自己（也能帮助同学）快速判断化合反应、分解反应、置换反应、复分解反应
📏 最低标准
至少包含四类反应的"表达式模板+关键特征+至少1个例子+易混淆点提醒"

建构过程示范（6步）

点击步骤圆点切换，跟随引导完成你的辨析模型

1 提取证据
2 比较分类
3 类型模板
4 配例子
5 识陷阱
6 绘模型
第1步：回到现象，提取"化学变化"的证据
做什么：回顾铁丝燃烧实验的全过程，写下观察到的所有变化（发光、放热、生成黑色固体）
产出：一份"现象—证据"对应表
示范：你写下的现象中，哪些属于物理变化？哪些属于化学变化？区分依据是什么？——铁丝变红热是物理变化（没有新物质），生成黑色四氧化三铁是化学变化（有新物质生成）。
第2步：比较不同反应, 发现分类依据

对比铁丝燃烧 vs 其他反应的区别

反应反应物种数生成物种数类型猜测
铁丝 + 氧气 → 四氧化三铁21化合
硫 + 氧气 → 二氧化硫21化合
水通电 → 氢气 + 氧气12分解
铁 + 硫酸铜 → 铜 + 硫酸亚铁22置换
产出：反应物—生成物多种关系对比表
示范：铁丝燃烧是"两种物质→一种物质"；你能找到"一种→两种"或"两种→两种"的例子吗？
第3步：抽象出类型模板
做什么：把比较结果提炼成通用表达式

用 A + B → AB（化合）、AB → A + B（分解）等符号表达式

化合反应
A + B → AB
分解反应
AB → A + B
置换反应
A + BC → AC + B
复分解反应
AB + CD → AD + CB
产出：四大基本反应类型的符号表达式模板
示范：你的模板不仅要写字母表达式，关键是要说清"为什么要这样归类"
第4步：给每个类型配上"自己的例子"
做什么：为每个反应类型找 1—2 个具体的化学方程式

从教材和课堂笔记中搜索，优先选你亲眼看过实验的

化合反应：铁 + 氧气 点燃→ 四氧化三铁

分解反应：水 通电→ 氢气 + 氧气

置换反应：铁 + 硫酸铜 → 铜 + 硫酸亚铁

复分解反应：氢氧化钠 + 盐酸 → 氯化钠 + 水

产出：类型—例子对照卡
第5步：识别易混淆陷阱
做什么：找出哪些反应看起来像某一类、实际上不是
比如"甲烷燃烧"——反应物有氧气，生成物有二氧化碳和水，看起来像两种→两种，但这不是置换反应，而是氧化反应（不属于四大基本类型）。

你的辨析笔记要写清楚：它看起来像哪种类型 → 为什么不是 → 实际属于什么 → 判断依据
产出：易错辨析笔记
第6步：绘制完整的辨析模型
做什么：把前5步的成果整合成一张完整的模型图

用思维导图或流程图的形式，把"判断顺序、类型模板、例子、易错点"组织起来

判断一个化学变化是什么类型
化合反应
多种→一种
3Fe+2O₂→Fe₃O₄
分解反应
一种→多种
2H₂O→2H₂+O₂
置换反应
单质+化合物→新单质+新化合物
Fe+CuSO₄→Cu+FeSO₄
复分解反应
两种化合物交换成分
NaOH+HCl→NaCl+H₂O
产出：一张A4纸的完整辨析模型
