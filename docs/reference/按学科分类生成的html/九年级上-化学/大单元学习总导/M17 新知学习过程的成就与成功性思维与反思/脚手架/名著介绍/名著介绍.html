📖 经典名著

《化学基础论》 拉瓦锡

核心问题：如何用"元素"这个新概念重新定义整个化学？

核心观点

拉瓦锡提出"元素是不可再分的纯净物质"，这个定义统治了化学近100年。他首次把定量方法引入化学——"不称量，不化学"。

目录结构

第一篇：论气体的形成与分解 · 第二篇：论酸与碱的组合 · 第三篇：化学操作

关键片段导读

重点读前言——拉瓦锡写道："我无意在此给出最终答案，而是希望为后来的研究者铺一条更可靠的路。"

  • 电子书：超星电子书平台
  • 解读：得到APP听书解读
《科学革命的结构》 托马斯·库恩

核心问题：科学是怎么进步的？是知识慢慢积累，还是突然的革命性突破？

核心观点

"范式转换"——科学进步不是线性积累，而是从一个思维框架跳到另一个。当现有理论解释不了越来越多的异常时，"科学革命"就发生了。

关键片段导读

重点读"在显微镜下看科学革命"——库恩用拉瓦锡推翻燃素说作为典型案例，正好对应本模块的化学学习。

获取路径
  • 纸质书：北京大学出版社
  • 电子书：微信读书
  • 有声书：喜马拉雅FM
《为什么：关于因果关系的新科学》 朱迪亚·珀尔

核心问题：人类如何从数据中提取因果关系？

核心观点

珀尔提出"因果之梯"三层理论：观察 → 干预 → 反事实思维。这和化学学习中"描述现象→设计实验→推理结论"完全对应。

获取路径
  • 纸质书：中信出版社
  • 电子书：Kindle商店
  • 解读：B站系列讲解视频
《天工开物》 宋应星（中国著作类比）

推荐理由：被称为"中国17世纪的工艺百科全书"，记载了古代中国的化学工艺——冶铁、制盐、陶瓷、造纸等。

与拉瓦锡的呼应

如果说拉瓦锡代表西方科学的"理论建构"传统，《天工开物》则代表东方科学的"工艺实证"传统。两者都重视观察和记录，但侧重点不同——前者追问"为什么"，后者记录"怎么做"。

获取路径
  • 纸质书：中华书局版
  • 电子书：国学大师网可在线阅读
