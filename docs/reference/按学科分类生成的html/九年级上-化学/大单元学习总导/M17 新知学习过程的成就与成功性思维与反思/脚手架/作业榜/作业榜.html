任务 A1：概念填空与配对

  • 完成化学变化、物理变化、化合反应、分解反应、氧化反应的填空式判断

B1：图示任务（偏向表达型）
用示意图展示"从铁丝燃烧现象到判断它是化合反应"的完整思维过程。
评价标准：至少包含 5 个信息节点、3 个判断箭头、1 句总结。
B2：写作任务（偏向解释型）
写一段 300 字的"化学变化判断指南"，教一个生病缺课的同学如何判断一个变化是不是化学变化。
评价标准：包含 3 个判断依据、2 个对比例子、1 个易错提醒。
B3：视频/演讲任务（偏向表达型）
录制一段 3 分钟讲解视频，主题："为什么说化学变化中一定同时发生物理变化？"
评价标准：有观点、有证据、有类比、有总结。

🔥 挑战任务 C1：概念跨界对比表 开放

将"化学变化的特征"和"物理变化的特征"与一个你感兴趣的非化学领域过程（如烹饪、植物生长、手机充电等）进行类比，分析它们的"变化本质"。

  • 成果形式：类比分析短文（500 字以内）或一幅类比画
  • 评价标准：类比有说服力、点出了本质相似性和差异性

🕵 挑战任务 C2：设计"化学侦探谜题" 开放

设计一个虚构的"化学案件"（如：一瓶标签不明的无色液体是什么？），给出线索和检测方法。

  • 成果形式：谜题卡片（含情境描述 + 线索 + 提示）+ 答案页
  • 评价标准：线索指向合理、检测方法可操作、答案唯一且有理有据
💡 提示：可以参考侦探小说的结构——有一个"谜案"、一些"线索"、一个"破案方法"和一个"最终答案"。
