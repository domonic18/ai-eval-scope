🔬 微观粒子运动观察

在VR环境中"缩小"到分子尺度，观察水分子在不同状态中的行为：

  • 固态（冰）—— 分子排列整齐，仅在固定位置振动
  • 液态（水）—— 分子间距增大，可以滑动和转动
  • 气态（水蒸气）—— 分子高速飞散，几乎不受约束
