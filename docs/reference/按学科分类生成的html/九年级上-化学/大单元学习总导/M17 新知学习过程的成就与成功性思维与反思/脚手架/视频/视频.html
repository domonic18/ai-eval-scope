📺 《化学变化与物理变化——一张图说清楚》

来源：B站 / YouTube 科普频道

时长：约8分钟

观看目标：看完后能画出区分化学变化和物理变化的"标准判断流程图"

📝 关键问题

① 视频中用什么例子来区分化学变化和物理变化？
视频中用了蜡烛燃烧、铁器生锈、纸张撕碎、冰块融化等日常例子。核心判断标准是：有没有新物质生成。有新物质的是化学变化，没有的是物理变化。例如纸张撕碎只是形状改变（物理），但纸张燃烧生成二氧化碳和水（化学）。
② 如果只用一句话告诉家人"什么是化学变化"，你会怎么说？
参考示例："化学变化就是物质变成另外一种物质的过程——比如铁变成了铁锈，木头变成了灰，它们'变'成了完全不同的新东西。" 你可以用自己的话写下来！

🧪 《铁丝在氧气中燃烧——实验全程分析》

来源：化学实验教学视频

时长：约6分钟

观看目标：记录实验中每一步操作的目的（不要只记操作本身）

🔍 关键问题

① 为什么铁丝要打磨？为什么缠成螺旋状？为什么瓶底要放水或细沙？
  • 打磨铁丝：去除表面的氧化膜（Fe₂O₃），露出纯净的铁，便于燃烧。
  • 缠成螺旋状：增大铁丝与氧气的接触面积，同时有利于集中热量。
  • 瓶底放水或细沙：防止高温熔融物溅落炸裂集气瓶底。
② 如果实验失败了（铁丝没有燃烧），可能的原因有哪些？

可能原因：①氧气纯度不够（集气瓶中空气未排尽）；②铁丝未打磨干净；③铁丝太粗；④火柴燃尽太快，未引燃铁丝（应待火柴快燃尽时再伸入）；⑤集气瓶未盖玻璃片，氧气逸出。

③ 这个实验证明了氧气的什么性质？

铁丝在空气中不能燃烧（只能红热），但在氧气中能剧烈燃烧——证明氧气具有助燃性（支持燃烧），是一种化学性质比较活泼的气体。

💬 深度思辨问题

① 演讲者认为化学和日常生活的关系是什么？你同意吗？有没有你觉得他忽略的方面？

思考框架：

  • 演讲者列举了哪些日常生活中的化学应用？
  • 这些应用在你自己的生活中真实存在吗？
  • 你觉得他忽略的方面——可能是化学的负面影响（污染）？还是化学学习的困难？
② 如果你要反驳他，你会从哪个角度切入？

可能的反驳角度：①化学入门门槛高，过早接触可能让一些学生失去兴趣；②化学实验有安全风险；③演讲者可能过度乐观地描绘了化学的"美好一面"，忽略了化学工业的环境问题；④化学需要数学基础，数学薄弱的学生可能感到挫败。
