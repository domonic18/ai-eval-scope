🤝 学术社区

在协作、对话与真实连接中深化化学学习

化学不是一个人的孤岛探索。通过与同伴的讨论、与家长的分享、与社会资源的连接，你会发现化学学习可以更丰富、更有趣。点击下方标签切换不同的连接方式。

🔗 三类连接

讨论与协作

议题A：

"化学变化中一定有能量变化"——你能举例说明吗？你想到的例子和同桌想到的，互补性如何？

议题B：

你从建构性示例中提炼的"建构性原理"是什么？和同学的原理对比，谁的更通用？谁的更贴近化学学科特点？

议题C：

本模块的反思环节中，你觉得自己最大的"学习突破"是什么？你是如何实现这个突破的？

👫 两人搭档

互相讲解各自的"化学变化类型辨析模型"，用对方的模型判断一个未接触过的反应，提出改进建议。

👥 四人小组

分别完成7个建构性案例中的不同案例，互相展示成果，评选"最佳建构案例"。

角色分工建议

角色职责
🎯 主持人控制讨论节奏，确保每人都有发言机会
📝 记录员记录小组讨论的关键观点和分歧
❓ 质疑官对每个结论提出"你怎么知道？你的证据是什么？"
📊 总结者汇总小组共识，准备向全班汇报

支持与反馈

📸 展示成果

每完成一个建构成果（模型图、方案书），拍照发给家长，用3句话解释："我做了什么，我学到了什么"。

💬 家长反馈单

请家长在成果照片下方写一句反馈——"你最惊奇的一点是什么？"

🧪 家庭化学小实验：

如果条件允许，在家长陪伴下完成一个安全家庭小实验（如白醋+小苏打），拍摄短视频并在班级群分享。

⚠️ 注意安全，请在家长陪同下操作
