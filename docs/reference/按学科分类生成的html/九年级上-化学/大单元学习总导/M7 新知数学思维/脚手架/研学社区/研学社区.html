🫖 基础案例：家里的"白醋除垢"效率 基础 ▾

🧊 背景：水壶中的水垢（主要成分 CaCO₃）可以用白醋去除。但怎样做最高效？

❓ 核心问题：用多少白醋、泡多久才能完全去除水垢？

方案 A 大量白醋 + 短时间浸泡
方案 B ✅ 少量白醋 + 长时间浸泡

📊 结果：方案 B 更高效！定量分析显示：除垢效果取决于"醋的总用量 × 接触时间"的乘积，而非单纯用量。

💡 可迁移启示

生活中很多"效率问题"可以通过简单的变量控制实验来解决——控制一个变量，测量一个结果。

🧪 进阶案例：食品添加剂安全争议 进阶 ▾

🧊 背景：关于食品添加剂（如防腐剂苯甲酸钠）的争议——一方说"有毒"，一方说"在规定范围内安全"。

❓ 核心问题："有毒"和"安全"的边界在哪里？关键是——剂量。"一切毒物都离不开剂量。"（Paracelsus）

🔍 决策路径：

调查 ADI（每日允许摄入量） → 计算日常摄入量 → 与 ADI 对比 → 做出判断

📊 结果：多数食品添加剂在标准用量下是安全的，关键问题是"是否超标"。

💡 可迁移启示

遇到"有毒/安全"的争议时，先问"剂量是多少"——这是化学定量思维的第一反应。

🚗 挑战案例：新能源汽车真的更环保吗？ 挑战 ▾

🧊 背景：有人说电动车"零排放"最环保，有人说电池生产污染更大。矛盾的观点哪个正确？

❓ 核心问题：如何从全生命周期角度比较电动车和燃油车的环境成本？

🔍 决策路径：

🚗 生产阶段 → ⚡ 使用阶段 → ♻️ 废弃阶段

列出三个阶段的排放数据 → 统一计算单位（g CO₂/km）→ 综合比较

📊 结果：全生命周期内电动车碳排放更低（约低30%~50%），但电池回收仍是重大挑战。

🚗 燃油车 约 200 g CO₂/km
（全生命周期）
⚡ 电动车 约 100~130 g CO₂/km
（全生命周期）
💡 可迁移启示

复杂问题要"全链条"定量分析，不能只看一个环节——这是系统思维的核心。

📌 注意：数据因国家能源结构不同而有差异
