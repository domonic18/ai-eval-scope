基础层 再现与理解

📝 A1 · 装置简图

阅读教材中"蜡烛燃烧实验"部分，画出实验装置简图，标注测量工具

📊 A2 · 数据记录表

📖 A3 · 概念辨析

写出"定量分析"与"定性观察"的定义，各举一个化学中的例子

分层层 选择你的路线

🌱 A路线 · 基础巩固

给定5组实验数据，计算平均耗氧速率，绘制柱状图

📈 B路线 · 能力提升

设计一个方案，用定量方法比较不同燃料（蜡烛、酒精灯、木炭）的耗氧速率

🚀 C路线 · 挑战拓展

查阅资料，分析"拉瓦锡测定空气中氧气含量"的原始实验数据，写一篇200字的方法评析

💡 选择建议： 感觉自己还需要练手 → A路线；想要挑战自己 → B路线；想深入了解化学史 → C路线

开放层 创造性表达

✍️ 写作

🎨 图示

🎬 视频

🎤 演讲

📏 评价标准（Rubric）

维度优秀（3分）达标（2分）待改进（1分）
数据准确性 全部数据正确，单位规范 大部分正确，单位有个别遗漏 数据错误较多
分析方法 使用了2种以上分析方法 使用了一种分析方法 缺少分析方法
表达清晰度 图表+文字结合，逻辑清晰 表达基本清楚 表达混乱
反思深度 有独立思考和质疑 有思考但不深入 只有结论没有思考
