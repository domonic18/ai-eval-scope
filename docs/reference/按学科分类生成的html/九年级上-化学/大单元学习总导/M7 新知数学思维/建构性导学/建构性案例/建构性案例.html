1 我的"喝水"定量记录 ▾

情境：医生说你每天要喝8杯水，但"一杯"到底是多大？你决定用定量方法记录自己一周的饮水量。

任务：建构一个"个人饮水量监测与评价"模型

调用原理： 变量识别法 对比测量法

成果：一周饮水量数据表 + 日变化折线图 + 自我评价结论
