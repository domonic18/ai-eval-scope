🕯️ 建构性示例

从"蜡烛燃烧"到"质量变化"——用数学思维重新认识化学变化

情境引入

你点燃一支蜡烛，看着它慢慢燃烧。蜡烛越来越短，但你有没有想过：燃烧后的蜡烛去哪儿了？

如果称一称蜡烛燃烧前后的总质量（蜡烛+空气），结果会怎样——变轻了？变重了？还是不变？

这个问题看似简单，却隐藏着一个深刻的化学定量问题。让我们用数学思维来建构对这个"日常奇迹"的定量理解。

建构目标

建构一个"蜡烛燃烧质量变化"的定量解释模型，能说明蜡烛燃烧前后质量变化的真正原因，并能用数据和图表支持你的结论。

成果用途：向同学解释"蜡烛燃烧后为什么质量会变"

最低标准：包含 猜想→证据→推理→结论 四个环节，至少使用2项定量数据

建构过程示范（6步）

点击每一步展开详细说明

1

观察与提问

点燃蜡烛，仔细观察：蜡烛变短了、蜡油滴落、火焰上方有黑烟……

核心问题：燃烧后蜡烛的质量去哪儿了？燃烧前后总质量会变化吗？

2

定性猜想

先凭直觉猜一猜：

🔹 猜想A：质量变轻——蜡烛烧掉了，所以总质量减少

🔹 猜想B：质量不变——根据质量守恒定律，物质不会凭空消失

🔹 猜想C：质量变重——燃烧消耗了空气中的氧气，结合成了新物质

你的猜想是？______ （没有对错，先记录直觉）

3

设计测量方案

如何测量"参与反应的物质质量"？

方案：将蜡烛放在天平上，记录燃烧前总质量 → 点燃燃烧一段时间 → 熄灭后再次称量

注意：需要控制燃烧时间一致、使用同一台天平

4

收集定量证据

假设实验数据如下：

测量项目质量（g）
蜡烛燃烧前总质量（蜡烛+托盘+空气）52.4
蜡烛燃烧3分钟后总质量51.8
质量变化-0.6

数据解读：燃烧后总质量减少了0.6g。这说明有物质"离开"了系统。

5

建立解释模型

用数据和化学原理解释：

📊 数据：燃烧后质量减轻了0.6g

🧪 化学原理：蜡烛的主要成分是碳氢化合物（石蜡），燃烧时与氧气反应生成二氧化碳和水蒸气。这些气体扩散到空气中，离开了天平测量的范围。

🔗 推理链接：质量减少 ≠ 质量消失。如果能收集所有燃烧产物（CO₂+H₂O），总质量应该等于蜡烛减少的质量 + 消耗的氧气质量（符合质量守恒定律）。

6

表达与交流

用图表+文字呈现完整推理：

52.4 燃烧前
51.8 燃烧后
-0.6 质量差

结论：蜡烛燃烧后质量变轻，是因为燃烧产物（CO₂和水蒸气）扩散到空气中。但这并不违反质量守恒定律——如果能收集所有产物，总质量是不变的。
