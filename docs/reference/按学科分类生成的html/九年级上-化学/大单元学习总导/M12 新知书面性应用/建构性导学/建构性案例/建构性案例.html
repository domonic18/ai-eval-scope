案例一 · 个人理解型

难度★☆☆ 基础
字数150-200字

📖 情境

你刚学完"物理变化与化学变化"的概念。同桌好奇地问你："物理变化和化学变化到底怎么区分？我老是搞混。你能用最简单的话给我说明白吗？"

🎯 任务

写一段150-200字的说明性文字，帮助同学区分物理变化和化学变化。要求包含：两个概念的定义、区分的核心依据（有无新物质生成）、分别给出2个例子（至少1个来自日常生活）、总结一句"判断口诀"。

📖 原理调用

  • 建构性示例中"三段式"结构：概念说明→区分方法→实例佐证
  • 先收集素材再组织：先列出你知道的变化例子，再判断哪个最有说服力
  • 读者意识：你的读者是"搞混了"的同学，表达要通俗易懂

🧩 脚手架

概念模板："______变化是指______过程中______的变化。例如，______就是______变化。"
对比表格：在文字中加入简单的对比能让概念更清晰

物理变化是指没有新物质生成的变化，如水变成冰、蔗糖溶于水，只是形态或状态改变。化学变化是指有新物质生成的变化，如镁条燃烧生成氧化镁、铁生锈生成铁锈，物质的种类发生了改变。区分的关键是看"有没有新物质生成"——记住：物理变化易复原，化学变化变新物。

✏️ 你的写作练习

📊 自评量规

维度优秀达标待改进
概念准确定义完全正确，术语规范基本正确有概念混淆
举例恰当例子典型，能支持定义例子正确例子不匹配
逻辑清晰从定义到例子再到总结，层层递进基本有序缺乏逻辑结构

案例二 · 生活应用型

难度★★☆ 进阶
字数200-300字

📖 情境

你妈妈看到厨房里的铁锅生锈了，随口问了一句："这个锈是怎么回事？能不能用醋洗掉？"你刚刚在化学课上学过"铁生锈是化学变化"以及"酸能与某些金属氧化物反应"，你知道这个知识点。妈妈等着你的回答——不是随便说一句，而是给她一份"铁锅除锈小指南"。

🎯 任务

写一份200-300字的"铁锅除锈小指南"，面向家庭读者。内容包含：铁锈是什么（化学本质）、铁生锈的原因（化学变化过程）、如何用醋除锈（简单操作说明）、如何预防铁锅再生锈。要求用通俗的语言但保持化学准确性。

📖 原理调用

  • 建构性示例中的"5W1H"检查表：确保涵盖"是什么、为什么、怎么做"
  • 优化建议中的"让术语更规范"：写给家人看，化学术语仍需准确
  • 从日常现象出发、用化学知识解释的建构思路

🧩 脚手架

框架建议："是什么（铁锈）→为什么生锈→怎么除锈→怎么预防"
操作步骤模板：用"首先/接着/最后"写清楚除锈步骤
铁锈化学本质：Fe₂O₃·xH₂O（氧化铁水合物）
除锈原理：Fe₂O₃ + 6CH₃COOH → 2Fe(CH₃COO)₃ + 3H₂O
预防方法：保持干燥、涂食用油隔绝空气和水分

📊 自评量规

维度优秀达标待改进
科学准确化学原理准确，无知识错误基本准确有明显错误
可读性语言通俗易懂，家人能看懂部分需要解释过于晦涩
实用性步骤清楚，操作可行基本可行缺乏可操作性

案例三 · 校园问题解决型

难度★★☆ 进阶
字数200-250字

📖 情境

学校实验室的废液缸中，不知谁倒入了过氧化氢溶液（H₂O₂）。你发现废液缸边上有几根用过但未清洗的铁钉，铁钉表面有气泡冒出。你意识到这个现象值得探究，但你需要写一份实验观察记录提交给化学老师。

🎯 任务

写一份200-250字的"实验观察记录与初步分析"，格式参考科学实验记录的标准格式（日期/观察人/观察现象/初步分析/待探究问题）。要求：准确描述气泡现象，推测气泡成分并说明推测依据，提出至少1个值得进一步探究的问题。

H₂O₂在金属（铁）催化下分解：2H₂O₂ → 2H₂O + O₂↑。气泡成分很可能是氧气(O₂)。待探究问题：铁钉是否起了催化作用？

📊 自评量规

维度优秀达标待改进
观察描述具体、准确、有细节基本完整过于笼统
分析推理有理有据，逻辑严密有基本推理缺乏证据支撑
记录规范格式标准，要素齐全格式基本正确缺少关键要素

案例四 · 学科表达型

难度★★☆ 进阶
字数250-350字

📖 情境

化学课上老师展示了"水的电解"实验——通直流电后，两个电极上都有气泡产生，正极产生约5mL气体，负极产生约10mL气体。正极气体能使带火星的木条复燃，负极气体能燃烧发出淡蓝色火焰。

🎯 任务

写一段250-350字的实验现象描述与结论分析。要求：准确描述正负极的现象和差异，说明两种气体的验证方法及结果，写出水的电解化学方程式，分析该实验说明的结论，讨论变化类型并说明理由。

化学方程式：2H₂O 通电 2H₂↑ + O₂↑。正极（O₂）：使带火星木条复燃；负极（H₂）：燃烧产生淡蓝色火焰。体积比 H₂:O₂ = 2:1。说明水由氢元素和氧元素组成。

📊 自评量规

维度优秀达标待改进
现象描述有数据、有对比、有细节基本完整缺少关键现象
气体检验验证方法正确，结果准确基本正确方法或结果有误
结论分析推理严密，结论准确结论正确推理跳跃

案例五 · 跨学科整合型

难度★★★ 挑战
字数200-300字

📖 情境

你在生物课上学到"植物的光合作用"，化学课上刚学了"化学变化中质量守恒"。你突然发现——光合作用同时涉及生物和化学两个学科的知识。你想写一篇短文，从化学的视角解释光合作用为什么是一个化学变化，并运用质量守恒定律分析反应物和生成物的关系。

🎯 任务

写一篇200-300字的跨学科短文，主题为"从化学视角看光合作用"。要求：写出光合作用的化学方程式，指明反应物和生成物，解释为什么光合作用是化学变化，用质量守恒定律分析反应前后元素种类不变，提出"生物视角"和"化学视角"的异同点。

光合作用：6CO₂ + 6H₂O 光能/叶绿体 C₆H₁₂O₆ + 6O₂。反应物（CO₂、H₂O）和生成物（葡萄糖、O₂）种类不同 → 化学变化。碳、氢、氧元素在反应前后种类、质量不变 → 质量守恒。

📊 自评量规

维度优秀达标待改进
化学知识方程式正确，质量守恒分析准确基本正确有错误
跨学科理解明确指出两个学科的联系与区别有联系但不够深入只讲了化学部分
表达创意角度新颖，引人思考较常规缺乏个人思考

案例六 · 社会/未来情境型

难度★★★ 挑战
字数200-300字

📖 情境

你和家人一起看新闻，报道说"城市空气质量指数(AQI)达到重度污染"。你对化学课上学过的"空气组成"有了感觉——你知道空气中本来有氮气、氧气、稀有气体等成分，污染物是什么？它们从哪里来？你想写一段说明文字，帮助家人理解空气质量报告中的"化学信息"。

🎯 任务

写一段200-300字的"空气质量化学解读"。要求：说明空气的正常组成成分，列举2-3种常见空气污染物及其主要来源，用化学知识解释CO为什么是有毒气体，提出2条基于化学知识的空气保护建议。

空气组成：N₂ 78%、O₂ 21%、稀有气体 0.94%、CO₂ 0.03%。污染物：PM2.5（颗粒物）、SO₂（燃煤产生）、NOₓ（汽车尾气）、CO（不完全燃烧）。CO与血红蛋白结合能力比O₂强200倍→缺氧中毒。

📊 自评量规

维度优秀达标待改进
科学知识空气组成准确，污染物描述正确基本准确有知识错误
社会关联能联系实际生活，有现实意义有联系脱离实际
建议可行性建议具体、可操作基本可行过于空泛

案例七 · 综合创新型

难度★★★ 挑战
字数400-500字

📖 情境

学校要举办"化学与生活"科学小报比赛，你决定报名参加。小报的主题是"水——最普通的化学物质"。你需要写一篇500字以内的科普短文，面向全校同学（包括未学过化学的低年级同学），从化学角度介绍"水"。你不能只是罗列事实，而要用一个"故事线"把水的化学知识串联起来。

🎯 任务

写一篇400-500字的科普短文。要求：用一个吸引人的标题，设计一个叙事线索（如"一滴水的旅程"），包含水的化学式及含义、水的物理性质和化学性质、水的分解与合成、水在生活中的重要性。语言要生动但不失准确。

📖 原理调用

  • 综合运用前6个案例习得的所有建构技巧
  • 面向低年级读者的"可读性"要求最高
  • 叙事线索的建构：不是"写成一篇化学论文"，而是"讲一个化学故事"

🧩 建议步骤

  1. 确定一个有趣的叙事线索（如"从一杯水出发，探索它的化学秘密"）
  2. 列出你想涵盖的化学知识点：H₂O的含义、沸点冰点、水的电解、水的合成
  3. 把知识点"串"到故事线中
  4. 写初稿，注意语言既要准确又要有画面感
  5. 请一位低年级同学试读：他/她感兴趣吗？能看懂吗？

📊 自评量规

维度优秀达标待改进
科学内容知识准确丰富，有深度知识正确内容不够或错误
创意表达叙事线吸引人，语言生动有基本叙事平铺直叙
面向读者让没学过化学的同学也能读懂大部分能理解过于专业或简单
综合运用综合运用多种建构方法运用了部分方法建构方法单一

🎯 小结

七阶案例从"帮同学区分概念"到"写科普短文"，难度逐步升级。你可以根据自己的情况选择适合的起点：

  • 如果对概念还不够清晰，从案例一开始，逐步推进
  • 如果已经有一定基础，可以从案例三或案例四开始
  • 如果希望挑战自己，直接尝试案例七——它将检验你综合运用所有建构技巧的能力

每个案例完成后，请务必对照自评量规进行自我评价，找到你最需要改进的方向。
