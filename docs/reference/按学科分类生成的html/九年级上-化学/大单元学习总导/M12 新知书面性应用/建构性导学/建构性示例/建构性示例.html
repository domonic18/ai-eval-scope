二、建构性原理提炼

以下5条原理是从完整过程中提炼的可迁移方法，每条都配有示例体现和操作指南。

原理一

先积累素材，再组织表达

核心：动笔之前先"倒"出所有观察，不筛选不排序，先写下来。

示例体现：先把镁条颜色、白光、放热、变短、白色粉末等全部记录，不急于组织语言。

操作指南：

  • 准备草稿纸或空白文档
  • 用"我看到/听到/闻到……"句式列出原始观察
  • 加上已知的化学知识（术语、概念）
  • 不要急着删除任何条目
原理二

用化学概念给素材"贴标签"

核心：将原始观察转化为化学语言——用学过的概念对素材分类和解释。

示例体现："银白色"→物理性质，"白色粉末"→新物质（氧化镁）。

操作指南：

  • 对照素材，逐个问："这对应课本上哪个概念？"
  • 用箭头把原始观察和化学术语连接
  • 筛选出"有化学价值的素材"
原理三

搭建框架再填充，按时间顺序最保险

核心：先确定结构框架，再逐块填充内容。

示例体现："三段式"框架：反应物描述→反应过程描述→分析与结论。

框架类型：

  • 时间顺序型：反应前→反应中→反应后
  • 概念解释型：定义→特征→例子→总结
  • 问题解决型：现象→原因→方案→预防
原理四

初稿求完整，定稿求规范——"先写完，再改好"

核心：写作分为两个阶段——第一阶段"把话说完"，第二阶段"把话说准"。

操作指南：

  • 初稿只关注完整性
  • 修改逐项检查：准确性、规范性、简洁性、可读性
  • 至少修改两轮——自己改一遍，再找同学看一遍
原理五

读者意识决定表达方式

核心：写给谁看，决定怎么写。时刻问自己："我要写给谁看？"

操作指南：

  • 同学：注重可理解性
  • 老师：注重规范性和分析深度
  • 家人：注重通俗性，减少专业术语
