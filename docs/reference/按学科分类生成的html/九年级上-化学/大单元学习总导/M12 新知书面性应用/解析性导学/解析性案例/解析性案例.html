第1阶 · 生活情境

食醋说明书分析

案例材料：

某品牌食醋瓶身说明——"含醋酸（乙酸）约3.5g/100mL。醋酸是一种有机酸，具有刺激性气味。可用作调味品，也可用于去除水垢。注意：请勿与碱性物质混合使用，以免失效。"
解析任务：分析准确性、规范性。关注浓度表达是否规范、"失效"含义是否科学。
[+] 查看解析要点
浓度表达：'3.5g/100mL' 是日常生活中常见写法，学术上应写为 '3.5 g/100 mL'（空格），或质量分数形式。此处基本可接受。
'失效'一词不够精确：醋酸与碱性物质中和生成盐和水，只是失去了酸性（失效），但并非化学上的'失效'。更准确的表述应为'以免降低去污效果'或'以免中和失效'。
整体描述基本准确，规范性略有不足，科学性基本正确。
第2阶 · 校园情境

实验报告互评

案例材料（李华同学"探究蜡烛燃烧产物"实验记录）：

"点燃蜡烛，用干燥烧杯罩在火焰上方，看到内壁有水珠。再用沾有澄清石灰水的烧杯罩上去，看到石灰水变浑浊。所以蜡烛燃烧产物是水和二氧化碳。"
解析任务：分析完整性、逻辑性、规范性。关注推理严谨性、是否缺乏对照实验。
[+] 查看解析要点
推理有逻辑跳跃：仅凭水珠和石灰水变浑浊就断定产物，缺少'产物是什么'的排除性论证。
缺少对照实验：未用空烧杯做空白对照，无法排除空气中水蒸气的影响。
操作描述不规范：'用沾有澄清石灰水的烧杯罩上去'——石灰水只是沾在杯壁还是盛在杯内？应明确表述。
改进建议：增加空白对照，描述具体操作细节，结论措辞改为'可以推断……'更严谨。
第3阶 · 家庭/社区情境

安全用气说明

案例材料（社区安全宣传栏）：

"天然气主要成分是甲烷。泄漏时会闻到特殊气味。此时要立即开灯检查，打开窗户通风。如果发现有人中毒，立即进行人工呼吸。"
解析任务：分析安全性准确性。关注"开灯检查"在泄漏环境下是否安全、"人工呼吸"是否合适。
[+] 查看解析要点
严重安全性错误：'立即开灯检查'——开灯产生的电火花可能引燃泄漏的甲烷与空气混合物，导致爆炸。正确做法是严禁开关任何电器，立即关闭阀门、打开窗户。
'人工呼吸'不当：发现有人中毒，应先将其转移到通风处，再判断呼吸心跳情况；若直接人工呼吸，施救者也可能吸入有毒气体。
建议修改：'泄漏时切勿开关任何电器，立即关阀开窗。若有人员不适，迅速转移至通风处并拨打急救电话。'
第4阶 · 学科核心情境

氧气制取实验描述

案例材料：

"将高锰酸钾放入试管中，塞上带导管的单孔橡皮塞，固定在铁架台上。用酒精灯对着试管底部加热，用排水法收集气体。"
解析任务：全面分析规范性。关注试管口方向、预热、棉花团、收集时机、结束操作顺序。
[+] 查看解析要点
试管口方向未说明：应略向下倾斜，防止冷凝水回流炸裂试管。
缺少预热步骤：直接'对着试管底部加热'易导致局部受热不均炸裂。
缺少棉花团：高锰酸钾制氧气时试管口应塞一团棉花，防止粉末进入导管。
收集时机不明确：应在导管口气泡连续均匀时才开始收集（先排尽空气）。
结束操作顺序：应先从水槽中移出导管，再停止加热，防止倒吸。
该描述缺少多个关键操作细节，完整度不足。
第5阶 · 跨学科情境

观察日记分析

案例材料（校园牵牛花观察记录）：

"牵牛花在早晨是蓝色的，下午变成红色。细胞中含有花青素，遇酸变红遇碱变蓝。光合作用消耗CO₂，CO₂减少，碱性增强，所以花变蓝。"
解析任务：跨学科角度分析。关注花青素指示剂性质描述是否正确、因果逻辑是否完整。
[+] 查看解析要点
花青素性质描述有误：花青素遇酸变红、遇碱变蓝是正确的指示剂性质，但牵牛花实际从蓝变红是因为夜间呼吸作用积累CO₂，使细胞液酸性增强，花青素变红；早晨光合作用消耗CO₂，酸性减弱，花变蓝。
因果逻辑方向反了：文中说'碱性增强，所以花变蓝'方向正确，但缺少中间环节——CO₂减少使酸性减弱（pH升高呈弱碱性），花青素显蓝色。
整体逻辑链条不完整，需补充呼吸作用和光合作用的昼夜交替如何影响细胞酸碱度。
第6阶 · 社会/未来情境

环保倡议书分析

案例材料（学校环保社团倡议书节选）：

"工厂排放的废气中含有大量一氧化碳、二氧化硫等有害气体。一氧化碳有毒，二氧化硫溶于水形成酸雨。这些气体都会破坏臭氧层，导致全球变暖。"
解析任务：化学知识准确性分析。关注"破坏臭氧层"和"全球变暖"的概念混淆。
[+] 查看解析要点
概念混淆：一氧化碳和二氧化硫并非破坏臭氧层的主要物质。破坏臭氧层的主要是氟氯烃（CFCs）、氮氧化物等。
二氧化硫导致酸雨正确，但酸雨与臭氧层破坏是两类不同的环境问题。
'导致全球变暖'：一氧化碳和二氧化硫都不是主要的温室气体。CO和SO₂对全球变暖贡献极小，主要温室气体是CO₂、CH₄等。
正确表述：'一氧化碳有毒，二氧化硫是形成酸雨的主要原因之一。这些气体的排放对环境和人体健康有严重危害。'
第7阶 · 综合迁移情境

综合实践报告评审

案例材料（"学校附近水质调查"报告节选）：

"用pH试纸测了酸碱度，试纸变绿，说明水是中性偏碱的。加热闻到有淡淡的气味，说明水里有有机物。用显微镜看到了微生物。结论：水质不太好，有污染。"
解析任务：综合运用五个维度（准确性、规范性、完整性、逻辑性、安全性）全面解析。关注检测方法合理性、结论是否缺乏依据。
[+] 查看解析要点
准确性：'试纸变绿'——pH试纸变绿通常对应pH约7-8，说'中性偏碱'基本合理。但绿色在不同批次试纸上可能对应不同pH范围，应说明具体数值或对照比色卡读数。
规范性：缺少pH试纸使用的标准操作描述（如用玻璃棒蘸取、点在试纸上、半秒内比色等）。
完整性：加热闻到气味不能单纯说明'有有机物'——很多无机物也有气味；显微镜看到微生物属于生物指标，缺乏定量数据；未检测氨氮、COD等常规水质指标。
逻辑性：结论'水质不太好，有污染'缺乏充分证据。检测项目有限，不足以支撑污染定性，结论过于主观宽泛。
安全性：加热水样时应注意通风，避免吸入不明气体；取样应有安全防护措施。
