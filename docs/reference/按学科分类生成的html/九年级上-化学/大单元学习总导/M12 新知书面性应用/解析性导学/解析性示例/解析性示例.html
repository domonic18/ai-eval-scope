化学书面表达质量分析 — 五步解析法示范

一、Slot 定位

本Slot作为"解析性导学"Tag的第一个下属环节，承载示范引领功能。学生将通过一个完整的化学书面表达解析实例，直观理解"如何分析一段化学书面表达的质量"，建立"五步解析法"的操作模型。

二、解析任务情境

小明同学在完成"加热碳酸氢铵"实验后，写了一篇实验观察记录。假设你是化学课代表，老师请你帮小明看看这篇记录存在哪些问题，并提出修改建议。

小明同学的实验观察记录：

实验名称：加热碳酸氢铵
今天我们在实验室做了一个实验，把碳酸氢铵放在试管里加热。碳酸氢铵是一种白色的固体，把它放进试管，用试管夹夹住，在酒精灯上加热。过了一会儿，看到固体好像不见了，变成了一些白色的东西在试管壁上。还闻到一股刺鼻的味道。我觉得这个实验说明了碳酸氢铵受热会分解。

实验现象：固体减少，试管壁上有白色物质，有气味。
实验结论：碳酸氢铵受热会分解。

五、专家解析过程（五步法）

第一步：明确解析任务

对九年级化学实验观察记录进行全面质量分析，从准确性、完整性、逻辑性、规范性、可读性五个维度诊断。

第二步：提取关键信息

关注信息缺口：是否使用检验手段？加热时长和温度？白色物质具体位置？

第三步：逐项诊断

准确性："固体好像不见了"表述模糊，应改为"固体逐渐减少"；"白色物质"未说明具体成分；"刺鼻味道"应表述为"刺激性气味"。

完整性：缺少实验日期、具体操作细节、气体检验手段、化学方程式、质量对比。

逻辑性：现象描述顺序杂乱；"固体不见"与"白色物质"前后矛盾；结论缺乏证据链支撑。

规范性：未使用化学式（NH₄HCO₃）、缺少化学方程式、"刺鼻的味道"用语不规范。

可读性：偏口语化、缺乏分段、现象与结论混合在一起。

第四步：形成结论与改进建议

对比查看 原文 与 改进版：

实验现象：固体减少，试管壁上有白色物质，有气味。
实验结论：碳酸氢铵受热会分解。
实验名称：加热碳酸氢铵 — 探究碳酸氢铵的热稳定性
实验用品：试管、试管夹、酒精灯、碳酸氢铵(NH₄HCO₃)、澄清石灰水、湿润的红色石蕊试纸

分析与结论：碳酸氢铵受热易分解，产生氨气
化学方程式：NH₄HCO₃ △ NH₃↑ + H₂O + CO₂↑
