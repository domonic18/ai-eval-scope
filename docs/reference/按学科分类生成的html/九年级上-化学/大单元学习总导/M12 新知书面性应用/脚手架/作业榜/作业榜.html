🎯 基础层目标

用规范用语描述基本化学现象和概念

用词准确
无色/白色、气体/烟雾等区分到位
结构完整
时间→地点→现象→结论脉络清晰
书写规范
化学式、元素符号书写无误
✍️ 任务 A · 现象描述 写作
用 100-150 字描述指定实验现象（如锌粒与稀盐酸反应）
📏 字数：100-150 字📋 形式：写作
💡 写作提示：
  • 观察要点：反应前固体颜色形态 → 加入液体后的变化 → 产生的气体/沉淀
  • 用词示例：银白色金属 → 表面产生大量气泡 → 固体逐渐减小
  • 避免使用"冒烟"（应区分"白雾"与"白烟"）
✍️ 任务 B · 概念复述 写作
用自己的话写出 3 个化学基本概念的定义（物理变化、化学变化、物理性质）
📏 字数：每个概念 30-50 字📋 形式：写作
💡 写作提示：
  • 定义格式：概念名称 + 属（属于什么范畴）+ 种差（区别于其他概念的特征）
  • 例如："物理变化是物质在变化过程中没有生成其他物质的变化"
  • ⚠️ 注意：不能直接抄课本，必须用自己的语言重组
✍️ 任务 C · 纠错练习 写作
给出一段含错误的实验描述，圈出错处并修正
📋 形式：批注 + 修改
💡 示例片段：

"将锌粒放入稀盐酸中，观察到锌粒表面产生白烟，溶液温度降低。"

🔍 找一找： 几处错误？正确的应该怎样写？
  • "白烟" → 应为"气泡"（氢气是无色气体，不是白烟）
  • "温度降低" → 应为"温度升高"（置换反应放热）
📊 图示任务 · 物质分类树状图 图示
绘制"物质分类树状图"并配文字说明
📋 形式：图示 + 100-200 字说明
📐 结构建议：
  • 顶层：物质 → 分支为 纯净物 与 混合物
  • 纯净物 → 分支为 单质（金属/非金属）与 化合物
  • 化合物 → 分支为 氧化物、酸、碱、盐等
  • 配图说明：用箭头 + 关键词 + 简单例句

培养对比分析、因果推理的书面表达能力

✍️ 任务 A · 比较分析 写作
比较氧气与二氧化碳的物理性质和化学性质，写成 200 字对比说明
📏 字数：200 字📋 形式：对比说明
📋 比较框架：
  • 物理性质：颜色、状态、气味、密度（与空气比较）、溶解性
  • 化学性质：是否支持燃烧、与什么物质反应、反应现象
  • 结尾：用一句话总结二者的本质差异
✍️ 任务 B · 因果解释 写作
解释"为什么铁丝在空气中不能燃烧而在氧气中可以"，要求用"因为……所以……"结构
📋 形式：因果推理段落
🔬 关键知识点：
  • 空气中氧气体积分数仅约 21%，氧气浓度低
  • 铁丝燃烧需要达到一定的氧气浓度（助燃物浓度）
  • 纯氧中氧气浓度高，反应剧烈，放出热量足以维持持续燃烧
  • 写作结构：现象描述 → 原因分析 → 实验证据 → 结论
✍️ 任务 C · 实验分析 写作
分析实验失败的可能原因并提出改进方案（如"收集的氧气不纯，可能是因为……"）
📋 形式：诊断式分析
🧪 分析模板：
  • Step 1：明确指出实验失败的具体表现
  • Step 2：列举可能的失败原因（2-3 条）
  • Step 3：针对每一条原因提出对应的改进措施
  • Step 4：用"因此，改进后的实验应该……"做结
🎬 任务 · 实验解说视频 视频
录制 2 分钟"实验现象解说"短视频，需提交视频脚本文字稿
⏱ 时长：2 分钟📋 产出：视频 + 文字脚本
🎥 脚本结构建议：
  • 开场（15s）：实验名称 + 目的
  • 准备（20s）：介绍实验器材和药品
  • 操作（40s）：边做边解说操作步骤
  • 现象（25s）：描述观察到的现象
  • 结论（20s）：解释现象背后的化学原理

🎯 开放层目标

综合运用化学知识进行创造性、辩论性的书面表达

✍️ 任务 A · 创意写作 写作
"假如我是一个水分子，我的奇妙旅程"（不少于 300 字）
📏 字数：≥300 字📋 形式：创意故事
💡 创作思路：
  • 以第一人称"我"（H₂O 分子）的视角展开
  • 经历不同的状态变化（固态→液态→气态）
  • 经历化学变化（如电解水：2H₂O → 2H₂↑ + O₂↑）
  • 融入真实的地理/生活场景（河流→云→雨→自来水厂）
  • 至少用到 5 个化学专业术语并标注
✍️ 任务 B · 科学小论文 写作
"日常生活中有哪些化学变化？任选 3 例进行科学分析"
📏 字数：400 字左右📋 形式：小论文
📝 论文结构：
  • 引言：化学变化就在我们身边（50 字）
  • 例 1：如铁生锈（化学方程式 + 条件分析）
  • 例 2：如食物消化（酶的作用 + 化学变化本质）
  • 例 3：如燃烧（燃烧三要素 + 产物分析）
  • 结论：化学变化在日常生活中的意义
✍️ 任务 C · 辩论稿 写作
"化学工业的发展是利大于弊还是弊大于利？"撰写正方或反方一辩稿
📏 字数：500-600 字📋 形式：辩论一辩稿
🎯 辩论稿结构：
  • 立论（100 字）：明确我方观点 + 核心论点
  • 论据 1（150 字）：事实论据（数据/案例）
  • 论据 2（150 字）：理论论据（化学原理）
  • 论据 3（100 字）：补充论据或驳对方可能的观点
  • 结论（50 字）：升华观点，掷地有声
✍️ 任务 D · 实验方案设计 写作
设计一个实验证明"空气中氧气约占 1/5"，写出完整的实验方案
📋 形式：实验方案
🧪 方案框架：
  • 实验目的：定量测定空气中氧气的体积分数
  • 实验原理：利用红磷燃烧消耗氧气，压强差使水倒流
  • 实验器材：集气瓶、燃烧匙、红磷、水槽等
  • 操作步骤：6-8 步，逻辑清晰，可复现
  • 注意事项：装置气密性、红磷足量等
  • 预期结果与结论：水倒流至约 1/5 刻度处
