💬 名人名言

从化学史的思想轨迹中，理解知识是如何被发现的

🏛️ 奠基型 · 近代化学之父

安托万·拉瓦锡

Antoine Lavoisier
1743 — 1794

核心贡献：推翻燃素说、发现氧气、建立质量守恒定律、统一化学命名法

写作特色：精确的实验记录 + 定量的数据分析 + 清晰的逻辑推理

📖 深度阅读

《化学基础论》序言——关于化学命名法的论述

核心观点：拉瓦锡强调"命名要反映物质组成"。

思考题：拉瓦锡的化学命名法如何影响了我们今天书写化学式和化学反应方程式？

✍️ 仿写练习

模仿拉瓦锡用"定义+组成+性质"写化学说明

练习要求：用"定义 + 组成 + 性质"的三段结构写一段关于"水"的化学说明。

示例框架：

  • 定义：水是一种由氢元素和氧元素组成的化合物
  • 组成：每个水分子由 2 个氢原子和 1 个氧原子构成（H₂O）
  • 性质：无色无味液体，沸点 100°C，凝固点 0°C，密度 1 g/cm³

⚖️ 对比思考

比较拉瓦锡对"燃素说"的批判性论证与我们现在的实验论证
两者在证据获取方式、推理逻辑链条上有何异同？

提示：拉瓦锡用定量实验（称重）的核心方法是"控制变量"的早期雏形。与现代实验论证相比，区别主要在于仪器精度和数据统计方法，但逻辑框架（假设→实验→证据→结论）是一致的。

🧩 跨界型 · 元素周期表之父

德米特里·门捷列夫

Dmitri Mendeleev
1834 — 1907

核心贡献：发现元素周期律，预测未知元素性质

写作特色：宏观的图表思维 + 大胆的预测性写作 + 严谨的周期性归纳

📖 深度阅读

门捷列夫关于元素周期律的原始论文中的预测段落

核心启示：门捷列夫用"空白"来预测未知元素——这种"留白式"的表达具有强大的指向力量。

思考：他是如何用"空白"来预测未知元素的？"留白式"表达有什么力量？

✍️ 模式迁移

借鉴"分类→排列→预测"的思维框架

练习要求：借鉴门捷列夫"分类→排列→预测"的思维框架，对已学过的物质进行分类表达。

示例：

  • 分类：将常见物质分为单质、氧化物、酸、碱、盐
  • 排列：按组成元素的种类和数量排序
  • 预测：根据排列规律推测未学过的物质可能具有的性质

🔮 预测写作挑战

假如你发现了一种新元素
模仿门捷列夫，用"表格 + 文字"描述它的性质和可能存在的化合物

模板：

  • 元素名称：______（你命名）
  • 推测原子量：______
  • 可能存在的化合价：______
  • 预测物理性质：颜色____ 状态____ 密度____
  • 预测化学性质：与____反应，生成____
  • 可能形成的化合物：如 XO、XCl₂ 等
🌱 当代型 · 诺贝尔奖得主

屠呦呦

中国药学家
1930 —

核心贡献：发现青蒿素，为疟疾治疗提供新药

写作特色：跨学科的研究笔记 + 传统文献与现代实验的结合 + 严谨的科研论文

📖 深度阅读

《青蒿素的发现——传统中医献给世界的礼物》诺奖演讲

关键启发：屠呦呦从古籍"青蒿一握，以水二升渍，绞取汁，尽服之"中获得启发。

思考：她的文字推演过程是怎样的？从古籍到现代实验的转化逻辑是什么？

✍️ 论文引述练习

学习"背景→方法→结果→结论"的四段式写作

四段式框架：

  • 背景：为什么要研究这个问题？前人做了什么？
  • 方法：用了什么实验方法？如何操作的？
  • 结果：得到了什么数据？观察到什么现象？
  • 结论：从结果中推断出什么？有什么意义？

📝 创意表达

"从古籍到诺奖"
写一段 200 字的短文，讲述屠呦呦发现青蒿素的故事，体现从古代文献到现代科学发现的跨越

素材线索：

  • 东晋葛洪《肘后备急方》中的原文记载
  • 屠呦呦的灵感：低温提取避免高温破坏有效成分
  • 1972 年成功提取青蒿素（C₁₅H₂₂O₅）
  • 2015 年获得诺贝尔生理学或医学奖
