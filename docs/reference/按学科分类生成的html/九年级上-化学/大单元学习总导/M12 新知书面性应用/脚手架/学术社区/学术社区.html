💬 讨论议题

▼
议题一 · 物理变化与化学变化的本质区别
请用准确的文字界定二者边界
  • 核心判断标准：是否有新物质生成
  • 举例：蜡烛熔化（物理）vs 蜡烛燃烧（化学）
  • 写作要求：用"是否……"句式精确定义
议题二 · "蜡烛燃烧"的证据链
哪些现象可以证明产生了新物质？请列出证据链
  • 证据 1：蜡烛燃烧产生水（烧杯壁有水滴）
  • 证据 2：蜡烛燃烧产生 CO₂（澄清石灰水变浑浊）
  • 证据 3：蜡烛质量减少（部分转化为新物质）
议题三 · "化学变化中原子不变"的论证
你怎么证明这句话？写一篇微型论证
  • 理论依据：化学反应的实质是原子的重新组合
  • 实验依据：质量守恒定律（反应前后总质量不变）
  • 微观示例：2H₂ + O₂ → 2H₂O，原子数量不变

🧪 家庭科学实验项目

▼

🥄 食醋与小苏打反应

产生二氧化碳，观察气泡现象

化学原理：CH₃COOH + NaHCO₃ → CH₃COONa + H₂O + CO₂↑

书面任务：撰写实验报告 + 以"写给家长的信"解释原理

🔮 紫甘蓝汁做酸碱指示剂

观察不同酸碱性溶液中的颜色变化

化学原理：紫甘蓝中的花青素在不同 pH 下结构变化导致颜色变化

书面任务：记录颜色变化表格 + 解释背后的化学原理

🥚 鸡蛋壳与醋反应

观察碳酸盐与酸的反应现象

化学原理：CaCO₃ + 2CH₃COOH → (CH₃COO)₂Ca + H₂O + CO₂↑

书面任务：描述实验现象 + 解释反应本质
