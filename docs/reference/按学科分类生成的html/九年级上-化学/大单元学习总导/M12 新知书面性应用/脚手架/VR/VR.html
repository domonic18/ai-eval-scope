🔬 仪器识别与操作

VR中操作试管、酒精灯、量筒、托盘天平、铁架台。写出仪器的使用步骤和注意事项。

写作重点："先……再……然后……最后……"的时序表达，"禁止""防止""避免"等警示用语。

🔥 规范实验操作

VR模拟加热液体、倾倒液体、量取液体、称量固体等操作。为每个操作写出规范步骤说明（≥100字/操作）。

⚖️ 对比分析

写出"正确操作"与"错误操作"的结果差异。例如：试管加热时先预热 vs 直接加热固定部位。

3 挑战层：记录与报告任务

实验报告：探究蜡烛燃烧的产物

实验目的：探究蜡烛燃烧后生成哪些物质

实验用品：蜡烛、火柴、烧杯、澄清石灰水

实验步骤：

  1. 点燃蜡烛，观察火焰分层情况
  2. 用干燥烧杯罩在火焰上方，观察杯壁现象
  3. 将烧杯迅速翻转，倒入澄清石灰水，振荡观察

实验现象：火焰分三层（外焰、内焰、焰心）；干燥烧杯内壁出现水雾；澄清石灰水变浑浊。

结论：蜡烛燃烧生成水和二氧化碳。因此，石蜡中含有氢元素和碳元素。

✏️ 注意区分"现象"（观察到的）与"结论"（推理出的）

✓ 书面复盘任务

观看 VR 回放，对照标准实验报告评价自己的记录。找出书面记录中的3个问题，写出改进方案。

1. 问题：漏掉了"火焰分三层"的细节 → 改进：以后观察时先看整体再看局部
2. 问题：把"水雾"写成了"水珠"→ 改进：用词要准确，水雾≠水珠
3. 问题：没有写"澄清石灰水变浑浊"→ 改进：变浑浊是CO₂的关键证据，不能遗漏
