📜 名著一 · 《天工开物》—— 宋应星

▼
❓ 中国古代工匠如何用文字记录化学工艺？

目录焦点：第五卷"作咸"（盐的提取）、第七卷"陶埏"（陶瓷烧制中的化学变化）

"研精覃思，寻其大要" —— 宋应星强调深入思考后才动笔记录
🔍 导读方向
  • 学习古人观察和记录物质变化的方法
  • 对比古人的表达方式与现代化学表达方式的异同
  • 思考：从古代的"匠人记录"到现代的"科学论文"经历了怎样的演变？
📎 获取路径：中华书局版《天工开物译注》（图书馆借阅或在线文库）

🔬 名著二 · 《化学基础论》—— 拉瓦锡

▼
❓ 拉瓦锡如何用一本书推翻了统治化学界上百年的燃素说？

目录焦点：前言——关于化学命名法的思考；第一章"关于元素"的定义和分类

"我期望读者发现，这里所采用的命名法，能免除一切含糊的表达"
拉瓦锡认为"清晰的语言"是化学进步的基础
🔍 导读方向
  • 拉瓦锡的命名规则——化合物的名称如何反映其组成
  • 学习拉瓦锡"定义先行"的写作策略：先定义核心概念，再展开论述
  • 写作训练：模仿"定义 + 分类 + 举例"三段式写一段化学说明
📎 获取路径：商务印书馆《化学基础论》（李醒民译本），或 Project Gutenberg 英文版

🌍 名著三 · 《科学的旅程》—— 斯潘根贝格 & 莫泽

▼
❓ 科学思想如何在历史上演变？科学家如何用文字推动科学进步？

目录焦点：第四章"化学物质的变化——从炼金术到化学"；第六章"定量时代的到来"

科学进步不仅是新发现，更是"表达方式的革新"——从模糊的炼金术语言到精确的化学语言
🔍 导读方向
  • 关注科学写作风格的演变：炼金术的隐喻语言 → 近代化学实验记录语言 → 现代科技论文标准格式
  • 写作训练：选择一个化学概念（如元素、原子、分子），追踪其书面描述的演变过程
📎 获取路径：北京大学出版社《科学的旅程》（中文版）

✍️ 综合写作任务

1名言运用练习
从名言精选的 4 句中任选 2 句，用到同一篇化学短文中
例如：以拉瓦锡"Nothing is lost…"开头引入质量守恒，再引用戴安邦"实验的价值在于能转化为理论"强调化学理论传播的重要性。
2名著读书笔记
选择一本名著，写一篇 300 字的读书笔记，含"核心观点 + 原文引述 + 个人思考"
结构模板：
· 书名 + 作者 + 选择理由（50 字）
· 核心观点概述（100 字）
· 原文引述 + 你的理解（80 字）
· 与当前学习内容的连接（70 字）
3引述写作挑战
以"我读过的一本化学书"为题，写 500 字短文，至少引述一处名著内容、一句名言
写作框架：
开头（80 字）：引出书籍和阅读背景
主体（300 字）：核心内容介绍 + 引述 1 处名著原文 + 1 句名言
结尾（120 字）：这本书对你化学学习的启发和影响
💡 学习建议：名言与名著是化学书面表达的"活水源泉"。先熟悉名言（短小精悍易上手），再深入名著（系统阅读促思考），最后完成综合写作任务——这是一条从"输入"到"输出"的完整路径。
