基础实验操作指南

  • 酒精灯使用：检查灯芯→添加酒精（不超过2/3）→点燃→用灯帽盖灭
  • 试管操作：液体不超过1/3，加热时试管倾斜45°，管口不对人
  • 药品取用：不触、不尝、不直接闻，用后立即盖紧瓶塞
