基础层 生活案例
基础

为什么铁锅会生锈？

背景：家里的铁锅用完后没擦干，第二天出现红褐色斑点。

核心问题：铁生锈是化学变化吗？什么条件促进了生锈？

💡 可迁移启示：很多化学变化需要特定条件才能发生。

决策路径：观察现象 → 提出假设（水+氧气共同作用）→ 设计验证实验 → 得出结论

结果：铁在有水和氧气的环境中才会生锈，这是一个氧化还原反应。

进阶层 历史案例
进阶

拉瓦锡的质量守恒实验

背景：18世纪，燃素说主导化学界。

核心问题：物质燃烧后质量是增加还是减少？

💡 可迁移启示：封闭系统和开放系统的差异——实验设计的关键。

决策路径：设计密闭容器实验 → 精确称量反应前后物质质量 → 发现质量不变

结果：推翻了燃素说，建立了质量守恒定律，奠定了现代化学的基础。

挑战层 跨领域案例
挑战

雾霾中的化学

背景：城市雾霾天气的化学成分和形成机制。

核心问题：工厂排放、汽车尾气如何通过化学反应形成 PM2.5？

💡 可迁移启示：化学知识能帮助理解和解决环境问题。

决策路径：收集空气样本 → 分析化学成分 → 追踪反应途径 → 提出治理建议

结果：理解二次污染物的形成机制，为环保决策提供科学依据。
