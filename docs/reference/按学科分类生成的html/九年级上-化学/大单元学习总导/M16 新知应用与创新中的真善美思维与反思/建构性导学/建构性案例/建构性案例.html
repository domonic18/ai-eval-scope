🧩 建构性七阶案例

7个不同层级的建构任务，运用真善美思维创造属于你的化学成果。

🌱 案例一 · 个人理解型

★ 入门

真实建构情境：你已经学习了"走进化学世界"的核心内容，也了解了真善美思维框架。现在，请你用一张概念图把这两者整合起来——画出你心中的"真善美化学"是什么样子的。

建构任务：制作一张A4概念图，展示你对"化学中的真善美"的个性化理解

🔬 原理调用：三维目标法——以"真、善、美"为三个一级分支，从每个分支延伸出具体的化学实例

建构步骤：

  • 中心主题定位——确定"真善美化学"为核心
  • 三级分支展开——真（科学事实）、善（环保安全）、美（神奇之美）
  • 化学实例填充——每个分支至少连接2个实例
  • 视觉美化——布局清晰，颜色区分

成果表达：A4概念图（手绘或XMind）

✅ 自评量规： [ ] 是否体现了3个维度？ [ ] 每个维度是否至少连接2个化学实例？ [ ] 整体布局是否清晰美观？

🏠 案例二 · 生活应用型

★ 入门

真实建构情境：你家厨房里有小苏打（NaHCO₃）、纯碱（Na₂CO₃）和烧碱（NaOH）。奶奶分不清它们，有一次差点用烧碱洗菜。请你设计一张"厨房用碱安全指南"。

建构任务：为家人设计一份图文并茂的厨房用碱安全指南

🔬 原理调用：证据链法（查清每种碱的化学性质）+ 问题锚定法（聚焦"误用"风险）

建构步骤：

  • 调查家中厨房用品种类
  • 查清每种碱的化学性质与安全性
  • 设计对比表格（名称、性质、用途、安全等级）
  • 标注安全注意事项
  • 设计误用后的紧急处理指南

成果表达：A4安全指南海报

✅ 自评量规： [ ] 是否包含了3种碱的化学名称和俗名？ [ ] 是否标注了每种碱的安全等级？ [ ] 是否有误用后的紧急处理步骤？

🧪 案例三 · 校园问题型

★★ 进阶

真实建构情境：学校化学实验室的废液桶里经常被乱倒各类废液（酸、碱、有机溶剂混合），存在安全隐患。实验老师请你设计一套学生容易执行的分类投放方案。

建构任务：设计校园化学实验室废弃物分类投放指南

🔬 原理调用：问题锚定法（先分析乱倒的原因）+ 迭代优化法（先出方案，请同学试读后改进）

建构步骤：

  • 调查实验室常见废弃物种类
  • 设计分类标准
  • 设计视觉标识系统
  • 制定简易操作流程
  • 同学试用反馈并改进

成果表达：分类指南手册（A4 1页海报 + 1-2张贴纸图样）

✅ 自评量规： [ ] 是否包含了主要废弃物类别和处理方式？ [ ] 是否有清晰的操作流程图？ [ ] 标识贴纸是否一目了然？ [ ] 是否征求了同学的意见并做了改进？

🎨 案例四 · 学科表达型

★★ 进阶

真实建构情境：学校科技节需要化学展板，但文字太多没人看。请你设计一套"一句话 + 一个图"的科普卡片系列。

建构任务：设计3张科普卡片，分别展现化学中的"真·善·美"

🔬 原理调用：三维目标法（每个卡片对应一个维度）

建构步骤：

  • 选择3个化学知识点（分别对应真、善、美）
  • 为每个知识点设计"一句话抓住眼球"
  • 配示意图
  • 设计统一风格模板

成果表达：3张统一风格的科普卡片（手绘或电脑制作）

✅ 自评量规： [ ] 每张卡片是否不超过20个字？ [ ] 是否图文匹配？ [ ] 3张卡片是否风格统一？

🌍 案例五 · 跨学科整合型

★★ 进阶

真实建构情境：学校要举办"环保周"，化学组负责展示"化学如何帮助环境保护"。仅仅展示负面信息效果不好，需要从"真（科学原理）善（环保价值）美（创意表达）"三个角度来设计。

建构任务：设计一份结合化学、美术、信息技术的"绿色化学"宣传方案

🔬 原理调用：三维目标法（同时考虑科学准确、环保价值、视觉美感）

建构步骤：

  • 确定3个绿色化学核心信息
  • 为每个信息设计科学原理说明
  • 设计视觉呈现方案（展板草稿）
  • 设计互动环节（小实验或问答）
  • 整合成"展位方案书"

成果表达：展位方案书（含展板草稿+互动环节描述）

✅ 自评量规： [ ] 是否包含至少3个绿色化学的科学原理？ [ ] 是否有互动设计？ [ ] 方案是否同时考虑了真善美三个维度？

🏭 案例六 · 社会情境型

★★★ 挑战

真实建构情境：你选择一种日常生活中常见的化工产品（如塑料瓶、洗衣液、保鲜膜），它的"一生"——从石油开采到生产、使用、废弃——对环境和人类有什么影响？

建构任务：选择一个化工产品，完成其"从摇篮到坟墓"的全生命周期真善美评价

🔬 原理调用：证据链法（搜索全生命周期数据）+ 三维目标法（从真善美三个角度评价）

建构步骤：

  • 选择一个日常化工产品
  • 画出生命周期流程图（开采→生产→使用→废弃）
  • 在每个环节标注环境影响
  • 从真（科学）角度分析
  • 从善（伦理）角度评价
  • 从美角度提出改进建议

成果表达：A4生命周期评价海报

✅ 自评量规： [ ] 是否画出了完整的生命周期流程？ [ ] 是否标注了每个环节的环境影响？ [ ] 是否从真善美三个维度分别提出了见解？

🚀 案例七 · 综合创新型

★★★ 挑战

真实建构情境：如果你是一位化学工程师，公司给你一个任务——设计一款"真善美满分"的化学产品。你可以从解决一个现实问题出发（如"夏天冰淇淋融化太快"、"塑料垃圾太多"），自由发挥。

建构任务：设计一份体现真善美思维的化学创新产品方案

🔬 原理调用：综合运用全部4条建构性原理

建构步骤：

  • 确定要解决的现实问题
  • 设定真善美三维目标
  • 收集相关化学原理和数据
  • 设计产品概念（成分+形态+用法）
  • 设计生产工艺流程
  • 评估产品影响（真善美）
  • 输出方案说明书

成果表达："我的理想化学产品"方案说明书

✅ 自评量规： [ ] 问题是否真实且有意义？ [ ] 产品设计是否基于化学原理？ [ ] 是否同时考虑了真善美三个维度？ [ ] 方案是否具有可行性？ [ ] 表达是否清晰美观？
