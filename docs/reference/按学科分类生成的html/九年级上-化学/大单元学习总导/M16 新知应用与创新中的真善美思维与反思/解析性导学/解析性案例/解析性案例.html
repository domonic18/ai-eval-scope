🧪 案例一：生活情境 — "小苏打去污"的科学分析

📖 案例材料：你家厨房里，妈妈用小苏打（NaHCO₃）加白醋来清洗水槽和灶台。她说这是"纯天然去污"，比化学清洁剂安全。这个小苏打+白醋的组合真的有去污效果吗？

🎯 解析任务：从"真（科学原理）"的角度分析：小苏打+白醋为什么可能去污？这个原理是什么？

🔬 解析路径：

① 识别核心问题：小苏打+白醋的化学反应是什么？
② 确定分析对象：反应产物和去污效果
③ 拆解关键要素

NaHCO₃ + CH₃COOH → CO₂↑ + H₂O + CH₃COONa

④ 分析过程：产生CO₂气泡 → 物理松动污渍；醋酸钠溶液 → 助溶
⑤ 寻找证据：查阅教材中酸碱中和反应、CO₂产生的相关内容

证据链追踪法 — 找到"去污效果"背后的化学证据。

💭 参考思路：小苏打是弱碱性，白醋是酸性，中和产生CO₂气泡，气泡产生的物理冲击力和反应后溶液对油脂的溶解作用共同实现了去污效果。但"纯天然"是否就等于"安全"？用这个方法清洗铝锅会有什么后果？

💡 分析记录表：反应物 NaHCO₃+CH₃COOH → 生成物 CO₂+H₂O+CH₃COONa → 去污机理 物理松动+助溶 → 证据来源 教材酸碱中和 → 局限 铝锅腐蚀风险

⚠️ 案例二：校园情境 — 化学实验事故分析

📖 案例材料：某校化学实验室，一名学生在无教师在场的情况下，将未经标定的浓硫酸加入水中（而不是将水加入浓硫酸），导致液体飞溅灼伤手臂。

🎯 解析任务：从"真（安全原理）"和"善（安全责任）"两个维度分析这起事故。

🔬 解析路径：

① 识别核心问题：为什么"酸入水"而不是"水入酸"？
② 确定分析对象：浓硫酸稀释的放热原理
③ 拆解：浓硫酸遇水剧烈放热 → 水密度小于硫酸 → 沸腾 → 飞溅
④ 解释：正确的操作应该是"酸入水，沿壁流，不断搅拌"

三维审视法利益相关者分析法

💭 参考思路：可以从"为什么教材多次强调这个操作规范？""如果学校在实验室显眼位置贴上安全警示标志，事故是否可以避免？"等角度思考。
三维审视：真（放热原理）；善（实验室安全管理，谁该负责）；美（操作规范的设计智慧——这个规定背后是无数教训换来的）。

☣️ 案例三：家庭社区 — 洁厕灵+84消毒液的危险混用

📖 案例材料：网上流传着一段视频，有人将洁厕灵（含盐酸）和84消毒液（含次氯酸钠）混合，产生了黄绿色气体。这位博主说"这样可以增强消毒效果"。

🎯 解析任务：从"真（化学反应原理）"和"善（安全风险评估）"分析这个操作的危险性。

🔬 解析路径：

① 识别核心问题：混合后产生了什么气体？为什么有毒？
② 确定分析对象

NaClO + 2HCl → Cl₂↑ + NaCl + H₂O

③ 拆解：氯气（Cl₂）的性质——黄绿色、有刺激性、有毒
④ 结论：这个操作不仅不能增强消毒效果，还会产生有毒气体

证据链追踪法系统效应分析法

💭 参考思路：可以从"如果你在社交媒体上看到这种危险操作，你会怎么提醒他人？"的角度拓展思考。
系统效应：短期 → 立即中毒；长期 → 公众对消毒产品的误解。

⚡ 案例四：学科核心 — "电解水实验"的科学之美

📖 案例材料：在教材中，电解水实验（2H₂O → 通电 → 2H₂↑ + O₂↑）是化学入门的经典实验。这个实验看似简单，但它同时蕴含了科学之真和设计之美。

🎯 解析任务：从"美（科学逻辑之美、实验设计之美）"的维度分析电解水实验。

🔬 解析路径：

① 识别核心问题：这个实验美在哪里？
② 确定分析对象：实验装置设计、氢气与氧气体积比（2:1）
③ 拆解：每个实验环节的设计巧思
④ 解释：为什么用直流电？为什么加少量硫酸？为什么用排水法收集？

三维审视法（重点用"美"的维度）

💭 参考思路：电解水实验的美在于——2:1的体积比恰好揭示了水的组成（H₂O），这个简单比例背后是化学计量的精确之美。实验装置虽简单，但每一步设计都有科学依据：加稀硫酸增强导电性、排水法收集气体、用直流电确保定向分解。和拉瓦锡早期的水分解实验相比，现代装置更安全、更精确。

🎨 案例五：跨学科 — 颜料的化学成分分析

📖 案例材料：古人在壁画中使用了各种矿物颜料——红色的朱砂（HgS）、绿色的石绿（CuCO₃·Cu(OH)₂）、蓝色的青金石。这些颜料历经千年而不褪色。

🎯 解析任务：从"真（化学成分决定稳定性）"和"美（颜料色彩和化学结构的关系）"进行分析。

🔬 解析路径：

① 识别核心问题：为什么矿物颜料能千年不褪色？
② 确定分析对象：颜料的化学结构
③ 拆解：HgS的晶体结构稳定 → 不易和空气中的氧气反应
④ 解释：化学稳定性决定了颜料的耐久性

三维审视法系统效应分析法

💭 参考思路：矿物颜料千年不褪色的秘密在于——这些颜料多是金属硫化物或碳酸盐，化学性质非常稳定，常温下不易与氧气、水反应。现代合成颜料虽然色彩更鲜艳、成本更低，但部分合成颜料的耐光性、耐候性不如传统矿物颜料。
延伸思考：从化学结构的角度看，颜色和电子跃迁有什么关系？金属离子的d-d跃迁如何决定颜料颜色？

♻️ 案例六：社会议题 — 塑料回收的化学方案评价

📖 案例材料：目前塑料回收主要有三种方法：机械回收（粉碎再加工）、化学回收（裂解成单体再聚合）、焚烧发电。每种方法都有支持者和反对者。

🎯 解析任务：从真善美三个维度综合评价三种塑料回收方法。

🔬 解析路径：

① 识别核心问题：哪种回收方法最"好"？
② 确定分析对象：三种方法
③ 拆解：每种方法的科学原理、成本、环境影响
④ 诠释：真（原理是否可靠）、善（环境效益）、美（循环经济设计之美）

三维审视法利益相关者分析法系统效应分析法

💭 参考思路：三种方案各有利弊——
机械回收：物理变化，成本低但质量逐次下降（降级回收）；
化学回收：化学变化，质量高但成本高、能耗大；
焚烧发电：彻底解决垃圾问题，但产生CO₂，且公众反对"烧垃圾"。
没有完美的方案，只有"当下最优"的选择——最好的方案是减少使用。
