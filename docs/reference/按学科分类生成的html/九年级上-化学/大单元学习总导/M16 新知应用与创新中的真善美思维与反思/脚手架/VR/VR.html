🔬 水的电解

进入VR场景，观察水分子在电场作用下如何分解：

  • H₂O 分子 键断裂 → 氢原子与氧原子分离
  • 氢原子两两结合 → H₂ 气体（在阴极产生）
  • 氧原子两两结合 → O₂ 气体（在阳极产生）
  • 关注：化学键断裂和形成的微观过程

🧪 中和反应的微观过程

  • H⁺（来自酸）和 OH⁻（来自碱）在溶液中相遇
  • 结合成 H₂O 分子——这就是"中和"的本质
  • 剩下的阳离子和阴离子结合成 盐
  • 例：H⁺ + Cl⁻ + Na⁺ + OH⁻ → H₂O + NaCl
