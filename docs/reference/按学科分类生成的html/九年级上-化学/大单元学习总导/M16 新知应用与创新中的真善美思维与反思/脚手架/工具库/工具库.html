📊 真善美三维评估矩阵 基础

用表格形式系统评估化学应用在真善美三个维度上的表现

使用步骤：

  1. 在表格第一列列出你关注的每个方面
  2. 在后续三列分别从真、善、美三个维度评分（1-5分）
  3. 计算总分并分析得分原因

示例：

评估项真（科学）善（伦理）美（审美）备注
原理可靠性5—3原理清晰但表现不佳
环境影响—4—成本偏高
设计美感——4流程简洁

适用：分析任何化学产品、方案或创新案例

🔲 SWOT分析 进阶

从优势、劣势、机会、威胁四角度全面分析一个化学方案

使用步骤：

  1. 在纸上画一个田字格
  2. 左上：优势（S）——方案的科学优势在哪里？
  3. 右上：劣势（W）——方案有哪些科学/伦理/审美缺陷？
  4. 左下：机会（O）——这个方案在什么条件下会更好？
  5. 右下：威胁（T）——这个方案面临什么外部挑战？
优势 (S)劣势 (W)
科学性、创新性成本、副作用
机会 (O)威胁 (T)
政策支持、技术进步竞争、监管

适用：设计方案前的全局分析

🤖 AI助手使用指南 全层级

用AI工具辅助检索信息、提供灵感、帮助整理思路

使用步骤：

  1. 清晰描述问题（如："请列举3个绿色化学的实际应用案例"）
  2. 追问细节（如："这些案例的成本分别大概是多少？"）
  3. 交叉验证（对AI的回答保持批判性）

示例指令：

  • "帮我找2-3个近年来的绿色化学创新案例，并说明它们的真善美表现"
  • "用初中生能懂的语言解释什么是'酯键水解'"

⚠️ AI的回答可能不准确，请结合教材和老师讲解进行交叉验证

📋 对比分析表 进阶

用对比表清晰展示不同方案的差异

使用步骤：

  1. 确定对比对象（2-3个方案）
  2. 确定对比维度（真善美各维度）
  3. 逐项填写并标注差异
  4. 写出对比结论

适用：比较不同化学方案的优劣
