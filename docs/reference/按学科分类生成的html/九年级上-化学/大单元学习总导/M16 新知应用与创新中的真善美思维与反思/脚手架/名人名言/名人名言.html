🧪

奠基人物：侯德榜 1890—1974

▶
身份中国化学工业先驱，侯氏制碱法发明人 关键问题如何摆脱西方对纯碱生产技术的垄断？ 思维突破点突破索尔维法专利封锁，将制碱和制氨两大工业体系融合，创造"联合制碱法" 失败/争议早期实验多次失败，工厂爆炸风险，在资金和技术双重困境中坚持了5年 当下启示科学创新需要智慧（真）与对国家和人民的责任（善）
💭 如果你是他

如果你是1930年代的中国化学工程师，面对西方技术封锁和国内工业空白，你会选择从哪里突破？是利用现成的西方技术路线，还是走一条全新的路？

📖 推荐阅读：《侯德榜传》· 纪录片《大国工匠》侯德榜专题

🚀

跨界人物：钱学森 1911—2009

▶
身份系统工程科学家、"中国航天之父" 关键问题如何从全局视角整合不同学科的知识解决复杂问题？ 思维突破点提出"系统工程"思想——任何创新都不能只看单一维度，要从整体系统出发 与M16关联钱学森的系统工程思想帮助你在评价化学创新时跳出二元判断，从系统效应角度全面分析
💭 如果你是他

钱学森说"不要人云亦云"——当所有人都在追捧某项化学新技术时，你会如何运用系统思维来保持独立思考？

当代人物：屠呦呦 1930—

▶
身份中国药学家，2015年诺贝尔生理学或医学奖得主 关键问题如何从传统中医药中发现抗疟新药？ 思维突破点从东晋葛洪《肘后备急方》记载中获得灵感，改用乙醚低温提取，保留青蒿素活性 失败/争议前期用水煎法失败，实验重复190次后才成功 当下启示科学发现需要"坚持之真"（190次失败不放弃）+ "济世之善"（拯救数百万生命）

屠呦呦从古籍中寻找科学的灵感——如果你穿越回古代，你想用现代化学知识解决古人的哪个难题？
