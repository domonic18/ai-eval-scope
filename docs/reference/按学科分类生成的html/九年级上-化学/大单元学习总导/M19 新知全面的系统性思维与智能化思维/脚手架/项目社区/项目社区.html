短周期微项目（1—3天）

🍳 微项目A：厨房化学发现日志

⏱ 3天 🎯 观察厨房里的化学现象

项目目标：用3天时间，每天观察厨房里的1个化学现象或化学用品，用"现象—原理—用途"框架记录。

阶段任务：

  • 第1天：选择一个厨房现象（如：为什么泡打粉能让蛋糕蓬松？）
  • 第2天：查找资料，写出化学原理（用化学式或文字说明）
  • 第3天：整理为"厨房化学卡片"（卡片格式：现象描述 + 化学原理解释 + 发现心得）

成果形式：

3张"厨房化学卡片"，可以画图+文字，拍照上传

📋 评价标准：每张卡片至少包含1个明确的化学原理，原理解释准确

🏷 微项目B：我的"化学物品安全标签"设计

⏱ 3天 🎯 为家里的化学品设计安全标签

项目目标：为家里的3种化学品（如84消毒液、洁厕灵、酒精）设计新的"安全标签"。

阶段任务：

  • 第1天：查看现有标签，列出关键信息（成分、危险标识、使用方法、急救措施）
  • 第2天：根据自己理解，设计一个更清晰、更易懂的标签样式
  • 第3天：制作成品（手绘或电脑排版），贴在对应瓶子上

成果形式：

3张实物标签（照片）或电子版设计图

📋 评价标准：标签包含主要安全信息，设计美观，适合家庭成员阅读

中周期专题项目（1—2周）

📑 专题项目：家庭常用物质"系统性研究报告"

⏱ 约10天 🎯 运用系统性思维完成对比研究

项目目标：选择3种家庭常用化学物质（如小苏打、白醋、食盐），运用系统性思维完成一份对比研究报告。

阶段任务：

  • 阶段1（2天）：建立研究框架——确定分析维度（物理性质/化学性质/主要用途/安全注意事项/环境影响）
  • 阶段2（3天）：收集信息——查阅教材、资料、产品包装、网络
  • 阶段3（2天）：对比分析——用表格对比3种物质在各维度上的异同
  • 阶段4（2天）：绘制"物质关系系统图"——展示3种物质之间可能发生的化学反应和相互作用
  • 阶段5（1天）：撰写报告结论——从"系统性"角度总结规律

成果形式：

一份完整的系统性研究报告（文字+图表，不少于3页A4纸）

📋 评价标准：① 分析维度不少于4个 ② 有跨物质对比 ③ 有系统关系图 ④ 结论有独立的思考发现

📎 教材第1章 · 思维导图模板 · VR工具中的分子结构查看

长周期综合项目（1个月以上）

🌍 "未来化学与我的生活"主题探究

⏱ 1个月以上 🎯 深度探究 + 可展示成果

项目目标：选择一个你感兴趣的化学相关主题，进行为期至少1个月的深度探究，最终产出一件可展示的成果。

可选主题方向（三选一）：

🌱 绿色化学在行动

研究一种环保化学技术（如：生物降解塑料、CO₂捕集技术、天然植物染料），分析它的原理、现状和推广难点

🍶 厨房化学革命

深入研究一种厨房化学品的"前世今生"（如从发现小苏打→工业化生产→未来替代方案），写一份"物质传记"

🏠 我家的一日化学全景

记录你从起床到睡觉的一整天中，所有接触到的化学品/化学现象，建立"人与化学物质互动的系统模型"

👆 点击选择一个主题方向

阶段任务：

  • 第1周：选题并制定研究计划（确定研究问题、研究方法、预期成果）
  • 第2周：搜集资料，完成信息整理（查阅不少于3种来源）
  • 第3周：深入分析，形成核心观点（运用解析性原理和建构性原理）
  • 第4周：成果制作和展示准备

成果形式（四选一）：

📄 图文报告（PDF ≥5页） 🎬 短视频（3-5分钟） 🖼 海报/手抄报（A2以上） 📊 PPT演讲稿（≥10页）
📋 评价标准：① 主题明确，有探究价值 ② 有化学原理解释（不少于3个核心概念） ③ 有系统性分析（不仅仅是知识罗列） ④ 有个人观点和反思 ⑤ 呈现美观、表达清晰
