⬜观察氯化钠晶体结构：从宏观的食盐晶体"钻进去"，观察离子在三维空间中的周期性排列

🔄 替代方案（无VR设备时）

方案一：3D模型网站

打开 molview.org，在浏览器中旋转缩放任意分子模型（支持鼠标拖拽操作）

方案二：3D眼镜版

搜索"水分子3D结构图"查看立体图像

方案三：纸模型

用黏土 + 牙签制作水分子、二氧化碳分子的物理模型
