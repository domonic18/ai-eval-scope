📊 分层题 — 三选一

A层·基础巩固 列出水的3种用途并说明分别利用了水的什么性质 ▼

示例：水可用于灭火（利用了水能吸热降温、隔绝氧气的性质）；水可用作溶剂（利用了水能溶解许多物质的性质）；水可用于发电（利用了水的重力势能转化为电能）。

B层·应用分析 用表格比较白醋、小苏打、食盐的酸碱性、溶解性和日常用途 ▼

提示：白醋—酸性、可溶于水、调味；小苏打—弱碱性、可溶于水、烘焙膨松；食盐—中性、易溶于水、调味防腐。
