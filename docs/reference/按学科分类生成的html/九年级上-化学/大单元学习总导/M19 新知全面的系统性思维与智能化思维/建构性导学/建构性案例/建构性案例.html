案例一：我身边的"化学宝藏"地图

📌 真实建构情境：你的书桌上有一瓶水、一支铅笔、一块橡皮、一个玻璃杯。这4样看似普通的东西，其实都是"化学物质"。它们是由什么材料制成的？这些材料又来自哪里？

建构任务：围绕书桌上的4样物品，建构一幅"化学物质来源与用途"关联图。

原理调用：问题驱动法 + 证据锚定法

建构步骤：

  1. 观察4样物品，记录外观、手感等物理特征
  2. 猜测它们的主要成分（水瓶→塑料/高聚物，铅笔→石墨+木材，橡皮→橡胶，玻璃杯→二氧化硅）
  3. 查阅教材或资料，验证你的猜测
  4. 画一幅"物品—成分—来源"关联图

脚手架：用思维导图形式，中心写"我的书桌"，向外辐射4个物品分支

📄 成果表达：我的"化学宝藏"地图 — A4纸，中心图+4个物品分支，每个分支标注：物品名称→成分→来源→用途。用不同颜色区分不同维度，标注查证过的资料来源。

案例二：厨房里的"酸碱大侦探"

📌 真实建构情境：厨房里有白醋（酸性）、小苏打（弱碱性）、纯碱（碱性）、柠檬汁（酸性）。如果不说名字，只给你一杯"神秘液体"，你能用厨房里的材料判断它是酸性还是碱性吗？你能建构一套"家庭酸碱检测方案"吗？

建构任务：设计一份"家庭酸碱检测指南"，包含检测方法、操作步骤和结果判断标准。

原理调用：系统建模法

建构步骤：

  1. 查阅教材，了解酸碱指示剂的原理
  2. 寻找家庭替代品（紫甘蓝汁、红茶、花青素等可作为天然指示剂）
  3. 设计检测流程（取样→加指示剂→观察颜色变化→判断酸碱性）
  4. 用流程图形式呈现完整的检测方案

脚手架：流程图模板——"开始→取样品→加指示剂→观察颜色→对照色卡→得出结论"

📄 成果表达：家庭酸碱检测指南（流程图+色卡）— 包含检测原理、材料清单、操作步骤、颜色对照表、安全提示。流程图用箭头串联，色卡用彩笔标记预期颜色变化。

案例三：校园水质"体检"报告

📌 真实建构情境：学校最近发布了直饮水检测报告，显示pH值为6.8，TDS为120mg/L。这个数据是什么意思？学校的饮用水水质到底怎么样？你作为班级"水质观察员"，需要建构一份校园用水水质分析简报。

建构任务：建构一份"校园饮用水水质简易分析简报"，利用教材中的物质性质知识，解释关键水质指标的含义。

原理调用：证据锚定法

建构步骤：

  1. 收集学校水质检测数据（找后勤老师或查看公示牌）
  2. 查阅教材和资料，了解pH值、TDS等指标的含义
  3. 对照国家饮用水标准，判断数据是否在安全范围
  4. 用通俗语言写成一句话"水质结论"，并解释为什么

脚手架：教材附录的国家饮用水标准；"数据—标准—判断—结论"四栏分析表

📄 成果表达：校园饮用水水质简报 — 数据表格+指标解释+结论+改进建议。用"红黄绿"三色标注各指标是否达标。

案例四：构建"化学变化"分类系统

📌 真实建构情境：铁钉生锈→化学变化；蜡烛燃烧→化学变化；小苏打加醋冒泡→化学变化……生活中到处是化学变化，但它们的"变化方式"一样吗？一个燃烧、一个生锈、一个冒泡，你该怎么系统地区分和描述它们？

建构任务：建构一套"化学变化分类系统框架"，能够把至少6种化学变化按一定标准分类。

原理调用：系统建模法 + 迭代优化法

建构步骤：

  1. 列出6种以上你见过的化学变化
  2. 思考分类标准（按反应类型？按能量变化？按参与物质？）
  3. 画出分类树状图（第一级、第二级分类）
  4. 把每种变化填入对应类别
  5. 检查是否有变化无法归入——迭代补充

脚手架：分类树模板——"化学变化"根节点，分2—3级子节点

📄 成果表达：我的化学变化分类树 — 树状图/层级图，根→一级→二级→具体实例。用不同颜色区分不同分类层级，每个实例旁边标注简短变化描述。

案例五："跨学科"视角——水从源头到水杯

📌 真实建构情境：一杯自来水从江河湖泊到你的水杯，经历了取水→混凝→沉淀→过滤→消毒→输送的完整过程。这不仅是物理过程，更涉及大量化学原理（混凝剂的作用、消毒剂的反应、pH调节……）。你能用化学+物理+生物的综合视角，建构"一滴水的旅程"系统模型吗？

建构任务：建构"一滴水的旅程"跨学科系统模型，涵盖从水源到水杯的全过程，标注每个环节涉及的化学变化和原理。

原理调用：系统建模法

建构步骤：

  1. 查阅资料，了解自来水处理的标准流程（6—8个环节）
  2. 标注每个环节的化学变化（如：加氯消毒→氧化还原反应）
  3. 用流程图+文字说明的方式呈现全链条
  4. 标注每个环节涉及的学科（化学/物理/生物）

脚手架：用"环节—化学原理—学科标签"三列表格辅助整理

📄 成果表达：一滴水的旅程——跨学科系统模型 — 横轴为"从水源到水杯"的环节链条，纵轴标注每个环节的化学原理和涉及的学科。不同颜色代表不同学科，在每个环节节点弹出化学原理说明。

案例六：碳中和时代的"化学方案"

📌 真实建构情境：国家提出2060年前实现"碳中和"。作为化学学习者，你知道二氧化碳的排放和吸收都涉及化学反应吗？你能从化学角度分析：哪些反应"制造"了CO₂？哪些反应可以"吸收"CO₂？你能不能建构一个"碳循环化学方案"？

建构任务：建构一份"碳循环化学方案"，列出至少3种产生CO₂的化学反应和3种吸收CO₂的化学反应，并分析哪种吸收方案最可行。

原理调用：问题驱动法 + 证据锚定法

建构步骤：

  1. 查阅资料，列出产生CO₂的常见化学反应（燃烧、呼吸、分解等）
  2. 列出吸收CO₂的化学反应（光合作用、CO₂与碱反应、矿物碳化等）
  3. 用化学方程式表达每个反应
  4. 分析各吸收方案的优缺点（成本、效率、副产物）
  5. 写出你的"推荐方案"并说明理由

脚手架：用"反应名称—化学方程式—碳排放/吸收—可行性评价"四栏表格

📄 成果表达：碳中和时代的"碳循环化学方案" — 产生端和吸收端双栏对比表+化学方程式+可行性评估。产生端用红色箭头，吸收端用绿色箭头，形成循环图。

案例七：综合创新——设计"家庭化学实验安全手册"

📌 真实建构情境：你有一个8岁的表妹，她喜欢在家做各种"科学小实验"。但她在网上看到的实验不一定安全——有的用了不该混合的化学品，有的操作不当有危险。你决定运用本学期学到的化学知识和系统性思维，为她设计一份安全、易懂的"家庭化学实验安全手册"。

建构任务：建构一本"家庭化学实验安全手册"，包含安全原则、常见危险物质清单、安全操作指南和简易急救措施。

原理调用：问题驱动法 + 证据锚定法 + 系统建模法 + 迭代优化法 + 外化表达法（全部5条）

建构步骤：

  1. 查阅教材和资料，列出家庭常用化学品的危险组合
  2. 归纳3—5条核心安全原则
  3. 设计手册结构（封面→安全原则→危险清单→安全操作→急救措施）
  4. 用通俗语言和插图完成初稿
  5. 请表妹试读，根据反馈修改——迭代优化
  6. 形成定稿，并画一个"安全标识"封面

脚手架：使用"安全原则—化学原理—案例说明—操作建议"四栏格式

📄 成果表达：家庭化学实验安全手册（送给表妹的礼物）— 小册子形式（A4纸对折），4—6页，包括封面页。文字+手绘简笔画，红色标注危险、绿色标注安全操作。
