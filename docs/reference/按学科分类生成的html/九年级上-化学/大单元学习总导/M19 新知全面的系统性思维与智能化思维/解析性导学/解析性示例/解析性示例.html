1

第一步：先看见什么

看见：木柴被点燃，发光发热，最终变成灰烬。

为什么这样想：分析任何一个现象的第一步不是"解释"，而是客观描述——不加解释地去描述你看到了什么。这能帮你避免"还没看清楚就想当然地解释"的误区。

2

第二步：判断问题属于哪一类

判断：这是一个"化学变化"类问题。

依据：产生了新物质（灰烬、烟）——这是化学变化的本质特征。

为什么这样想：判断"这是什么类型的问题"是最关键的一步。化学问题可以分为：化学变化/物理变化、物质性质类、物质制备类……不同类型的分析路径不同。

3

第三步：调用哪些概念、原理或方法

调用：

  1. 化学变化与物理变化的区分标准（是否产生新物质）
  2. 燃烧的三要素（可燃物、助燃物、温度达到着火点）
  3. 质量守恒思想（燃烧前后物质总质量不变）
  4. "条件→过程→结果"分析框架

为什么这样想：系统性思维的核心就是不看一个点，而看一个"知识网络"——燃烧不是孤立概念，它与"化学变化""能量变化""质量守恒"等多个概念有联系。

4

第四步：如何进行推理或分析

推理过程：

  1. 分解要素：木柴（可燃物）+ 氧气（助燃物）+ 点燃（达到着火点）
  2. 分析关系：三者缺一不可→如果隔绝氧气（盖上沙子），火会熄灭
  3. 推理结果：燃烧不是"木柴自己烧完了"，而是"木柴和氧气发生了化学反应"
  4. 拓展思考："木柴变灰"看似质量减少，如果把烟和水蒸气都收集起来，总质量应该等于"木柴质量+消耗的氧气质量"
  5. 质疑与验证：为什么"看起来"灰烬比木柴轻？因为大部分变成了气体散逸到空气中

关键思维链：要素→关系→结果→质疑→验证，而不是一步跳到结论。

5

第五步：怎样形成结论

结论：木柴燃烧是一个化学变化，本质是木柴中的碳、氢等元素与空气中的氧气发生反应，生成了二氧化碳、水蒸气和其他氧化物，同时释放出热能和光能。看似"质量减少"是因为产物以气体形式散逸了，实际上遵循质量守恒定律。

现象 → 本质 → 条件 → 质量关系

木柴发光发热生成灰烬 → 化学变化（C+H+O→CO₂+H₂O+能量）→ 可燃物+氧气+着火点 → 反应物总质量 = 生成物总质量

💡 方法提炼：七步解析法

以后遇到类似问题，可以这样做：

  1. 描述现象：先不加解释地描述你看到了什么
  2. 判断类型：这是哪一类问题？（化学变化/物理性质/……）
  3. 拆解要素：涉及哪些要素？（木柴、氧气、点燃）
  4. 分析关系：要素之间的关系是什么？（三者缺一不可）
  5. 调用原理：涉及哪些化学概念？（燃烧三要素、质量守恒）
  6. 推理结论：综合以上分析形成结论
  7. 质疑验证：有没有反例？结论是否站得住脚？

🔄 这个方法可以迁移到分析任何化学现象！

⚠️ 易错提醒

❌ 错误：把"看起来怎样"当作"本质怎样"——木柴燃烧后灰烬变轻，误以为质量不守恒
✅ 正确：气体产物散逸了，收集起来总质量不变
❌ 错误：忽略了条件的影响——以为燃烧都产生一样的产物
✅ 正确：氧气充足→完全燃烧（CO₂），氧气不足→不完全燃烧（CO 和碳颗粒——黑烟）
❌ 错误：孤立地看一个现象——只看"木柴燃烧"，不看它与"能量""质量"的关系
✅ 正确：把孤立现象放到"知识网络"中分析
