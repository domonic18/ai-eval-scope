🕯️ 从"蜡烛燃烧"到"化学变化的概念建构"

化学 · 九年级上 · 建构性示例

🎯 真实建构情境

你观察过蜡烛燃烧吗？蜡熔化、灯芯变黑、火焰跳动、玻璃杯壁出现水珠……这些现象看起来很简单，但你真的能回答"蜡烛燃烧时发生了什么"吗？

这个问题看似简单，却蕴含了化学最核心的概念——物理变化与化学变化的区别。化学家们花了几个世纪才真正理解这一点，而你要在接下来的30分钟内，重新走一遍这个发现之旅。

🥅 建构目标

建构自己对"化学变化本质"的理解模型，能够清晰区分物理变化和化学变化，并能用自己的方式表达出来。

📋 建构任务说明

成果名称：化学变化本质理解模型

用途：展示你对化学变化核心特征的理解

对象：你的同桌或学习伙伴（他们需要通过你的模型看懂什么是化学变化）

形式：手绘图 + 文字说明（A4纸大小）

最低标准 至少包含1个具体案例 + 3个关键特征 + 1个对比说明

🚶 建构过程六步走
观察 分类 概念 证据 模型 表达

第1步：观察与记录

做什么：点燃一支蜡烛，仔细观察并记录所有你能看到的现象。
怎么做：从固体到液体、从液体到气体、颜色的变化、新物质的出现。
生成什么：现象观察记录表
示范：蜡→熔化（固体变液体）；烛芯→变黑；火焰→发热发光；杯壁→出现水珠

第2步：提问与分类

做什么：把这些现象分成两类——"只是变了样子"和"变成了新东西"。
怎么做：用"能不能恢复原状"来判断。
生成什么：分类表
分类示范：
🔄 物理变化（变了样子但还能变回来）：蜡熔化、蜡液凝固
🔥 化学变化（变成了新东西，变不回去了）：烛芯变黑、杯壁水珠、发光发热

第3步：建立概念

做什么：给你的两类现象起名字——物理变化和化学变化。
怎么做：查教材找到准确定义，但用自己的话写出来。
生成什么：个人版概念定义
我的定义：
物理变化——物质的外形或状态改变了，但物质本身没变。比如蜡烛的蜡熔化了，但它还是蜡。
化学变化——物质变成了全新的物质。比如烛芯燃烧后变成了灰（碳），杯壁上的水珠不是蜡烛里的蜡变的，而是蜡烛中的氢和空气中的氧结合生成的。

第4步：寻找证据

做什么：为"化学变化产生了新物质"找证据。
怎么做：杯壁的水珠→水从哪里来？烛芯变黑→黑色物质是什么？
生成什么：证据链示意图
证据链：
① 杯壁水珠 → 蜡烛燃烧生成水（氢气+氧气→水）✅
② 烛芯变黑 → 生成碳（不完全燃烧产生碳颗粒）✅
③ 火焰发热发光 → 化学能转化为热能和光能 ✅
④ 蜡越来越少 → 蜡（碳氢化合物）与氧气反应，变成了CO₂和H₂O

第5步：建构模型

做什么：把你的发现整合成一张完整的理解图。
怎么做：用箭头、关键词、简笔画表达"蜡烛燃烧时发生了什么"。
生成什么：理解模型手绘图
模型框架建议：
🔹 画面中央画一支燃烧的蜡烛
🔹 左边用蓝色箭头标注物理变化（蜡熔化、蜡液流淌）
🔹 右边用红色箭头标注化学变化（碳生成、水生成、发光发热）
🔹 底部标注：蜡烛（碳氢化合物）+ 氧气 → 二氧化碳 + 水

第6步：表达与反思

做什么：用你的模型向别人讲解你的发现。
怎么做：指着图中每一个部分说出你的推理过程。
生成什么：口头/书面表达稿
口头表达示范：
"我观察到蜡烛燃烧时蜡会熔化，这是物理变化，因为熔化的蜡冷却后还能变回固体。同时我注意到杯壁上出现了水珠、烛芯变黑了，这是化学变化——因为水是氢原子和氧原子重新组合生成的，碳是碳原子从蜡烛中释放出来的。所以，化学变化的本质是原子的重新组合，生成新物质。"
🧰 支架工具推荐

建构模板

现象→分类→概念→证据→模型→表达

关键句式

"我观察到……这说明……因为……"
"如果……那么……"

检查清单

是否包含了现象观察？是否做了分类？
是否找到了证据？是否有可视化表达？

推荐格式

A4纸横放，中间画蜡烛，周围用箭头和文字标注各步骤

✨ 成果优化建议
  • 检查你的模型有没有遗漏关键证据——水珠、碳黑、质量变化
  • 尝试用不同颜色的笔区分"物理变化"和"化学变化"（如蓝红双色）
  • 让别人看你的模型，如果他们能看懂蜡烛燃烧时发生了什么，那就成功了！
