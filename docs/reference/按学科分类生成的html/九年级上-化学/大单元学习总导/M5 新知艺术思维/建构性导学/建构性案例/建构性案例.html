🧴 案例二：家庭清洁剂的"成分侦探"

证据驱动表达外显
情境

厨房台面有水垢，白醋能去除水垢，但洗洁精不行。

建构任务

建构一个"为什么不同清洁剂对不同污渍效果不同"的解释方案。

建构步骤
  • 收集家中清洁剂成分表
  • 设计简单对比实验（白醋 vs 洗洁精 vs 水）
  • 记录现象
  • 寻找"酸碱性"概念
  • 建构解释方案
📌 成果：家庭清洁剂档案表 + 酸碱性作用原理说明图
💡 关键洞察：白醋显酸性，与水垢（碳酸钙）反应生成可溶物；洗洁精显中性或碱性，对酸性水垢无效。这就是"相似相溶"之外的另一条化学原理——酸碱中和。

⚗️ 案例三：实验室废液处理方案

证据驱动迭代优化
情境

化学实验室每天产生含酸、碱的废液，直接排放会污染环境。

建构任务

设计一套安全、可行的实验室废液中和处理方案。

建构步骤
  • 调查废液来源与成分
  • 查找酸碱中和原理
  • 设计处理流程

  • 优化方案
📌 成果：废液处理流程图 + 操作注意事项清单
⚖️ 安全提示：酸碱中和会放热，操作时应缓慢添加、不断搅拌、佩戴护目镜。先用pH试纸测试废液酸碱性，再确定中和剂的用量。

📜 案例四：化学史"发现时刻"情景再现

情境锚定表达外显
情境

拉瓦锡用定量实验推翻了燃素说，建立了氧化学说。

建构任务

用时间轴 + 实验示意图，建构拉瓦锡的发现过程。

建构步骤
  • 阅读化学史资料
  • 提取关键实验（曲颈甑加热汞实验）
  • 画出实验装置
  • 标注证据链（气体体积变化、质量守恒）
  • 梳理推理逻辑
📌 成果：拉瓦锡发现过程可视化时间轴
🔬 拉瓦锡的贡献：他第一次用定量方法研究化学反应——称量反应前后物质的总质量，发现质量守恒，证明燃烧不是"燃素逃逸"，而是物质与氧气结合。

🌱 案例五：植物生长需要的"化学营养"

情境锚定原理迁移
情境

教室里的绿萝叶片发黄，园丁说它缺"氮、磷、钾"。

建构任务

结合化学和生物知识，建构"植物如何从土壤中获取化学元素"的整合模型。

建构步骤
  • 分析缺素症状（叶片发黄→缺氮）
  • 查找三种元素的化学性质
  • 绘制"土壤→根→叶"元素迁移图
  • 提出施肥建议
📌 成果：植物营养元素迁移示意图 + 家庭施肥指南
🌿 跨学科视角：氮（N）是蛋白质和叶绿素的组分——缺氮叶片失绿；磷（P）是ATP能量货币的组分——促根系和花果；钾（K）调节渗透压——增强抗病性。化学元素 → 生物功能 → 宏观症状，三者环环相扣。

🌍 案例六：2030年"碳中和"中的化学贡献

证据驱动迭代优化
情境

中国承诺2030年实现碳达峰，化学家正在研究将CO₂转化为燃料。

建构任务

建构一个"化学如何帮助实现碳中和"的未来方案模型。

建构步骤
  • 了解碳循环现状
  • 研究CO₂转化技术原理（电催化、光催化）
  • 画出"碳捕获—转化—利用"链条
  • 评估可行性
  • 提出你的创意
📌 成果：碳中和化学解决方案概念图
🔋 前沿链接：电催化CO₂还原可以将二氧化碳转化为甲酸、甲醇、乙烯等燃料和化工原料。如果这个过程用太阳能发电驱动，就实现了"人工光合作用"——真正的负碳技术。

💧 案例七：社区环境监测与改善提案

情境锚定证据驱动表达外显迭代优化
情境

你所在社区的河水颜色异常，居民怀疑附近工厂排放污水。

建构任务

设计一套完整的社区水环境监测方案 + 改善建议。

建构步骤
  • 确定监测指标（pH、溶解氧、浊度、氨氮）
  • 设计采样方案（上游、排放口、下游三处取样）
  • 学习简易水质检测方法（pH试纸、比色法）
  • 分析结果
  • 撰写改善提案
📌 成果：水质检测报告 + 社区水环境改善提案
🏘️ 综合应用：这是本模块的终极挑战——你需要综合运用全部5条建构性原理。从"河水异常"这个真实情境出发，用证据（检测数据）驱动分析，用图表外显你的发现，多次迭代优化提案，最后将整个方法迁移到其他环境问题中。
