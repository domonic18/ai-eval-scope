🧠 化学家如何思考

三位改变了科学史的人物 · 他们的思维突破与对你的启示

⚗️
拉瓦锡 · Lavoisier
1743 — 1794
关键问题

物质燃烧时真的会「失去」某种元素（燃素）吗？

思维突破点

用天平做定量实验——称量反应前后所有物质的重量，发现总重量不变。

失败 / 争议点

被错误理论（燃素说）主导多年，直到他大胆质疑「大家都这么说」的结论。

对当下的启示
不要相信权威，要相信证据。 拉瓦锡之所以能推翻燃素说，不是因为他更聪明，而是因为他做了别人没做的事——称重。

如果你生活在18世纪，所有人都相信燃素说，你会从哪里开始质疑它？你会设计什么实验来检验这个理论？

💻
埃达·洛芙莱斯 · Ada Lovelace
1815 — 1852
关键问题

机器只能算算术，还是能创造知识？

思维突破点

她看到分析机不仅能计算数字，还能处理符号——这意味着「机器可以创造音乐和科学」。

失败 / 争议点

她的远见在当世被认为「不切实际」，直到一百年后才被验证。

对当下的启示
跨界的价值。 埃达用诗人和数学家的双重眼光看问题，发现了纯数学家看不到的可能性。

如果你是埃达，你会把分析机用在化学的哪个方面？如何用「计算」来帮助化学家发现新知识？

🌿
屠呦呦
1930 —
关键问题

如何在数千种传统草药中找到治疗疟疾的有效成分？

思维突破点

从东晋葛洪《肘后备急方》获得灵感——「青蒿一握，以水二升渍，绞取汁，尽服之」。她意识到高温可能破坏有效成分，改用低温乙醚提取。

失败 / 争议点

经历了190次失败才在第191次实验中成功。

对当下的启示
从失败中学习。 每一次失败都在排除一个不可能的方向——190次失败不是190次错误，而是190次「找到了不对的路」。

如果你要在2000种植物中找到一种有效成分，你会怎么设计「排除路径」？如何保证不遗漏真正有效的植物？
