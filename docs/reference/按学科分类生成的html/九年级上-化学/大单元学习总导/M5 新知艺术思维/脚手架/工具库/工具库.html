📋 KWL 表格
明确你已经知道什么、想知道什么、学到了什么
  1. K（已知）— 学习前，写下你已经知道的
  2. W（想知）— 写下你想知道的
  3. L（学到）— 学习后，写下你学到的
适用场景：模块开始前和完成后各填一次，对比你的成长

📌 K: 我知道化学反应会产生新物质
📌 W: 我想知道如何判断是否发生了化学变化
📌 L: 我学到了变化前后质量不变
