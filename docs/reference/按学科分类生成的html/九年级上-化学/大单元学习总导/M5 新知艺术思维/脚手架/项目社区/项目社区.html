动手做化学 — 短周期 · 中周期 · 长周期 三级项目体系

📌 项目1：家庭化学侦探

⏱ 短周期 · 1—3天
🎯 项目目标：找出家中5种日常用品，判断它们通过什么化学变化制造出来的。
  1. 第1天：调查与记录 — 列出用品清单，研读成分表
  2. 第2天：研究与制图 — 查找每种用品的制造过程，画出"原料→化学变化→成品"流程图
  3. 第3天：展示与交流 — 向家人展示你的发现
📦 成果形式：家庭用品化学变化地图（A3手绘海报）
⭐ 评价标准：至少5种用品、每种含化学变化类型标注、信息准确、图文清晰

📌 项目2："走进化学世界"班级展览策划

⏱ 中周期 · 1—2周
🎯 项目目标：策划一场面向低年级同学的化学科普微展览。
  1. 第1周：主题确定+资料收集 — 确定展览主题、收集实验方案和安全规范
  2. 第2周：展品制作+解说词撰写 — 制作展示海报、准备演示实验、撰写解说词
📦 成果形式：展览设计方案书（含主题、展品清单、实验步骤、解说词）
⭐ 评价标准：科学性、安全性、可操作性、趣味性

📌 项目3："我家旁边的水"环境研究报告

⏱ 长周期 · 1个月以上
🎯 项目目标：调查社区周边水源的水质情况，撰写一份科学报告。
  1. 第1周：确定监测方案 — 确定监测指标、学习检测方法、准备工具
  2. 第2周：实地采样与检测 — 定期采样、使用pH试纸等工具检测、记录数据
  3. 第3周：数据分析与制图 — 整理数据、绘制变化趋势图、查找国家标准对比
  4. 第4周：报告撰写与建议 — 撰写研究报告、提出改善建议、准备展示
📦 成果形式：水质监测研究报告（含数据、图表、分析与建议）
⭐ 评价标准：数据真实完整、分析有依据、建议切实可行、格式规范
资源链接：当地环保部门公开数据 | 水质检测简易工具包（pH试纸、TDS笔）
