M5 新知艺术思维 · 选择适合你的挑战层级

A1 · 概念对应填空 必做

用关键词填空，巩固核心概念。点击空白处查看答案：

1. 没有生成新物质的变化叫 ______。
2. 生成新物质的变化叫 ______。
3. 不需要发生化学变化就能表现出来的性质叫 ______。
4. 在化学变化中表现出来的性质叫 ______。

A2 · 现象分类 必做

判断下列现象属于物理变化还是化学变化，点击选择：

🔩 铁生锈
🧊 水结冰
🍞 面包发霉

分层任务 · 三选一

从以下三个任务中选择一个完成：

C1 · 概念设计 挑战

「如果世界上没有化学变化」 —— 写一篇短科幻或画一幅概念图，展开你的想象。

C2 · 探究提案 挑战

提出一个你想验证的化学相关问题，设计简单的验证方案。
