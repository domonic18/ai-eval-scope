1物理变化和化学变化的本质区别是什么？封闭型
思路：本质区别在于"是否有新物质生成"。化学变化有新物质，物理变化没有。关键词是"新物质"——不是"不同的物质"，而是"原来不存在的物质"。
2下列现象中，哪些是化学变化？
铁生锈、水结冰、面包发霉、玻璃破碎、酒精挥发。识别型
答案：化学变化——铁生锈、面包发霉。物理变化——水结冰、玻璃破碎、酒精挥发。
判断逻辑：铁生锈产生了铁锈（新物质），面包发霉产生了霉菌（新物质）；其他只是形态变化。
3化学性质与物理性质有什么区别？请各举2例说明。概念型
区别：化学性质需通过化学变化表现（如可燃性、稳定性）；物理性质可直接观察或测量（如颜色、密度）。
举例：化学性质→铁会生锈、氢气可燃；物理性质→水的沸点100℃、铜呈紫红色。
4为什么说"蜡烛燃烧既包含物理变化又包含化学变化"？请你用证据说明。开放型
分析框架：物理变化——蜡受热熔化（蜡油→液态蜡，无新物质）。化学变化——蜡燃烧生成水和二氧化碳（有新物质）。证据：杯壁出现水珠（水）、使澄清石灰水变浑浊（二氧化碳）。
5比较铁生锈和铁熔化的异同点——两者的变化类型、条件、产物分别是什么？对比型
对比表：
铁生锈——化学变化、需水+氧气、产物为铁锈（Fe₂O₃·xH₂O）。
铁熔化——物理变化、需高温（1538℃）、产物为液态铁（成分不变）。
相同点：都和"铁"有关，都需要一定条件。
不同点：有没有新物质生成。
6小明说"食物腐烂是化学变化"，你同意吗？你能用哪些证据支持或反驳他的观点？应用型
同意。证据：①食物腐烂后颜色改变（从新鲜色变为暗褐色）——新物质产生导致；②气味改变（产生恶臭气体如氨、硫化氢）；③口感改变（变软、变酸）；④微生物分解食物中的有机物，产生新的化学物质。这些都是化学变化的标志。
7如果化学反应不能产生新物质（即所有反应都只是物理混合），世界会变成什么样？反事实型
思考方向：①没有新物质→没有药物（药物通过化学反应合成）、没有塑料、没有合金。②光合作用无法进行（CO₂+H₂O→葡萄糖+O₂ 需要化学反应）。③生命不可能存在——新陈代谢本质就是化学反应。④世界将是一个"纯混合"的世界。
8有人说"AI可以取代化学家"，你同意吗？化学家不可替代的思维特质是什么？批判型
不同意完全取代。化学家不可替代的特质：①提出"好问题"的能力——AI擅长优化已知路径，但难以提出颠覆性问题。②直觉与联想——A物质和B物质的微妙联系往往来自经验直觉。③实验设计与意外发现——实验室的"意外"往往带来重大突破。AI是工具，不是替代者。
9设计一个简单实验，证明"水不是一种元素，而是由氢和氧组成的化合物"。创造型
实验思路（电解水）：①用直流电源连接两个电极放入水中（可加少量硫酸钠增强导电性）。②观察：正极产生气体A，负极产生气体B，体积比为1:2。③检验气体A——使带火星木条复燃（氧气）。④检验气体B——点燃产生淡蓝色火焰（氢气）。⑤结论：水分解为氢气和氧气，说明水由氢、氧元素组成。
10当你判断一个现象是"化学变化"时，你的思考过程是怎样的？用了什么判断标准？这个标准可靠吗？元认知型
元认知反思：我的思考过程→①观察现象（颜色/气味/温度等变化）②问"有没有新物质产生？"③寻找证据（气泡、沉淀、颜色变化等）④下结论。
判断标准："是否有新物质生成"。
可靠性：这个标准是科学的，但需要注意——有些化学变化没有明显现象（如酸碱中和），需要更专业的检测手段。所以"观察不到"不等于"没有发生"。
