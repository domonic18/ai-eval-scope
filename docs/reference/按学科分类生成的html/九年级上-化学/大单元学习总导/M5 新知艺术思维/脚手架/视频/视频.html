视频1：化学变化与物理变化

🎯 观看任务：区分视频中所有现象，哪些是化学变化，哪些是物理变化。

❓ 关键问题

  • 视频中哪些现象说明"产生了新物质"？
  • 有没有一个现象既有物理变化又有化学变化？
@1:15 暂停思考
铁钉生锈——你认为这属于哪种变化？你的依据是什么？
输出任务：列出视频中出现的5种现象，分别标注变化类型 + 判断依据。
基础 · 5分钟

视频2：走进微观世界——原子与分子

🎯 观看任务：画出水分子（H₂O）和二氧化碳分子（CO₂）的结构示意图。

❓ 关键问题

  • 分子和原子是什么关系？
  • 化学变化中原子会变吗？
@2:30 暂停思考
猜一猜：氧气分子由几个原子组成？
输出任务：用纸笔画出3个不同分子的结构，并用文字说明"化学变化中原子如何重新组合"。
进阶 · 8分钟（纪录片）

视频3：拉瓦锡的燃烧实验

🎯 观看任务：记录拉瓦锡实验的3个关键步骤。

❓ 关键问题

  • 为什么拉瓦锡要用"密封容器"？
  • 他的实验如何证明质量守恒？
@5:10 暂停思考
在你知道结果之前，先预测：加热后总重量会变吗？
输出任务：用"问题→假设→方法→证据→结论"五步法，分析拉瓦锡的实验。
挑战 · 15分钟（TED演讲）

视频4：AI如何帮助化学家发现新材料

🎯 观看任务：找出AI在化学研究中承担了什么角色。

❓ 关键问题

  • AI能替代化学家吗？不能替代的部分是什么？
  • 你从这个视频中学到了什么"新的思考方式"？
@8:00 暂停思考
如果让你用AI设计一种新材料，你会怎么向AI描述需求？
输出任务：写一段150字的小短文"当化学遇见AI——我的思考"。
