🕯️ 解析性示例

蜡烛燃烧——它真的"消失"了吗？

核心问题：蜡烛燃烧时，蜡烛真的"消失"了吗？如果没消失，它变成了什么？如何证明？
像化学家一样，用结构化的分析方法重新审视这个熟悉的现象。

💡 关键信息提取

情境信息说明什么为什么重要
蜡烛变短固体蜡减少了需要追踪"减少的部分"去哪了
杯壁出现水珠有液态水生成蜡中含氢元素，燃烧后生成水
烛芯变黑生成碳蜡中含碳元素，燃烧不充分时产生碳
发热发光有能量释放化学变化伴随能量变化

🧩 专家五步解析法

1 先看见什么——全面观察 ▾

做什么：观察并列出所有能直接看到的"线索"。

为什么这样想：分析必须从事实出发，而不是从结论出发。

🔑 关键提示：先别急着回答"蜡烛去哪了"，先仔细看、全面看。
2 判断问题属于哪一类 ▾

做什么：判断这是物理变化还是化学变化，或者两者都有。

为什么这样想：不同的问题类型需要不同的分析方法。

🔑 关键提示：固体蜡→液体蜡（物理变化）；蜡→CO₂+H₂O（化学变化）——蜡烛燃烧包含两种变化。
3 调用哪些概念 ▾

做什么：回忆相关化学概念——物理变化、化学变化、质量守恒、元素。

为什么这样想：分析需要理论工具，就像修车需要合适的扳手。

🔑 关键提示：质量守恒定律是核心——"消失"的物质不会凭空消失。
4 如何进行推理 ▾

推理链条：

① 蜡烛变短 → 蜡被消耗了

② 质量守恒 → 蜡不可能消失，一定变成了其他物质

③ 杯壁有水珠 → 其中有水生成

④ 烛芯有碳黑 → 其中有碳生成

⑤ 蜡烛成分是碳氢化合物 → 燃烧→CO₂+H₂O

🔑 关键提示：推理链条需要环环相扣，每一步都要有证据支撑。
5 怎样形成结论 ▾

做什么：将分散的证据整合成一个完整的解释。

为什么这样想：单独的证据只是线索，整合在一起才是答案。

🔑 关键提示：结论需要覆盖所有现象，不能有"解释不了"的部分。

📋 结论表达

蜡烛燃烧不是"消失"了，而是发生了化学变化——蜡（碳氢化合物）与空气中的氧气反应，生成了二氧化碳和水蒸气（以及微量的碳黑）。你看不到它们，是因为CO₂和H₂O是气体，飘散到空气中去了。

蜡烛 + 消耗的氧气 = 生成的CO₂ + 水 + 碳黑 —— 质量既没有增加也没有减少，这就是质量守恒定律！

📌 方法提炼

  • 全面观察：写下所有你能看到的现象，一个都不要漏
  • 先分类，后分析：先判断这属于"哪类问题"
  • 找证据链：每个结论至少找2个独立证据
  • 画概念图：把分散的信息画成关系图，思路会清晰很多
  • 用守恒思想检查：特别是涉及"变化"的问题

⚠️ 易错提醒

❌ 看到"东西少了"就认为"消失了"——其实是变成了看不见的其他形态
❌ 把物理变化和化学变化完全割裂——很多现象两者同时发生
❌ 认为"看不见=不存在"——气体是看不见的，但它是真实存在的

🏋️ 自主迁移小练习

妈妈在厨房炒菜时，发现锅里的油"变少了"。请你用今天学到的解析方法，分析"炒菜时油去哪了"。

提示：炒菜时油的温度是多少？食用油的主要成分是什么？高温下油脂可能发生什么变化？

请用"观察清单→分类→证据链→结论"的方法自己分析。
