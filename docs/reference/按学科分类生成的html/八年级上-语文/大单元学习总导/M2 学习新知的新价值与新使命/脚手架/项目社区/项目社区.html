🧪 我在班级"开"了一个新闻实验室

🎯 项目目标

用一节课+课后时间，完成一次"新闻要素拆解挑战"。

📌 阶段任务

  1. 自选或教师指定一篇校园新闻/社会短讯（约200字）
  2. 用5W1H表格拆解新闻六要素
  3. 用双色笔标注"事实"句和"观点"句
  4. 写一句"这则新闻最有价值的信息是______"并说出理由

📰 班级新闻简报

🎯 项目目标

以小组为单位，制作一份"班级/年级新闻简报"（纸质或电子版）。

📌 阶段任务

  1. 选题：搜集2-3个校园内值得报道的话题
  2. 采访：分工采访当事人或见证者
  3. 写作：完成新闻稿（每人至少200字）
  4. 编辑：整合排版，添加标题和插图
  5. 展示：在班级内展示并收集反馈

📊 "我们身边的信息生态"调研报告

🎯 项目目标

调查并分析校园/社区的信息传播环境，提出改进建议。

📌 阶段任务

  1. 调研设计（3天）：确定调研范围（如"班级微信群信息传播"），设计调查问题
  2. 数据收集（1周）：收集一周内班级群的分享信息，分类统计（有用/娱乐/广告/可疑）
  3. 分析诊断（3天）：分析信息质量，找出问题（虚假信息、信息过载等）
  4. 报告撰写（3天）：撰写调研报告，提出"改善信息环境"的建议
  5. 成果展示（1天）：在班级/年级做展示

点击下方按钮查看进度

📦 成果形式

📄 1500字以上调研报告 含数据、分析、建议

✅ 数据真实
来源可靠、采集规范
✅ 分析深入

操作性强、有针对性
