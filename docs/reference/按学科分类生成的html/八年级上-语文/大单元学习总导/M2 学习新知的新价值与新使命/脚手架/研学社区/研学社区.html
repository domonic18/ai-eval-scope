🌱 基础层 · 生活案例

生活类

🥦 一条"养生文章"的传播之路

背景：一篇《震惊！这种常见蔬菜竟然致癌》在家长群疯传。
核心问题：这是真的吗？为什么传播这么快？
三条决策路径：
➡️ 路径A 相信并转发
🔍 路径B 先查证再决定
🚩 路径C 举报或提醒
结果：经核查为断章取义、夸大其词。
💡 可迁移启示（点击展开）
面对"震惊体"标题，先停顿3秒再转发。这三秒足够你问一句：这来源可靠吗？有权威出处吗？

📈 进阶层 · 历史案例

历史类

🏛️ 水门事件——两名记者如何改变历史

背景：1972年，华盛顿邮报记者伍德沃德和伯恩斯坦追踪一起看似不起眼的"入室盗窃案"。
核心问题：如何从蛛丝马迹中挖出惊天政治丑闻？
策略：运用"深喉"线人、交叉验证、步步紧逼的报道策略。
结果：尼克松总统被迫辞职，新闻界赢得"第四权力"的美誉。
💡 可迁移启示（点击展开）
两名记者始终在问同一个问题："这件事不对劲，背后到底发生了什么？" 坚持追问、交叉验证——这就是新闻调查的核心。

🧩 挑战层 · 复杂系统案例

跨领域

🦠 2020年"信息疫情"

背景：疫情期间，"喝醋能杀死病毒"等假信息与病毒同步传播。
核心问题：信息传播速度远快于科学验证速度，如何应对？
对策：WHO提出"信息疫情"概念；各国采取事实核查 + 辟谣平台 + 公众教育的综合治理策略。
结果：辟谣平台有效降低了有害信息的传播范围。
💡 可迁移启示（点击展开）
信息辨别不仅是个人能力，更是社会治理的一部分。当你看到一条可疑信息，不转发就是在为社会做贡献。

🎯 "如果你是决策者"分支任务

🏫 设计班级"信息辟谣"方案

假设你是班级信息管理委员，针对同学中流行的假消息问题，设计一套方案。

需要包含：问题识别 → 对策制定 → 执行步骤

框架参考：
① 问题识别：收集同学们最近遇到过/转发过的假消息案例，分类统计类型
② 对策制定：建立班级"辟谣角"；培训3名"信息委员"负责核查；制定"转发前必查三步"规则
③ 执行步骤：第1周宣传培训 → 第2周试运行 → 第3周正式施行 → 每月回顾改进
🔗 与脚手架体系的衔接： 案例可作为"学术社区"讨论素材；分支任务可发展为"项目社区"微项目；使用"工具库"中的事实核查四步法完成分析。
