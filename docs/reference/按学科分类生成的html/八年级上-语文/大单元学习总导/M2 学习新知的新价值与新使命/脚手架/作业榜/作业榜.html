📌 基础层：再现与理解 必选

A1 · 新闻要素拆解表
从教材第一单元中任选一篇课文（如《我三十万大军胜利南渡长江》），用表格拆解5W1H六要素。
格式：要素名称 → 原文摘录 → 你的分析。书写/电子文档均可。
A2 · 事实与观点分类卡
从同一篇课文中摘出5句话，判断每句是"事实"还是"观点"，说明依据。
五张分类卡片：原句 → 分类 → 判断依据（如：有无主观判断词、能否被验证）。

🎯 分层任务 三选一

B1 · 新闻改写 基础难度
将课文中的一条消息改写成适合校园广播站播报的短讯（200字以内）。
核心要素完整，语言通俗化，适合"听"的节奏。
B2 · 新闻对比分析 中等难度
找同一新闻事件的两篇不同媒体报道，从标题、导语、信息侧重、立场倾向四个维度做对比分析。
产出：对比分析表 + 200字分析短文。
B3 · 新闻评论写作 较高难度
就近期一则校园新闻或社会新闻，写一篇300字左右的短评。
必须包含"事实陈述 + 观点表达 + 理由说明"，观点不能凭空判断。

🚀 开放层：创造性表达 选做

C1 · 校园新闻调查与报道
以小组形式，选择一个校园真实话题（如食堂新菜品、社团招新、教室空调问题），完成"选题→采访→写作→编辑"全流程，产出完整校园新闻稿（400–600字）。可附加1分钟出镜报道视频。
C2 · 新闻播报短视频
用手机录制一段2分钟以内的新闻播报视频，内容可以是"本周校园新闻摘要"或"一则假新闻辟谣"。要求：有开场白、主体内容、结束语，语言清晰客观。
C3 · 新闻知识图解
用信息图 / 思维导图 / 手抄报呈现"如何分析一篇新闻"的方法体系。必须包含：5W1H、事实与观点区分、语言特征分析、常见新闻结构（如倒金字塔）。

✅ 学生自评表

作业提交前，逐项检查：

  • 我是否按要求完成了所选作业的全部要素？
  • 我的分析/表达是否有事实依据，不是凭空判断？
  • 我的语言是否尽量客观，避免情绪化表达？
  • 如果是对比/评论类作业，我是否同时呈现了正反两面的信息？
  • 我的成果是否可以被没学过本模块的同学看懂？
