🟢 基础工具

📊 5W1H分析表

快速锁定新闻核心信息
  1. Who 谁
  2. What 什么事
  3. When 何时
  4. Where 何地
  5. Why 为什么
  6. How 怎么发生的
示例：分析《我三十万大军胜利南渡长江》—— Who: 解放军 / What: 南渡长江 / When: 1949年 / Where: 长江流域 / Why: 解放全中国 / How: 分三路渡江

🔵🔴 事实-观点标注法

区分客观信息和主观判断
  1. 🔵 蓝色标 = 事实（可验证的陈述）
  2. 🔴 红色标 = 观点（作者的个人判断）
示例："今天气温35℃"（🔵事实）→ "今天热死了"（🔴观点）。同一件事，事实和观点的表述完全不同。

🟡 进阶工具

专业 · 需要练习

📰 新闻要素分析模板

深度拆解一篇新闻的结构
  1. 标题 → 概括主旨
  2. 导语 → 核心信息
  3. 主体 → 展开细节
  4. 背景 → 补充信息
  5. 结语 → 收束全文
按"标题→导语→主体→背景→结语"逐段分析每部分的功能。适用于教材新闻分析和任何新闻报道。

🔍 事实核查四步法

判断一条消息是否可信
  1. ① 确定信息来源
  2. ② 追溯原始出处
  3. ③ 寻找佐证
  4. ④ 交叉验证
示例：朋友圈说"喝醋能杀死病毒"→①来源是"养生群"→②查不到权威出处→③WHO没有相关说法→④多家辟谣平台已辟谣。结论：假消息！

🔴 高阶工具

研究级 · 挑战自我

⚖️ 报道立场分析矩阵

比较不同媒体的报道立场
  1. 选择同一事件的2—3篇报道
  2. 比较用词选择
  3. 比较信息取舍
  4. 比较情感倾向
同一事件，不同媒体的标题、选用的引语、突出的细节可能完全不同。对比之后你会发现"立场"如何塑造了"故事"。

🤖 AI辅助分析工具

借助AI快速提取新闻关键信息
  1. 将新闻文本输入AI
  2. 用提示词让AI提取5W1H
  3. 人工核对准确性
提示词示例："请提取以下新闻的6个要素（5W1H），并用表格输出。"
输出后——最关键的一步：人工核验AI输出的每个要素是否准确。
