📋 讨论议题

议题 A

你更相信官方媒体还是社交媒体上的新闻？为什么？

可以从信息源头、编辑审核机制、发布速度、利益立场几个角度对比。
议题 B

遇到一条"像假新闻"的消息，你会怎么办？设计一个"三步处置流程"。

三步参考：①看来源——是谁发布的？②找佐证——其他渠道有报道吗？③等一等——让子弹飞一会儿。

🏠 家校连接

📰 素材收集

请家长提供1—2份报纸或手机新闻截图，作为课堂分析素材。

🍽️ 家庭话题晚餐

"今天最有趣/最可疑的一条新闻"——与家人分享你的发现。

饭桌上问一句："爸/妈，今天你看到什么让你觉得不太对劲的新闻了吗？"然后一起用四步法分析一下。
