📊 案例一：我的"新闻画像"

🌱 个人理解型
情境：你在手机里翻到一条同学转发的"重大新闻"，但你发现自己其实不知道什么是"好新闻"——它真不真？新不新？重要不重要？你突然意识到，自己好像缺少一套"判断标准"。
建构任务：用"我的新闻标准坐标图"判断5条新闻片段的质量 原理调用：要素拆解 + 证据支撑
要素拆解证据支撑
建构步骤：
  1. 识别新闻5要素（时间、地点、人物、事件、原因）
  2. 给每条新闻的完成度打分（每个要素0~2分）
  3. 把得分画在坐标图上，横轴为"要素完整度"，纵轴为"信息可信度"
  4. 总结自己的"好新闻"标准——什么才算一条好新闻？
成果：个人新闻标准坐标图（可手绘或电子版）

🔍 案例二：辨析一条朋友圈

🏠 生活应用型
情境：妈妈转给你一条朋友圈文章《注意！这些食物一起吃会致癌》，配图是几张触目惊心的截图。你有点怀疑——转发的人是你妈妈的朋友，不是权威机构。这条消息是真的还是谣言？
建构任务：制作一份"信息核实报告"，判断文章可靠性
证据支撑
建构步骤：
  1. 拆解文章的核心论点——它到底说了什么？
  2. 定位信息来源——原文引用了什么"研究"？能追溯到原始论文吗？
  3. 交叉验证——用专业平台（如果壳、辟谣平台、科普中国）查证
  4. 形成核实结论——是"基本可信""存疑"还是"假消息"？
  5. 把结论写进"信息核实报告"，发给妈妈
成果：信息核实报告（含结论 + 查证过程）
✅ 自我评价：

🏃 案例三：校运会报道方案

🏫 校园问题解决型
情境：学校即将举办运动会，班主任让你负责班级的新闻报道。你是第一次做这件事，完全不知道从哪儿开始——需要采访谁？写什么？怎么发布？
建构任务：设计一份校园新闻报道方案（选题 + 分工 + 采访提纲 + 发布计划）
情境锚定表达转化
建构步骤：
  1. 确定报道角度——关注比赛成绩？运动员故事？观众反应？
  2. 制定采访提纲——针对每个角度设计3~5个问题
  3. 分配角色——谁去采访、谁拍照、谁写稿、谁编辑
  4. 设计发布形式——做一篇公众号推文？一期校园广播？一期短视频？
成果：校园新闻报道方案书（含分工表 + 采访提纲 + 排期）

📝 案例四：改写一篇新闻

📚 学科表达型
情境：教材中的经典新闻《人民解放军百万大军横渡长江》写得气势磅礴，但语言风格偏书面。如果要在校园广播里播报，需要更口语化、更有节奏感的版本。
建构任务：将其改写为适合校园广播的"口语版新闻"
表达转化反思迭代
建构步骤：
  1. 分析原文的语言特点（句式结构、用词风格、修辞手法）
  2. 确定改写策略——哪些长句要拆分？哪些书面词要替换？
  3. 逐段改写，保持信息完整的同时让语言更口语化
  4. 大声朗读一遍——听起来像不像广播新闻？
成果：口语版广播新闻稿（可录制为音频）

🧪 案例五：新闻中的科学

🔗 跨学科整合型
情境：你读到一则"新型疫苗研发取得突破"的科技新闻，里面提到"mRNA技术""三期临床试验""抗体滴度"等术语。你大概知道这是好消息，但新闻在说什么？它说的靠不靠谱？
建构任务：制作一份"新闻中科学知识"解读笔记
要素拆解证据支撑
建构步骤：
  1. 识别新闻中的科学术语——圈出所有你不懂的名词
  2. 查阅资料——用课本、科普网站理解每个术语
  3. 用自己的话解释——不是复制粘贴，是真的"讲清楚"
  4. 制作术语卡片——术语 + 通俗解释 + 新闻中的具体数据
成果：科技新闻术语解读笔记（含术语卡片3~5张）

🕵️ 案例六：你当"假新闻调查员"

🌍 社会情境型
情境：某社交平台上一条"XX地发生重大灾害，已造成上百人伤亡"的假新闻正在快速传播，评论区一片恐慌。你知道这是假的——但你该怎么办？
建构任务：设计一条假新闻破解帖子的完整方案
情境锚定要素拆解表达转化
建构步骤：
  1. 分析假新闻中的传播要素——它用了什么"套路"让你觉得是真的？
  2. 追溯原始来源——这条消息最早从哪儿来的？
  3. 设计反驳逻辑——证据1、证据2、证据3逐条拆穿
  4. 设计传播方案——你的辟谣帖发在哪儿？用什么标题吸引人看？
成果：假新闻破解方案（含分析 + 逐条反驳 + 传播建议）

🌟 案例七：创办"校园新闻角"

🚀 综合创新型
情境：经过前面的练习，你已经成了半个"新闻专家"。现在你想在班级或校园里创建一个定期更新的"新闻角"，让更多同学关注真实新闻、学会辨别信息。这需要一整套方案。
建构任务：设计"校园新闻角"整体运营方案
情境锚定要素拆解证据支撑表达转化反思迭代
建构步骤：
  1. 确定定位——新闻角的核心目标是什么？传递信息？培养素养？
  2. 设计内容模块——今日头条、校园快讯、深度报道、谣言粉碎机？
  3. 制定排期——每周几更新？谁负责哪个板块？
  4. 分配角色——主编、记者、编辑、美工、审核
  5. 设计反馈机制——怎么知道同学爱看什么？怎么收集意见？
成果：校园新闻角运营方案（含内容设计 + 分工表 + 排期表 + 反馈机制）
✅ 自我评价：
八年级上 · 语文 · 统编版五四制 · 第一单元（新闻阅读与写作）
