📏 时间轴模板

展示事件在时间线上的先后顺序和持续时间
  1. 画一条水平线，标注起点和终点时间
  2. 按比例在线上标记关键时间点
  3. 在每个时间点上方标注事件名称
  4. 用彩色箭头标注持续时间
适用：我的一天 · 植物生长 · 历史事件

🥧 时间饼图模板

展示各类活动占总时间的比例
  1. 计算一天总分钟数（1440分钟）
  2. 计算每类活动的分钟数
  3. 活动时间÷1440×360°，用量角器分区
  4. 给每个区域涂上不同颜色
适用：分析时间分配 · 比较时间投入

📊 时间柱状图模板

比较不同活动或不同日期的用时多少
  1. 确定横轴（活动名称）和纵轴（分钟数）
  2. 每项活动画一个柱条
  3. 高度代表分钟数
  4. 在柱条上方标注具体数字
适用：对比一周运动量 · 各科作业用时
