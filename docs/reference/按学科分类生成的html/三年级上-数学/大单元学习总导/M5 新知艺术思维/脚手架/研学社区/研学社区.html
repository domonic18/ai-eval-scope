🏠 小明的周末时间安排

三年级 · 周六上午 8:00-11:00 · 自由时间管理
小明有 3 小时自由时间（8:00-11:00），想安排：数学作业30分钟、语文阅读20分钟、练字15分钟、拼乐高45分钟、看动画片20分钟。总时长看似只要130分钟（2小时10分钟），但他忘了算"换活动"和"休息"的时间——实际需要约3小时20分钟。怎么办？
方案A
减少乐高时间到30分钟 → 刚好3小时
方案B ✅ 推荐
拼乐高移到下午 → 上午只做学习类（65分钟）
方案C
保持所有活动，提前15分钟开始（7:45）
✅ 结果：方案B最合理——上午专注做学习类，下午拼乐高当作奖励。
💡 可迁移启示：安排活动时，除了活动本身的时间，还要加上"间隙时间"（换活动、休息、发呆……）。
🎯 如果你是决策者： 如果你是时间规划师，你会建议小明怎么安排？请画出"你的版本"的时间轴方案！
💡 提示：可以先画出8:00-11:00的轴线，把每项活动用彩色方块标在线上，别忘了留出10-15分钟的"缓冲时间"哦！

🏃 100米赛跑的秒表计时

学校运动会 · 秒表计时 · 精度到0.1秒
4位选手的100米成绩：小明18.5秒、小红17.8秒、小刚19.2秒、小丽18.0秒。谁跑得最快？你能把成绩排序吗？
思路A
直接看秒数——时间越短跑得越快：17.8 < 18.0 < 18.5 < 19.2
思路B
换算成秒+毫秒比较——这里精度已到0.1秒，不需要
✅ 结果：小红最快（17.8秒），小丽第二（18.0秒），小明第三（18.5秒），小刚最慢（19.2秒）。
💡 可迁移启示：在比"快慢"时，时间越小 = 速度越快——这个"反向思维"在体育、交通等领域经常用到。
🎯 如果你是决策者： 如果小红17.81秒，小丽17.85秒，谁赢了？作为计时裁判，怎么保证公平？
💡 17.81 < 17.85，所以小红胜！要保证公平，可以用电子计时器代替手动秒表，每位选手配3个计时员取平均值，减少人为误差。

🌱 绿豆生长周期观察

科学课 · 植物生长记录 · 分时段测量
绿豆生长记录：第1天播种→第3天破土（0.5cm）→第5天2cm→第7天5cm→第10天10cm。从发芽到10cm用了几天？哪个阶段长得最快？
算一算
第3天到第10天 = 7天（从发芽到10cm）
比一比
第3-5天长1.5cm，第5-7天长3cm，第7-10天长5cm → 第7-10天最快！
✅ 结果：绿豆苗的生长速度不是均匀的——它会越長越快（"加速生长"）。
💡 可迁移启示：很多自然现象都不是"匀速"的。用"分时段记录+比较"的方法，可以发现隐藏在数据中的规律。
🎯 如果你是决策者： 如果同时种了绿豆、黄豆和红豆各10株，你如何设计实验表格来比较三种豆子的生长时间差异？
💡 设计一张表格：横轴=天数（第1天到第14天），纵轴=三种豆子各10株的平均高度。每天同一时间测量并记录，最后画三条不同颜色的折线做对比。
