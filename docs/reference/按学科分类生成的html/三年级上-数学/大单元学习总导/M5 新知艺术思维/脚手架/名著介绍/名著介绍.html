"一寸光阴一寸金，寸金难买寸光阴。"
——中国古谚
冲突型
为什么金子和光阴能画等号？你的哪一分钟最"值钱"？
"逝者如斯夫，不舍昼夜。"
——孔子
启发型
请你观察一杯水从水龙头流下——你能"看见"时间在流吗？
"时间就是生命，浪费时间就是浪费生命。"
——鲁迅
批判型
你同意吗？你上一次觉得"浪费时间"是什么时候？
"时钟在走，生命在过。"
——林清玄
启发型
时间不是"用来过的"，而是"正在过"的。你的时间作品就是"记录生命在过"的一小部分。
"要生活啊，不是只活着。"
——塞涅卡（古罗马哲学家）
批判型
他一句话区分了"生存"和"生活"。你的时间是在"活着"还是在"生活"？
