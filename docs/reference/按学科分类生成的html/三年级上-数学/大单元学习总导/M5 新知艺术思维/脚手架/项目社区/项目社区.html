短周期 1-3天

项目1："我的一天"时间折线图

用折线图展示一天中不同时间段的"精力变化"，发现自己的精力高峰和低谷！

📋 阶段任务

  1. 设计记录表：把一天分为上午、中午、下午、晚上四个时段
  2. 每2小时记录一次"精力值"（1-10分）
  3. 把数据点在坐标纸上，连成折线
  4. 标注"高峰期"和"低谷期"
  • 是否有至少6个时间点的数据？
  • 折线是否清晰连接？
  • 是否标注了高峰和低谷？
工具库中的"时间记录表""坐标纸模板"

💡 想一想：你的精力高峰在什么时候？为什么有的人早上精神好，有的人晚上精神好？

中周期 1周

项目2："时间小侦探"——家族时间大调查

调查家人一周的时间安排，发现不同人的时间使用模式差异。你可能会大吃一惊——原来家人的一天是这样的！

📋 阶段任务

  1. 采访家中的2-3位成员"你的一天怎么过？"
  2. 设计并分发"时间记录卡"，请每人记录3天
  3. 汇总数据——做一张"家庭时间对比图"
  4. 写一份"时间调查报告"（至少200字）
  • 是否收集了至少2位家人的时间数据？
  • 对比图是否用了"统一的分类标准"？
  • 报告是否包含"发现"和"感受"？
工具库中的"时间记录卡模板""数据汇总表"

小明周末时间安排参考：

周六：9:00-10:00写作业 / 10:00-11:30打球 / 14:00-16:00课外班 / 19:00-20:30看电视
周日：9:00-11:00写作业 / 14:00-15:00阅读 / 16:00-17:00自由活动

发现了什么？小明周末"屏幕时间"占了很大比例！你的家人呢？

中周期 1-2周

项目3：创建"班级时间银行" 🏦

用"时间银行"的概念，和同学互相交换"时间"——你帮我补习10分钟数学，我帮你值日15分钟。时间就是"货币"！

📋 阶段任务

  1. 讨论：班级里有哪些可以互相帮助的事情？每件事大概需要几分钟？
  2. 设计："时间存折"模板（记录谁帮了谁、花了多少时间）
  3. 试运行：全班试行一周"时间银行"
  4. 总结：一周后分析"谁存的时间最多""谁取的时间最多"
  • 时间存折是否记录清晰（帮助人、帮助内容、所用时间）？
  • 存入和取出时间计算是否正确？
  • 是否在"时间银行"中帮助了至少1位同学？
工具库中的"记录表模板"

💡 延伸思考：时间银行的"利率"是多少？如果A帮B 10分钟，B帮A 8分钟，公平吗？

长周期 1个月

项目4："时光胶囊"——记录一个月的成长变化

用一个月的时间，记录自己在学习、身体、习惯三个方面的变化，制作专属于你的"时光胶囊"！

📋 阶段任务

  1. 设计记录维度（阅读时长、运动时长、练字时长、早睡次数等）
  2. 每天花2分钟记录各项数据
  3. 汇总30天数据，计算总时长、日均时长、变化趋势
  4. 制作"时光胶囊"展示板
  5. 向全班展示"这个月我花时间做了什么"
  • 数据记录是否覆盖至少4周？
  • 时间图是否包含了所有记录维度？
  • 个人反思是否真诚、具体？（"我发现自己……"）
  • 是否在班级展示中分享了自己的发现？
工具库中的"月度记录表"；作业榜中的"时间写作"任务参考

⏳ 你的"时光胶囊"里要放进什么？

一本读完的书？一张满分试卷？一张运动照片？还是……你坚持了30天的记录表本身？
