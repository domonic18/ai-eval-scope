🎬 视频：时间的秘密

跟随6个精心挑选的视频，从"一秒有多长"到"太空中的时间"，一步步探索时间的奥秘。每个视频都配有观看任务、思考问题和输出任务，帮助你边看边学、边学边做。

基础层 解释型 — 感知时间

视频1：《一秒有多长？》

🎬 科普动画 ⏱ 3-5分钟

一秒钟可以发生多少事情？蜜蜂扇动翅膀、心脏跳动一次……让我们一起来感受"一秒"的真实长度，了解"秒"这个时间单位是怎么来的。

🎯 观看任务：
  1. 数一数：视频中说1秒内可以发生哪些事情？
  2. 想一想：1秒对你来说有多长？你眨一下眼大约是多少秒？
💬 关键问题：
  • 为什么"1秒"被定义为一个固定的时间？以前的人怎么定义的？
  • 如果世界上没有"秒"这个单位，会有什么不方便？
自己数5秒——你的"5秒"和视频里的"5秒"一样长吗？试试看！
画一张"一秒图鉴"——画5件"1秒内能发生的事"（如蜜蜂扇翅膀、心脏跳一下……）

视频2：《一分钟能做什么》

🎬 科普短片 ⏱ 3分钟

一分钟能跳绳多少下？能读多少字？能写完一道数学题吗？不同的人、不同的情境下，"一分钟"的价值完全不同。

🎯 观看任务：
  1. 记录视频里提到了哪些"一分钟完成的事情"
  2. 选择其中一项，自己实际测试一下是不是真的
💬 关键问题：
  • 一分钟对世界上不同的人来说，价值一样吗？
  • "一分钟"对"等公交车"和"吃零食"来说，感觉为什么不一样？
设定手机计时器1分钟——你在这1分钟里能读多少字？试试看！
写"我的1分钟纪录"——做一件事（跳绳、写字、拍球……）计时1分钟，记录你的成绩

有的同学1分钟能跳绳120下，有的同学1分钟能写35个字。你的1分钟纪录是多少？点击这里输入！

（在纸上记录你的成绩，和同学比一比吧！）

进阶层 分析型 — 理解时间

视频3：《时间的历史——从日晷到原子钟》

🎬 纪录片片段 ⏱ 8-12分钟

人类是如何从看太阳的位置来判断时间，到如今用原子钟精确到亿分之一秒？这趟穿越数千年的时间之旅，将带你认识各种巧妙的计时工具。

🎯 观看任务：
  1. 画一张"时间工具进化时间轴"——从最早的计时工具到最新的，按顺序排列
  2. 每样计时工具旁边标注：它用什么原理来计时
💬 关键问题：
  • 日晷有什么缺点？为什么阴天和晚上它就没用了？
  • 哪种计时工具让航海成为可能？为什么？
  • "精度"的提高对人类有什么实际好处？
暂停视频，想一想——你房间里有哪些东西能用来计时？你能用它们做什么样的"钟"？
选择视频中一种计时工具（日晷/水钟/沙漏/机械钟/石英钟/原子钟），画它的结构图并标注原理

视频4：《动物眼中的时间》

🎬 科学纪录片 ⏱ 6分钟

为什么苍蝇那么难打？为什么狗不能理解"5分钟后"？不同动物感知时间的方式完全不同——让我们走进动物的"时间世界"。

🎯 观看任务：
  1. 视频中说，动物"看世界"的速度和人类不一样——记录至少3个例子
  2. 想一想：为什么有些动物"看东西很快"，有些"看东西很慢"？
💬 关键问题：
  • 苍蝇为什么很难被打到？这和它的"时间感"有什么关系？
  • 狗能理解"5分钟后"这个概念吗？
  • 如果人类也像蝴蝶那样感知时间，世界会是什么样子？
暂停——你觉得你和猫"谁更懂得时间"？为什么？
写一段想象短文《如果我是（选一种动物）眼中的时间世界》

挑战层 辩论/前沿型 — 驾驭时间

视频5：《时间管理：如何利用你的24小时》

🎬 TED演讲（青少年版） ⏱ 8-10分钟

每个人都有24小时，为什么有的人能做更多的事？时间管理的秘密不是"挤时间"，而是"选对事"。

🎯 观看任务：
  1. 演讲者说了哪3个时间管理的"秘密"？
  2. 她的方法"一个土豆时间法"是什么意思？和60进制有什么关系？
💬 关键问题：
  • 她说"你没时间做X，只是因为你没把X放在优先位置"——你同意吗？
  • 你用过什么时间管理方法？有用吗？
暂停，列出你今天的To-Do List——按照"重要"和"紧急"分四个格子
用视频中学到的方法，设计一份你自己的"一周时间管理方案"（含学习、运动、娱乐、休息四大板块）

视频6：《太空中的时间——和地球不一样？》

🎬 NASA少儿频道 ⏱ 5分钟

空间站的宇航员一天能看到16次日出！在太空中，时间的流逝方式和我们在地球上完全不同。让我们一起探索太空中的时间奥秘。

🎯 观看任务：
  1. 空间站的宇航员一天看几次日出？为什么？
  2. 视频中如何解释"时间膨胀"这个概念？（用你能听懂的话记下来）
💬 关键问题：
  • 如果火星一天是24小时39分钟，你在火星上怎么安排"一天"？
  • 你认为未来人类在火星上会用什么方式来计时？
暂停——如果从地球到火星要飞7个月，你怎么让自己不觉得"时间太长了"？
设计一份"火星一天时间表"（以24小时39分钟为一天）
