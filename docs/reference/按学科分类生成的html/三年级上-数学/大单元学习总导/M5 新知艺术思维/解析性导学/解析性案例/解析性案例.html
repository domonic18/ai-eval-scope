🏠 案例1：我的上学路上要多久？

▼

情境：小光说"走路10分钟就够了"，但他每天8:00上课，7:45出门却常常迟到。请你当时间侦探，帮他分析问题出在哪儿。

任务：记录从"关门出发"到"走进教室"每一个环节，看看"15分钟"只是一个环节还是总时间？

解析路径：①识别核心问题 → ②确定解析对象 → ③拆解关键要素 → ④梳理关系 → ⑤解释原因 → ⑥迁移判断

🔬 原理调用：原理1（时间定位法）· 原理2（分解法）
环节起点终点用时备注
出门→走到电梯———
等电梯———
坐电梯———
走到校门口———
走进教室———

💡 从关门到坐下可能有5-8个小环节，固定时间+浮动时间都要算。没有唯一答案，关键是把模糊概念变成可记录的时间点。

📚 案例2：课表的时间分析

▼

任务：分析课间休息几分钟？一天几节课？上课总时长？午休占比？和你学校的课表有什么不同？

🛒 案例3：家庭周末购物时间计算

▼

情境：9:00出发→走路15分钟→购物40分钟→走路回家→11:00前到家。但10:00-10:30有"乐高拼搭比赛"想参加！

任务：列出多种方案并计算是否可行。

🔬 原理调用：原理1（时间定位法）· 原理2（分解法）
方案9:00-9:159:15-9:5510:00-10:3010:30-10:45到家可行?
A去超市购物比赛回家10:45✅
B去超市—比赛购物→回家11:25❌
C你还能想到其他方案吗？

💡 注意购物和比赛之间的切换时间。列出所有可能，用时间线验证可行性。

🏃 案例4：体育测试成绩分析

▼

情境：三年级一班50米跑成绩（秒）——男：小张9.8、小刘10.2、小王9.5、小李11.0、小陈9.9；女：小周10.5、小吴10.1、小郑11.2、小冯9.7、小赵10.8。达标：男≤10.5秒，女≤11.0秒

任务：①男女生分别几人达标？②全班平均成绩？③最快和最慢差多少？④按成绩排序

🔬 原理调用：原理3（分类法）· 原理5（可视化法）
姓名性别成绩(秒)是否达标排名
小王男9.5✅1
小冯女9.7✅2
小张男9.8✅3

💡 数字越小跑得越快（反向思维最易错）！最快的不一定都是男生。关注数据分布特征比单纯算数字更有意思。

🚀 案例6：火星上的时间

▼

情境：一个火星日≈24小时39分钟35秒。未来人类在火星建立基地，需适应"火星时间"。

任务：①一个火星日比地球日长多少？②地球"上午8:00"火星对应几点？③每过一个火星日，时钟偏差多少？

🔬 原理调用：原理4（60进制法）· 原理5（可视化法）
天数火星累积(h:m:s)地球累积偏差
124:39:3524:00:00+39:35
5123:17:55120:00:00+3:17:55
30739:47:30720:00:00≈19:47:30

💡 39分35秒看起来不多，累积30天≈20小时偏差！第30天"火星中午12:00"≈地球"第二天早8:00"。如果你是时间规划师，怎么设计火星人的一天？

🎉 案例7：活动策划的时间统筹

▼

情境：班级"六一"庆祝会，14:00-15:30（共90分钟）。开场5min → 合唱10min → 小品15min → 游戏20min → 抽奖10min → 总结5min

任务：①总时长多少？有余地吗？②游戏超时6分钟怎么压缩？③插入神秘嘉宾8分钟怎么调整？④画时间轴

🔬 原理调用：原理1（时间定位法）· 原理2（分解法）· 原理3（分类法）
序号活动时长开始结束可调?
1开场5min14:0014:05固定
2合唱10min14:0514:15固定
3小品15min14:1514:30微调
4游戏20min14:3014:50灵活
5抽奖10min14:5015:00微调
6总结5min15:0015:05固定

💡 活动总长65分钟，但别忘了节目间切换时间（5-10分钟"消失"了）和3-5分钟保险时间。实际可用≈75-80分钟。超时问题：压缩后面还是延长时间？不同选择不同结果。

💭 综合反思

完成7个案例后，问问自己：

  1. 哪个案例最容易？为什么——课表分析还是上学路上？哪个最贴近你的生活？
  2. 哪个案例最难？是火星时间的60进制换算，还是活动统筹的多变量思考？
  3. 哪条原理最常用？分解法和可视化法用到次数最多？
  4. 哪条原理还不太会用？60进制法有没有在哪卡住你？打算怎么加强？
