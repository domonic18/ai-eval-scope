🏷️ 案例1：设计你的"时间名片"
🎯真实建构情境

每个人的"时间节奏"都不一样——你是"百灵鸟型"（早睡早起）还是"猫头鹰型"（晚上精神好）？用一周记录来发现自己的类型。

📋建构任务

用一周时间记录每天最有精力的时间段，建构一张属于你自己的"时间名片"。

🔬原理调用
原理1：从真实困惑出发原理2：用数据说话
📝建构步骤
  1. 确定两个指标——"最有精神"和"最没精神"的时段
  2. 每天睡前花1分钟，在表格里打勾记录
  3. 一周后汇总，看"最有精神"的时段集中在哪
  4. 用色块时间轴展示结果
  5. 标注"我的高效时段"
🎨成果表达

名称：我的专属时间名片 · 格式：A5卡片，正面精力时间轴色块图，背面3条时间使用建议

✅自评量规
  • 是否连续记录了至少5天的数据？
  • 是否展示了"高精力"和"低精力"两个时段？
  • 背面是否写了至少3条给自己的时间使用建议？
  • 数据是否真实（没有为了"好看"而编造）？
  • 是否用颜色清楚地区分了不同状态？
👨‍👩‍👧‍👦 案例2：家庭周末时间规划方案
🎯真实建构情境

全家人周末愿望撞在一起——与其"抢时间"，不如一起来建构一份"家庭周末时间规划方案"。

📋建构任务

记录一个周末家庭活动的时间数据，建构一份让每个人的愿望都有时间的规划方案。

🔬原理调用
原理3：分类让数据变信息原理5：先完成再完善
📝建构步骤
  1. 采访每位家人的"周末愿望"和"预计时长"
  2. 把所有愿望分类汇总，看总共需要多少时间
  3. 比较"需要的时间"和"实际有多少时间"
  4. 用色块时间轴画出方案A
  5. 根据家人反馈调整出方案B
🎨成果表达

名称：我们的快乐周末时间规划图 · 格式：时间轴展示周六周日，不同颜色标注不同家庭成员

✅自评量规
  • 是否采访了至少3位家人的周末愿望？
  • 是否给每位家人都安排了至少1项喜欢的活动？
  • 是否用不同颜色标注了不同家庭成员？
  • 是否对比了方案A和方案B的差异？
  • 总活动时间是否没有超过周末的实际可用时间？
⏰ 案例3：课间10分钟最优利用方案
🎯真实建构情境

下课铃一响，你经常发现：刚跑到操场上课铃就响了，刚打开课外书还没看完一页老师就进来了……课间10分钟到底怎么利用才最划算？

📋建构任务

用"时间记录法"追踪3天的课间活动，建构一份"10分钟最优利用指南"。

🔬原理调用
原理2：用数据说话原理4：选对表达方式
📝建构步骤
  1. 设计一份"课间记录小卡片"
  2. 连续3天，每个课间记录：做了什么、做了多久、感受如何
  3. 统计"最常做的事"和"最开心的事"
  4. 设计"最优课间10分钟"方案（含喝水、上厕所、放松、准备下节课）
  5. 试用新方案并改进
🎨成果表达

名称：我的10分钟课间魔法方案 · 格式：A5卡片，正面写分配方案，背面画实际执行记录

✅自评量规
  • 是否记录了至少3天的课间活动？
  • 方案是否包含：上厕所、喝水、活动、准备上课这4项？
  • 是否实际执行并记录了至少2次"新方案"？
  • 方案是否能在10分钟内完成？
  • 是否留了"缓冲时间"？
📖 案例4：用时间轴讲一个数学故事
🎯真实建构情境

时间轴不仅可以讲故事，还可以"算"故事！比如："小明7:30出门，走路到学校需要20分钟，那他几点到校？"

📋建构任务

选择你熟悉的一天经历，用时间轴讲述故事，在关键节点设置"时间计算题"。

🔬原理调用
原理4：选对表达方式原理3：分类让数据变信息
📝建构步骤
  1. 选定一个你印象深刻的一天
  2. 回忆这一天的事件和时间线
  3. 在时间轴上标注每个事件及开始/结束时间
  4. 在关键节点设置"时间计算题"
  5. 加上标题、色彩和装饰
🎨成果表达

名称：我的一天时间故事轴 · 格式：横向时间轴，至少6个节点，至少3道时间计算题

✅自评量规
  • 是否包含了至少6个时间节点？
  • 每个节点是否标注了确切的时间？
  • 是否设置了至少3道时间相关计算题？
  • 能否正确回答自己设置的题目？
  • 图形是否清晰、颜色是否区分了事件类型？
🌍 案例5："一分钟可以做什么"跨学科图鉴
🎯真实建构情境

一分钟在不同领域能做的事完全不同——眨60次眼睛？跑200米？心跳大约80次？让我们做一份"一分钟图鉴"。

📋建构任务

研究"一分钟"在数学、体育、科学、生活中分别能做什么，建构跨学科百科图鉴。

🔬原理调用
原理1：从真实困惑出发原理3：分类让数据变信息
📝建构步骤
  1. 头脑风暴"一分钟能做哪些事"（至少8个想法）
  2. 按"数学/体育/科学/生活/艺术"分类
  3. 用实验验证：实际测试"真的能做到吗？"
  4. 把你验证过的结果整理成"一分钟图鉴"
  5. 添加趣味发现和"冷知识"
🎨成果表达

名称：一分钟能做什么——跨学科图鉴 · 格式：A3海报，5个学科领域分区，每区至少2条纪录

✅自评量规
  • 是否覆盖了至少4个不同的学科/领域？
  • 每条纪录是否经过实际测试验证？
  • 是否展示了测试方法或数据来源？
  • 全图鉴是否包含至少10条"一分钟纪录"？
  • 是否配有图标或简笔画帮人快速理解？
🏙️ 案例6：未来城市时间管理创意方案
🎯真实建构情境

如果100年后人们住在未来城市里，空中汽车飞、地下有超级地铁——大家的时间够用吗？未来城市该怎么帮助人们"节约时间"？

📋建构任务

基于你对"时间"的理解，为未来城市设计一份"时间管理创意方案"。

🔬原理调用
原理5：先完成再完善原理4：选对表达方式
📝建构步骤
  1. 发现问题——列出未来城市中至少3个"浪费时间"的场景
  2. 创意构思——针对每个场景想一个"节省时间"的发明
  3. 画草图——画出你的"时间发明"
  4. 添加说明——用时、分、秒写出能节省多少时间
  5. 方案展示——整理成完整的"未来时间方案书"
🎨成果表达

名称：未来城市时间管理方案书 · 格式：至少3个场景分析+3个方案（问题→想法→效果→时间账）

✅自评量规
  • 是否分析了至少3个未来城市中的时间浪费场景？
  • 每个场景是否都配了解决方案？
  • 是否用到了时、分、秒的"时间账"计算？
  • 每个方案是否配了草图或示意图？
  • 方案是否体现了"创造性思维"？
📘 案例7：创作《我的时间密码》迷你手册
🎯真实建构情境

经过前面6个案例的练习，你已经积累了很多关于"时间"的发现。现在，把所有成果整合到一起——建构一本属于你自己的《我的时间密码》迷你手册！

📋建构任务

从前6个案例成果中精选3项，整合成一本完整的《我的时间密码》迷你手册。

🔬原理调用
原理3：分类让数据变信息原理5：先完成再完善
📝建构步骤
  1. 回顾前6个案例，选出你最满意的3个成果
  2. 设计手册结构——封面→目录→成果展示→我的反思
  3. 为每个成果写一段"我为什么要做这个"的说明
  4. 写一篇"学习反思"——通过建构时间作品，你学到了什么？
  5. 设计封面、排版、美化
🎨成果表达

名称：《我的时间密码》迷你手册 · 格式：A4对折小册子，不少于6页，含封面、3个成果、反思页、致谢页

✅自评量规
  • 手册是否包含封面、目录、3个精选成果、反思页？
  • 每个成果是否有标题和"我为什么要做这个"的说明？
  • 反思页是否写了至少3句关于"建构学习"的感悟？
  • 手册是否有统一的排版风格和色彩？
  • 是否体现了"这是我的作品"的个性化表达？
