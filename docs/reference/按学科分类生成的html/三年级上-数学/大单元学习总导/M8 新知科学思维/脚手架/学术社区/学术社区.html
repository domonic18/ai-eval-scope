👥 同伴连接 — 讨论

你每天「感觉」到的最短时间是多少？点击话题展开讨论提示

1 你猜「3分钟」有多久？
⏳ 试一下：闭上眼睛，感觉过了3分钟就睁眼，看看和钟表差了多少秒？试试3次，取个平均值！
2 不做钟表的计时法？
🕰️ 如果让你设计一个不用钟表的计时方法，你怎么做？数心跳？看影子？还是用沙漏？和同学比一比谁的创意最棒！
3 你的1分钟 vs 同桌的1分钟
🤝 你感觉到的1分钟和同桌感觉到的1分钟一样长吗？为什么不一样？来，做个实验对比一下！
