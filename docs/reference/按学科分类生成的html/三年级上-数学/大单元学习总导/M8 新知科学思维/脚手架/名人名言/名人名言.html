🤔 「如果你是他」思考

🔭
如果你是伽利略：看到教堂的灯在摆，你注意到了什么？
✅ 你可能注意到：不管灯摆得多大，每次摆动的时间好像一样。这就是「等时性」的起点——观察细节，才能发现规律。
⚡
如果你是爱因斯坦：怎么向一个三年级小学生解释「时间变慢」？
✅ 试试这样说：想象你在一个飞快的宇宙飞船上，你的手表走得比地球上的人慢一点点——不是表坏了，是速度让时间变慢了！
