视频四

📂 专题演讲 · 认知科学/心理学 ⏱️ 建议 1-1.5 课时
🎯 观看任务：记录视频中提到的至少2种"让时间变快/变慢"的心理情景，对照自己的生活经验。
❓ 关键问题：

02:15 — 讲到"等待时时间过得很慢"——"你等红灯30秒的感觉，和玩手机30秒的感觉，为什么差那么多？注意看接下来的实验。"
05:40 — 展示时间知觉实验数据时暂停——"你觉得年龄越大时间过得越快是真的吗？先写下你的猜测，再往下看。"
做一个"时间感知对比表"——左边列出3件你觉得"过得很快"的事，右边列出3件"过得很慢"的事，每件事旁估算实际时长。思考：感觉和实际之间的差值说明什么？

🤔 M10 哲学核心追问

我们用来"测量时间"的工具（日晷、钟表、原子钟），
本身是否也在塑造我们对"时间是什么"的认知？
当测量工具变得足够精确，我们离"时间的真实"更近了，还是更远了？
