💬 同伴讨论

🗣️ 讨论一：24小时够用吗？
如果让你自己设计一天的长度，你会设计成多少小时？
思考方向：现在一天24小时，你觉得够用还是不够？如果变成30小时，你会多做什么？如果变成18小时呢？
🌟 小提示：想想你的日常——睡觉、上学、作业、玩耍，分别花了多少时间？
🗣️ 讨论二：时间快慢之谜
为什么同一个10分钟，有时候过得快，有时候过得慢？
思考方向：玩得开心时10分钟一眨眼就过去了，等下课铃时10分钟像1小时那么长……你有哪些"时间变快"或"时间变慢"的经历？
🌟 小提示：小组里每个人分享一个自己的例子。
🗣️ 讨论三：没有时钟的世界
如果世界上没有时钟，我们的生活会变成什么样？
思考方向：你怎么知道几点上学？怎么看动画片时长？怎么约朋友出去玩？
🌟 挑战：试着想5件没有时钟就做不到的事情。
