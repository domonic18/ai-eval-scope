🏆 作业榜

三层差异化作业设计，帮你巩固时间概念

📘 基础层：再现与理解 必做

所有同学都要完成，检测你是否掌握了基本的时间换算。

📝 第 1 题 · 填空题

（1）3 时 = 分

（2）120 秒 = 分

（3）1 时 30 分 = 分

🔢 第 2 题 · 比大小

2 时 ○ 120 分（填 >、< 或 =）

✅ 第 3 题 · 判断题

"1 分钟 = 60 秒，所以半分钟 = 30 秒" 这句话对吗？

🕐 第 4 题 · 看钟面说时刻

观察上面动画时钟：如果时针指向 3 和 4 之间，分针指向 6，这时是几点？

📗 分层任务 三选一

选择适合你的一级挑战，完成后可以继续挑战更高难度。

📋 A 级 · 基础巩固 记录你晚上 6:00 到 8:00 之间做的事情，精确到分钟，制作一张 "晚间时间记录表"
📊 B 级 · 能力提升 调查家人一天的时间分配（工作、休息、娱乐各多少小时），用柱状图展示并写一段分析
🔍 C 级 · 挑战拓展 查资料了解 "古代人如何计时"，对比古今计时的优缺点，写 200 字小报告
🎯 你选择了：
完成后记得和同学分享你的成果！

🎨 开放层：创造性表达 四选一

用你喜欢的方式，展示你对时间的理解。

📝 写作 以 "一分钟的秘密" 为题，写一首小诗或小故事
🎨 图示 画一张 "时、分、秒关系思维导图"
🎬 视频 录制一段 2 分钟的 "时间小课堂" 讲解视频
🗣️ 演讲 准备一份 3 分钟的 "珍惜时间" 演讲提纲
🌟 太棒了！ 期待你的作品！完成后可以邀请老师或同学为你打分。
