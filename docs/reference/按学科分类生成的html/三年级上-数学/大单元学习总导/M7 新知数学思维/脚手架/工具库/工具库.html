🧰 工具库

用好工具，让时间思考更高效

📋 时间记录表
用途：记录每天活动的时间分配
  1. 填上日期
  2. 列出今天的活动
  3. 记录开始时间和结束时间
  4. 算一算每件事花了多久
示例：
📅 日期：9月15日
📚 作业 16:00—16:45 → 45分钟
🎹 练琴 17:00—17:30 → 30分钟
🏃 玩耍 17:40—18:10 → 30分钟
📥 下载空白表格 ✏️ 在线填写
📊 比较表（T形图）
用途：对比两个事物的差异
  1. 左边写"事项A"
  2. 右边写"事项B"
  3. 中间写出对比的方面
示例：对比"感觉时间"和"时钟时间"
左边：玩的时候时间过得快
右边：上课的时候时间过得慢
中间比较：同样的30分钟感觉不一样
📝 制作我的T形图
🌳 思维导图模板
用途：梳理时间相关概念之间的关系
  1. 中间写"时间"
  2. 画出分支：时、分、秒
  3. 再画分支：它们之间怎么换算？
  4. 加上你学到的应用场景
示例：中心"时间"→ 分支"小时"→ "1小时=60分钟"→ "1节课40分钟"
中心"时间"→ 分支"分钟"→ "1分钟=60秒"→ "我跳绳1分钟能跳多少个？"
🧠 开始画思维导图
🕐 在线时钟模拟器
用途：可调节指针的虚拟时钟，观察时间变化
  1. 打开模拟器网站
  2. 拖动指针调整时间
  3. 观察数字时钟的变化
  4. 记录你的发现
🕐 当前：2:30
🌐 toytheater.com/clock 🌐 mathsframe.co.uk 时钟模拟器
⏱️ 秒表/计时器
用途：精确测量短时间间隔
  1. 点击"开始"
  2. 活动结束时点"停止"
  3. 读取结果
试一试：你能单脚站立几秒？深呼吸一次几秒？走10步要几秒？
📱 推荐：手机自带秒表

🧮 时间换算计算器
用途：快速进行时、分、秒之间的换算
  1. 输入数字
  2. 选择单位（时/分/秒）
  3. 点击"换算"
  4. 查看结果
自己动手：用Excel或纸笔，自己做一个换算表！
💡 2小时 = 120分钟 = 7200秒

📈 数据图表制作工具
用途：把时间数据变成漂亮的图表
  1. 收集你的时间数据
  2. 用折线图、柱状图或饼图展示
  3. 分析：什么活动花的时间最多？
推荐：WPS/Excel图表功能、Canva在线图表制作

📅 日期计算器
用途：计算两个日期之间的天数、推算N天后的日期
  1. 输入起始日期
  2. 输入结束日期（或要加的天数）
  3. 查看结果
试试：从今天到放寒假还有多少天？
🌐 推荐：timedatecalc.com
