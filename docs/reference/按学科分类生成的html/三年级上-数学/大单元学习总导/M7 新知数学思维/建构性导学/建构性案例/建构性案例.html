🌱 第一阶 · 个人理解型

⏱️ "一分钟能做什么"大挑战

调用原理：原理1 从感觉到测量原理2 用数字说话

你和同学打赌"一分钟能做多少事"。有人说可以跳100下绳，有人说只能跳30下。到底谁说得对？

👆 点击展开详情

建构任务：设计并执行"一分钟能力测试"，制作一张"一分钟能力卡"。

  1. 猜测：写下你觉得一分钟能做多少下跳绳
  2. 测试：用秒表计时，实际跳一分钟
  3. 记录：写下实际成绩
  4. 对比：猜测 vs 实际差多少？
  5. 制作成果："我的专属一分钟能力卡"
🎯 成果：一份图文结合的"一分钟能力卡"，记录你在一分钟内能做的各种事。
📋 第二阶 · 生活应用型

📅 设计一个周末时间计划表

调用原理：原理2 用数字说话原理4 画出关系图

周末有作业、兴趣班、玩耍、看书……怎么安排才不冲突？给自己设计一份精确到分钟的计划表！

👆 点击展开详情

建构任务：给自己设计一份合理的周末时间表

  1. 列出任务：作业、兴趣班、玩耍、看书、吃饭、睡觉……
  2. 估计时长：每个任务大概需要多久？
  3. 安排顺序：先做什么，后做什么？
  4. 反复调整：时间冲突了怎么办？
  5. 制作表格：画一张漂亮的周末计划表
🎯 成果：一份周末时间计划表 + 实际执行记录（看看你执行得怎么样）。
🔍 第三阶 · 校园问题解决型

🏃 课间10分钟能做什么

调用原理：原理1 用工具延伸感官原理3 找差异背后的规律

同学们总说课间时间不够用，真的是时间太少，还是我们没用好？

👆 点击展开详情

建构任务：做一个课间10分钟行为调查，给出"高效课间方案"。

  1. 记录一周课间：每次课间做了什么、用了多久
  2. 分类统计：上厕所、喝水、聊天、玩游戏各占多少时间
  3. 发现"时间黑洞"：哪个环节最浪费时间？
  4. 提出改进方案：怎样安排课间最高效？
🎯 成果：课间行为分析报告 + 高效课间建议清单。
📖 第四阶 · 学科表达型

🎨 用数学故事讲时间

调用原理：原理3 从比较到发现原理4 从具体到抽象

一年级小朋友问你"什么是秒？"——你能用一个故事讲清楚时间单位吗？

👆 点击展开详情

建构任务：创作一本"时间的秘密"数学小绘本（至少4页）。

  1. 确定故事线：比如"小秒针的冒险旅程"
  2. 设计角色：秒针弟弟、分针姐姐、时针伯伯
  3. 融入单位关系：60秒=1分，60分=1时
  4. 配图配文：每页画图 + 简单文字
  5. 制作成册：装订成一本小书
🎯 成果：一本4页以上的时间主题数学绘本。
🔬 第五阶 · 跨学科整合型

🏃‍♂️ 时间与速度的关系

调用原理：原理2 用数字说话原理3 比较发现规律

体育课上，小明跑50米用了10秒，小红用了12秒。谁跑得更快？快多少？

👆 点击展开详情

建构任务：用时间和距离的关系判断速度。

  1. 收集数据：记录几个同学跑50米的时间
  2. 计算速度：每秒跑多少米？50 ÷ 秒数 = 每秒___米
  3. 比较：谁最快？谁最慢？
  4. 画柱状图：横轴是谁，纵轴是速度
  5. 下结论：用数据说话
🎯 成果：速度对比分析图 + 文字结论。
🌍 第六阶 · 社会情境型

🌏 如果全世界的时间都不一样

调用原理：原理4 画出关系图

北京的小朋友早上8点上学，纽约的小朋友那时在干嘛？为什么同一个地球，时间不一样？

👆 点击展开详情

建构任务：了解时区的概念，制作"世界时间对照表"。

  1. 了解时区概念：地球自转一圈24小时，分成24个时区
  2. 选择5个城市：北京、东京、伦敦、纽约、悉尼
  3. 找到时差：每个城市比北京早还是晚几小时
  4. 制作对照表：同一时刻这5个城市分别是几点
  5. 提出设想：如果全世界用同一个时间，会怎样？
🎯 成果：世界时间对照表 + 你的发现笔记。
💡 第七阶 · 综合创新型

⏳ 设计未来的计时工具

调用原理：原理1 从感觉到测量原理2 数字说话原理3 比较发现原理4 画出关系图

如果有一天所有时钟都坏了，人类怎么知道时间？用身边的材料做一个计时器！

👆 点击展开详情

建构任务：利用身边材料设计并制作一个简易计时器。

  1. 研究古法计时：日晷、漏刻、沙漏、蜡烛钟
  2. 选择材料：水瓶、沙子、蜡烛、木棍、纸盘……
  3. 设计方案：画设计图，想好怎么做
  4. 制作测试：动手做出来，用时钟检验准确度
  5. 改进迭代：哪里不准？怎么改进？
  6. 写说明书：让别人也能照着做
🎯 成果：一个可以实际运行的简易计时器 + 使用说明书。
