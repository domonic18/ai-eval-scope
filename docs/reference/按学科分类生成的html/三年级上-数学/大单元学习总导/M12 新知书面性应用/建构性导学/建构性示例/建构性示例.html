从起床到上学

一张时间表诞生的故事 · 建构性示例导学

📖 真实建构情境

你是三年级的小明，每天早上都要经历"起床→洗漱→吃早饭→上学"的过程。今天妈妈给你出了一道题：

小明早上7:30从家出发步行去学校，走了25分钟后到达学校。请问小明到校的时刻是多少？

这不是一道普通的计算题，而是你亲手构建一个"时间应用书面成果"的起点。

🎯 建构目标

构建一个完整的"时间问题书面解答"成果，包含三大要素：

📊
时间轴图示
✏️
算式书写
💬
答句表达

成果名称：《我的时间书面应用示例》（第一版）

最低标准：不少于80字，包含时间轴、算式和答句

🔧 建构过程示范（6步）

步做什么怎么做产出
1 圈信息 圈出"7:30""25分钟""到校时刻" 圈注后的题目
2 画时间轴 左端写"7:30"，右端写"？"，中间注"走路25分钟" 时间轴草图
3 想关系 判断"起点+时长=终点" 算式思路
4 列算式 7时30分 + 25分 = 7时55分 完整算式
5 写答句 答：小明到校的时刻是7:55。 规范答句
6 检查 7:55减去25分钟是不是回到7:30？是的！ 验证结论
📝 第1步：圈信息

在原题上圈出所有时间词和数据。妈妈给出了四个关键信息：时间点"7:30"、时长"25分钟"、问的是"到校时刻"、人物是"小明"。

📝 第2步：画时间轴

画一条带箭头的横线，左端点标注"7:30"，中间标注"走路25分钟"，右端点标注"？到校时刻"。

🧰 支架工具

⏱ 时间轴模板

起点____→（经过____）→终点____

📝 关键句式

"___时___分 + ___分 = ___时___分"

✅ 检查清单

  • ☐ 圈了数据
  • ☐ 画了时间轴
  • ☐ 写了算式
  • ☐ 写了答句
  • ☐ 做了验算

⭐ 建构成果样例

📋 《小明上学时间书面分析》
时间轴：7:30 →（25分钟）→ 7:55
算式：7时30分 + 25分 = 7时55分
答句：答：小明到校的时刻是7时55分。
验算：7时55分 − 25分 = 7时30分 ✓

✨ 成果优化建议

  • 把"7:30"写成"7时30分"更规范
  • 时间轴上加小箭头表示时间方向
  • 答句中写上单位"时"和"分"
