二、名著部分

📖 《中国古代计时器》
基础阅读

核心问题：在没有时钟的古代，中国人怎么知道时间？

核心观点：介绍了日晷（看太阳影子）、漏刻（滴水计时）、更鼓（夜间打更报时）等古代计时方法。
"日晷利用太阳投影的方向变化来指示时刻，一天中晷针的影子从西到东转动一圈——这就是最早的'时针'。"

推荐章节：👉 "古人如何知道时间"

🪐 《时间简史》（儿童版 / 图解版）
进阶阅读

核心问题：什么是时间？有开始和结束吗？时间旅行可能吗？

核心观点：从大爆炸讲起的宇宙时间故事，解释了为什么时间有方向。
"时间箭头"——为什么鸡蛋能变成煎蛋，煎蛋不能变回鸡蛋？这就是时间的"单向性"！

建议：配合图解版阅读更容易理解。

🧠 《给孩子的关于时间的书》
基础 进阶

核心问题：为什么星期一到星期五感觉好慢，周末却过得飞快？

核心观点：从心理学、生物学、物理学的角度解释我们对时间的感受。
