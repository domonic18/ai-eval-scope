A层 — 下课时间计算

题目：一节课40分钟，9:00上课，几点下课？

提示：9:00 + 40分钟 = ？ 先想9:00过40分是几时几分。

解答：9:00 + 40分钟 = 9:40。答：9时40分下课。

B层 — 多任务时间推算

题目：小明8:15开始写作业，写了45分钟，又读了25分钟书。他完成所有任务的时间是几点？

提示：分两步算——先算写完作业的时间，再加读书时间。

解答：8:15 + 45分 = 9:00（写完作业）。9:00 + 25分 = 9:25。答：他完成所有任务的时间是9时25分。

C层 — 秋游行程安排

题目：学校组织秋游，8:30出发，往返车程各45分钟，景点游览2小时。按时间顺序写出完整的行程安排表。
