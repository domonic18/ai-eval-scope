🧰 工具库

时间书面应用工具 — 把你的时间思维武装起来！

💡 思维工具

工具1：时间轴模板 必备

🎯 用途：把抽象的时间变成看得见的线，再也不怕算错时间！

📋 使用步骤：

  1. 画一条带箭头的横线 → 表示时间在向前走
  2. 左端写 起点时刻（比如7:30）
  3. 右端写 终点时刻（比如7:55）
  4. 在线段中间标注经过的时间和关键事件
✏️ 试试看：小明7:30出门，走路25分钟到学校，到校时间是？ 起点 7:30 ———(走路25分钟)———→ 终点 7:55 ✅

适用场景 所有时间应用题都能用！

工具2：解题四步模板 超实用

🎯 用途：规范书面表达，让你的解题过程清晰又完整。

📋 四步法：

  1. 读题圈数据 — 把题目中的时间数字圈出来
  2. 画时间轴 — 把圈出的数据标在线段上
  3. 列算式写单位 — 计算并写上单位（时/分/秒）
  4. 写答句并验算 — 完整回答，再倒推检查一遍
已知：起点____ 经过____ 终点____ 求：____ 时间轴：[画在这里] 算式：______ 答句：_______ 验算：_______
📝 复制上面的模板，自己出题试试填一遍！

🖥️ 数字工具

基础工具：在线时钟模拟器

🔗 推荐：Toy Theater Clock（访问 toytheater.com/clock）

🎯 用途：拨动指针学习认读时刻

👆 点击按钮观察时针和分针怎么联动

进阶工具：计时器 APP

📱 推荐：手机自带计时器 / Forest APP

🎯 用途：计时练习，训练你的"时间感"——猜猜1分钟有多长？

⏱️ 挑战：闭眼默数60秒，感觉到了就睁眼，看看手机秒表准不准！

🔬 实验工具

实物操作：机械手表或闹钟

⏰ 准备：找一个家里带指针的钟表（机械的更好！）

👀 观察任务：轻轻拨动分针，仔细观察——

  • 分针走一圈，时针走了多少？
  • 分针走半圈（30分钟），时针走几小格？

🔍 发现记录：

分针走1圈 → 时针走____大格 = ____分钟

分针走半圈 → 时针走____小格 = ____分钟
