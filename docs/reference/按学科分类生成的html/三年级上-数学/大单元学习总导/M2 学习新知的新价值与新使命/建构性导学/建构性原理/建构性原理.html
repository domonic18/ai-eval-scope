📖 4条建构性原理

把能量球拖入对应的气缸，点燃你的学习动力！

  1. 经验锚定法 — 新知识"挂在"已有经验上才能被理解
    用"刷牙两分钟""一节课40分钟"等生活经验理解时间单位。适用：任何新概念学习。注意：经验需验证。
  2. 动手体验法 — 亲身体验比被动听讲更能建立深刻理解
    闭眼数秒、走100步测时。适用：需要建立量感的概念学习。注意：体验需有目标和记录。
  3. 模型建构法 — 把抽象关系转化为可视化模型
    用纸条拼贴"秒→分→时"长度对比模型。适用：理解比例关系、层级结构。注意：模型要真实反映关系。
  4. 自述检验法 — 用自己的话说一遍是最有效的检验方式
    要求"用自己的话写一段关系说明"。适用：任何知识学习后自查。注意：说不清就是没真懂。
