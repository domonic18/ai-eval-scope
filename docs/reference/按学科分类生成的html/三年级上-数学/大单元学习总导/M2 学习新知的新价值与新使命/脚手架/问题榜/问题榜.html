❓ 问题榜

从基础到挑战，用问题驱动思考！点击筛选不同难度层级的题目。

理解
1时等于多少分？1分等于多少秒？怎么换算的？
识别
秒针、分针、时针在钟面上各有什么特征？
想一想：哪根针走得最快？哪根针最短？它们分别代表什么？
应用
哪些生活场景用"秒"来计量？哪些用"分"？哪些用"时"？
应用
你能从钟面上读出"3时15分"吗？怎么看的？
短针（时针）过了3一点点，长针（分针）指向3——分针每大格是5分钟。
💭开放性问题
开放
为什么一个小时是60分钟，而不是100分钟？你觉得这个设定合理吗？
和古巴比伦人有关——他们使用60进制。60可以被2、3、4、5、6整除，很方便！
开放
如果没有钟表，给你一杯水和一个碗，你能设计出计时1分钟的方法吗？
⚖️对比性问题
对比
感觉上的"1分钟"和实际测量的"1分钟"一样长吗？为什么会有差异？
闭上眼睛，当你觉得过了1分钟时睁眼，看看实际过了多久？玩游戏时和等待时，感觉一样吗？
对比
心跳计数和秒表计时，哪个更准？为什么？
🔄反事实问题
反事实
如果1分钟只有30秒，世界会变成什么样？
反事实
如果你能控制时间快慢，你最想用在什么场景？
💡创造与批判型问题
创造
时间明明看不见摸不着，我们凭什么说"它真的存在"？你怎么向外星人解释"时间"？
我们可以用"变化"来证明时间——树叶变黄、人变老……如果没有变化，时间还存在吗？
创造
为什么人越长大，感觉时间过得越快？你能设计一个实验来验证这个现象吗？
🧠元认知问题
元认知
遇到一个"与时、分、秒有关的新问题"时，你通常先做什么？
元认知
你怎么判断自己是不是真的理解了一个时间换算问题？
试试"教给别人"——如果你能清楚地讲给同桌听，让他也学会，那你就真的理解了！
元认知
从"不会做时间换算题"到"会做"，你经历了怎样的思考过程？
