挑战 · 前沿/跨领域
🛰️ 给地球拍"延时摄影"——卫星如何用时间看地球变化？
NASA的卫星每天拍摄同一地点的照片，用时间差来发现变化
如何利用时间维度来观察地球变化？
① 确定观察目标（冰川、城市、森林……）
② 设定时间间隔（每天、每月、每年）
③ 对比分析变化（同一地点不同时间）
通过多年对比，可以看到冰川消融、城市扩张、森林变化
时间维度是理解"变化"的关键；很多问题放在时间轴上才能看清趋势
找两张同一地点不同年代的照片，分析变化

可以搜索"城市 老照片 对比"或"冰川 对比 照片"。
