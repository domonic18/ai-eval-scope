🔬 奠基人物

🔭
伽利略·伽利莱 1564—1642
意大利 · 现代科学之父 · 发现单摆等时性
关键问题
如何精确测量时间？（当时钟表非常不精确）
思维突破
不改造钟表，而是发现摆的规律——摆长固定，每次摆动时间相同
失败与争议
他的"思考实验"方法被当时学者质疑，被认为不够严谨
对当下的启示
解决问题有时不用改造工具，而是发现自然规律

你面前有一根绳子和一个螺母，你能用它来计时 1 分钟 吗？试试看，用摆的规律来设计！

⚙️ 跨界人物

⏰
克里斯蒂安·惠更斯 1629—1695
荷兰 · 物理学家 · 发明摆钟
关键问题
如何让钟表走得更准？
思维突破
把伽利略发现的摆的规律做成实用机器——摆钟，精度从每天差15分钟提升到10秒
失败与争议
摆钟在海上不好用——船一摇晃，摆就不准了
对当下的启示
基础科学发现 → 工程应用 → 发现新问题，这是科学进步的循环
伽利略 发现摆的规律（科学） → 惠更斯 发明摆钟（技术） → 发现海上计时不准确（新问题） → 催生航海钟 → 再到今天的原子钟。每一次解决一个问题，就打开一扇新的大门！

⚛️ 当代人物

🕰️
路易斯·埃森 & 原子钟团队 1949
英国 · 第一台实用原子钟 · 重新定义"秒"
关键问题
如何用原子震荡（而不是机械摆动）来计时？
思维突破
铯-133 原子每秒跃迁 9,192,631,770 次——用它定义"秒"
历史意义
1967 年起，全世界的"1 秒"用原子常数定义，不再依赖天文观测
对当下的启示
时间不再是模糊的感觉，而是精确的自然常数

如果你来给"秒"下一个新定义，你会用什么参照物？心跳？沙漏？还是别的什么？
