🗣️ 讨论与协作

和同学们一起，围绕"时间"展开有趣的讨论吧！

⏱ 10分钟 辩论：5分钟很长 VS 5分钟很短

为什么每个人的"时间感受"不一样？等下课铃的时候5分钟像1小时，玩游戏的时候1小时像5分钟！

⏱ 15分钟 小组互评：分享"时间分析报告"

把自己的时间记录表展示给组员，找2位同学给你提建议。看看谁的时间安排最合理？

⏱ 8分钟 头脑风暴：如果没有秒针……

"如果没有秒针，只看时钟，生活会有什么变化？" 大胆想象，什么答案都可以！

🏠 家庭参与

📊全家"等待时间"大调查

和父母一起算一算：全家一天花在"等待"上的时间有多少？点击查看提示

提示：等公交车、等上菜、等下课、等妈妈化妆……把这些"等待时间"加起来，你和家人会不会吓一跳？
👴采访爷爷奶奶的"时间故事"

他们小时候没有电子表，是怎么看时间的？点击查看采访小贴士

可以问：您小时候怎么知道几点起床？➡ 那时候如果迟到了怎么办？➡ 您觉得现在的钟表和以前的有什么不同？
📵"无屏幕周末"挑战

设定一个没有手机、电视的周末，省出的时间用来做什么？点击查看挑战规则

规则：周六上午8点—周日晚上8点，不看任何电子屏幕。省下的时间可以用来：读书、户外运动、做手工、陪家人聊天……记录你省出了多少"额外时间"！

🔗 社会连接

🔧手表维修店探访
