🧰 工具库

用好工具，让时间分析更轻松、更有趣！

🧠 思维工具

这些"大脑工具"帮你把时间想清楚、理明白。点击工具卡片查看使用步骤。

📅
时间轴
按时间顺序排列事件
三步搞定：
① 确定起止时间 → ② 标刻度 → ③ 填入事件
示例：7:00起床 → 7:30出发 → 8:00上课
✅ 适用：记录一天安排
📊
双栏对比表
对比两个时间段异同
三步搞定：
① 左边列A时段 → ② 右边列B时段 → ③ 对比差异
示例：昨天vs今天的起床时间对比
✅ 适用：分析时间变化
🐟
鱼骨图
分析时间浪费的原因
三步搞定：
① 鱼头写问题 → ② 画出原因分支 → ③ 找出根本原因
示例："作业写太慢"的原因分析
✅ 适用：找时间管理问题

📊 条形图

比较不同活动的时间消耗，一眼看出哪件事花时间最多！

🥧 饼图

展示时间分配比例——学习占了多少？游戏呢？一目了然！

📋 统计表

每天记录各项活动的时间数据，连续记录一周，看看你的时间花在哪了。
