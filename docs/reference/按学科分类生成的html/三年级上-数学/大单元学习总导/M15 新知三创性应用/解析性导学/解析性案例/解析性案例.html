🏠 生活情境

案例一：早餐时间够不够？

7:10起床，洗漱10分钟，吃早餐15分钟，7:50要到校。时间够不够？

🎯 解析任务：计算各项活动总时长，判断是否来得及
调用原理：信息拆解 + 分步计算
先算总需要多少分钟，再和"7:10到7:50共多少分钟"比较
🏫 校园情境

案例二：一节课到底上了多久？

第一节课8:00开始，下课铃8:40响，但老师说"再讲2分钟"。

🎯 解析任务：计算8:00到8:40的时长，再加2分钟
调用原理：单位转化 + 分步计算
两种方法都能算：8:00→8:40=40分，+2分=42分；或直接8:00→8:42=42分。关键在于分钟满60要进位吗？
🏪 家庭/社区

案例三：超市营业时间多长？

超市门口写"营业时间 8:30—21:00"，一天营业多长时间？

🎯 解析任务：计算从开始到结束的营业总时长
调用原理：信息拆解 + 逆向验证
上午8:30到中午12:00是多久？下午12:00到21:00是多久？两个相加
📐 学科核心

案例四：谁跑得更快？

3位同学跑100米——甲用25秒，乙用了半分钟，丙用了0.4分钟。谁第一？

🎯 解析任务：统一单位到秒，再比较大小
调用原理：单位转化（核心）+ 信息拆解
半分钟=30秒，0.4分钟=24秒。注意：跑步名次和所用时长的关系是相反的——用时最短的才是第一名！
🌍 跨学科

案例五：时差问题

北京和伦敦有8小时时差。伦敦上午9:00，北京是几点？北京晚上8:00，伦敦是几点？

🎯 解析任务：根据时差推算不同城市的时间
调用原理：情境锚定 + 逆向验证
向东加时差，向西减时差。伦敦9:00→北京17:00（下午5点）；北京20:00→伦敦12:00（中午12点）
🚗 社会议题

案例六：限时停车收费

停车场"15分钟内免费，超过后每小时8元"。10:05停好，11:20取，要交多少钱？

🎯 解析任务：算停车时长→判断免费→计算费用
调用原理：分步计算 + 信息拆解
停了1小时15分→减免费15分→收费1小时→交8元。不满1小时按1小时算
🚀 综合迁移

案例七：设计班级作息表

上午4节课，每节40分钟，课间10分钟，早读15分钟8:00开始，午餐11:40开始。请你设计作息表。

🎯 解析任务：推算每节课和课间的具体起止时间
调用原理：全部5条原理综合运用
从8:00开始，按"上课40分→课间10分"的节奏依次推算。8:00-8:15早读，8:15-8:55第一节课……
