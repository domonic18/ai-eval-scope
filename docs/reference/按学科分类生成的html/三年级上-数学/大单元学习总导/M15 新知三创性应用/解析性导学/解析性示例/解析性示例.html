🎬 电影到底什么时候结束？

跟着专家一步步学会"已知开始时间和经过时间，求结束时间"

📖 解析任务情境

周六下午，小明和妈妈去看电影。海报上写着"片长95分钟"，电影2:15开始放映。妈妈问小明："我们几点能出来？"小明看了看手表，犯难了——"2:15加上95分钟，怎么算？"

核心问题：从2:15开始，经过95分钟，结束时间是几点几分？

📋

关键信息提取

情境信息说明为什么重要
开始时间：2:15时针指向2和3之间，分针指向3这是计算的起点
片长：95分钟即1小时35分钟（60+35）这是要加上的经过时间
结束时间：未知需要计算这是我们要找的结果

🧩 专家解析过程（5步示范）

1

先看见什么

👀 观察：开始时间2:15，经过时间95分钟
🤔 思考："95分钟超过60分钟了，需要换算"
💡 为什么这样想：因为分钟满60就要进1小时
2

判断问题属于哪一类

📌 判断：这是"已知开始时间和经过时间，求结束时间"的时间计算问题
🔧 方法提示：用加法（开始时间 + 经过时间 = 结束时间）
3

调用哪些方法

📐 关键方法：把95分钟拆成"60分钟（1小时）+ 35分钟"
📏 换算关系：1小时 = 60分钟
📝 分步计算：先加小时，再加分钟
4

进行推理分析

2:15 + 95分钟 = 2:15 + 1小时35分钟
先加分钟：15分 + 35分 = 50分
再加小时：2时 + 1时 = 3时
✅ 结果：3:50
5

形成结论

🎬 电影大约在3:50结束
🔄 验证：3:50 − 2:15 = 1小时35分 = 95分钟 ✓

✨ 方法提炼

以后遇到类似问题，可以这样做：

① 先看经过时间是否超过60分钟 → ② 超过就换成'几小时几分' → ③ 分三步：先加分钟、分钟满60进1小时、再加小时 → ④ 最后检验

⚠️ 易错提醒

❌ 直接加数字：2:15 + 95 = 2:110 → 错！分钟不能超过60
❌ 忘记进位：2:15 + 95分 = 3:50，不是2:110
❌ 单位混淆：95分钟不是95小时，一定要先换算

✏️ 自主迁移小练习

一部动画片从4:40开始，播放了45分钟，几点结束？

💡 先看看45分钟有没有超过60分钟？需要用上面的哪一步方法？
答案：4:40 + 45分 = 5:25
45分钟 < 60分钟，不需要换算进位，直接加分钟即可：40分 + 45分 = 85分 → 85 − 60 = 25分，进位1小时 → 4时 + 1时 = 5时 → 5:25
