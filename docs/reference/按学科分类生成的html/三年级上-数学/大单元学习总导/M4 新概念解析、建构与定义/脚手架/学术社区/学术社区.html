🔥 核心议题 核心

为什么时间的进制是60，而长度、重量的进制是10？

📋 准备材料

60的因数有哪些？10的因数有哪些？我们来对比一下：

60 = 2×30 60 = 3×20 60 = 4×15 60 = 5×12 60 = 6×10 10 = 2×5

讨论目标：理解60进制的数学优势——60能被10个数整除，而10只能被4个数整除（1、2、5、10）

🧠 拓展议题 拓展

如果没有钟表，我们还能怎样知道时间？

讨论目标：激发对"时间测量本质"的思考——太阳、星星、沙子、水流、心跳……时间测量的方法远不止钟表一种。

💡 想一想：古人没有手机，他们怎么知道"该起床了"？

🎯 趣味议题 趣味

你觉得一分钟有多长？我们来做同步闭眼计时挑战！

讨论目标：感受主观时间与客观时间的差异——心里默数60秒，和实际60秒一致吗？为什么有时候感觉时间过得快，有时候过得慢？

🕐 试试看：全班一起闭眼，每个人觉得过了1分钟就坐下，看看谁最接近！
