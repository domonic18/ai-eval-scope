📊 分层任务

A层 基础巩固

  • 150分 = ___ 时 ___ 分
  • 4分15秒 = ___ 秒
  • 比较大小：120分 ○ 2时
  • 比较大小：100秒 ○ 2分
150分 = 2时30分 · 4分15秒 = 255秒 · 120分 = 2时 · 100秒 < 2分(120秒)

B层 综合应用

  • 小明7:30出门，步行到校需要15分钟，他到校是什么时间？
  • 一节课40分钟，如果9:00上课，什么时候下课？
7:30 + 15分 = 7:45 · 9:00 + 40分 = 9:40

C层 拓展挑战

  • 三个人赛跑：甲用120秒，乙用2分30秒，丙用145秒，谁最快？
  • 设计一个「时间换算小转盘」——把时、分、秒换算关系做成可旋转的工具
乙: 2分30秒 = 150秒 · 甲: 120秒 < 145秒 < 150秒 → 甲最快
