🌞 项目一：制作简易日晷

⏱ 短周期 · 1~3天

项目目标：亲手制作一个日晷，亲眼验证太阳与时间的神奇关系！古人没有钟表，就是靠太阳的影子来计时的哦。

📅 阶段任务

第1天 · 材料准备

准备材料：纸板（硬一点更好）、铅笔、量角器、指南针（或手机指南针App）。按下面步骤制作日晷底座和指针。

  1. 在纸板上画一个半圆（半径约10cm），用量角器每隔15°画一条线
  2. 在圆心处垂直插入铅笔（作为指针）
  3. 标注方向：用指南针找到正北，将日晷12点方向对准正北
  4. 标上时间刻度（6:00~18:00）
第2天 · 观察记录

选一个晴天！从早上8点到下午4点，每隔1小时在日晷上标记影子的位置，并在记录表上写下时间。

第3天 · 对比分析

对比你的日晷读数和真实钟表。写一份"我的日晷使用报告"——有哪些误差？为什么会有误差？

🏆 评价标准

  • 日晷结构完整，指针竖直稳固 ✓
  • 至少记录5个时间点的影子位置 ✓
  • 对比分析准确，能说清楚误差原因 ✓

📝 项目二：我的一周时间日记

⏱ 中周期 · 1~2周

项目目标：连续记录自己一周的时间使用情况，学会用数据"看见"时间，并优化自己的时间分配！

📅 阶段任务

第1~7天 · 每日记录

每天记录各项活动所用的分钟数：学习、游戏、运动、吃饭、睡觉、其他。

第8天 · 制作统计图

汇总一周数据，制作"一周时间饼图"或"时间条状图"。可以手绘，也可以用Excel。

第9~10天 · 分析与改进

分析你的时间使用情况：哪些时间用得好？哪些浪费了？写出你的时间优化方案。

🏆 评价标准

  • 连续记录7天，无缺漏 ✓
  • 统计图表清晰美观，标注完整 ✓
  • 改进方案具体可行，有具体行动步骤 ✓

🔍 项目三："时间的秘密"综合探究报告

⏱ 长周期 · 1个月+

项目目标：选择一个与"时间"相关的课题，像科学家一样做一次深度探究，形成属于你自己的研究报告！

🎯 可选课题

  • 选题A：古代文明如何计时——古埃及、中国、玛雅对比研究
  • 选题B：计时工具的发展史——从日晷到原子钟
  • 选题C：时间在自然科学中的角色
  • 选题D：自选课题（需经老师确认）

📅 阶段任务

第1周 · 确定选题 & 收集资料

确定研究方向，通过图书、网络、博物馆等渠道收集资料。至少收集3个不同来源。

第2周 · 整理提纲

筛选整理资料，形成报告提纲：引言 → 主体（3~5个方面）→ 自己的观点 → 结论

第3周 · 撰写探究报告

图文结合，不少于800字。要有自己的分析和观点，不只是抄资料。

第4周 · 展示与分享

准备展示作品（海报 / PPT / 小视频 三选一），在班级或项目社区中分享你的研究成果。

🏆 评价标准

  • 选题有意义，有探究价值 ✓
  • 资料来源可靠，至少3个不同来源 ✓
  • 有自己的观点和分析，不只是搬运资料 ✓
  • 表达清晰、美观，展示有吸引力 ✓
