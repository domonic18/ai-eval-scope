《周髀算经》中的历法思想

中国 · 约公元前1000年成书
基础
❓ 核心问题

古人怎么用数学计算一年有多长？

📌 核心观点

通过日影测量确定节气，划分时间——用一根竿子就能算出一年的长度！

📖 关键片段导读

阅读「日中立竿测影」段落——古人在地上立一根竿子，每天正午测量影子的长度。影子最短那天是夏至，最长那天是冬至，两次最短之间的天数就是一年。

霍金《时间简史》（儿童版）

英国 · 1988年初版
进阶
❓ 核心问题

时间是什么？它有起点吗？

📌 核心观点
  • 时间与空间是连在一起的——叫「时空」
  • 时间是相对的，不是绝对的
  • 黑洞附近的时间会变慢
📖 关键片段导读

「时间的箭头」章节——为什么时间只能向前，不能倒退？霍金用"三个箭头"解释：热力学箭头（东西会旧）、心理学箭头（我们会老）、宇宙学箭头（宇宙在膨胀）。三个箭头指向同一个方向——这就是为什么我们不能回到昨天。

🧪 自己试一试

把一个鸡蛋打碎→煎熟。你能让熟鸡蛋变回生鸡蛋吗？热力学箭头告诉我们：时间只能向前走，不能倒退。

《DK时间图谱》

英国 · DK出版社
基础
❓ 核心问题

从古到今，人类如何测量和记录时间？

📌 核心观点

计时工具的演变史——每一种新工具都让人类对时间的认识更精确一步

📖 关键片段导读

「从日晷到原子钟」章节——每个时代的计时突破都以图文形式呈现，特别适合三年级学生阅读。

☀️ 日晷（公元前3500年）

用太阳影子指向的刻度来看时间——晴天才能用

💧 水钟（公元前1500年）

利用水流速度均匀的特性来计时——晚上也能用了

⏳ 沙漏（中世纪）

沙子从一端流到另一端——便于携带的计时器

🕰️ 摆钟（1656年）

惠更斯利用伽利略的发现制造出第一台摆钟

⏱️ 石英钟（1927年）

利用石英晶体的压电效应——精度大幅提升

⚛️ 原子钟（1955年）

用铯原子振动定义秒——3000年误差不到1秒
