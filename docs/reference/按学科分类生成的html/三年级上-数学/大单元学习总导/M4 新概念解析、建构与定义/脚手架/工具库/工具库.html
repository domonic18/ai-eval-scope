概念建构

概念图模板

把时、分、秒的关系可视化，一目了然

  1. 中心写"时间单位"
  2. 分出 "时""分""秒" 三个分支
  3. 标注换算关系：1时=60分，1分=60秒
📏
顺序分析

时间轴模板

按时间顺序排列事件，看懂时长

  1. 画一条横线 → 标注起点和终点
  2. 标记中间关键时间点
  3. 计算每段间隔的时长
⏱️
精确测量

秒表 / 计时器

亲手测量时间，验证你的"感觉"

  1. 按 开始 按钮
  2. 按 暂停 读数
  3. 记录秒数，和你的估计对比
00:00

熟练之后，用这些工具做对比分析

对比分析

T-Chart 对比表

同时对比多个活动的时间数据

  1. 左栏列项目（跑步、跳绳、写字）
  2. 右栏列时间（秒数）
  3. 计算差值和比例

💡 示例：跑步35秒 vs 跳绳50秒，相差15秒

学习管理

KWL 学习表

梳理"已知→想知→学知"全过程

  1. K栏：我已知的时、分、秒知识
  2. W栏：我还想知道什么
  3. L栏：学习后我学到了什么
