📖 背景故事

小明每天 7:00 起床，7:30 出门，8:00 上课。可他总是磨蹭——刷牙5分钟、吃饭15分钟、穿鞋3分钟……最后总是迟到。你有没有类似的经历？

❓ 核心问题

怎样规划早晨的时间才能不迟到？

🔀 决策路径
  1. 方案A：提前30分钟起床 → 但起不来 😴
  2. 方案B：计算每件事的时间，重新排列顺序 ⏳
  3. 方案C：给每件事设定闹钟限制 ⏰

🏛️ 进阶案例 · 行业/历史类
进阶

古罗马的"水钟"计时制度

📖 背景故事

古罗马没有手表，人们用 水钟 计时。法庭上每个人的发言都有时间限制——律师说完一段话，旁边的人就换一个水瓶，水漏完就轮到下一个人说话了！

❓ 核心问题

没有分、秒概念，古人怎么约定"发言不能太长"？

🔀 决策路径
  1. 用沙漏计时 → 沙子流速不稳定
  2. 用滴水量计时 → 更稳定，可以标准化
  3. 用燃香计时 → 一根香烧完约等于一个时辰
🧩 可迁移启示：计时工具的演变反映了人类对"精确"的追求——每一次进步都让我们的时间测量更准、生活更有序。

🚀 挑战案例 · 跨领域类
挑战

航天发射的"倒计时"为什么精确到秒？

📖 背景故事

火箭发射有 "发射窗口"——必须在某一秒精确点火，差一毫秒都不行！全球的测控站用 原子钟 同步时间，确保万无一失。

❓ 核心问题

发射早一秒或晚一秒会怎样？

🔀 决策路径
  1. 早一秒：可能撞上其他航天器
  2. 晚一秒：错过最佳轨道对接点，燃料不够调整
  3. 解决：用原子钟同步全球测控系统，精度达纳秒级
🧩 可迁移启示：精确计时是现代科技的基础。从火箭发射到手机信号，都离不开对"秒"的精确把握。
