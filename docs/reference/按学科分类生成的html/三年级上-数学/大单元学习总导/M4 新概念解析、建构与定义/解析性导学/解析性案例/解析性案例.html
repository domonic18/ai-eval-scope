🏫 谁来得更早？

情境：晨读8:00开始。小明到校是7:45，小红到校是7:50，小刚到校是8:05

📋 解析任务：谁来得最早？谁迟到了？分别早到或迟到了多少分钟？

建议原理：

要素拆解法 单位统一法
姓名到校时间距8:00的差距排名
小明7:45
小红7:50
小刚8:05
先把所有时间转化成"距8:00还有多少分钟"再比较：小明早到15分钟，小红早到10分钟，小刚迟到5分钟。用减法算出时间差，最早到的是小明。
2简单

📝 谁作业做得快？

情境：数学课上老师布置了3道题。A同学用了2分15秒做完，B同学用了130秒做完，C同学用了2分30秒做完

📋 解析任务：把他们按完成速度从快到慢排序

建议原理：

单位统一法 进率判断法
姓名原始用时转化为秒排名
A同学2分15秒
B同学130秒130秒
C同学2分30秒
先把"2分15秒"转换成秒：2×60+15=135秒；"2分30秒"=2×60+30=150秒。统一后：A=135秒，B=130秒，C=150秒。速度排名：B最快→A→C最慢。
3简单

🍕 一份披萨需要多久？

情境：妈妈烤披萨，说明书上写着"预热烤箱5分钟，烤制180秒"。爸爸说"预热加烤制一共8分钟就够了"

📋 解析任务：爸爸说得对吗？实际需要多少分钟？

建议原理：

要素拆解法 单位统一法 验证还原法
步骤原始时间统一后（分钟）
预热5分钟5分钟
烤制180秒
合计
180秒=180÷60=3分钟。5+3=8分钟——爸爸说得对！ 验证：8分钟减去5分钟等于3分钟，3分钟=180秒。前后一致，答案正确 ✅
4中等

⏰ 钟表的秘密

情境：秒针走一圈（60秒），分针走一小格（1分钟）。分针走一圈（60分钟），时针走一大格（1小时）

📋 解析任务：①秒针走2圈，分针走几小格？②分针走3圈，时针走几大格？

建议原理：

要素拆解法 进率判断法 验证还原法
问题已知关系进率计算过程答案
秒针2圈→分针?秒:分=60:160
分针3圈→时针?分:时=60:160
秒针和分针的关系是60:1——秒针走60圈，分针走1圈。所以秒针走2圈，分针走2小格。
分针和时针的关系也是60:1——分针走60圈，时针才走1圈（5大格）。所以分针走3圈，时针走3÷12=0.25大格（即1/4大格）。可以用"三针关系图"帮助理解。
5中等

☀️ 光的速度

情境：科学资料：光速约30万千米/秒。太阳离地球约1.5亿千米

📋 解析任务：太阳光到达地球需要多少分钟？

建议原理：

进率判断法 验证还原法
已知数据公式计算结果（秒）换算结果（分）
光速30万km/s
距离1.5亿km		时间=距离÷速度
先算时间（秒）：150000000÷300000=500（秒）。再把秒换算成分钟：500÷60≈8.33（分钟）。
所以我们现在看到的阳光，是大约8分钟前从太阳出发的！🌞
6中等

🚄 高铁和飞机谁更快？

情境：北京到上海约1200千米。高铁时速约300千米/时，飞机时速约800千米/时。高铁加进出站约需4小时30分，飞机加安检登机约需3小时

📋 解析任务：比较从北京到上海，坐高铁和坐飞机的总时间

建议原理：

要素拆解法 单位统一法
交通方式行驶时间附加时间总时间分析
高铁1200÷300=4小时进出站约30分4小时30分
飞机1200÷800=1.5小时安检登机约1.5小时3小时
虽然飞机飞得快（时速800km vs 300km），但加上安检、候机、登机的时间，实际总时间差距并没有想象中大。 启示：不能只看"速度"这一个因素，要全面考虑所有时间成本。

🏛️ 古人的一天

情境：① 古埃及人把白天分成12小时 ② 中国古代用"刻"计时，1天=100刻 ③ 欧洲中世纪用"祈祷钟"，一天分7个时段

📋 解析任务：分析三种计时方式的优缺点，和今天的24小时制比较

建议原理：

要素拆解法 单位统一法 进率判断法 验证还原法
计时体系划分方式优点缺点你的评价
古埃及白天12小时
中国"刻"1天=100刻
欧洲祈祷钟7个时段
现代24小时制24小时
今天的24小时制更精确、更方便全球统一；古埃及分法跟日照相关但不精确；中国的"刻"很有意思——后来改成了96刻（每刻15分钟）；古人的分法跟当时的生活需求密切相关，没有绝对的对错之分。
