📖 建构性示例：小明的"时间学习幸福地图"

一个三年级同学的真实故事——看看"好成果长什么样"

🎯 任务定位

通过一个完整的示范案例，展示"如何把学习收获的幸福转化为具体成果"。这个示例给你一个"半开放"的范例，让你看到——好成果就是这样一步步做出来的！

📌 建构目标

建构一份属于你自己的"学习幸福地图"——用图画和文字记录你在学习"时、分、秒"过程中最开心的时刻，展示你的收获和成长。

成果名称：我的"时间学习幸福地图"

形式：一张A4纸大小的手绘地图，包含至少3个"幸福站点"

用途：记录和展示你在学习时间单位过程中的幸福收获

👦 小明的故事

小明和你一样，也是三年级的学生。在学"1 时、分、秒"这个单元时，他一开始觉得"1分钟=60秒"好难记，总是搞混。

有一天，他在家里对着钟表数了60秒——滴答、滴答、滴答……他发现60秒其实并没有想象中那么短，而且数完之后，他居然真的记住了！那一刻，他高兴得跳了起来。

从此，小明开始留意：学习中还有哪些这样的"开心时刻"？他想把它们都记录下来。

✏️ 建构任务说明

项目内容
成果名称我的"时间学习幸福地图"
用途记录和展示你在学习时间单位中的幸福收获
对象你自己（也可以和同学、家人分享）
形式A4纸大小手绘地图
最低标准至少3个"幸福站点"，每个站点有图画和文字说明

🔄 建构过程示范（6步完成法）

1回忆——寻找你的"幸福站点"

做什么：回顾整个学习过程，找出那些让你开心、自豪或"原来如此"的时刻。

小明列出了：①第一次看懂钟表上的时间；②做对第一道时间换算题；③教会同桌看时间；④在1分钟跳绳比赛中跳了130下；⑤自己画出了时间单位关系图。
2选择——挑选最精彩的3个站点

做什么：从列表中挑选最想记录的3个时刻。问自己——哪个最开心？哪个最有收获？哪个最值得记住？

小明选了：站点①"原来钟表是这样看的"（最开心）、站点②"我做对了！"（最有成就感）、站点③"我教会了同桌"（最自豪）。
3构思——为每个站点设计内容

做什么：每个站点回答三个小问题：①当时发生了什么？②我是什么感觉？③我学到了什么？

站点①：画一个大钟表，写"原来分针走一圈就是1小时！"
站点②：画一道做对了的换算题，写"150分=2时30分，我算出来了！"
站点③：画两个人，写"我把看时间的方法教给了同桌"
4画图——把地图画出来

怎么做：把纸想象成一张"藏宝图"，3个站点之间用路线连接起来，画上路标、箭头和小装饰。

小明把纸横过来用，左边画起点"第一节课"，右边画终点"单元结束"，中间依次排列3个站点，用彩色箭头连起来。
5配文——为每个站点写"幸福注释"

怎么做：用"当时……我……因为……"的句式来写，说清楚为什么这是"幸福时刻"。

站点①旁边写："当时我盯着钟表看了好久，突然发现分针走一圈时针就走一小格——原来1小时=60分钟是这个意思！我特别开心，因为我真的看懂了！"
6美化与分享——让它更完整

做什么：给地图加上标题、日期、签名，然后分享给同学或家人。

小明在地图上方写上标题"我的时间学习幸福地图"，右下角签上名字和日期。

🏆 成果样例评价

评价要求：

  1. 科学性：时间知识表达准确（如1小时=60分钟不能写错）
  2. 完整性：包含至少3个站点，每个站点图文结合
  3. 美观性：色彩搭配协调，布局合理，字迹工整

推荐布局：纸张横放，标题在上方居中，从左到右依次排列站点用箭头连接，每个站点左图画右文字。

建议：先在第1-5步中把草稿写在这份导学案的表格里，再正式画到纸上。画图时先用铅笔打草稿，再用水彩笔上色。
