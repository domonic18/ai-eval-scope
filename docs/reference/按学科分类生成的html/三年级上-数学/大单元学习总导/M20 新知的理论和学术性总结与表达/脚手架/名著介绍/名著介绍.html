《周髀算经》节选

中国 · 古代天文数学经典
挑战级（需家长辅助）
❓ 核心问题

古人是怎么测量时间的？在没有钟表的情况下，他们如何知道"几点"？

📋 目录结构（简化）

全书分"天文测量"和"数学计算"两大部分，涉及用日影长度判断时辰的方法。

💡 核心观点摘要

古人用"立竿见影"的方法——竖一根杆子，观察影子长度和方向的变化，就知道时间。

🔍 关键片段导读

找一找书中关于"晷影"的记载，看看古人是如何用8尺高的竿子测量出一年的长度的。

《时间简史》（儿童版）

史蒂芬·霍金 · 湖南科技出版社
基础级（三四年级可读）
❓ 核心问题

时间有起点吗？时间会停止吗？为什么我们不能回到过去？

📋 目录结构（简化）

宇宙是什么样的 → 时间是什么 → 黑洞 → 时间旅行可以吗？

💡 核心观点摘要

霍金用最通俗的语言解释了"时间"在物理学中的含义——时间不是固定不变的，它和空间一起构成了"时空"。

🔍 关键片段导读

第三章"时间是什么"——霍金用"烤面包"的例子说明时间的方向性：面包烤好了就不能变回生面团。

《时间的奥秘》——科普绘本

跨学科 · 当代作品
基础级（独立阅读）
❓ 核心问题

人类是怎么从"看太阳"到"原子钟"的？

📋 目录结构

日晷 → 沙漏 → 水钟 → 蜡烛钟 → 机械钟 → 石英钟 → 原子钟

💡 核心观点摘要

计时工具的发展史，就是人类追求"越来越精确"的历史。

🔍 关键片段导读

关于"为什么钟表向右转"的解释——是因为日晷的影子在北方是顺时针走的！
