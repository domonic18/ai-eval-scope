⏱ 作业榜：时、分、秒

通过分层练习，检验你对"时、分、秒"知识的掌握程度，并挑战更高层次的应用和表达。

基础层：再现与理解

一、填空题

  1. 1时 = _____ 分
  2. 1分 = _____ 秒
  3. 1时 = _____ 秒
  4. 2时 = _____ 分
  5. 3分 = _____ 秒
答案：① 60 ② 60 ③ 3600 ④ 120 ⑤ 180

二、换算题

  1. 120分 = _____ 时
  2. 180秒 = _____ 分
  3. 3600秒 = _____ 时
  4. 2时15分 = _____ 分
  5. 1分30秒 = _____ 秒
答案：① 2 ② 3 ③ 1 ④ 135 ⑤ 90

分层任务

选择适合你难度的层级，点击展开对应题目。

A 基础巩固 B 能力提升 C 思维挑战
A 基础巩固

计算经过时间：

  1. 从 8:00 到 8:45，经过了 _____ 分钟
  2. 从 9:30 到 10:15，经过了 _____ 分钟
  3. 下午 2:00 到下午 4:30，经过了 _____ 小时 _____ 分钟
答案：① 45分钟 ② 45分钟 ③ 2小时30分钟
B 能力提升

解决实际问题：

  1. 小明 7:30 出门，路上用了 25 分钟，他几点到校？
  2. 一列火车 9:15 从 A 站出发，10:02 到达 B 站，行驶了多少分钟？
  3. 小丽跳绳，1 分钟跳了 86 下，她跳 30 秒能跳多少下？（估算）
答案：① 7:55 到校 ② 47分钟 ③ 约43下（30秒是1分钟的一半，86÷2=43）
C 思维挑战
  1. 三个同学赛跑，甲用了15秒，乙用了¼分，丙用了12秒，谁最快？请写出比较过程。
  2. 小华看一场电影，电影从下午2:10开始，共放映1小时45分，电影几点结束？结束后他走路15分钟到家，到家时是什么时间？
  3. 请你设计一个问题，让解题者同时用到"时→分"和"分→秒"两种换算。
思路提示：
① ¼分 = 15秒，所以甲15秒、乙15秒、丙12秒 → 丙最快。
② 2:10 + 1:45 = 3:55 结束，3:55 + 0:15 = 4:10 到家。
③ 示例：一场电影2小时15分钟，合多少秒？（2时15分 = 135分 = 8100秒）
