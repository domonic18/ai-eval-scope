🎯 本任务定位

通过互动观察和操作，感受钟表内部精密的机械结构，理解齿轮传动与指针运动的数学关系。

看看齿轮是怎么转动的，感受秒针、分针、时针之间的转速比例关系——秒针转一圈 = 分针转一小格。

👆 操作
