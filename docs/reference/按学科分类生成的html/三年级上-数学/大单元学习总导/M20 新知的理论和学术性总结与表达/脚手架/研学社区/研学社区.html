🎯 核心问题：电视台怎么保证倒计时和新年到来那一刻完全吻合？

⏳ 历史案例

航海经度难题

18世纪，时间精度曾经是关系到生死的重大问题

🎯 核心问题：如何在摇晃的船上精确计时，从而算出经度？
✕ 方案A：看月亮的位置——天气不好就看不到
✕ 方案B：在大海上摆个钟悬着——摇晃导致走不准
✓ 方案C：哈里森的航海钟——用弹簧和平衡轮解决了晃动问题
⌛ 哈里森花30年做出了H4航海钟，精度达到每天误差不到1秒
💡 可迁移启示：时间精度在历史上曾经是关系到生死的重大问题

GPS与时间同步

你手机里的导航App能告诉你位置，靠的是卫星发来的"时间信号"

🎯 核心问题：为什么需要极精确的时间？
📡 4颗卫星同时发出时间信号
📱 手机收到信号的时间差，反映了卫星到手机的距离差
📍 通过距离差计算出你的位置
⚠️ 关键：如果时间误差1秒，定位偏差就是30万公里（光速）
🛰️ GPS卫星搭载原子钟，精度达到纳秒级
💡 可迁移启示：你学的"时、分、秒"是理解这一切的基础
