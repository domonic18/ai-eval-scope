为什么时间用60进制？

60可以被2、3、4、5、6、10、12、15、20、30整除，是1~6的整数中除了7以外都能整除的最小数字。古人选择60进制，是因为它特别适合平均分配——把1小时平均分成2份、3份、4份、5份、6份，都能得到整数分钟。如果是10进制，把1小时平均分成3份就得不到整数了！
