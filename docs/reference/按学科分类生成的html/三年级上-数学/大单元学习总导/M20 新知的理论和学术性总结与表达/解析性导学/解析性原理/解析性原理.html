🧰 解析性原理

从示例到原理——把具体的分析步骤提炼成可以反复使用的"分析工具"

前面你跟着解析性示例，一步一步拆解了"时间知识总结"的结构和方法。现在，你需要把这些具体的分析步骤提炼成可以反复使用的解析性原理。这些原理就像你工具箱里的"分析工具"——以后遇到任何需要分析的知识文本或学习材料，你都可以拿出来用。

🔧 五大解析性原理

1整体感知法

▾
原理说明

拿到一段知识总结，先不急着看细节。先快速浏览一遍，问自己三个问题：这段在说什么？分了几部分？整体是什么结构？

在示例中的体现

示例第一步就是"先看整体"——先确定这是一篇时间单位介绍，分为"秒—分—时"三个部分，采用"定义—换算—举例"的格式。

如何使用

拿到任何新文本，先花30秒"扫读"标题、首句、加粗词、图表标题。

可迁移场景

阅读理解题课前预习分析说明性文章

⚠️ 别急着划线——先看完一遍再说。

2结构拆解法

▾
原理说明

把完整的文章按"结构单元"拆开，看清每一部分的作用。常见的结构有"总—分—总""定义—分类—举例""时间顺序"等。

在示例中的体现

示例将知识卡片拆解为"标题—符号—定义—换算—举例"5个结构单元。

如何使用

用"/"标出段落边界，给每个边界段加一个小标题。

可迁移场景

写作文提纲分析课文段落整理笔记

⚠️ 不是所有的文章都有明显的结构标记——要学会找过渡句。

3要素标记法

▾
原理说明

从文本中标记出关键要素——关键词、关键数据、核心概念、连接词。这些是理解内容的"路标"。

在示例中的体现

示例标记了"60""进制""进率""换算""乘""除"等关键词。

如何使用

用荧光笔或符号，圈出文中所有的数字、专业术语、因果关系词（因为、所以、因此）。

可迁移场景

复习备考整理知识点读教材

⚠️ 不要一看到"关键词"就划线——只标真正帮助理解的信息。

4逻辑追踪法

▾
原理说明

理清文章的思路——作者是先说原因再说结果？还是先举例再总结？用箭头画出逻辑链条。

在示例中的体现

示例追踪了"为什么1小时=60分钟"的逻辑链——从历史原因（古巴比伦60进制）到数学定义（进率）到生活应用（换算）。

如何使用

用"→"连接关键句，画出作者的论证路径。

可迁移场景

分析数学证明题理解科学原理写议论文

⚠️ 有时候作者的逻辑不是直线而是"树形"的——要学会分叉标记。

5评价对照法

▾
原理说明

对照标准（如教材、权威资料、评价量表）判断文本的表达质量——有没有错误？够不够清晰？完整吗？

在示例中的体现

示例最后对照课本原文检查了知识卡片的准确性，并对照"5要素标准"检查了完整性。

如何使用

先定标准（要检查什么），再逐条对照打分。

可迁移场景

修改自己的作业评价同学的作品自我检查

⚠️ 标准要具体可判断，不要用"好""不好"这种模糊词。

解析任务：用整体感知法+结构拆解法分析这份总结，找出它的结构问题，给出修改建议。

分析提示：先看整体结构是否完整→再拆每一部分→看看是不是"定义+换算+举例"都有→找出重复或遗漏的部分

分析提示：标记文中的"数据"和"研究引用"→对照课本里的时间知识，看有没有错误→对照评价标准打分
