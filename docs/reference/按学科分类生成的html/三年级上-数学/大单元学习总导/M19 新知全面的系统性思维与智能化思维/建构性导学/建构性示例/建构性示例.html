看别人怎么做"时间系统"概念图

🔍 建构情境

你知道吗？时间其实是一套60进制系统。长度用10进制（10毫米=1厘米，100厘米=1米），但时间用的是60进制（60秒=1分，60分=1时）。为什么人类会选择60这个数字？这背后藏着古老的智慧。

🎯 你的任务：亲手建构一张"时间系统图"，把时、分、秒的关系画得清清楚楚。
🎯 建构目标

建构一张"时间系统关系图"，让人一眼看懂：

🔄时、分、秒之间的换算关系（×60或÷60）
🏠三个单位在生活中的典型应用
💡为什么60进制是一个聪明的设计

👣 建构过程示范（6步）

💡 点击每一步展开详细示范

第一步：收集信息
做什么：翻看教材中关于时、分、秒的内容，找出所有进率关系
怎么做：用铅笔在草稿纸上列出所有你能找到的关系式
示范：1时=60分，1分=60秒，1时=3600秒
点击展开示范 →
第二步：确定结构
做什么：决定用"什么形状"来组织信息
怎么做：想想是串联（→→→）、并联（并列三个框）还是环形（循环关系）
示范：决定用"从上到下"的层级结构——时在最上，分在中间，秒在最下
点击展开示范 →
第三步：画出框架
做什么：用铅笔在A4纸上画出三个层级框
怎么做：上面写"时"，中间写"分"，下面写"秒"，左右留出标注位置
示范：三个长方形，从上到下排列
点击展开示范 →
第四步：标注进率
做什么：在框之间的连线上标注换算规则
怎么做：从上到下标注×60，从下到上标注÷60
示范：时→分：×60；分→秒：×60；秒→分：÷60；分→时：÷60
点击展开示范 →
第五步：加入生活例子
做什么：在每个单位框里或旁边写一个生活例子
怎么做：想一想你在什么时候用到这个单位
示范：时——节课40分钟（约0.67时）；分——广告时长3分钟；秒——百米冲刺12秒
点击展开示范 →
第六步：完善美化
做什么：用黑笔描深，用红笔标重点，用彩色标注例子
怎么做：铅笔改为签字笔，进率箭头改红色，例子用蓝色
示范：红色箭头清晰标注×60和÷60，蓝色小字写例子
点击展开示范 →
