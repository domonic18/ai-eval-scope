🌱 基础层

解释型视频

📺 推荐视频：《为什么要学习时间单位换算？》（B站 / 腾讯视频搜索）
  • 观看任务：认真看视频中如何解释 ×60 和 ÷60 的换算规则

关键问题：

  • 为什么说时间换算跟人民币换算不一样？
  • 视频中使用了什么"记忆口诀"帮你记住进率？
暂停思考点：@1:30 处暂停 —— 在纸上写下 1时 = ？分、1分 = ？秒
输出任务：用你自己的话写一段"时间换算小结"（50字以内）
🚀 进阶层

分析型视频

📺 推荐视频：《古代人是怎样计时的？》（央视纪录片 / 科普短片）
  • 观看任务：找出古代至少 3 种计时工具，说清它们的工作原理

关键问题：

  • 日晷在阴天和晚上还能用吗？
  • 水钟的精度受什么影响？
暂停思考点：@3:00 处暂停 —— 预测一下沙漏是什么时候发明的
输出任务：画一个"计时工具进化时间线"（从古到今，至少 5 种）
🔥 挑战层

辩论 / 前沿型视频

📺 推荐视频：《时间是什么？霍金的时间简史片段》（BBC 纪录片选段）
  • 观看任务：听懂"时间箭头"的概念 —— 为什么时间只能向前？

关键问题：

  • 视频中提到了几种"时间箭头"？
  • 如果时间可以倒流，会出现什么问题？
暂停思考点：@5:20 处暂停 —— 你觉得"时间旅行"可能实现吗？写下你的理由
输出任务：写一段 100 字的"我对时间的理解"—— 结合视频和你的课堂所学
