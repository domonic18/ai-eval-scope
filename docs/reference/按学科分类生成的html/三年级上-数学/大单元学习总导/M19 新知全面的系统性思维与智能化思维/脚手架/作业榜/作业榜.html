📊 分层任务：差异化训练

★ 易三步计算

"小红从家到学校需要15分钟，她8:00出发，几点到校？"

解题过程：
第一步：8:00 + 15分钟 = ?
第二步：8:00的"00分"加上15分 = 15分
第三步：所以到达时间是 8:15 ✅

★★ 中多步换算

"一场电影从下午2:30开始，共放映1小时45分钟，几点结束？"

解题过程：
第一步：2:30 + 1小时 = 3:30
第二步：3:30 + 45分钟 = 3:30 + 30分 + 15分 = 4:00 + 15分
第三步：结束时间是 下午4:15 ✅

★★★ 难逆向思考

"小明从学校走回家用了20分钟，如果他从家走到学校要走30分钟——为什么差了10分钟？这可能是什么原因？"

💡 提示：路是一样的，为什么时间不同？
可能原因：
1. 🏃 去学校时在赶路（走得快），回家时慢慢走（走得慢）
2. ⛰️ 去学校是下坡路，回家是上坡路
3. 🚶 路上遇到了朋友，停下来聊天耽搁了
你还能想到其他可能吗？
