早餐时间够不够？

★☆☆ 入门 生活情境
👆 点击展开详情

📋 案例材料：小红7:00起床，洗漱10分钟，做早餐15分钟，吃早餐20分钟，学校8:00上课，步行15分钟。

🎯 解析任务
小红最早几点能到校？能准时上课吗？哪段时间可以缩短？
🛤️ 解析路径
识别时刻与时长 → 画时间轴 → 加法计算到达时刻 → 验证
📊 分析记录表
时刻/时长分类表，区分时间点和时间段
💡 参考思路
关注"做早餐"和"吃早餐"能否重叠？如果能同时进行，时间会缩短多少？

时刻→开放→等待→开放时长的记录表
💡 参考思路
15:50到16:00才10分钟！别着急，看看时钟的指针怎么走的。

公交发车时间表

★★☆ 进阶 学科核心
👆 点击展开详情

📋 案例材料：某路公交首班6:30，末班21:30，每15分钟一班。

🎯 解析任务
①末班比首班晚多少小时？②一天发几班？③8:05到站，下一班几点？
🛤️ 解析路径
识别规律 → 用除法求班次 → 用"周期+余数"推算下一班
📊 分析记录表
时刻序列记录+规律发现
💡 参考思路
21:30−6:30=15小时，每小时4班，15×4=60班。下一班在8:15！

谁跑得更快？

★★★ 挑战 跨学科
👆 点击展开详情

📋 案例材料：3个同学跑200米——甲用了45秒，乙用了52秒，丙用了48秒。

🎯 解析任务
①排名并说明理由 ②乙比甲慢多少秒？③有同学用时0.8分钟，他排第几？
🛤️ 解析路径
统一单位（都换成秒）→ 比较数字大小 → 判断快慢与时间的关系
📊 分析记录表
姓名→用时→排名→分析理由
💡 参考思路
时间越少跑得越快！0.8分=48秒，所以0.8分和丙一样快！

闰秒——这个世界为什么需要它？

★★★ 挑战 社会议题
👆 点击展开详情

📋 案例材料：地球自转在变慢，每隔几年就要加1个"闰秒"。2017年1月1日加了第27个闰秒。

🎯 解析任务
①为什么需要闰秒？②如果一直不加闰秒，100年后钟表时间比实际时间差多少？
🛤️ 解析路径
识别问题 → 分析原因 → 追踪影响 → 判断必要性
📊 分析记录表
现象→原因→频率→影响→结论
💡 参考思路
每天慢几毫秒，100年累积约1分钟——虽然不多，但对导航等系统影响极大！
