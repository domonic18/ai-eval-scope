📖 三层案例

🌟 基础 · 生活 小明等绿灯
📌 背景： 小明走到路口，红灯还有45秒。他急着上学，是该等还是绕路？
45秒有多长？值不值得等？
🛤️ 决策路径： 方案A：干等45秒 方案B：绕路要多走2分钟
等红灯45秒 < 绕路2分钟，等更划算！
学会估算小段时间，帮助日常决策——当你觉得"等好久"的时候，先看看实际走了多少秒！
🚀 进阶 · 行业 为什么马拉松要计时到千分之一秒？
📌 背景： 马拉松比赛用专业计时芯片，精度达0.001秒。但跑一场马拉松要2个多小时啊！
跑步比的是谁更快，但差距往往只有几秒甚至不到一秒——怎么公平地测量？
🛤️ 决策路径： 方案A：人工掐表 方案B：电子计时 方案C：芯片计时 + 终点摄像
现代比赛用芯片计时，每个选手从起跑线就开始计时，终点摄像辅助判断名次
"精确"到哪个级别，取决于你需要解决的问题——不是越精确越好，是"够用"最好。
🏆 挑战 · 科学 GPS为什么需要"相对论修正"？
📌 背景： GPS卫星上的钟每天快38微秒——0.000038秒，比眨一下眼睛还快一万倍。
不到万分之四秒的误差，会导致定位偏差多少？
🛤️ 决策路径： 方案A：不修正（误差每天累计10公里） 方案B：软件修正（按时调整卫星时钟）
必须修正，否则导航全错！卫星上的时间只要差一点点，地面定位就会偏出好几条街
极小的误差在复杂系统中会放大——"差不多"有时差很多，尤其是和时间打交道的时候！

🚀 进阶
如果你是马拉松计时员，你怎么证明今年的冠军确实比去年快？
需要考虑：两次比赛的天气一样吗？赛道一样吗？计时设备一样吗？排除这些差异之后，才能确定"真的快了"。
