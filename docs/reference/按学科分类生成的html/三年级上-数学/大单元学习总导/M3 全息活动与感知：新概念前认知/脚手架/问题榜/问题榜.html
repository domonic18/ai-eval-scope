🌱 基础层 · 理解与识别

1下面的活动中，哪些需要用"秒"来计时？哪些需要用"分"？

选项：刷牙、眨眼、吃一顿饭、跑100米、打一个哈欠

基础识别类
21分钟和1小时比，差多少倍？你能找到生活中差60倍的东西吗？
基础比较类
提示：1小时=60分钟，1分钟=60秒，都是60倍的关系。生活中还有哪些东西也用"60"这个数？
3闭上眼数10秒，睁开眼看计时器——你数快了还是慢了？
基础感知类
