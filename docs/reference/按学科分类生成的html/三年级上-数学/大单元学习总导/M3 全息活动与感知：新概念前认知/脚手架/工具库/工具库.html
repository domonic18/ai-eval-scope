🧰 工具库 · 认知放大器

这里是你的"时间探索工具箱"——每个工具都能帮你更好地体验、记录和分析时间。选一个你感兴趣的工具，开始探索吧！

🧠 思维工具

🥧
时间饼图模板
把一天的时间分配画成饼图
适用：感知24小时怎么分配
步骤：①列出活动 → ②估算时长 → ③画饼图
📊
预测—实际对比表
猜想 vs 验证
适用：1分钟挑战等量感建立活动
步骤：①写下预测 → ②计时实践 → ③对比记录
📏
时间线图
把一段时间分成"格子"
适用：理解"一段路分成几段"
步骤：①画横线 → ②标起点终点 → ③均分格子

📱 数字工具

⏱️
手机秒表/计时器
精确计时
基础：点击开始/停止
▶ 进阶用法
计次功能——记录多段时长，比如记录你跑一圈、两圈、三圈各用了多少秒
🔴
Time Timer（APP）
可视化时间流逝
基础：设置时长，看红色区域逐渐减少
▶ 进阶用法
调整速度——在不同速度下感受时间"变快"或"变慢"
🌲
Forest专注森林
把时间管理游戏化
基础：种一棵树 = 专心25分钟
▶ 进阶用法
统计每日专注时长，看看自己一天真正"专注"了多久
🤖
AI助手（ChatGPT/文心一言等）
时间相关问题咨询
基础：问"1秒钟我能做哪些事？"
▶ 进阶用法
分析你的时间数据，让AI给出"时间使用建议"

📊 数据工具

📋
Excel/表格模板 — 记录一周每天的"时间使用日记"，看看时间都去哪儿了
📈
手绘图表模板 — 用柱状图比较"不同活动用了多长时间"，一目了然

🧪 实验工具

⏳
沙漏套装
1分钟 / 3分钟 / 5分钟
实体感受不同时间长度
🎵
节拍器APP
设置60拍/分钟
感受一秒一拍
