🏗️ 项目社区

在真实的时间研究项目中，成为"时间探索家"！每个项目都有一份专属的挑战地图等你解锁。

⚡ 短周期 · 1-3天
我的"时间秤"
🎯 项目目标：建立个人的时间参照系统——找到你身上最准确的"生物计时器"。
  • Day 1：测试"闭眼数30秒"的准确度（用手机计时器验证）
  • Day 2：测试"听一首歌判断3分钟"的准确度
  • Day 3：汇总数据，制作"我的时间秤"报告

成果形式：一份个人时间感知报告

评价标准：数据完整有对比分析有改进策略

📆 中周期 · 1-2周
一周时间侦探
🎯 项目目标：记录并分析你一周的时间分配，发现自己的时间使用模式。
  • 第1-6天：每天记录"时间使用日记"（何时、何事、用时）
  • 第7天：汇总并分析——画饼图或时间线，看看时间都去哪儿了
  • 反思：哪些时间用得好？哪些可以改进？

成果形式："一周时间日记" + 分析报告

评价标准：7天完整记录数据可视化有反思结论

🏆 长周期 · 1月+
校园时间优化提案
🎯 项目目标：观察并记录学校一天中的时间安排，提出真正的优化建议。
  • 第一周：观察并记录（课表、上课用时、课间、放学时间）
  • 第二周：采访同学和老师——"你觉得学校时间安排合理吗？"
  • 第三周：数据分析和方案设计
  • 第四周：形成提案并展示

成果形式：校园时间优化提案（含数据、分析、建议）

评价标准：数据真实建议合理表达清晰有可行性
