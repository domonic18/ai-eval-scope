1

同单位优先比较法

单位相同，直接比数值

👆 点击翻转查看详情

⚖️ 同单位优先比较法

原理说明：比较时间大小时，先看单位是否相同；单位相同，直接比数值。

示例体现：三位同学成绩都是"秒"，所以直接比较16、14、15的大小。

如何使用：先看单位 → 单位相同 → 直接比较数值

📌 可迁移：长度、重量、时间比较 ⚠️ 注意：确认单位名称是否真的相同

2

单位统一换算法

不同单位，先换算再比较

👆 点击翻转查看详情

🔄 单位统一换算法

原理说明：单位不同时，先统一换算成同一单位，再比较。

示例体现：如果有人的成绩是"0.3分钟"，需换算成"18秒"再比较。

如何使用：确定目标单位 → 用进率换算（×或÷）→ 比较数值

📌 可迁移：时间、长度、货币换算 ⚠️ 注意：熟记1时=60分、1分=60秒

3

多角度验证法

换种方式再算一遍

👆 点击翻转查看详情

🔍 多角度验证法

原理说明：换一种方式再算一遍，验证结论是否正确。

示例体现：先用"秒"直接比较，再换算成"分钟"验证，结论一致。

如何使用：换一种单位 → 重新比较 → 看结论是否一致

📌 可迁移：数学验算、科学实验验证 ⚠️ 注意：多角度验证≠胡乱验证，要选有意义的验证方式

4

情境判断法

根据实际情境选方法

👆 点击翻转查看详情

🎯 情境判断法

原理说明：根据实际问题情境，选择合适的单位和比较方法。

示例体现：运动会中用"秒"记录成绩合理——记成"0.233分钟"反而不直观。

如何使用：看问题情境 → 判断哪个单位最合适 → 选择对应的分析方法

📌 可迁移：生活中的一切测量和比较 ⚠️ 注意：不要机械套用，要结合具体情境思考

📖 应用练习一：图书馆借书时间

🏛️ 图书馆的难题

图书馆规定每次借书最多 14天。小华借了一本书，从3月5日下午3:00开始，到3月18日下午3:00还书。

问题：他超期了吗？

📌 情境判断法 📌 单位统一换算法
💡 脚手架提示：可以画一条时间轴来帮助理解——从3月5日15:00到3月18日15:00。
🔎 点击展开分析 ▾

第一步：情境判断——这是"借书天数是否超限"的问题，需要用"天"为单位来衡量。

第二步：计算时间差——从3月5日15:00到3月18日15:00，正好是13天（不是14天哦）。

第三步：比较——13天 < 14天。

结论：小华没有超期。

💡 如果你算出来是14天，可以这样检查：3月5日到3月6日是1天，到3月7日是2天……到3月18日就是13天。

🏃 应用练习二：三个人的跑步成绩

🏟️ 操场上的混单位数据

操场上有三个人跑步：

  • A跑一圈用了 3分钟
  • B跑一圈用了 200秒
  • C跑一圈用了 2分40秒

问题：谁跑得最快？

📌 单位统一换算法 📌 同单位优先比较法
💡 脚手架提示：现在"分钟""秒""分秒"混在一起了——先把所有数据变成同一个单位。
🔎 点击展开分析 ▾

第一步：统一单位——全部换算成"秒"为单位。

  • A：3分钟 = 3 × 60 = 180秒
  • B：200秒（已经是秒）
  • C：2分40秒 = 2 × 60 + 40 = 160秒

第二步：同单位比较——160秒 < 180秒 < 200秒。

第三步：得出结论——用时最短 = 跑得最快 → C最快（160秒），A第二（180秒），B第三（200秒）。

💡 C用了2分40秒=160秒就跑完一圈，比A的3分钟（180秒）快了20秒！
