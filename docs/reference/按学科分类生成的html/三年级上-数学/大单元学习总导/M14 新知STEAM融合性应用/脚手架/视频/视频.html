🎬 视频脚手架

观看指南 · 带着任务看视频，从入门到深度思考 · 数学 · 三年级上 · 时、分、秒

📖 基础层 · 解释型视频

视频1：时间单位换算动画

▶ 观看任务
看完你能做到：自己说出 1时=？分、1分=？秒 的换算关系，并理解为什么是60进制。
  • 视频里用什么比喻解释了时、分、秒的关系？（比如"时针是爷爷，分针是爸爸，秒针是孩子"）
  • 为什么1小时是60分钟而不是100分钟？这和什么有关？
⏸ 暂停 视频演示换算过程时（约1:30），暂停！自己先算一遍，再对答案。
📝 输出任务：画一张时、分、秒"家庭关系"图，标出它们之间的换算关系。
🔍 进阶层 · 分析型视频

视频2：纪录片《秒的诞生》

▶ 观看任务
看完你能做到：记录下"秒"这一单位在历史上被重新定义了哪几次。
  • 最初的"秒"是怎么定义的？（提示：和地球自转有关——1/86400）
  • 现在的"秒"是怎么定义的？（提示：和原子有关——原子钟）
⏸ 暂停 视频介绍古代计时器时（约3:00），暂停！想一想：如果是你，在没有工具的情况下怎么测量1秒？
📝 输出任务：画一个"秒的定义演变时间轴"，从古代到现在，标注每一次变化。
🎤 挑战层 · 辩论/前沿型视频

视频3：TED演讲《为什么时间感知会变化？》

▶ 观看任务
看完你能做到：找出至少2个影响人类时间感的因素。
  • 为什么快乐的时候时间过得快、害怕的时候时间过得慢？
  • 你同意演讲者的观点吗？你有什么自己的例子可以证明或反驳？
⏸ 暂停 演讲者分享第一个实验时（约4:30），暂停！如果你也参加这个实验，你觉得自己的时间感会是怎样的？
📝 输出任务：写一段100字的小短文"我的一次时间感知经历"——什么时候你觉得时间特别快？什么时候特别慢？
