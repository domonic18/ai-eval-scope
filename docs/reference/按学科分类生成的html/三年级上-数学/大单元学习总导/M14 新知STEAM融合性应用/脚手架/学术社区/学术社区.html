🤝 同伴连接 讨论 · 互评 · 接龙

1时间感知讨论

"你觉得上课40分钟和下课的10分钟，哪个感觉更短？为什么？"

💡 引导方向：讨论"主观时间"和"客观时间"的区别——时间是一样的，但感觉不一样！
👍 上课40分钟感觉更短 ⏰ 下课10分钟感觉更短 🤔 说不上来
2方案互评

把你的"一分钟挑战方案"或"课间时间优化方案"发给同桌，互相提改进建议。

📋 互评标准：时间计算是否正确？方案是否可行？有没有更有创意的方法？
3时间换算接龙

每人出一道时间换算题，传给下一个人做，看谁不出错！

🎮 玩法：A出题→B作答并出下一题→C作答并出下一题→……谁接不上就淘汰，最后一人为"换算之王"👑

👥 协作任务

🗺️ 校园时间地图
4人一组，合作完成"校园时间地图"——标注学校一天中所有关键时间节点：到校、上课、课间、午餐、放学……
🏫 小组任务
📌 成果分享
把小组的"时间地图"贴在班级墙上，看看各组有什么不同！谁的标注最全？谁的创意最佳？
🏆 展示评比

🏠 家校连接

🗣️ 采访家长："你小时候是怎么知道时间的？"
对比古今计时方式的差异——家长小时候有电子表和手机吗？他们用什么看时间？把采访结果记录下来，和同学们分享！
📸 作品展示
把你在本模块完成的时间作品（时间规划图、换算速查表、日晷模型等）拿给家长看，请家长写一段评价。拍一张"作品+评价"的照片上传班级相册！

✅ 完成以下步骤后打勾：

  • 完成采访并记录
