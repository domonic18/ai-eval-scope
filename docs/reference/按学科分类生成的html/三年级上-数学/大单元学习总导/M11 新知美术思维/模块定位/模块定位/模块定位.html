📌 模块定位 — M11 新知美术思维

用美术的方式表达时间——把"看不见的时间"变成"看得见的画面"

1

本模块学习什么

不是学画画技巧，而是学一种全新的思维方式：把抽象的时间变成可视的画面。

  • 用 颜色 代表不同的时间长度或活动类型
  • 用 线条、色块、图形展示时间分配
  • 用 比例 把"1小时"变成"1厘米"
  • 创作属于你自己的时间可视化作品
2

本模块为什么重要

前面 10 个模块你都在"算时间"——计算小时和分钟的换算、解决时间应用题。

关键一步：从"用数字理解时间"到"用画面感知时间"。这种可视化思维能力能迁移到数学统计图、科学实验记录、语文思维导图……
3

与大单元主题的关系

"时、分、秒"这个大单元的主题是认识时间 → 计量时间 → 运用时间。

M11 承担的是"运用时间"中最有创意的一环——表达时间。

数学的精确（1小时=60分钟）和美术的感性（用色彩表达时光）在这个模块相遇。
