🌳 第1阶：我的放学后时间地图

🎯 真实情境：你每天放学后到晚睡前有大约5小时的自由时间。但每次到了睡觉时间你才发现"怎么什么都没做就过完了"？请找到"时间都去哪儿了"的答案。

✅ 任务：画一张从放学（16:30）到睡前（21:00）的时间流程图，清晰标注每个活动及其时长。

📐 原理调用：时间分层法 + 色彩编码法

📝 建构步骤：

  1. 先记录一周每天的放学后时间消费（做什么、做了多久）
  2. 给活动分类（作业/阅读/运动/娱乐/吃饭/洗漱等）
  3. 选取有代表性的一天，用不同颜色标注不同类型
  4. 按时间顺序画流程图
  5. 标出"时间占比最大"和"可以优化"的地方
  6. 给作品加标题和图例

🪜 脚手架：提供4.5小时（16:30-21:00）时间轴模板底图

📋 自评量规：

  • 我记录了至少3天的放学后时间使用情况
  • 时间轴上每个活动都标注了开始和结束时间
  • 我用了至少4种颜色区分不同类型的活动
  • 图中有图例，别人能看懂颜色代表什么
  • 我标出了"时间最多的活动"和"想要改进的地方"

🏠 第2阶：家庭周末活动日历

🎯 真实情境：你家周末经常上演"你想去公园，爸妈想打扫卫生，弟弟想看动画片"的冲突。如果能画一张家庭周末活动日历，让每个人都能看到周末的时间安排，会不会好很多？

✅ 任务：为你的家庭设计一张周六/周日的时间日历图，平衡全家人的活动需求。

📐 原理调用：比例映射法 + 视觉叙事法

📝 建构步骤：

  1. 和家人一起列出每个人周末想做的事
  2. 把活动分为"全家一起做"和"各做各的"两类
  3. 用不同颜色代表不同的家庭成员
  4. 按时间顺序排列，注意活动之间的重叠和冲突
  5. 标注每个活动的时间长度
  6. 留出"自由时间"应对突发情况

🔔 第3阶：校园铃声时间设计

🎯 真实情境：学校每天用铃声告诉我们上下课。每节课40分钟，课间10分钟。如果让你把一天的铃声节奏画出来，你会怎么设计这张"声音时间图谱"？

✅ 任务：设计一张学校一上午（8:00-12:00）的"铃声时间视觉图"，用图形的大小、颜色反映不同时间段的长短和性质。

📐 原理调用：比例映射法 + 色彩编码法

📝 建构步骤：

  1. 记录今天上午的铃声时间表（上课/下课/做操）
  2. 上课时段用一种颜色，下课用另一种，做操用第三种
  3. 色块宽度=时间段长度（40分钟比10分钟宽4倍）
  4. 在每个色块上标注具体时间（如"8:00-8:40 语文"）
  5. 加上铃声音效标注（🔔代表上课铃，🔕代表下课铃）

⏱️ 第4阶：一分钟能做什么

🎯 真实情境：1分钟=60秒。跳绳能跳120下？心跳约72次？光能走1800万公里？请用一幅"1分钟信息图"来回答这个问题。

✅ 任务：创作一幅"1分钟信息图"，用视觉方式展示3-5个"1分钟内发生的事"——既可以是亲身体验，也可以是科学事实。

📐 原理调用：比例映射法 + 色彩编码法

📝 建构步骤：

  1. 列出3-5个"1分钟内发生的事"
  2. 给每个事件分配一种颜色
  3. 用图形大小表示数值大小（120下比72次更大）
  4. 在图形内或旁边标注具体数字
  5. 加标题和图例

🌸 第5阶：时间与季节的对话

🎯 真实情境：一年有四季，每季约3个月。春天的"一天"和冬天的"一天"有什么不同？日出时间、日照时长、活动安排都会随着季节变化。

✅ 任务：设计一个"四季时间轮盘"，展示同一个活动在春、夏、秋、冬四季的时间变化。

📐 原理调用：时间分层法 + 视觉叙事法

📝 建构步骤：

  1. 选3个代表性活动（如"起床""放学""晚餐"）
  2. 查找或回忆每个季节这些活动的时间变化
  3. 画一个圆分成4个扇形（春夏秋冬）
  4. 在每个扇形内标出活动的时间点
  5. 用不同颜色代表不同季节
  6. 观察时间的季节变化规律

🚌 第6阶：城市交通时间地图

🎯 真实情境：从你家到学校需要多长时间？如果是走路、坐公交、骑自行车分别要多久？时间与空间的关系在地图上是怎么"画"出来的？

✅ 任务：画一张从你家到学校的"出行时间关系图"，用线段长度表示时间（而非实际距离）。

📐 原理调用：比例映射法 + 事件锚定法

📝 建构步骤：

  1. 确定起点（家）和终点（学校）
  2. 测试并记录不同交通方式所需时间
  3. 以家为中心，按时间比例画线段
  4. 不同交通方式用不同颜色
  5. 在线段上标注具体分钟数

🌟 第7阶：我的"理想一天"设计

🎯 真实情境：如果有一天的"自由支配权"，你会怎么分配学习、游戏、运动、休息和探索的时间？把它"画出来"，是一次对时间管理的深度思考。

✅ 任务：创作"我的理想一天"时间图谱作品，综合运用前6阶学到的全部建构性原理。

📐 原理调用：全部5条建构性原理综合运用

📝 建构步骤：

  1. 列出你理想一天中想要包含的所有活动
  2. 给活动分配时间比例
  3. 选择一种表现形式（时间轴/圆形/表格）
  4. 使用色彩编码法分类活动
  5. 用比例映射法确保时间准确
  6. 用视觉叙事法让一天有起承转合
  7. 加标题、图例，美化完善
