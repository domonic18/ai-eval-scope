📌 议题推荐

🎨 议题一："时间能用颜色表示吗？"

如果你要给"1分钟"配一个颜色，会选择什么颜色？为什么？

⏳ 议题二："哪个时间最容易被浪费？"

看看你的时间图谱，找出"你以为很短其实很长"和"你以为很长其实很短"的时间段

💬 议题三："设计吐槽大会"

互相看对方的时间图谱作品，每人找"一个亮点 + 一个槽点"
