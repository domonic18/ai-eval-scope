📖 基础任务：看懂一幅时间可视化作品

看一幅时间可视化作品（如教室作息表、地铁线路图），回答：

  • 这幅作品用了什么形式？（时间轴/表格/饼图/其他）
  • 它用了哪些颜色？每种颜色代表什么？
  • 如果缺了图例，你能从内容推断出颜色的含义吗？

🎯 选择 A / B / C，找到最适合你的挑战

A. 时间轴临摹 → 推荐基础较弱者

  • 照着一幅已有的时间作品临摹一遍
  • 目标：看懂原作的时间标注、颜色分配和比例安排
  • 输出：临摹作品 + 50字感悟

B. 时间配色方案设计 → 推荐对颜色敏感者

  • 自创一套"时间颜色语言"——给 12 种常见活动各配一种颜色
  • 输出：一张配色方案卡（12色色块 + 活动名称）
  • 要求：颜色搭配美观，含义直观

C. 时间作品改进方案 → 推荐有批判思维者

  • 找一幅你觉得"不够好"的时间可视化作品
  • 用"要素完整性检查法"找出至少 3 个不足
  • 输出：改进前后的对比草图

✨ 开放任务：回答"时间是什么？"

用你喜欢的方式——可以画、写、拍、说！

  • 🎨 画 — 时间图谱 / 时间动物 / 时间建筑
  • ✍️ 写 — 一首关于时间的小诗 / 一段时间的奇妙想象
  • 📸 拍 — 延时摄影 / 时间主题的摄影作品
  • 🎤 说 — 录一段 1 分钟视频，用 5 句话描述时间

📊 评价标准（Rubric）

维度优秀 3分合格 2分需改进 1分
时间准确性时间标注完全正确大部分正确有明显错误
视觉表达色彩和谐，布局美观色彩清晰，布局合理色彩混乱，布局随意
可读性有图例，一读就懂基本能看懂需要口头解释才能懂
创意性有自己的独特设计有部分创意完全照搬模板
