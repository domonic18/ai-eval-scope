基础层：解释型

观看并理解——时间是怎么被"看见"的？

🎥 5min

《一天的时间是如何被"看见"的》

👁 观看任务 找到视频中 3种以上 表现时间流逝的视觉方法
① 延时摄影如何让"看不见的时间流动"被看见？
② 为什么不同时段（早晨/正午/黄昏）的光影颜色不同？
⏸ [2:15] 暂停——你觉得这段延时摄影花了多长时间拍摄？
提示：延时摄影1秒画面可能对应真实世界几分钟甚至几小时！猜完后继续看，片尾可能有答案。
输出任务
✏ 用文字描述你看到的一种最震撼的时间视觉化方法。是什么画面？为什么震撼？
🎥 8min

《钟表是怎么工作的——齿轮的秘密》

👁 观看任务 画出钟表齿轮传动的 简化示意图
① 秒针、分针、时针的齿轮比例是多少？
② 为什么60秒=1分钟？这个60进制和齿轮齿数有什么关系？
⏸ [4:30] 暂停——看看秒针齿轮和分针齿轮，猜猜它们的齿数比是多少？
秒针齿轮齿数 : 分针齿轮齿数 = 1 : 60。秒针转60圈，分针才转1圈！
输出任务
✏ 用图画表示"秒针转一圈 = 分针转一格"的关系，标出齿轮齿数。

进阶层：分析型

分析优秀的信息图表——好在哪里？

🎥 12min

《TED：信息图表的魅力》

👁 观看任务 记录演讲者展示的 3种 信息图表类型
① 演讲者说"好的图表是一个故事的浓缩"——你怎么理解？
② 哪个图表示例让你印象最深？它好在哪里？
⏸ [7:00] 暂停——这个图示用了什么颜色方案？为什么选这些颜色？
观察色调、饱和度对比——暖色突出关键数据，冷色做背景衬托。
输出任务
✏ 一段话 + 一张草图：把视频中的信息图表方法"翻译"成你自己的时间作品创意。

挑战层：辩论 / 前沿型

用延时摄影看世界——时间是最好的画家

🎥 15min

《延时摄影中国——时间是最好的画家》

👁 观看任务 找出延时摄影中 3种不同的"时间尺度"（秒级？分钟级？小时级？）
① 延时摄影改变的不仅是对时间的感知——它还改变了什么？
② 如果让你用延时摄影拍"1分钟"，你会拍什么？
③ 这些画面能不能被"翻译"成你的时间图谱？
⏸ [10:00] 暂停——想一想：如果你要把这个镜头画成时间图，会是什么样？
想一想色彩的渐变、色块的比例、以及时间的流动方向。
输出任务
✏ 选一个延时摄影镜头，把它"翻译"成一张时间图谱草图，标注时间尺度和色彩含义。
