基础工具

入门必备

时间轴模板

思维工具
用途：提供标准的横向 / 纵向时间轴线框，降低"从0画起"的难度
  1. 选横轴或竖轴
  2. 标定时长总跨度
  3. 在刻度上分配活动色块
  4. 添加图例
一张A4纸上的24h水平线，每3cm标一个整点
📌 适用场景：创作"我的一天""我的周末"等基础时间作品
🎨

色彩决策卡

思维工具
用途：帮你决定每种活动用什么颜色
  1. 列出所有活动类型
  2. 为每种活动选颜色（学习用冷色、运动用暖色、休息用中间色）
  3. 填入卡片
学习=🔵蓝 运动=🟢绿 休息=🟡黄 娱乐=🔴红 饮食=🟤棕
📌 适用场景：任何需要色彩分配的时间作品

技能提升

Canva 在线设计

数字工具
用途：在电脑 / 平板上快速制作时间可视化作品
  1. 注册 / 登录 Canva
  2. 搜索"时间轴"模板
  3. 拖拽编辑内容
📌 适用场景：不想 / 不能手绘时，可用数字方式完成作品
📐

色块比例尺

思维工具
用途：确定不同时长对应的色块长度
  1. 确定"1小时 = ? 厘米"的比例
  2. 按比例计算其他时长
  3. 画出精确的色块
1h=3cm，则30min=1.5cm，15min=0.75cm，10min=0.5cm
📌 适用场景：需要精确比例的时间作品

快速计算器

20min = 1cm

时间统计分析表

数据工具
用途：先用表格统计一周时间分配，再决定"画什么"
  1. 列出一周7天的每项时间
  2. 计算每类活动总时长
  3. 找出"占比最大的活动"和"最零散的时间段"
📌 适用场景：长周期项目或需要数据分析的时间作品
⏱️

秒表计时器

实验工具
用途：测量你做一件事平均需要多少时间
  1. 准备好计时器
  2. 做5次相同动作
  3. 记录每次时间
  4. 取平均值
写一行字——第1次8秒、第2次7秒、第3次9秒、第4次7秒、第5次8秒 → 平均7.8秒
📌 适用场景：当你需要准确数据来画"1分钟能做什么"类型的信息图时
