快速上手，从画自己的一天开始！

🎨 设计"我的一天"时间图谱 ⭐ 推荐
🎯 项目目标 用A4纸创作一幅"我的一天"时间可视化作品
1Day 1 记录：用表格记录一天的时间分配

📋 班级课程表可视化改造

中周期项目（1-2周）· 专题项目

全班合作，办一场时间主题展览！

🖼 时间主题班级展 👥 合作
🎯 项目目标 全班合作，策划一期"时间的美术表达"主题展览
1确定展览主题（"我的一天""一分钟能做什么"等）

5邀请其他班级同学参观！

长周期项目（1月+）· 综合项目

深度创作，打造你的个人时间作品集！

📖 "时间的形状"个人作品集 🔥 挑战
🎯 项目目标 在一个月内，创作一本"时间的形状"个人作品集（至少5幅作品）
第1周基础练习——"我的一天""我的一周"两幅时间轴
第2周专项训练——"一分钟能做什么"信息图 + "四季时间轮盘"
第3周主题创作——选你最想表达的时间主题（成长/自然/城市）
第4周作品集整理——封面设计 + 目录 + 序言写作
📋 评价标准
作品集中包含至少5幅独立作品
每幅作品有时间标注、图例、标题
作品集有封面、目录、序言
整体体现"新知美术思维"——用视觉表达时间概念
