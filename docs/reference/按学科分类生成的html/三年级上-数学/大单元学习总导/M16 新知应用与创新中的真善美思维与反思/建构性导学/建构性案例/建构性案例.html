案例一

我的"一分钟"时间日记

真实建构情境
你的一天中有很多个"一分钟"——起床穿衣服大约一分钟、等红灯大约一分钟……但你真的留意过吗？请你选择一天中的5个不同时刻，记录下你对那一分钟的感受。
建构任务
制作一份"我的一天一分钟日记"，记录5个不同时刻的一分钟体验，并分析为什么同一分钟在不同时刻感受不同。
调用原理
🔬 亲身体验驱动原理 · 🔍 对比发现原理
建构步骤
  1. 选5个不同时刻（如：早上起床、中午吃饭、下午体育课等）
  2. 设计记录表，包含"时刻""活动""感觉时长""实际时长""心情"
  3. 在每个时刻先记录感受，再核对实际时间
  4. 比较5个时刻的数据，找出规律
  5. 写一段话总结你的发现
成果表达
成果名称：我的一天一分钟日记
要求：记录表 + 数据分析 + 规律总结，至少5组数据
自评量规

案例二

家庭"时间小管家"计划

真实建构情境
妈妈做饭时说"再等五分钟就好"，结果过了十分钟；爸爸说"马上到"，结果过了半小时。请你当一回家庭的"时间小管家"，帮家人提高时间估计的准确度。
建构任务
设计一份"家庭时间估计训练计划"，帮助至少2位家人提高对1分钟、5分钟、10分钟的估计准确度。
调用原理
🔬 亲身体验驱动原理 · 📊 数据可视化原理
建构步骤
  1. 设计3个时长（1分钟、5分钟、10分钟）的估计测试
  2. 先测试家人对这三个时长的初始准确度
  3. 制定训练方案：每天练习3次
  4. 用表格记录一周的训练数据
  5. 比较训练前后的数据，总结进步情况
成果表达
成果名称：家庭时间小管家训练报告
要求：训练前后对比数据 + 总结建议
自评量规

案例三

课间活动时间规划师

真实建构情境
课间只有10分钟，可你常常刚跑到操场就听到上课铃响了。怎样才能在10分钟内既上厕所、又喝水、还能玩一会儿？
建构任务
设计一份"课间10分钟最优活动方案"，精确规划每个活动的时间分配，并实际测试你的方案是否可行。
调用原理
🎯 规律总结原理 · 📊 数据可视化原理
建构步骤
  1. 连续3天记录你的课间活动和时间分配
  2. 分析哪些活动占用了太多时间
  3. 用时间轴规划理想的10分钟分配
  4. 按方案执行2天，记录实际用时
  5. 根据测试结果调整优化方案
成果表达
成果名称：我的课间10分钟最优方案
要求：现状分析 + 时间轴方案 + 测试结果
自评量规

案例四

时间单位换算"魔法转盘"

真实建构情境
1小时=60分钟，1分钟=60秒，那2小时是多少秒？120分钟是几小时？时间单位换算容易搞混。请你设计一个能帮助同学们快速换算的"魔法转盘"。
建构任务
制作一个"时间单位换算魔法转盘"（实物或图纸），能快速进行时、分、秒之间的换算。
调用原理
📊 数据可视化原理 · 🎯 规律总结原理
建构步骤
  1. 列出时、分、秒之间的所有换算关系
  2. 设计3层同心圆，分别代表时、分、秒
  3. 在每层标出对应的数值
  4. 用卡纸制作转盘
  5. 用5道换算题测试转盘的准确性
成果表达
成果名称：时间单位换算魔法转盘
要求：3层同心圆，每层标注对应数值，能旋转对照
自评量规

案例五

时间主题数学绘本创作

真实建构情境
一年级的小朋友刚认识钟表，但他们常常分不清时针和分针，也不知道一小时有多长。请你用数学+美术的方式，创作一本给一年级小朋友看的"时间故事绘本"。
建构任务
创作一本至少6页的"时间主题数学绘本"，用故事和图画帮助低年级同学理解时间概念。
调用原理
🔬 亲身体验驱动原理 · 🎯 规律总结原理
建构步骤
  1. 确定主题（如：小乌龟的一天、时钟王国历险记）
  2. 设计1—2个主角
  3. 把时间知识融入故事情节
  4. 每页配一幅图
  5. 标注用到的时间知识点，装订成册
成果表达
成果名称：我的时间主题数学绘本
要求：至少6页，每页有图有文字，包含至少3个时间知识点
自评量规

案例六

社区"时间便利站"设计方案

真实建构情境
小区门口经常有人问"现在几点了？""到学校要走多久？"如果有一个"时间便利站"能帮大家快速了解各种时间信息，那该多好！
建构任务
设计一份"社区时间便利站"方案，包含至少3种时间信息服务。
调用原理
🔍 对比发现原理 · 🎯 规律总结原理
建构步骤
  1. 采访5位邻居，了解他们最常问的时间问题
  2. 设计便利站的外观和功能
  3. 确定提供哪些时间信息
  4. 画出便利站的草图
  5. 用纸盒等材料制作一个模型
成果表达
成果名称：社区时间便利站设计方案
要求：需求调查结果 + 设计图 + 功能说明

案例七

"时间银行"创意方案

真实建构情境
如果时间可以像钱一样存起来、取出来，你会怎么管理你的时间？请你设计一个"时间银行"的创意方案来思考如何更高效地利用时间。
建构任务
设计一份"时间银行"创意方案，包含"存储规则""取款规则""利息计算"等核心机制。
调用原理
🎯 规律总结原理 · 📊 数据可视化原理
建构步骤
  1. 确定时间银行的基本单位（如：1分钟=1时间币）
  2. 设计存储规则（什么行为可以"存入"时间？）
  3. 设计取款规则（什么行为需要"取出"时间？）
  4. 设计"时间利息"机制
  5. 制作宣传海报展示方案
成果表达
成果名称：我的时间银行创意方案
要求：规则说明 + 示例交易记录 + 宣传海报
自评量规
