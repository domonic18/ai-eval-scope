📊 分层任务（三选一）

🌱

A 层 基础巩固

B 层 能力提升

🎨 开放层：创造性表达

从以下任务中任选一项完成，展示你的创意！

✍️

写作任务

以"我的一天"为题，写一篇 200 字左右的短文，用上时、分、秒描述一天中各个活动的时间安排。至少使用 5 个时间数据。

📊

图示任务

画一张"我的周末时间饼图"，用扇形图展示周末各项活动的时间分配。精确到半小时，标注每个扇形的角度和时间。

🎬

视频任务

录制一段 2 分钟的"时间知识小讲堂"视频，讲解一个时间换算技巧。要求有板书或演示，语言清晰。

🎤

演讲任务

准备一段 1 分钟的"时间名言"演讲，分享一句你最喜欢的时间名言，并说明为什么。
