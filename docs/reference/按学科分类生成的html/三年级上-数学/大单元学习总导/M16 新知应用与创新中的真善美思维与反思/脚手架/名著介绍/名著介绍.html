📚 名言及名著脚手架

从名言到名著，带你在文字中感受时间的力量。

💬 名言部分

冲突型
"时间就是生命。"
——富兰克林
🤔 思考：为什么说时间就是生命？如果浪费时间，等于浪费了什么？
启发型
"少年易老学难成，一寸光阴不可轻。"
——朱熹
🤔 思考："一寸光阴"到底有多长？为什么古人用"寸"来形容时间？
批判型
"人们常说时间在流逝，其实是我们自己在流逝。"
——佚名
🤔 思考：这句话和"时间就是生命"有什么不同？你同意哪种说法？
启发型
"完成工作的方法是爱惜每一分钟。"
——达尔文
🤔 思考：达尔文用了 20 年写《物种起源》，他是怎么"爱惜每一分钟"的？
批判型
"时钟的指针不会停，但你可以选择怎么度过每一秒。"
——现代格言
🤔 思考：这句话告诉我们对时间应该持什么态度？

📖 名著部分

📜 中国著作：《论语》中的时间智慧

核心问题：古人如何看待时间和人生？
核心观点：孔子在河边感叹"逝者如斯夫，不舍昼夜"，意思是时间像流水一样日夜不停地流逝。这句话告诉我们：时间一去不复返，要珍惜当下。

📋 目录结构（简化版）

学而篇：学习的态度 为政篇：为政的道理 里仁篇：仁德的修养 子罕篇：孔子的感叹

📖 适读人群建议

基础：《论语》儿童版（带拼音和插图）
进阶：《论语》白话注释版
挑战：《论语》原文

🐛 外国经典：《毛毛——时间窃贼和一个小女孩的故事》

核心问题：如果时间可以被偷走，会发生什么？
核心观点：德国作家米切尔·恩德通过这个故事告诉我们：时间就是生命，生命就在我们心中。那些让我们"节省时间"的东西，反而偷走了我们的时间。

📋 目录结构（简化版）

🌌 跨学科/当代作品：《时间简史》（儿童版）

核心问题：时间有起点吗？时间会结束吗？
核心观点：霍金在《时间简史》中探讨了时间的本质——时间有没有起点？有没有终点？时间为什么只能向前不能向后？
