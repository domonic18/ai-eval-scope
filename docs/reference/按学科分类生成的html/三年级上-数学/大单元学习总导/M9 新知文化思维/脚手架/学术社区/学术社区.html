📌 讨论议题（结构化）

1 你相信"时间感知"有快慢之说吗？

同一节40分钟的课，为什么有人觉得"一眨眼"有人觉得"度秒如年"？——用你的"一分钟体验"数据来说话。

2 如果你的手表现在突然停止不走了，你能用哪些方法估计现在大概几点？

排除看手机！你能想出多少种方法？越多越好！

想一想：太阳的位置、自己的影子、肚子饿不饿、公鸡打鸣……古人可没有手表哦！

3 古人的"日晷""水钟"和今天的"原子钟"，你觉得哪个最厉害？为什么？

三种计时工具各有优劣，你的选择是？

日晷：需要太阳，晚上和阴天不能用

水钟：不受天气影响，但水会蒸发

原子钟：精确到每亿年误差1秒，但需要高科技
