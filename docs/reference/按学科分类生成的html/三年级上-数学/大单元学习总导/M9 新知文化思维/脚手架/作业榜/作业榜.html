A档 ★

用"时、分、秒"记录你从起床到出门上学的时间安排，写一段80字左右的介绍。

💡 提示：6:30起床→6:45洗漱→6:55早餐→7:20出门，用上时间单位哦！

B档 ★★

你从家到学校走路需要15分钟，今天你起床晚了8分钟。请你算一算：如果你把走路速度提高一倍，能不能准时到校？请写出推理过程。

推理过程：
① 正常步行15分钟 → 速度提高一倍 → 时间减半 → 15÷2=7.5分钟
② 迟到8分钟 → 只需比正常快8分钟以内就能准时
③ 7.5分钟 < 8分钟 → ✅ 可以准时到校！
C档 ★★★

设计一道 "谁跑得快" 的数学题，包含至少两个运动员的不同时间数据（用不同单位），并写出解答过程。

示例题：
小明跑200米用了40秒，小华跑200米用了1分钟。谁跑得快？快多少秒？
解答：
1分钟=60秒，60秒 > 40秒，小明用时更短 → 小明跑得快。
快了60-40=20秒。
