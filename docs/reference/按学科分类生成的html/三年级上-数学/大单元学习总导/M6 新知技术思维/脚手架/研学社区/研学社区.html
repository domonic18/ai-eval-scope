🟢 基础：生活案例
📚 生活案例

做作业用了多长时间？

背景：小华每天做语文作业、数学作业各需要多少时间？他想知道自己做作业的效率，是变快了还是变慢了？
核心问题

怎么准确测量做作业的时间？一周的数据能发现什么规律？

决策路径
A 凭感觉估计 → ❌ 不准
B 看钟表记开始和结束时间 → ✅ 较准
C 用秒表分段计时 → ✅ 最准
📊 结果：用方案B发现，周一到周四作业时间稳定在40-50分钟，周五需要1小时以上
💡 可迁移启示：记录一周的真实数据，比"我觉得"更有说服力
🤔 "如果你是他" —— 记录自己一周的作业时间，看看哪天效率最高？
👆 点击展开思考提示
思考引导：你可以先猜猜自己每天作业花多久，再用计时器实际测一周。对比一下——你的"感觉"准吗？哪一天的差距最大？
🟡 进阶：历史案例
🏛️ 历史案例

古人如何知道"午时三刻"？

背景：古人说"午时三刻"行刑、"一炷香"的时间——他们到底靠什么计时？那时候没有电子表，他们是怎么知道时间的？
核心问题

古代计时方法准确吗？误差有多少？

决策路径
A 日晷（看太阳影子）→ 白天能用，阴天不行
B 水钟（滴水计时）→ 任何时间都能用，但水会蒸发
C 香钟（燃香计时）→ 便携但燃烧速度受风影响
⚖️ 结果：各有利弊，古人会根据场合选择不同方法
💡 可迁移启示：选择测量工具要看"使用场景"——没有完美的工具，只有合适的工具
🏮 "如果你是他" —— 如果你是古时候的更夫，你要用什么方法来确保准时打更？
👆 点击展开思考提示
思考引导：更夫需要夜间工作，不能用日晷。水钟和香钟哪个更适合？如果遇到大风天，你的备选方案是什么？
🔴 挑战：跨领域案例
🚀 跨领域案例

火箭发射为什么要"倒计时"？

背景：火箭发射前，指挥中心会喊"10、9、8……3、2、1、点火！"——这个全世界都熟悉的倒计时，背后有什么科学道理？
核心问题

为什么需要精确到秒甚至毫秒的倒计时？如果差1秒会怎样？

决策路径
A 提前1秒点火 → 可能错过最佳发射窗口
B 推迟1秒点火 → 轨道计算全错
C 精确到毫秒级同步 → 所有系统在同一瞬间启动 ✅
🛰️ 结果：只有方案C能确保火箭成功入轨
💡 可迁移启示：越是重大的事情，越需要精确的时间控制
🚀 "如果你是他" —— 设计一个"班级倒计时系统"——什么场合需要精确到秒的倒计时？
👆 点击展开思考提示
思考引导：想想看——运动会起跑、班级活动倒计时、考试结束提醒……哪些场合差一两秒没关系？哪些场合必须分秒不差？为什么？
