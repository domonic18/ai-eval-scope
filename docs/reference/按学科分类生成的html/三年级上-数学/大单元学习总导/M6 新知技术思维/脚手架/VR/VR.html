🏺 场景一：走进古代计时器博物馆

☀️

日晷模拟 —— 点击太阳，看影子变化！

🌅 早晨：影子很长，指向西方

📝 观察任务

👀观察：观察日晷、水钟、沙漏、香钟等古代计时工具的结构和运行方式

🖱️ 操作任务

🖱️操作：点击上方太阳图标，模拟日晷的影子随太阳移动而变化

📋 记录任务

✏️记录：记录每种工具的工作原理和优缺点

🥽
VR资源：Google Arts & Culture中的计时器展览
📺
替代方案：无VR时用3D展示动画或纪录片《时间的奥秘》

⚙️ 场景二：钟表内部结构探索

⏰
⚙️
🔧

齿轮联动：大齿轮转1圈，小齿轮转好几圈 → 秒针比时针转得快！

当前速度：正常

👀 观察任务

👀观察："钻入"机械钟表内部，观察齿轮如何联动，秒针如何转动

🖱️ 操作任务

🖱️操作：点击按钮调整齿轮大小，观察对指针速度的影响

📋 记录任务

✏️记录：画出齿轮传动简图，标注动力传递路径

🥽
VR资源：交互式钟表3D模型软件
📺
替代方案：拆解动画视频《钟表的工作原理》

🏃
🏃‍♂️
🏃‍♀️
👆 点击"开始比赛"，记录每位运动员的成绩

👀 观察任务

👀观察：站在计时员视角，观察百米赛跑的起跑和冲线瞬间

🖱️ 操作任务

🖱️操作：按下计时按钮，记录不同运动员的成绩

✏️记录：制作成绩排行榜，比较0.01秒的差距有多大

📺
替代方案：在线计时模拟器或视频分析工具
