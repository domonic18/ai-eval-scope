🟢 基础工具（易上手）
⏱️ 工具 1

秒表 / 计时器

精确测量一段时间的长短（精确到 0.01 秒）

使用步骤：

  1. 按"开始"按钮启动计时
  2. 结束时按"停止"
  3. 读取显示的时间
  4. 按"复位"归零

💡 试试：测量自己数 30 个数需要多长时间

📈 工具 3

时间轴 / 时间线图

  1. 确定起点和终点时间

  4. 标注时间刻度
