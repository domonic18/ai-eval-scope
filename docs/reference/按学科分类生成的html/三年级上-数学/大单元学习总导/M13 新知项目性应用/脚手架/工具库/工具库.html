1 时间四象限图 🎯
用途：区分事情的紧急和重要程度，决定先做什么、后做什么
  1. 画一个十字，分成四格
  2. 左上：重要且紧急（马上去做）
  3. 右上：重要不紧急（计划去做）
  4. 左下：不重要但紧急（快速处理或请人帮忙）
  5. 右下：不重要不紧急（尽量少做）

2 番茄计时法 🍅
用途：用"25分钟专注 + 5分钟休息"的节奏高效学习
  1. 设定25分钟倒计时
  2. 专注做一件事（不碰手机、不走神）
  3. 铃响后休息5分钟（站起来活动一下）
  4. 每完成4个"番茄"，休息15分钟

🚀 进阶工具 · 专业
3 甘特图 📊
用途：用条形图展示多项任务的时间安排，一眼看清全局
  1. 列出所有需要完成的任务
  2. 在时间轴上为每个任务画一条"条形"
  3. 用不同颜色区分任务类型（学习用蓝色、运动用绿色）
项目规划、周末安排、假期作业计划
4 时间日志 📝
用途：记录真实的时间使用情况，找到你的"时间漏洞"
  1. 以30分钟为单位，记录一天中每个时段的活动
  2. 睡前统计各项活动的总时长（睡觉、学习、玩耍、吃饭各多少）
  3. 比较"计划"和"实际"的差距
发现自己的时间都花在哪里了
