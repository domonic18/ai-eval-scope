🤝 学术社区 — 学习共同体

构建"学习共同体" —— 通过与同伴、家庭和社会的真实连接，把你的反思分享出去，也从别人的经验中学习。

👥 同伴连接 · 讨论与协作

💬 讨论议题

1 你觉得"1分钟"到底有多长？
每人先闭眼估时，然后一起对答案
2 你在"时、分、秒"单元中最大的一次进步是怎么发生的？
3 分享你的"学习成功档案"
别人看你的档案能看懂吗？

🎭 角色分工

⏱️
计时员
控制讨论时间
📝
记录员
记录"成功经验"
🎤
发言员
代表小组汇报
❓
质疑员
推动讨论深入

计时员：准备一个计时器，每个议题限时5分钟。

记录员：用关键词记录，讨论结束后整理成完整句子。

发言员：汇报前先复述一遍，确保说清楚了。

质疑员：用"我有个不同的想法…"开头，不否定别人，只补充新角度。

🏠 家校连接 · 支持与反馈

📋 家庭任务：采访家人 —— "您觉得10分钟能做什么？"
对比你和家人的答案，看看有什么不同。
📬 家长反馈：请家长看你完成的"时间账单"，听听他们的建议。

🌐 社会连接 · 真实世界

🔬 专家连线（可选）：了解精算师 / 航天工程师 / 体育教练如何与时间打交道

（车站大屏、商场营业时间表、红绿灯倒计时……）
