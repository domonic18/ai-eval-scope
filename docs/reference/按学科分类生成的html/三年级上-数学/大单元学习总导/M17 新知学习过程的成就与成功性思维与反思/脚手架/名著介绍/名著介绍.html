📖 名著推荐（3 本）

🇨🇳 中国著作

《毛毛》—— 米切尔·恩德

核心问题：如果时间被偷走了，生活会变成什么样？

故事主线：时间窃贼出现 → 人们越来越忙 → 毛毛发现真相 → 夺回时间

核心观点：时间不是用来"节省"的，而是用来"体验"的。

灰衣人偷走人们的时间 —— 为什么人越"省时间"，反而越没时间？
想一想：你身边有没有"越忙越没时间做喜欢的事"的人？

适读：三年级以上 纸质书·各大书店 微信读书 喜马拉雅有声书
🌍 外国经典

《时间之谜》（科普版）

核心问题：时间到底是什么？

核心观点：时间在物理学、生物学、心理学里有不同的面貌。

为什么蜂鸟觉得时间过得比人慢？
小型动物的新陈代谢更快，它们感知时间的"帧率"更高——1秒里能"看到"更多画面！

适读：三年级可读（基础篇） 图书馆借阅 电子书平台
🔗 跨学科作品

《时间的秩序》卡洛·罗韦利（儿童改编版）

核心问题：如果没有时钟，人类还能感知时间吗？

核心观点：时间不是"嘀嗒嘀嗒"走的东西——它和空间、引力紧密相连。

"山顶的时间比山脚过得快一点"——这是真的！
根据爱因斯坦的相对论，引力越强的地方时间越慢。山脚离地心近，引力更大，时间过得更慢！虽然差得微乎其微，但原子钟可以测出来。

适读：挑战级·亲子共读 纸质书精装版推荐
