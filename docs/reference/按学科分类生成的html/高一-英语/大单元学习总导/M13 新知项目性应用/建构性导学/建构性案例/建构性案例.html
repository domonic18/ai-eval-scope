案例 1 ⭐ 我的友谊画像 — My Ideal Friendship
▼
🎯 情境 你在新班级认识了一位好朋友，想用英语给 TA 写一段自我介绍和友谊画像——你心中最理想的友谊是什么样的？
📋 建构任务 写一段 150 词的"My Ideal Friendship"陈述，表达你对友谊的理解和期待。
🔧 原理调用 情境锚定法 方案建构与英语表达
📝 建构步骤 ① 回忆你和这位朋友相处的 2—3 个具体瞬间 → ② 确定你认为友谊最重要的 3 个品质 → ③ 用关键词汇（loyal, trustworthy, supportive, honest, caring）组织语言 → ④ 配上简单的插画或照片，做成友谊画像卡片
✅ 自评 [ ] 我是否使用了至少 5 个描述品质的形容词？
[ ] 我是否写了一个具体的回忆事例而不是空谈？
案例 2 ⭐ 解决一个小矛盾
▼
🎯 情境 你和朋友因为周末安排产生了分歧：你想去看电影，TA 想去打篮球。双方都坚持己见，气氛有点尴尬。
📋 建构任务 用英语写一封 200 词的沟通邮件，理性友善地表达你的想法并邀请对方商量。
🔧 原理调用 需求拆解策略 证据收集与共情表达
📝 建构步骤 ① 描述事实（"Last Saturday we had a disagreement about..."）→ ② 表达你的感受并猜测对方的感受 → ③ 提出折中方案（"What if we go to the movie first, then play basketball?"）→ ④ 邀请对方回复
✅ 自评 [ ] 我是否用了"I understand..."句式表达共情？
[ ] 我是否提出了至少 1 个可执行的具体方案？
案例 3 ⭐⭐ 班级友谊调查与行动建议
▼
🎯 情境 你发现班级里有一些同学在课间总是一个人待着，似乎被孤立了。你想了解情况并提出改进建议。
📋 建构任务 完成一份英文"Class Friendship Survey & Action Plan"：先记录你观察到的情况，再给出 3 条具体的行动建议。
🔧 原理调用 所有已学原理
📝 建构步骤 ① 观察并记录 2—3 个具体现象 → ② 分析可能的原因（交友困难？性格内向？）→ ③ 提出 3 条具体行动建议（如 pairing system, lunch group rotation）→ ④ 用英语写成简报
✅ 自评 [ ] 我的建议是否基于观察到的具体现象？
[ ] 我是否至少用了 3 个本单元的功能句型？
案例 4 ⭐⭐ 教材人物书信重写
▼
🎯 情境 教材 Unit 1 中的人物 Anne 收到了一封朋友的回信，但回信写得太简短了。Anne 需要一封更完整、更有温度的回信。
📋 建构任务 以教材中某个人物的身份，给 Anne 写一封 250 词的英文回信，表达关心、分享自己的经历、给出建议。
🔧 原理调用 情境锚定法 共情表达
📝 建构步骤 ① 分析你要扮演的人物的性格特点 → ② 确定写信目的（安慰/建议/分享）→ ③ 模仿教材中的语言风格 → ④ 完成书信并检查格式
✅ 自评 [ ] 我写的是否符合该人物的身份和口吻？
[ ] 书信格式是否正确（日期、称呼、正文、落款）？
案例 5 ⭐⭐ 友谊心理学英语小报
▼
🎯 情境 心理学研究表明，友谊的建立通常经历 3 个阶段：接触→了解→信任。你能否用英语向同学们介绍这个有趣的发现？
📋 建构任务 设计一份 A4 大小的英文"Friendship Knowledge"小报，包含定义、三个阶段说明、一个研究案例和插图。
🔧 原理调用 方案建构与英语表达 成果自检与优化
📝 建构步骤 ① 搜集友谊三阶段的信息 → ② 设计小报布局（标题→定义→阶段图→案例→名言）→ ③ 用英语写出各板块内容 → ④ 添加手绘插图
✅ 自评 [ ] 我的小报是否包含 3 个以上的信息板块？
[ ] 语言是否准确且适合同龄人阅读？
案例 6 ⭐⭐⭐ 国际笔友通信计划
▼
🎯 情境 学校与国外友好学校开展了笔友项目。你被分配了一位美国笔友，TA 对中国文化很感兴趣。
📋 建构任务 写一封 250—300 词的英文交友信，介绍中国文化的一个有趣方面，同时询问对方的文化。
🔧 原理调用 证据收集与共情表达 方案建构与英语表达
📝 建构步骤 ① 选一个你想介绍的文化主题（中秋/书法/火锅）→ ② 用具体事例说明它的特别之处 → ③ 表达你对美国文化的兴趣并提问 → ④ 用书信格式完成
✅ 自评 [ ] 我是否介绍了中国文化的一个具体方面而不仅是笼统描述？
[ ] 我是否提出了 2—3 个关于对方文化的问题以促进对话？
案例 7 ⭐⭐⭐ 班级友谊文化建设方案
▼
🎯 情境 你被选为班级文化建设的负责人。你想在下个月举办一个"友谊月"活动，通过系列活动增进同学之间的了解和信任。
📋 建构任务 完成英文版"Friendship Month Proposal"，包含活动名称、目的、至少 3 项活动流程和简单预算。
🔧 原理调用 需求拆解策略 方案建构与英语表达 成果自检与优化
📝 建构步骤 ① 分析班级现状：同学们之间有什么友谊问题？→ ② 确定活动目标 → ③ 设计 3—4 项具体活动（如"Secret Friend"一周配对、"Friendship Story"分享会）→ ④ 用英语写成提案 → ⑤ 自检完善
✅ 自评 [ ] 我的提案是否包含活动名称、目的、流程和预算？
[ ] 我的活动方案是否切实可行、适合班级实际情况？
[ ] 我是否使用了连接词让提案结构清晰？
