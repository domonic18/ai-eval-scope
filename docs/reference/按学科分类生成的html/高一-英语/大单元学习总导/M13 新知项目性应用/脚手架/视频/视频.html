🎬 视频 — 三层观看任务

通过聚焦型观看任务，训练从视频中提取关键信息的能力，将理解应用到友谊沟通创作中

📖 基础层：解释型

TED-Ed What makes a good friendship? ⏱ 4min

👁 观看任务：记录视频中提到的 3个 良好友谊的特征。

  • 视频中认为友谊的核心要素有哪些？
  • 为什么说 "quality time" 对友谊很重要？
⏸ 暂停思考点（2:15）
停在 "reciprocity" 概念出现时——你有这样的朋友吗？
📝 输出任务：用英文写出 50 词总结

🔍 进阶层：分析型

Friends S2E8 Ross 和 Chandler 的误会与和解 ⏱ 3min

👁 观看任务：分析两人的矛盾起因和和解方式。

  • 他们的误会是怎么产生的？
  • 最后是怎么和解的？用了哪些沟通技巧？
⏸ 暂停思考点（1:30）
停在冲突爆发点——如果你是他们，你会怎么说？
📝 输出任务：用英文写出 200 词的分析，对比剧中的做法和你自己的处理方式

🧠 挑战层：辩论 / 前沿型

TED Talk Nicholas Christakis — The hidden influence of social networks ⏱ 18min

👁 观看任务：理解社交网络如何影响个人行为和幸福感。

  • Christakis 的研究发现了关于友谊网络的什么有趣现象？
  • 这些发现对你选择朋友有什么启示？
⏸ 暂停思考点（5:00）
停在 "emotional contagion" 概念——你的情绪会受朋友影响吗？
📝 输出任务：用英文写下你的观点：你认为"友谊是相互影响的"这个观点有多大的道理？
