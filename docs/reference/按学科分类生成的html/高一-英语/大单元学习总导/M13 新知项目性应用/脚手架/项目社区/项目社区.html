🚀 项目社区

通过真实问题驱动的项目式学习，完成短、中、长周期项目，深入理解和运用关于友谊的英语知识

🟢 短周期 · 1-3天

Friendship Quote Collection

🎯 目标：收集 5 句关于友谊的英文名言，每句配一段英文解释
  • Day 1：搜索并选出 5 句名言
  • Day 2：为每句写 50 词的英文评论
  • Day 3：排版制作成电子 / 纸质"A4 名言集"
成果：A4 单页（手绘或电子） 评价：来源准确 · 语法基本正确 · 排版美观
🔗 Goodreads quote search
🟡 中周期 · 1-2周

Friendship Advice Blog

🎯 目标：创建一个 3-5 篇英文建议帖的迷你博客
  • 设计博客主题和风格（给新生的建议？关于异地友谊？）
  • 收集或构思 3 个真实的友谊问题
  • 为每个问题写一篇 200 词的英文建议帖
  • 配上英文标题和好友图片 / 图标
  • 邀请一位同学阅读并回复
成果：3-5 篇帖子的博客 评价：结构完整 · 建议可行 · 语言准确
🔗 Padlet / Canva / Google Sites
🔴 长周期 · 1月+

Class Friendship Handbook

🎯 目标：全班合作编写一本英文"班级友谊手册"
  • Week 1：确定目录（Welcome, Tips, Profiles, Advice, Memories）
  • Week 2：每人负责一个章节的内容撰写
  • Week 3：同伴互审、修改润色
  • Week 4：排版设计、打印 / 发布
成果：10-20 页英文手册（PDF 或实体） 评价：内容完整 · 语言统一 · 排版专业
🔗 Canva / Google Docs / Book Creator
