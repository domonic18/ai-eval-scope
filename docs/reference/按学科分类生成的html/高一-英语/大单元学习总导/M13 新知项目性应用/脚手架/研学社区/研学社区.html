🌟 基础

案例：小明和小华的小组合作冲突

背景：两人在准备英语展示时，一人想做PPT，另一人想写剧本
核心问题：合作中意见不合，如何达成共识？
决策路径：
方案A：轮流决定 方案B：每人负责一部分 ✓ 方案C：请教老师
结果：选择方案B，每人做自己擅长的事，展示效果很好
💡 可迁移启示：友谊≠完全一致，而是尊重差异、合理分工
任务：如果你是小明，你会怎么做来避免冲突升级？
提示：可以先倾听对方的理由，再提出折中方案

🔥 进阶

案例：海伦·凯勒和安妮·莎莉文的师生友谊

背景：海伦·凯勒是聋盲人，安妮是她的家庭教师
核心问题：一个无法沟通的人，如何建立深厚的情感连接？
决策路径：
安妮用触觉字母教海伦拼写 → 海伦突破"水"的概念 → 建立信任 → 形成终身友谊
结果：海伦成为作家和社会活动家
💡 可迁移启示：真正的友谊建立在相互理解和信任上——即使没有语言也能连接
任务：如果你是安妮·莎莉文，在海伦不理解"水"这个词时，你会怎么做？
提示：想想还有哪些感官途径可以建立理解

🧩 挑战：复杂系统类案例

🚀 挑战

案例：一个班级的友谊网络分析

背景：一个40人班级中，有些学生被孤立，有些是社交中心
核心问题：如何改善班级整体的友谊氛围？
决策路径：
绘制社交关系图 → 找到孤立学生 → 设计结对活动 → 评估效果
结果：经过3个月的结对计划，班级社交网络更加均衡
💡 可迁移启示：友谊不仅是个人的事，更是一个系统问题——关注被忽视的人
