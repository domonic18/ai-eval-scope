📊 评价量规

参考
维度 优秀 (4分) 良好 (3分) 合格 (2分) 待改进 (1分)
内容完整度 包含全部要素 缺1个要素 缺2个要素 要素缺失较多
语言准确性 语法基本正确 少量错误 较多错误 影响理解
情感表达 真诚自然 较真诚 略显生硬 缺乏情感
结构组织 逻辑清晰 较清晰 有些混乱 无逻辑结构
