分层作业（三选一）

🅰️ 基础作业

根据给定的框架模板，完成中西方友谊观对比表。模板会给出对比维度（如：交友方式、朋友定义、忠诚表现等），你只需填入对应内容。

🅱️ 进阶作业

独立设计并完成一份中西方友谊观对比分析。你需要：
1) 自己确定至少 4 个对比维度
2) 每个维度用 2—3 句英文分析
3) 给出自己的判断和总结

🅲 挑战作业

进行小型调查，采访 2—3 个不同背景的人（如：家长、外教、笔友），写出调查结果分析。包含：
1) 采访提纲（英文）
2) 采访记录摘要
3) 你的对比分析结论
