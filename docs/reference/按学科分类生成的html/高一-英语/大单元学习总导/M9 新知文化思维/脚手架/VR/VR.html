观察任务

进入 VR 场景，仔细观察不同文化中人们的交友行为。对比以下两个场景：

🇬🇧 场景一：英国校园午餐时间

观察英国学生如何聊天、如何约人、如何组成小圈子。注意他们在对话中的肢体距离、眼神交流、话题选择。

🇨🇳 场景二：中国校园课间

观察中国学生与朋友互动的方式。注意群体结构、对话方式、合作与竞争的表达。

观察要点：

📩
