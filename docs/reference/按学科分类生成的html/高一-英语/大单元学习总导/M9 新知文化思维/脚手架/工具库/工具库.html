进阶
📊 文化维度评分表

用途：用霍夫斯泰德文化维度理论定量分析文化差异

1查找中西方在个人主义指数、权力距离等维度的评分
2用柱状图对比
3分析评分与友谊表现的关系

霍夫斯泰德关键维度：

  • 个人主义 vs 集体主义：西方偏个人主义（独立交友），东方偏集体主义（圈子交友）
  • 权力距离：影响友谊中的平等意识
  • 不确定性规避：影响交友中的风险偏好
