🎬 视频脚手架

以"观看焦点"为核心的视频学习资源 · 跨文化友谊

基础层 解释型视频

▼

TED-Ed《What is friendship?》

📺 YouTube TED-Ed频道 ⏱ 约5分钟
👁 观看任务
  • 找出亚里士多德的三种友谊分类，并用自己的话解释
❓ 关键问题
  • 亚里士多德认为哪种友谊最持久？为什么？
  • 你的友谊属于哪种类型？
⏸ 2:30 暂停思考 — 想一想：你朋友圈里哪些属于"功利之友"？哪些属于"快乐之友"？
⏸ 4:15 暂停思考 — 你觉得"美德之友"在高中阶段可能存在吗？

📝 输出任务

用维恩图比较三种友谊，并注明你的朋友分别属于哪一类。

进阶层 分析型视频

▼

BBC纪录片《The Culture Shock》（文化冲击）

📺 BBC纪录片 / B站搜索"文化冲击 纪录片" ⏱ 约15分钟
👁 观看任务
  • 记录片中至少3个跨文化交友失败的案例，分析原因
❓ 关键问题
  • 文化冲击的四个阶段是什么？
  • 在跨文化交友中，为什么到了"adjustment"阶段友谊才能建立？
⏸ 多处标记 不同人物的"文化冲突时刻"，记录你的观察

📝 输出任务

绘制"跨文化友谊建立曲线图"，标注可能的文化冲突点和转折点。

挑战层 辩论 / 前沿型视频

▼

Sherry Turkle TED演讲《Connected, but alone?》

📺 TED.com / TED官方YouTube ⏱ 约20分钟
👁 观看任务
  • 找出Turkle对数字时代友谊的核心批判
❓ 关键问题
  • Turkle说"我们期望从技术中获得太多，从彼此身上获得太少"——你同意吗？
  • 数字时代的友谊算不算"真正的友谊"？中西方对这一问题的看法是否不同？
⏸ 演讲中对比"online connection"和"real conversation"时暂停 — 思考你的体验

📝 输出任务

写一篇英语回应文章（200词），同意或反驳Turkle的观点，引用你自己的经历。
