本模块为什么重要

点击下方按钮，展开每个视角：

很多同学背了大量单词，但遇到外国人时仍不知如何开启真正对话。原因不是语言能力不够，而是缺乏对文化差异的理解。本模块帮你填补这个"文化空白"。

跨文化理解力是21世纪核心素养。无论你将来是否出国，你都会在新闻、社交媒体、游戏中接触不同文化的人。学会理解他们，才能更好地与这个世界相处。

高考英语阅读理解和写作越来越注重"文化意识"和"思维品质"——这正是本模块要训练的能力。
