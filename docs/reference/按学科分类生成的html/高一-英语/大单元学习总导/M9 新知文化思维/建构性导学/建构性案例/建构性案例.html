案例一

个人理解型——我的"友谊定义"对比图

差异觉察原理证据分类原理
建构情境

回顾你与外国朋友的交流经历——"朋友"这个词在不同文化中分量不同。你的中国朋友可能只有2-3个"真正的朋友"，而你的外国同学可能有10个以上"friend"。

建构任务

建构一份"中西方'朋友'定义对比分析图"

建构步骤
  1. 列出你心中"朋友"的标准
  2. 列出你认为西方人眼中"friend"的标准
  3. 对比两者的相同和不同
  4. 用维恩图呈现对比结果
  5. 用英语写一段总结
脚手架推荐

维恩图模板、英语比较句型（"While... / In contrast..."）

成果表达

维恩图 + 60-80词英文说明

自评量规
  • 维恩图清晰展示异同
  • 至少2个维度对比
  • 英文说明没有语法错误
案例二

生活应用型——跨文化交友指南

文化溯源原理框架建构原理
建构情境

你的学校要来一批外国交换生，你需要为他们准备一份"在中国交朋友"的指南，帮助他们理解中国学生的交友方式。

建构任务

创作一份"在中国交朋友"实用指南（中英双语）

建构步骤
  1. 列出中国学生交友的3-5个特点
  2. 解释每个特点背后的文化原因
  3. 给出具体建议（如如何邀约中国朋友？如何看待AA制？）
  4. 整理为指南格式（tips形式）
脚手架推荐

建议话术模板、文化对比语料库

成果表达

一页指南（A4纸大小），中英双语

  • 包含至少3条建议
  • 每建议附文化解释
  • 中英双语表达准确
案例三

校园问题解决型——跨文化友谊冲突调解方案

差异觉察原理文化溯源原理
建构情境

班上一位中国同学和一位德国同学因为小组合作方式发生矛盾：德国同学坚持要提前一周完成，中国同学觉得"不用那么着急"。双方都不理解对方。

建构任务

设计一份"跨文化友谊冲突调解方案"

建构步骤
  1. 分析双方的行为差异
  2. 寻找各自的文化根源（时间观念、任务观念）
  3. 设计沟通策略
  4. 写出调解建议
成果表达

调解方案（约200词，含分析+建议）

  • 分析了双方立场
  • 提供了文化解释
  • 建议具体可操作
案例四

学科表达型——英语辩论稿

框架建构原理
建构情境

班里要举办一场英语辩论赛，辩题是："真正的友谊在网上可以建立吗？"你需要准备一份辩论稿，从文化视角分析不同人可能的不同看法。

建构任务

撰写一份英语辩论稿（正方或反方）

建构步骤
  1. 确定你的立场
  2. 收集支持论据（文化因素、社会现象）
  3. 组织论点结构（引言—论点—论据—结论）
  4. 撰写完整的辩论稿
成果表达

200-300词英语辩论稿

  • 结构完整
  • 至少3个论点
  • 每个论点有论据支持
案例五

跨学科整合型——友谊的文化心理学分析海报

框架建构原理文化溯源原理
建构情境

从心理学角度看，友谊是人类的社会需求（马斯洛需求理论）；从文化学角度看，不同文化满足这一需求的方式不同。请整合两个学科视角来解释：为什么中西方友谊观存在差异？

建构任务

制作一份"跨学科友谊观分析海报"

建构步骤
  1. 从心理学角度列出友谊对人类的普遍意义
  2. 从文化学角度列出中西方满足友谊需求的不同方式
  3. 将两个视角整合成一个框架
  4. 用海报形式呈现
成果表达

A3海报（手绘/电子均可），图文结合

  • 包含两个学科视角
  • 两者之间有逻辑连接
  • 视觉美观、结构清晰
案例六

社会情境型——社交媒体时代的跨文化友谊白皮书

差异觉察原理证据分类原理文化溯源原理框架建构原理
建构情境

社交媒体让跨文化交友变得前所未有的容易。但外国网友评论你的照片"amazing!"，你该认真回复感谢吗？对方发信息你24小时没回，对方会怎么想？

建构任务

撰写一份"社交媒体时代的跨文化友谊观察白皮书"

建构步骤
  1. 收集3个社交媒体跨文化交友的典型场景
  2. 分析每个场景中的文化差异表现
  3. 追溯差异的文化根源
  4. 给出"数字时代跨文化交友建议"
成果表达

一页白皮书（中英双语，300-400词）

  • 覆盖至少3个场景
  • 每个场景有文化分析
  • 建议具有可操作性
案例七

综合创新型——"我的跨文化友谊手册"项目

综合运用所有原理
建构情境

经过前6个案例的练习，你已有能力独立完成一个综合性的建构项目。请将所有成果整合为一个完整的"跨文化友谊手册"，分享给下一届学弟学妹。

建构任务

创作"我的跨文化友谊手册"（整合型项目成果）

建构步骤
  1. 回顾前6个案例的成果
  2. 选择最精华的部分整合
  3. 补充新的原创内容
  4. 设计手册的版式和装饰
  5. 撰写中英文前言和目录
成果表达

完整手册（4-6页，可打印/电子）

  • 内容涵盖文化差异分析
  • 包含实用建议
  • 中英文表达准确
  • 版式美观易读
  • 体现个人思考和创意
