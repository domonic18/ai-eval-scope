一、作业设计理念

友谊不仅是情感体验，更是一组值得追问的哲学命题：我们如何定义"朋友"？友谊的边界在哪里？当价值观冲突时，友谊还能维系吗？

本作业库以 "新知哲学思维" 为认知框架，引导你在英语学习过程中从"使用语言描述友谊"跃迁至"用语言思辨友谊"，实现知识的再现、内化与创造性表达。

Task 1.2 — Key Sentence Transformation

"I don't want to set down a series of facts in a diary as most people do, but I want this diary itself to be my friend."

  • 1. Anne considered her diary as her best friend. (T/F) → Evidence: _________

🟡 分层层 — 差异化任务

根据你的语言水平与思维深度，从以下三条路径中选择一条完成。

选择 A — 语言聚焦型

📝 Friendship Advice Letter

你的朋友 Tom 刚转学，在新学校交不到朋友，感到很孤独。请用本单元学到的词汇和句型，给他写一封 80—100 词的英文信，给出至少 3 条交朋友的建议。

🎯 评价重点：词汇运用准确度、书信格式、内容完整性

选择 B — 分析思辨型

💭 Friendship Philosophy Mini-Essay

阅读以下引语，选择其中一句，写一篇 120—150 词的英语短评，分析你是否同意该观点，并联系课文或生活经历论证。

  • "A friend to all is a friend to none." — Aristotle
  • "The only way to have a friend is to be one." — Emerson
  • "Walking with a friend in the dark is better than walking alone in the light." — Helen Keller

🎯 评价重点：论点清晰度、论据相关性、语言组织与逻辑连贯性

选择 C — 创造应用型

🎬 Friendship Paradox Analysis

安妮把日记当作最好的朋友。请分析："把无生命的物体当作朋友，是更安全的选择，还是更孤独的选择？"

  • 用英语录制一段 2—3 分钟的分析视频或音频
  • 包含：一个具体论点 + 至少两条来自课文或生活的证据 + 一个反方讨论

🎯 评价重点：论点新颖度、证据充分性、表达的清晰度与感染力
