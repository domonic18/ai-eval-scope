🧠 思维工具 基础

视觉思维导图

📋 用途说明

用中心辐射式图形梳理 Unit 1 Friendship 中的词汇网络、人物关系和情感线索，将抽象友谊概念转化为可视化结构。

📌 使用步骤
  1. 在白纸/白板中央写下 "Friendship" 或 "Anne's Best Friend"
  2. 画出主干分支（qualities of a friend, activities, conflicts, feelings）
  3. 从课文和阅读材料中提取关键词填入分支
  4. 用彩色标记情感词汇和主题词
以 "Anne's Best Friend" 为中心，辐射出 "nature → hiding place → feelings: lonely, hopeful, free" 等分支，直观呈现 Anne 的情感变化。
🎯 适用场景

课前词汇预热 / 课文结构梳理 / 写作素材收集

💻 数字工具 基础

Canva 海报制作

📋 用途说明

用 Canva 制作友谊主题的视觉化英语海报，融合图文排版锻炼美术思维与英语表达能力。

📌 使用步骤
  1. 打开 Canva 选择海报模板
  2. 确定主题句（如 "A true friend is…"）
  3. 搭配图片、图标和英语文案
  4. 运用配色和排版原则完成设计
制作一份 "Friendship Qualities" 海报：图标搭配英语短句——Trust（握手图标）、Support（拥抱图标）、Honesty（对话气泡图标）。
🎯 适用场景

课后创意输出 / 英语文化墙布置 / 小组分享展示

进阶工具 专业
📊 数据工具 进阶

情感色彩分析表

📋 用途说明

用颜色编码系统标注课文语篇中的情感词汇，分析友谊话题中的情感色彩分布和变化趋势。

📌 使用步骤
  1. 从 Unit 1 Friendship 课文中摘取重点段落
  2. 绿色标注积极词（joy, grateful, trust），红色标注消极词（lonely, anxious, betrayed），蓝色标注中性词
  3. 统计三类词汇的数量比例
  4. 用柱状图或饼图可视化呈现
分析 Anne Frank 日记选段：消极情感词占比超 60%，绿色词汇仅出现在 "nature" 相关段落 → "Anne 在自然中寻求情感慰藉"。
🎯 适用场景

深度文本分析 / 情感态度理解 / 写作情感铺垫分析

🧠 思维工具 进阶

视觉效果分析卡

📋 用途说明

分析课文中的视觉描写手法（比喻、意象、色彩词），理解作者如何用"画面感"表达友谊情感。

📌 使用步骤
  1. 选择一段文本
  2. 用方框标注视觉性词汇（颜色、形状、光影等）
  3. 用圆圈标注比喻和拟人修辞
  4. 分析这些视觉元素如何服务于情感表达
分析 "Nature is the only thing I can see out of the window"——提取视觉词 "sky, chestnut tree, roses, birds"，这些自然意象是 Anne 自由渴望的视觉化投射。
🎯 适用场景

文学赏析 / 修辞手法学习 / 英语写作中的细节描写训练

💻 数字工具 进阶

Miro 协作白板

📋 用途说明

使用 Miro 在线白板的便签、图形和连线功能，与同学合作建构友谊主题的概念地图。

📌 使用步骤
  1. 创建 Miro 白板并邀请小组成员
  2. 用不同颜色便签分类记录友谊相关要素
  3. 用箭头和连线展示要素之间的因果关系
  4. 用框架功能分区域组织信息
小组共创 "What Makes a Good Friend?"——黄色便签写性格特征，蓝色写行为，粉色写结果，连线展示 "honest → trust → deeper friendship" 的逻辑链。
🎯 适用场景

小组协作学习 / 概念建构活动 / 课堂互动

高阶工具 研究级
📊 数据工具 高阶

词汇语义网络图

📋 用途说明

用语义网络分析工具绘制 Unit 1 Friendship 核心词汇的语义关系图谱，发现词汇之间的隐含关联。

📌 使用步骤
  1. 提取课文中与友谊相关的核心词汇（20—30 个）
  2. 标注词之间的关系类型（同义、反义、上下义、搭配）
  3. 用工具（Gephi 或手动绘制）生成语义网络图
  4. 分析网络中的高连接度节点（hub words）
"friend" 同时连接 "trust"、"share" 和 "loyal"，"enemy" 连接 "betray" 和 "conflict"——两个子网络形成对比，揭示课文对朋友和敌人的二元对立表达。
🎯 适用场景

词汇深度研究 / 语篇语义分析 / 学术英语学习

🧪 实验工具 高阶

友谊视觉日记

📋 用途说明

连续 7 天拍摄/收集代表"友谊"的照片，并用英语撰写图注，训练用视觉媒介表达抽象概念的能力。

📌 使用步骤
  1. 确定每日主题（Day1: 陪伴；Day2: 分享；Day3: 成长）
  2. 每日拍摄一张能表达该主题的照片
  3. 用 50—100 词英语描述照片中的友谊元素
  4. 周末汇总成图文日记，分析自己的友谊理解演变
Day2 主题 "Sharing"——拍摄两张课桌并排的照片。图注："Two desks side by side, sharing the same sunlight. This reminds me of Anne sharing her thoughts with her diary — a different kind of sharing, but equally precious."
🎯 适用场景

项目式学习 / 综合语言实践 / 反思性学习

🧠 思维工具 高阶

设计思维工作坊

📋 用途说明

运用设计思维五步法——同理心→定义→构思→原型→测试，为课文中的友谊困境设计解决方案。

📌 使用步骤
  1. 同理心：分析人物的情感需求（Anne 需要什么？）
  2. 定义：用一句话定义核心问题
  3. 构思：头脑风暴 3—5 种解决方案
  4. 原型：用视觉化方式呈现方案（草图、流程图）
  5. 测试：回到文本检验方案是否符合人物性格和时代背景
针对 Anne Frank 的困境——同理心阶段：Anne 需要被倾听和理解；定义阶段：如何在被隔离的环境中维持情感连接？构思阶段提出"秘密语音日记"等创意方案。
🎯 适用场景

创造性问题解决 / 跨学科项目 / 高阶思维训练

📖 使用建议

  1. 按需取用：根据当前学习任务选择合适的工具，不必一次学完所有
  2. 循序渐进：从基础工具入手，熟悉美术思维方式后再尝试进阶和高阶工具
