🟢 基础工具

易上手

🧠 思维导图工具 — XMind

用途：用辐射状结构组织关于"Friendship"的零散想法，理清思路。

  1. 中心写上 Friendship
  2. 向外辐射 4 个一级分支：定义 / 类型 / 品质 / 行动
  3. 每个分支继续延伸二级分支
  4. 用不同颜色区分

示例：一级分支"品质"下延伸 → (honest, caring, funny, loyal)

适用场景：头脑风暴、整理概念、写作前构思

📋 概念图检查清单

🟡 进阶工具

专业

🤖 AI 写作助手 — ChatGPT / Claude / Kimi

用途：辅助英语写作——不是让 AI 替你写，而是让 AI 帮你检查和润色。

  1. 自己写完初稿
  2. 输入："Please check my English writing and suggest improvements"
  3. 把 AI 建议和你的原文对比，决定采纳哪些

示例对比：

你写：I very like my friend.
AI 提示：I really like my friend.

适用场景：英语写作后的自我修改

专业

📘 在线词典 — Cambridge Dictionary / Collins

用途：查单词不只是看中文意思，还要看英文释义、例句和搭配。

  1. 输入单词
  2. 看英文释义（English definition）
  3. 看 2—3 个例句
  4. 看常见搭配（collocations）

示例：查 "friendship" 可以看到 → strike up a friendship / friendship group / a lifelong friendship

适用场景：学习新词汇、确认用法

🔴 高阶工具（研究级）

研究级

📊 词频分析器 — VocabProfile

用途：分析你的英语写作中使用了什么水平的词汇。

  1. 把你的英语作文粘贴到网站
  2. 它会给出 K1（最基础）、K2、学术词汇占比
  3. 如果大部分是 K1 词汇，尝试升级词汇

升级示例：good → trustworthy ｜ nice → considerate

适用场景：想提升写作词汇丰富度时

研究级

🔍 语料库 — COCA

用途：查一个词在真实英语中是如何使用的。

  1. 搜索 "friendship"
  2. 可以看到数百条来自杂志、新闻、学术文章的真实例句
  3. 模仿这些例句使用

适用场景：想确认某个表达是否地道时
