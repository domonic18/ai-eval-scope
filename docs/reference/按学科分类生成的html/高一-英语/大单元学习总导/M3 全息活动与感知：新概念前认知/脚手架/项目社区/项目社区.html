1-3天

我的"友谊宣言"英语海报

设计并制作一张英语海报，主题为 "My Friendship Declaration"——用英语表达你对友谊的理解和承诺。

📅 阶段任务

  • Day 1（设计构思）：写下你想表达的3个核心观点（英语）→ 画草图布局
  • Day 2（制作）：用A4/A3纸张或Canva等工具制作海报 → 配图和排版
  • Day 3（分享优化）：拍照分享到班级群 → 收集2位同学的反馈 → 优化

✅ 评价标准

  • 包含至少3个英语核心陈述句
  • 版面设计有层次感（标题-正文-插图）
  • 英语表达语法正确、句式丰富
  • 展现个人风格和真实观点

1-2周

班级"友谊故事"英语采访特辑

采访3-5位同学/老师/家人，用英语整理一篇 "Friendship Stories" 特辑。

📅 阶段任务

  • 第1-2天（设计采访提纲）：设计5个英语采访问题
  • 第3-5天（执行采访）：采访3-5人，录音或记笔记
  • 第6-8天（整理撰写）：将采访内容整理成英语短文（每人100词）
  • 第9-10天（编辑排版）：将所有采访汇编成一份特辑文档
  • 第11-14天（分享展示）：在班级墙/学习群展示，收集投票评选

✅ 评价标准

  • 完成了至少3人的采访
  • 采访记录用英语撰写且语法基本正确
  • 有个人反思（"从这些故事中学到了什么？"）
  • 版面整洁美观

1月+

"Better Together" 班级友谊行动计划

设计并在班级中实施一项增进友谊的行动计划，用英语记录全过程。

📅 阶段任务

  • 第1周（调研诊断）：设计一份班级友谊调查问卷（英语），了解同学们的交友需求和困难
  • 第2周（方案设计）：基于调查结果设计行动方案，用英语写方案提案
  • 第3-4周（实施与记录）：执行方案，每周用英语写一份"进展日记"
  • 第5周（评估与总结）：收集反馈，用英语写一份项目总结报告

✅ 评价标准

  • 调查问卷回收率超过80%
  • 方案有清晰的目标和执行步骤
  • 进展日记不少于3篇（每篇100词以上）
  • 总结报告包含数据分析和反思
  • 对班级友谊氛围有实际改善
