建构性示例

友谊的直觉地图：从零开始绘制我的"Friendship Concept Map"

建构目标：建构一份个人化的 Friendship Concept Map + 一封 150 词英文回信
建构情境：你收到一位新同学的英语邮件。面对这封"半懂不懂"的信，你的任务不是查字典逐句翻译，而是通过自主建构，形成自己对"Friendship"概念的理解框架，然后用英语写一封回信。

0 任务概览

成果名称：Friendship Concept Map + 一封英文回信

用途：为后续学习和交流提供个人化的知识框架

最低标准：至少包含 6 个核心要素，5 个英语表达句，150 词的英文信

建构过程共 7 步：
  1. 感知原始材料
  2. 激活已有经验
  3. 观察情境证据
  4. 建构概念框架
  5. 语言建构
  6. 生成成果
  7. 反思与优化

1 感知原始材料

做什么：通读新同学的邮件，标记"看得懂的部分"和"看不懂的部分"
怎么做：用绿色笔划出已经认识的单词/短语，用红色笔标出陌生的表达
生成什么：一份"已知—未知"对比记录表

提示：先不用查字典——把你已经能看懂的部分写下来，看看自己能理解多少。

2 激活已有经验

做什么：回忆你已有的友谊经历，写下你觉得"朋友"最重要的 3 个特征
怎么做：用中文或简单英语写 3 个关键词（如 trust, fun, help）
生成什么：3 个关键词及简单说明

思考：你心中最好的朋友是什么样的？你们为什么能成为朋友？

3 观察情境证据

做什么：从邮件中找出作者对友谊的描述
怎么做：摘录 3—5 句关键句如 "A true friend is someone who...", "I believe friendship means..."，尝试猜测句子的含义
生成什么：关键句摘录 + 你猜测的含义
提示：不用完全理解每句话——找出你最有感觉的句子，猜猜作者想表达什么。

4 建构概念框架

做什么：将"已知经验"与"邮件证据"结合，画出 Friendship Concept Map
怎么做：中心写"Friendship"，向外辐射 4 个分支：Core Elements、Types、Feelings、Actions
生成什么：一份辐射状概念图

🧭 概念图模板

Friendship → 4 分支：Core Elements / Types / Feelings / Actions → 每个分支再展开具体内容

5 语言建构

做什么：从邮件中总结出描述友谊的英语句型和词汇
怎么做：建立自己的"Friend Talk"词汇表，分类记录
生成什么：个人词汇句型表（至少 10 个词汇，5 个句型）

📝 关键句式支架

  • "I think friendship means..."
  • "A good friend should be..."
  • "For me, the most important thing in a friendship is..."
  • "I really like it when my friend..."
  • "In my opinion, friendship is like..."

6 生成成果

做什么：利用概念图和词汇表，给新同学写一封英文回信
怎么做：使用邮件中出现的句型，表达自己的友谊观
生成什么：一封 150 词以上的英文回信（问候-主体-结尾）

建议：先画概念图，再写信；写信时反复参考概念图中的词汇。

7 反思与优化

做什么：检查回信是否充分表达了自己的友谊观，概念图是否完整
怎么做：对照自评检查清单逐项自评，优化终版
生成什么：优化后的终版概念图和回信

✅ 自我评价量表

概念图至少包含 4 个分支 从邮件中提取了至少 5 个句型并应用 回信超过 150 词且有完整结构 把已有经验和新材料结合起来了 对成果进行了反思和修改
