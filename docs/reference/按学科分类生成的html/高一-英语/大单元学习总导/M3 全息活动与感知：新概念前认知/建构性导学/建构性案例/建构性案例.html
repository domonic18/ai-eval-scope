案例一：我的友谊关键词

从自己已有的友谊经历中提取 3 个关键词，用英语写一段 50 词以上的短文解释每个词。

感知—联结原理
▼

建构步骤

  1. 回忆 1 个经历/关键词
  2. 选词
  3. 查证英语表达
  4. 写解释段
  5. 通读修改

成果

自评清单
