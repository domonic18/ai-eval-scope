同伴连接：讨论与协作

📌 讨论议题 2 如果你要设计一张"友谊海报"，你会从什么地方获得灵感？

家校连接：支持与反馈

🏡 家庭任务

向家人展示你的友谊海报，请他们用英语写一句评价（哪怕只是简单的 "It's beautiful!"）

📋 反馈收集 收集 3 位家人/朋友的反馈："看到这张海报，你对友谊的第一感觉是什么？"
🌍

社会连接：真实连接

🖼️ 线上展示 将你的 Friendship Visual Poster 拍照上传到班级英语角/学习平台，接受同学的点赞和评论
🔄 校际交流 如果有条件，与其他班级或学校的学生交换友谊海报，写下 "How does this poster make you feel?"
📤 输出机制

🖼️ 友谊视觉作品展（线上 / 线下）

📝 分享模板

"My poster is about ______.
I chose these colors because ______.
The most important word on my poster is ______."

⭐ 评价方式

同学互评 至少给 3 位同学的海报写英语评论，表达你的真实感受
自我反思 回顾你的设计过程：哪些地方做得好？如果再做一次你会改变什么？
