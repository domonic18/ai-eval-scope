🎯 任务 A（必选）

"My Friendship Keywords" 词汇卡片

A6 卡片尺寸每张含词汇+音标+图标+例句

"Friendship Visual Map" — A4 尺寸，用色彩和图标呈现你对友谊的理解结构

✍️ 任务 B2（文字表达型）

"My View on Friendship" — 150 词以上英语短文，配一张手绘插图

🎤 任务 B3（口语表达型）

"Friendship in My Eyes" — 3 分钟英语演讲视频，配合视觉辅助材料展示

🎭 任务 C（必选）

最低要求：至少 5 个英语句子 + 视觉元素覆盖
