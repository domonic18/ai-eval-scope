🧰 工具库脚手架

"认知放大器"——用合适的工具让你的美术思维学习更高效、更有创意

🎯 搭建意图

工具库脚手架为你提供"认知放大器"——用合适的工具让你的美术思维学习更高效、更有创意。从手绘到专业设计软件，找到适合你的那一款。

🟢 基础工具（易上手）
🧠

思维导图工具 — XMind / 手绘

基础 辐射式或层级式结构组织知识

用途说明：用辐射式或层级式结构组织和表达友谊相关知识

使用步骤：

  • 中心写核心词（如"Friendship"）
  • 向外发散 4—6 个分支（如 Trust、Humor、Support、Memory）
  • 每个分支加关键词和图标

📌 整理友谊词汇📌 规划海报内容结构

🎨

色彩搭配工具 — Color Hunt / Canva Color Wheel

基础 快速获取专业配色方案

用途说明：快速获取专业的配色方案，让海报更美观

使用步骤：

  • 选择主色（如代表友谊的暖橙色）
  • 系统自动生成配套色
  • 应用到海报设计中

📌 海报设计配色📌 视觉笔记配色

🟡 进阶工具（专业级）
🖼️

海报/视觉设计工具 — Canva / Adobe Express

进阶 在线拖拽式设计，含大量英语海报模板

用途说明：在线拖拽式设计工具，含大量英语海报模板

使用步骤：

  • 选择 "Poster" 模板
  • 替换文字为你的英语内容
  • 调整配色和布局
  • 导出作品
💡 示例：搜索 "Friendship poster" 模板，将中文内容替换为英语

📌 数字版友谊海报📌 线上展示作品

📊

信息图表工具 — Piktochart / Venngage

进阶 将数据信息转化为视觉化信息图

用途说明：将数据和信息转化为视觉化信息图

使用步骤：

  • 选择模板
  • 填充你的友谊主题内容
  • 导出分享

📌 友谊文化对比信息图📌 友谊数据可视化

🔴 高阶工具（研究级）
💎

专业设计工具 — Figma / Sketch（免费教育版）

高阶 专业UI/UX设计，精致数字作品

用途说明：专业 UI/UX 设计工具，适合制作精致数字作品

使用步骤：

  • 创建画板（Frame）
  • 使用图形工具和文字工具排版
  • 导出作品

📌 高质量数字化视觉作品
