💡 搭建意图：问题库是整个学习系统的"引擎"。通过理解、分析、创造三层次的问题设计，引导你对"友谊"主题从认知到批判，完成完整的学习闭环。

🌱 基础层 · 理解与识别

打开友谊概念的"第一扇门"

这些问题帮你建立对友谊主题的基础认知。

💬 开放性问题

你认为"友谊"的核心要素有哪些？请用 3 个英文关键词 回答。

试试从 trust（信任）、support（支持）、honesty（诚实）等角度思考。
⚖️ 对比性问题

英语中的 "friendship" 和中文的"友情"是同一个概念吗？有没有细微差别？

英语的 friendship 强调"关系状态"，中文的"友情"更强调"情感连接"——文化背景不同的两个概念。
🌀 反事实问题

如果没有"朋友"这个概念，人类社会会是什么样子？

想想人类历史上有哪些进步/发明是建立在友谊合作基础上的？
🧠 元认知问题

在回答这个问题时，你最先想到的是哪个朋友？为什么？

元认知就是"思考你的思考过程"——关注你为什么先想到某个人。
⬇ 进阶挑战 ⬇
🚀 进阶层 · 分析与应用

用美术思维分析友谊表达

这些问题帮你把友谊概念和视觉表达联系起来。

💬 开放性问题

一张好的友谊主题海报应该包含哪些元素？请分析 3 个关键要素。

从色彩、构图、文字层级、图标选择、留白等角度分析。
⚖️ 对比性问题

用文字写一段友谊描述 vs 用海报视觉化地呈现友谊——两种方式各有什么优势和局限？

文字擅长细节和情感深度，视觉擅长即时感染力和记忆点——想一想哪个更适合你的表达目标。
🌀 反事实问题

如果让你用"纯文字无图像"和"纯图像无文字"两种方式表达同一种友谊情感，哪一种更难？为什么？

纯文字需要精准用词，纯图像需要寻找普适的视觉符号——不同维度的挑战。
🧠 元认知问题

在设计海报时，你是先想内容再想视觉，还是先想视觉再填充内容的？

反思自己的创作流程有助于找到更适合你的设计方法。
⬇ 终极挑战 ⬇
🔥 挑战层 · 创造与批判

重新思考友谊与视觉表达的关系

这些问题挑战你的批判性思维和创造能力。

💬 开放性问题

如果 AI 能完美生成友谊主题的视觉作品，那人类创作这些作品的意义是什么？

AI 擅长"完美"但不擅长"真实"——人类的缺陷、情感记忆和个体视角是 AI 无法复制的。
⚖️ 对比性问题

你手绘的友谊海报和用 Canva/APP 生成的数字海报，从"自我表达"的角度看有什么本质不同？

手绘的每一笔都是个人选择，数字设计则是在选项中进行选择——两种"创作"的深度不同。
🌀 反事实问题

如果友谊可以用视觉"设计"出来（颜色、构图、布局），那人际关系是否可以像设计海报一样被"设计"？

这是一个哲学层面的问题——视觉设计有规律可循，人际关系呢？
🧠 元认知问题

对比你在本模块的学习过程和之前英语模块的学习过程，用"美术思维"学习让你感受到了哪些不同的体验？

可以关注：学习动力、理解深度、记忆持久度、趣味性等维度。
