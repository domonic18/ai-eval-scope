🗺️ 大单元学习路径选择

根据你的英语基础和学习风格，选择最适合你的学习路径

🌱 入门级

基础进入路径

适合对象：英语基础较弱（词汇量不足、语法概念模糊、听说有困难）的学生

学习重点：核心词汇与句型积累 → 语篇基本理解 → 简单口头和书面表达

模块2 · 价值认知 模块3 · 感知活动 模块4 · 概念解析 模块12 · 书面应用
每多记一个单词，就多一份表达的底气。不着急，慢慢来，每一步都算数。

成果要求：能朗读课文，理解主要内容，完成基础词汇和句型练习

支持工具：AI语音跟读工具、智能词汇本、句型模板库

📈 进阶级

标准发展路径

适合对象：英语基础中等，能完成基本理解和表达的学生

学习重点：深度理解语篇 → 语言运用 → 思维拓展 → 文化比较

模块2→3→4 模块5·艺术思维 模块6·技术思维 模块8·科学思维 模块9·文化思维 模块12→13 模块16·反思 模块17·成就
语言是用出来的，不是学出来的。多说、多写、多交流，每一次输出都是进步。

合作方式：与同伴组成2-4人学习小组，定期交流学习心得和作品

成果要求：能进行语篇分析，完成综合性语言任务，写出结构清晰的短文

🚀 挑战级

高阶挑战路径

适合对象：英语基础扎实，愿意深入研究和创新表达的学生

学习重点：跨学科融合应用 → 深度文化分析 → 创造性表达 → 学术写作

模块2→3→4→5→6 模块7·数学 模块8→9→10 模块11·美术 模块12→13→14·STEAM 模块15·三创 模块16→17→18→19→20
挑战是成长的阶梯。当我用英语思考和创造时，我不仅在学习语言，更在拓展我的认知边界。

挑战任务：用英语完成一项与本单元主题相关的小型研究，形成学术海报或短论文

成果要求：完成跨学科项目、写出英文研究报告、进行英文演讲展示

🧑‍🎓 个体学习建议

📖

概念薄弱型

从模块3-4入手，利用建构性导学框架梳理核心概念，配合AI学伴进行概念诊断。

✍️

表达困难型

从模块12入手，先用句型模板模仿写作，再逐渐过渡到自由表达。

🔧

实践兴趣型

直接进入模块13-15，在项目实践中边做边学。

🔬

研究挑战型

从模块5-11中选择2-3个感兴趣的深入探索，结合模块20进行学术输出。

🤝

合作成长型

组队学习，在小组讨论和互评中提升语言能力。

🤖

数智工具探索型

充分利用AI学伴、智能批改、语音识别等工具，探索个性化学习方式。
