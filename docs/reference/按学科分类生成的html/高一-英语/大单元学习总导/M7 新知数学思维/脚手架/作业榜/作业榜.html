🏆 作业榜

M7 新知数学思维 · 友谊关系分析 · 三层任务体系
▶ 基础层：理解与再现 必做
  • 用英语叙述一个朋友的友谊故事，运用本单元的关键词汇和句型（约 80 词）
  • 从给定的友谊关系文本中提取关键要素：信任、支持、冲突、共同兴趣等
  • 完成友谊关系要素分类表——将提取的要素归入不同友谊类型
▶ A 级：基础挑战 A
  • 制作一份友谊关系的词语思维导图（含 10 个以上关键术语）
  • 导图需包含友谊类型、品质、动词短语三个分支
  • 每分支至少 3 个相关词汇/短语
▶ B 级：进阶分析 B
  • 用英语写一篇 100 词短文，分析你和你最好的朋友之间的友谊模式
  • 运用亚里士多德的"三种友谊"分类来分析这段关系
  • 在文末列出 2–3 个体现你分析中思维方法的关键词
▶ C 级：挑战创造 C
  • 用概念图或流程图展示一段友谊关系的演变过程
  • 用英语口头介绍你的图示（录制视频或课堂展示均可）
  • 分析转折点：什么事件改变了这段友谊的走向？
