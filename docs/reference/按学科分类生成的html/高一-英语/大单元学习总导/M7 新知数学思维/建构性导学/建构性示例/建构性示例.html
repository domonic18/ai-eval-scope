建构性示例

从零开始建构友谊关系的完整路径示范 · 英语必修1

📖

示例故事："从同桌到朋友，再到更懂朋友"

高一开学第一天，李明和张华成了同桌。李明性格内向，不太主动说话；张华开朗健谈，总是先开口。第一个月，他们只是点头之交。第二个月，一次数学考试后李明忘带计算器，张华主动借给他——这是关系的转折点。之后他们开始一起吃饭、一起放学。但第三个月，李明发现张华总是打断他说话，这让他感到不被尊重。矛盾产生了……

这个故事看起来简单，但其中包含了友谊从建立到发展的完整要素。接下来，我们要像一个研究者一样，从这个故事中"建构"出一个友谊关系发展模型。

建构目标

你将亲手建构一个友谊关系发展模型——它能帮助理解友谊是如何从陌生走向亲密、又如何在矛盾中调整的。

🎯 建构任务说明

成果名称：友谊关系发展模型图

用途：用于解释朋友之间关系变化的规律，预测可能出现的矛盾

对象：写给同龄人看的，也可以向老师展示你的分析能力

形式：概念图/流程图 + 英语关键词标注

最低标准：至少包含3个发展阶段 + 每个阶段2个特征 + 1处转折点

建构过程示范（6步）

点击每个步骤查看其中的"简短示范"——展示每一步实际产出的样子。

1 观察与提问
做什么：观察情境中的关键现象——两个同学的关系发生了什么变化？
怎么做：逐段阅读故事，记录每次关系变化的时间点和事件。
生成什么：观察记录表。
📋 观察记录表示范：

第1周：初次见面，互动极少（打招呼仅2次）
第5周：借计算器事件，关系开始升温
第8周：一起吃饭、放学，互动频率上升至每天
第12周：打断说话事件，舒适度下降

2 收集与整理
做什么：从观察记录中提取友谊关系的关键要素。
怎么做：分类整理——哪些是"促进因素"？哪些是"阻碍因素"？
生成什么：信息分类表。
📊 信息分类表示范：

促进因素：主动帮助（借计算器）、共同活动（一起吃饭）、日常互动（同桌）

阻碍因素：性格差异（内向vs外向）、沟通方式（被打断）、期待不一致

环境因素：同桌座位安排、同一班级

3 建立联系
做什么：寻找要素之间的因果关系——哪个因素导致了关系变化？
怎么做：画关系图，用箭头表示"导致"关系。
生成什么：初步的关系草图。
🧩 关系草图示范：

"主动帮助" → "信任建立" → "互动增加" → "关系升温"

"性格差异" → "沟通方式不匹配" → "被打断" → "舒适度下降" → "关系紧张"

核心发现：信任是友谊升温的燃料，而沟通方式是关系的调节器。

4 形成模型
做什么：将关系图转化为结构化的友谊发展阶段模型。
怎么做：归纳出关系发展的规律——从陌生到熟悉经历了几个阶段？
生成什么：友谊关系发展模型。
🏗️ 模型示范：友谊四阶段模型

阶段1 — 接触期（Contact）：日常互动但缺乏情感投入，如点头之交

阶段2 — 破冰期（Break the Ice）：一个关键事件打破社交距离，如一次帮助

阶段3 — 亲密期（Intimacy）：互动频率和情感深度同时增加，形成信任

阶段4 — 调整期（Adjustment）：出现矛盾后需要重新调整关系模式

5 验证与调整
做什么：用故事中的证据检验你的模型是否站得住脚。
怎么做：逐段对比模型和故事事实，看是否有矛盾的地方。
生成什么：修订后的最终模型。
✅ 验证记录示范：

检验1：模型说"破冰期需要关键事件" → 故事中借计算器符合 ✔

检验2：模型说"亲密期后会有调整期" → 故事中打断事件导致紧张 ✔

修正：模型增加"性格差异"作为影响所有阶段的变量 ✔

6 表达与优化
做什么：将最终模型用规范的形式表达出来，并优化呈现效果。
怎么做：画出正式版概念图，配上英语关键词和简短说明。
生成什么：最终成果——友谊关系发展模型图。
🎨 最终成果示范：

标题：Friendship Development Cycle

Contact → (Key Event) → Break the Ice → (Trust) → Intimacy → (Conflict) → Adjustment → (Resolution) → Deeper Intimacy

每个阶段配上英语关键词：Greeting / Help / Trust / Misunderstanding / Adjustment

成果评价要求

📌 建构成果样例

成果名称：友谊关系发展模型图

评价要求：

  • 科学性 — 模型的阶段划分是否合理，是否有事实依据？
  • 完整性 — 是否覆盖了从建立到调整的全过程？
  • 逻辑性 — 阶段之间的过渡是否有合理的因果解释？

推荐表达格式：流程图 + 英语文字标注

完成建议：先用铅笔打草稿，用便利贴调整顺序，确认逻辑后再画正式版本。

成果优化建议

从"初稿"到"完成稿"的优化路径：

  • ① 初稿重点在完整性——先确保主要阶段都覆盖到
  • ② 二稿重点在逻辑性——检查每个阶段的过渡是否自然
  • ③ 三稿重点在表达——用英语关键词替换中文标注
  • ④ 终稿重点在美观——调整布局、配色和字号
