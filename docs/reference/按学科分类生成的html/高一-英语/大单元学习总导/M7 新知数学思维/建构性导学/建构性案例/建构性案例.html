案例1：我的友谊观自画像

真实情境 你刚刚进入高中，面对全新的同学群体。回顾过去交朋友的成功与失败经历，你需要形成一份属于自己的友谊观梳理。
建构任务 完成一份"我的友谊观自画像"——用概念图+英语关键词表达你对友谊核心要素的理解。
原理调用 情境锚定原理 · 经验联结原理
建构步骤：
  • 回忆你人生中3段印象深刻的友谊（好的或不好的）
  • 从每段经历中提炼出1个关键词（如信任、陪伴、冲突）
  • 思考这些关键词之间的关系，用连线画出概念图
  • 在概念图下方用3-5句英语写出你的友谊观
脚手架推荐 概念图模板（中心词"Friendship" + 分支关键词）
成果表达 A4纸手绘/电子概念图 + 英语短文
自评量规：
✓ 至少包含5个关键词
✓ 关键词之间有合理的逻辑连线
✓ 英语句子语法基本正确
✓ 观点包含个人真实经历
✓ 整体布局清晰美观
🏠 生活应用型

案例2：友谊能量加油站

真实情境 你发现最近好朋友情绪低落，不愿意和任何人交流。你想主动关心TA，但不知道如何开口才不会让对方有压力。
建构任务 设计一套"友谊能量加油站"方案，包含情绪识别、沟通方式和后续跟进三个环节。
原理调用 问题驱动原理 · 证据迭代原理
建构步骤：
  • 列出你观察到的3个TA情绪低落的信号
  • 设计2种不同的开口方式（直接询问 vs 间接关心）
  • 为每种方式写一段英语对话示例
  • 设计后续跟进计划（如：每周一次一起散步）
  • 请另一位同学给出反馈并修改方案
脚手架推荐 "I notice... I wonder... Would you like..." 句式模板
成果表达 一份方案清单（中英双语）+ 两段对话示例
自评量规：
✓ 情绪信号具体可观察
✓ 对话符合真实人际交往情境
✓ 方案考虑了对方感受
✓ 包含后续跟进计划
✓ 至少收到1份同学反馈
🏫 校园问题解决型

案例3：三人友谊的平衡难题

真实情境 你和两个好朋友组成了一个"三人组"，但你越来越觉得"你们两个人在聊天时，我像是局外人"。你开始怀疑自己是不是多余的。
建构任务 分析三人友谊中的失衡原因，提出改善方案，并制作一份"三人友谊相处指南"。
原理调用 情境锚定原理 · 表达外化原理
建构步骤：
  • 记录一周内三人相处的5次互动，标注你的感受（用1-5分的舒适度）
  • 找出让你感到被排除的具体行为模式
  • 查阅资料了解"三人友谊动力学"
  • 设计3条具体的改善策略
  • 将策略整理成一份"三人友谊指南"
脚手架推荐 感受记录表 + "三人友谊角色分析"模板
成果表达 一份图示化指南，包含问题分析图 + 策略清单
自评量规：
✓ 包含具体的互动记录数据
✓ 策略有可操作性
✓ 指南图文并茂
✓ 包含至少1条心理学依据
📖 学科表达型

案例4：A Letter to My Best Friend

真实情境 英语课上，老师要求写一封英语信给一位对你影响深远的朋友。你需要用英语准确表达你的情感和感激。
建构任务 写一封结构完整的英语信件（约200词），包含开场白、回忆片段、感谢表达和祝福语。
原理调用 经验联结原理 · 表达外化原理
建构步骤：
  • 用中文列出你想对朋友说的3个核心内容
  • 为每个内容匹配英语关键词和句型
  • 写出信件初稿（不必完美）
  • 用"信件检查清单"逐项修改
  • 请一位同学帮忙校对
  • 誊写终稿并配上插图
脚手架推荐 信件模板 + 情感词汇表（grateful / inspired / unforgettable 等）
成果表达 一封正式的英语信件（手写或打印）
自评量规：
✓ 信件格式完整（日期、称呼、正文、落款）
✓ 包含至少2个具体回忆细节
✓ 使用3个以上情感词汇
✓ 语法错误不超过3处
✓ 能让人感受到真挚情感
🔬 跨学科整合型

案例5：友谊的心理学与社会学分析

真实情境 你发现朋友圈中有些人朋友很多但都不深，有些人朋友很少但关系很深。你好奇：友谊的"数量"和"质量"之间存在什么关系？
建构任务 融合心理学与社会学视角，制作一份"友谊质量-数量关系分析报告"。
原理调用 证据迭代原理 · 情境锚定原理
建构步骤：
  • 在班级做一个小型调查（至少10人），收集每位同学的"好友数量"和"最信任朋友数"
  • 将数据整理成表格，计算平均值和中位数
  • 查阅邓巴数理论（150人社交上限）
  • 用英语写一段分析：为什么数量和质量难以兼得？
  • 提出你的结论并用数据支持
脚手架推荐 调查问卷模板 + 数据统计表 + 分析框架："数据→发现→解释→结论"
成果表达 A3尺寸分析海报，含数据可视化图表 + 英语分析文字
自评量规：
✓ 调查数据来自至少10人
✓ 包含数据图表
✓ 引用至少1个理论支持
✓ 英语分析逻辑清晰
🌐 社会/未来情境型

案例6：社交媒体时代的友谊图谱

真实情境 你有500个微信好友，但深夜想找人聊天时却不知道该@谁。社交媒体让"朋友"变多了，"友谊"却变浅了。
建构任务 分析社交媒体对友谊质量的影响，提出一份"数字化友谊健康指南"。
原理调用 问题驱动原理 · 全部原理综合
建构步骤：
  • 统计你的社交App好友，按"经常聊天""偶尔联系""从未私聊"分类
  • 采访2-3位同学，问他们"上一次和好朋友深度聊天是什么时候"
  • 列出社交媒体对友谊的3个好处和3个坏处
  • 参考相关研究报告（可搜索"social media friendship quality"）
  • 设计5条"数字友谊健康守则"
  • 制作成一份双语指南
脚手架推荐 好友分类统计表 + 五条守则模板
成果表达 一份折页指南（正反面），含统计数据、分析图表和健康守则
自评量规：
✓ 包含真实的社交数据统计
✓ 包含同伴访谈记录
✓ 守则可执行、可衡量
✓ 中英文版本匹配
🚀 综合创新型

案例7：FriendLink — 未来友谊连接器

真实情境 如果让你设计一个全新的社交产品来"提升友谊质量而非数量"，它会是什么样子？它不是为了获取更多"好友"，而是帮助人们维护深度友谊。
建构任务 设计"FriendLink"概念产品——包含核心功能、用户界面草图和运营理念的产品方案书。
原理调用 综合运用全部5条建构性原理
建构步骤：
  • 基于前6个案例的发现，列出你的产品设计原则（至少3条）
  • 画出产品核心功能界面草图（手绘或数字）
  • 用英语写一段"产品宣言"（Product Manifesto）解释核心理念
  • 设计1个具体的"友谊维护"功能并描述用法
  • 制作产品方案书（A3纸）
  • 向3位同学展示并收集反馈
脚手架推荐 产品方案书模板 + "如果…将会…"迁移思考框架
成果表达 产品方案书（含界面草图 + 功能说明 + 英语宣言）
自评量规：
✓ 产品理念与友谊质量相关
✓ 包含至少3个核心功能
✓ 有界面示意图
✓ 产品宣言英语表达清晰
✓ 收到至少3份反馈并标注改进
