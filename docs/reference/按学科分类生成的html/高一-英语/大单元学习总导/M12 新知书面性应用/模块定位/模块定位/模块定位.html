📍 M12 模块学习总体概要

新知书面性应用 · 高一英语 · Unit 1 Friendship

📌 本模块学习什么？

本模块是 Unit 1 Friendship 的书面表达实践环节。简单说——把你从前面各模块中学到的关于友谊的词汇、句式和阅读感悟，转化为你自己写的英语文字。

你的核心任务是：学会用英语写出一篇关于友谊的有真情实感的短文。

📖 阅读理解 → 📝 词汇句型 → ✍️ 书面表达

💡 本模块为什么重要？

从输入到输出——英语学习的关键一步

如果你只能"读懂"英语但不能"写出"英语，你的英语学习就只完成了一半。本模块就是要帮你跨出这关键的一步——从输入到输出。

情感表达的真实力量

能用英语表达自己的真实情感，比会背多少单词都更有意义。在未来的考试中，书面表达也永远是分值最高的题型之一。

🔗 本模块与大单元主题的关系

Unit 1 Friendship 这个大单元带你从不同角度理解友谊——通过阅读了解友谊的价值，通过词汇学习掌握表达工具。而本模块（M12）是整个单元的"产出时刻"——

前面积累的一切，最终都要在你的一篇文字中体现出来
