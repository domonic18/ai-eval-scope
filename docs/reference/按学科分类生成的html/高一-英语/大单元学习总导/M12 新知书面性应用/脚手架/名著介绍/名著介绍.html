📖 名著 · 结构化导读

📜 《论语》友谊篇节选

中国经典 · 先秦儒家

❓ 中国传统智慧如何看待交友之道？

核心观点：孔子提倡"友直，友谅，友多闻，益矣"——正直的朋友、诚信的朋友、博学的朋友，是有益的。

关键片段：子曰："道不同不相为谋"——志向不同的人无法共同谋划。这与 C.S. Lewis"一起朝同一个方向看"的观点不谋而合。

📌 基础级 · 全体学生
📎 获取：纸质书（中华书局版）/ 国学网站免费阅读

💞 The Four Loves · 《四种爱》

C.S. Lewis · "友爱"章节

❓ 友爱和其他类型的爱有何本质区别？

核心观点：友爱是"共同看向同一个方向"的伴侣关系——没有嫉妒、没有占有，只有共同的追求。

关键片段：Lewis 描述朋友相遇的经典场景——"What! You too?"，这个瞬间的共鸣是友谊诞生的标志。

📌 进阶级 · 英语基础较好的学生
📎 获取：纸质书（HarperOne）/ Kindle / Audible（推荐中英双语版）

🧮 How Many Friends Does One Person Need?

Robin Dunbar · 进化心理学

❓ 人类社交关系的数量和质量受什么限制？

核心观点：人类大脑皮层限制我们只能维持约150个社交关系，其中真正亲密的只有5人。

关键片段：Dunbar 解释"为什么异地友谊难以维持"——缺乏日常共情体验的"摩擦"。

📌 挑战级 · 对心理学感兴趣的学生
📎 获取：纸质书 / 电子书（各大图书平台）
