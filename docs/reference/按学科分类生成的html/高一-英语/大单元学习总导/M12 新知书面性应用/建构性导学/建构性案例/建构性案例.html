✍️ 建构性七阶适配案例

围绕「友谊」主题，7个循序渐进的写作建构案例。每个案例都提供真实情境、建构任务、原理调用提示和自评量规，帮助你一步步掌握英语写作建构能力。

案例一 个人理解型

🌟 My Ideal Friend

真实建构情境：你开始思考——"什么样的朋友才是真正的好朋友？"每个人的标准不同。请你用英语画出你自己的"理想朋友画像"。

建构任务：写一篇"My Ideal Friend"短文（60—80词），描述你心目中理想朋友的品质并解释为什么重要。

主题先行原理
事例支撑原理
建构步骤：
  1. 列出自己认为重要的3个友谊品质
  2. 选择最重要的2个品质
  3. 为每个品质想一个生活事例
  4. 按"总—分—总"结构写出短文
  5. 检查语法
🧱 脚手架推荐：形容词词库（kind, honest, funny, loyal, understanding...）、连接词列表（first of all, also, moreover, because）

成果表达：短文，60—80词，含标题和2个具体品质描述

📋 点击查看自评量规

  • ☐ 我的短文有明确主题句吗？
  • ☐ 我描述了至少2个具体品质吗？
  • ☐ 我用了至少3个英语形容词吗？
  • ☐ 我检查了拼写和句子结构吗？
  • ☐ 短文读起来像是在表达我的真实想法吗？
案例二 生活应用型

🏫 A New Friend at School

真实建构情境：班里来了一位交换生Tom，不太会说中文，课余总是一个人。你想主动和他交朋友，请策划一个"破冰计划"。

建构任务：用英语写一份"破冰活动建议书"（80—100词），包括2—3个活动及理由。

结构先行原理
读者意识原理
建构步骤：
  1. 分析Tom的可能需要（语言困难？文化差异？孤独感？）
  2. 头脑风暴适合两人的活动
  3. 选择2—3个可行性高的活动
  4. 按建议书结构写出
  5. 添加合适的语气词让文字更友好
🧱 脚手架推荐：建议书句式框架（I suggest that we... / How about doing... / This activity can help us...）

成果表达：建议书短文，80—100词，至少包含2个具体活动建议

📋 点击查看自评量规

  • ☐ 我的建议针对了Tom的实际情况吗？
  • ☐ 我提出了至少2个具体活动吗？
  • ☐ 我解释了为什么这个活动可以帮助交朋友吗？
  • ☐ 语气是否友好、有鼓励性？
  • ☐ 语法和用词是否基本正确？
案例三 校园问题解决型

🤝 Friendship Problem Helper

真实建构情境：你的好朋友小明最近和另一个同学产生了误会——小明觉得对方不尊重他的隐私。请你用英语帮小明写一封信给那个同学。

建构任务：写一封80—100词的英语信件，代表小明表达感受和希望解决问题。

主题先行原理
迭代修改原理
建构步骤：
  1. 分析问题本质（隐私被侵犯→感到不舒服→希望对方理解）
  2. 确定信的语气（礼貌但坚定）
  3. 起草：问候→表达感受→说明原因→提出建议→期待回应
  4. 检查语气是否合适
  5. 修改定稿
🧱 脚手架推荐：礼貌表达句式（I feel... when you... / I understand you may not realize... / Could we please... / I hope we can...）

成果表达：英语信件，80—100词

📋 点击查看自评量规

  • ☐ 我清楚地表达了"为什么不舒服"吗？
  • ☐ 语气是否既礼貌又真诚？
  • ☐ 我是否提出了解决问题的建议？
  • ☐ 书信格式是否完整？
  • ☐ 是否避免使用指责性语言？
案例四 学科表达型

📝 A Friendship Poem

真实建构情境：有时候，诗歌比散文更能表达内心的情感。请你试着用英语写一首关于友谊的短诗。

建构任务：创作一首英语短诗（6—10行），主题围绕"友谊"。

主题先行原理
迭代修改原理
建构步骤：
  1. 确定情感基调（温暖/感激/珍惜/希望）
  2. 写出5—10个与友谊相关的关键词
  3. 选择意象（如阳光、星星、桥、路等）
  4. 尝试写2—4行，注意节奏
  5. 朗读修改 → 完成
🧱 脚手架推荐：英语诗歌范例（如"Friends" by Abbie Farwell Brown）、押韵词表

成果表达：英语短诗，6—10行，手写配图（可选）

📋 点击查看自评量规

  • ☐ 我的诗有明确的情感主题吗？
  • ☐ 我使用了比喻或意象吗？
  • ☐ 诗的节奏是否流畅？
  • ☐ 用词是否与友谊主题相关？
  • ☐ 诗能传达出我的真实情感吗？

🌏 Friendship Across Cultures

真实建构情境：国际交友网站举办"跨文化友谊故事"征文比赛。你认为"真正的友谊可以跨越文化和语言的差异"。

建构任务：写一篇100—120词的英文故事，展示友谊如何跨越文化差异。

事例支撑原理
结构先行原理
建构步骤：
  1. 选择一个跨文化友谊的素材
  2. 确定"文化差异是什么？友谊怎么跨越它？"
  3. 按叙事结构组织：背景→矛盾→解决→升华

  • ☐ 使用了具体的细节描写吗？
  • ☐ 读完后能让别人感受到"友谊的力量"吗？
案例六 社会/未来情境型

📱 Friendship in the Digital Age

真实建构情境：现在的社交媒体让你可以拥有上百个"好友"，但有人说这种"数字友谊"很浅——点赞容易、交心难。作为数字原住民的你，怎么看？

建构任务：写一篇英语短文（100—120词），表达你对"数字时代的友谊"的看法，并给出理由。

主题先行原理
事例支撑原理
读者意识原理
建构步骤：
  1. 明确你的立场（网络友谊是真友谊/假友谊/视情况而定）
  2. 收集2—3个支撑理由（亲身经历或身边事例）
  3. 写成短文：表明观点→论证（2—3层次）→重申观点
  4. 检查论证逻辑
  5. 润色语言
🧱 脚手架推荐：议论文结构模板、观点表达句式（In my opinion / I believe that / On the one hand... On the other hand... / Therefore）

成果表达：议论文短文，100—120词

📋 点击查看自评量规

  • ☐ 我有明确的观点立场吗？
  • ☐ 我提供了至少2个支撑理由吗？
  • ☐ 我用了具体事例来说明吗？
  • ☐ 考虑了反方向的观点吗？
  • ☐ 结论是否自然有力量？
案例七 综合创新型

📜 My Friendship Manifesto

真实建构情境：经过整个单元的学习，你对"友谊"已经有了丰富的理解——现在请把这些理解和感悟整合起来，创作一份属于你自己的"友谊宣言"。

建构任务：创作一份英语友谊宣言（约150词），综合你对友谊的理解、感悟和承诺。

全部5条建构性原理
建构步骤：
  1. 翻看本单元的学习笔记，回顾你对友谊的所有理解
  2. 确定宣言的主题（如"A true friend is..." / "Friendship means..."）
  3. 列出3—5条你最重要的友谊信念
  4. 为每条信念配上一句有力的宣言
  5. 写出完整宣言 → 朗读修改 → 定稿
🧱 脚手架推荐：宣言写作范例、排比句式（Friendship is... Friendship is... Friendship is...）

成果表达：友谊宣言，约150词，可配图/设计版式

📋 点击查看自评量规

  • ☐ 宣言体现了你对友谊的深入思考吗？
  • ☐ 包含3—5条具体宣言内容吗？
  • ☐ 使用了有力量的英语表达吗？
  • ☐ 语言风格统一且有力吗？
  • ☐ 这份宣言代表了你真实的友谊观吗？
