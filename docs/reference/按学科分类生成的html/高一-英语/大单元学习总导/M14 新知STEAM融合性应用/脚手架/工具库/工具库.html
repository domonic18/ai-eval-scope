⭐ 基础

📝 思维导图工具 — 友谊关系可视化

将 Unit 1 中关于友谊的词汇、句型和观点用图形化方式组织，帮助理清思路、快速记忆。

使用步骤

  1. 确定中心主题（如 "Friendship"）
  2. 向外延伸分支（如 qualities / activities / problems）
  3. 在每个分支下添加关键词或例句
以 "A good friend" 为中心，分支写出 honest、supportive、funny 等形容词，并配课文例句。
课前预习 · 课后复习 · 写作前头脑风暴
⭐ 基础

📊 简易问卷工具 — 班级友谊调查

快速创建并收集同学间的友谊相关数据，用数字说话。

使用步骤

  1. 用 Google Forms / 问卷星设计 3-5 个问题
  2. 分发链接收集数据
  3. 查看统计图表分析结果
"What do you value most in a friend?"（honesty / humor / loyalty）统计全班占比。
课堂破冰 · 口语报告数据支撑 · 小组项目调研
🚀 进阶

🤖 AI 对话工具 — 英语口语陪练

借助 AI（如 ChatGPT、Kimi）模拟真实对话场景，用英语谈论友谊话题，提升口语和听力。

使用步骤

  1. 设定角色（"You are my new friend from the UK"）
  2. 输入话题（"Let's talk about how to make friends"）
  3. 跟读或自由回应 AI 的回答
AI 提问 "What do you usually do with your best friend?"，学生用课文句型回答。
课后口语训练 · 考试话题练习 · 缺乏语伴时的替代方案
🚀 进阶

📈 数据分析工具 — 友谊词频分析

用 Python / Excel 对 Unit 1 课文进行词频统计，找出高频词汇，辅助词汇学习。

使用步骤

  1. 将课文文本复制到文本文件
  2. 用 Python 的 collections.Counter 或 Excel 数据透视表统计词频
  3. 生成词云或柱状图
统计发现 "friend" 出现 15 次、"trust" 出现 8 次——帮助理解课文重点。
STEAM 融合统计学习 · 自主词汇分析 · 小组数据展示
🔬 高阶

🌐 社会网络分析工具 — 班级友谊网络图

收集班级内部"谁和谁是朋友"的数据，用网络图分析友谊结构——谁是核心人物、是否存在小团体。

使用步骤

  1. 匿名调查"请写出你在班上最要好的三位朋友"
  2. 用 Gephi 或在线工具（如 Flourish）导入数据
  3. 生成网络图并解读
某同学被 10 人提及（高中心度），说明他是班级社交核心；发现 3 个孤立小团体，讨论如何促进融合。
STEAM 项目式学习展示 · 班级凝聚力主题班会 · 社会学初步探究
🔬 高阶

💬 情感分析工具 — 友谊主题文本情感挖掘

用 Python 自然语言处理库（如 SnowNLP / TextBlob）分析学生对友谊的情感倾向，揭示群体情绪。

使用步骤

  1. 收集同学写的关于友谊的短文本（50-100 字）
  2. 调用情感分析 API 或本地库
  3. 输出积极/消极评分并可视化
全班 80% 的短文情感得分 > 0.8（高度积极），说明班级对友谊整体持正面态度。
STEAM 跨学科项目 · 英语写作反馈分析 · 心理健康教育融合
