搭建友谊的STEAM连接桥

英语必修1 Unit 1 Friendship · 协作 · 探究 · 展示

💡 讨论议题

议题 1：Friendship in the Digital Age

引导问题：社交媒体是拉近还是疏远了朋友之间的关系？请用数据或案例支持你的观点。

  • 🔬 科学视角：调查班级同学使用社交软件交友的频率，制作简易统计图表
  • 📝 英语表达：使用本单元核心词汇（trust, share, betray, communicate 等）展开讨论

议题 2：A STEAM Solution for a Friendship Problem

情境：你的好朋友因为搬家而感到孤独，请你设计一个"友谊维持方案"。

  • ⚙️ 工程思维：画出方案流程图（发送消息 → 计划视频通话 → 共享线上活动 → 定期回顾）
  • 🛠️ 技术工具：推荐使用在线协作白板（Miro / Jamboard）展示方案

👥 角色分工（每组 4 人）

👤 组长 / 协调员

控制讨论节奏，确保每人发言，记录关键观点

领导力 时间管理

📊 研究员 / 数据分析员

收集数据（问卷调查/网络资料），制作图表

数学素养 信息检索

🎨 创意官 / 设计师

构思方案流程图或海报草图，负责可视化呈现

艺术与设计思维

🎙️ 发言人 / 汇报员

汇总小组观点，用英语进行口头展示

英语表达 公共演讲

📤 输出机制：展示与分享

🏫 课内展示（15分钟）

每组3分钟英语汇报：讨论核心观点 + STEAM方案展示

其他组根据评价量规进行同伴互评

评分维度：内容完整性、英语表达、创意性

🏠 家校连接（课后延伸）

学生将方案带回家，与家长讨论"家长年轻时如何维持友谊"

写下对比感悟（50词英语短文），家长签名确认。优秀方案上传至班级云相册。

🌍 社会连接（真实世界）

邀请学校心理辅导老师或外教录制回应视频

将优秀方案制作成展板，在校园文化节或班级外墙展示

📐 评价量规（Peer Evaluation Rubric）

维度3分 优秀2分 良好1分 需改进
内容完整性观点清晰，论据充分，STEAM方案完整观点较清晰，论据基本充分观点模糊，缺少论据
英语表达流利自然，使用本单元核心词汇较流利，少量语法错误表达不流畅，词汇单一
创意与协作方案有创意，角色分工明确方案有一定创意，分工基本明确缺乏创意，分工不清

💬 本活动旨在通过同伴协作、家校互动与社会连接，帮助你在真实语境中运用英语与STEAM思维，深化对"友谊"主题的理解。
