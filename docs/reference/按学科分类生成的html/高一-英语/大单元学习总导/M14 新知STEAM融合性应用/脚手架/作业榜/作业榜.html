展示与表彰 · 分层评价 · 创意表达空间

📋 任务总览

经过前面多个模块的学习，你已经围绕 "Friendship" 完成了理解、分析、设计和产出。现在是时候把你的最佳成果搬上作业榜了！

💡 从以下三个层级中选择你的挑战路径

C级 · 全员必选 基础层：再现与理解

任务：用英文完成一份 "Friendship Core Map" — 梳理本单元关于友谊的核心词汇、关键句型和主题观点。

要求：

  • 包含至少 8个 单元核心词汇（如 trust, betray, share, forgive, communicate）
  • 包含 2个 本单元重点句型（如直接引语/间接引语转换）
  • 用思维导图或表格形式呈现

🎨 表达形式：A4纸手绘思维导图 / Canva数字海报

B级 · 三选一 分层任务层：差异化选择

根据你的兴趣和特长，从以下三项中选择一项完成：

✏️ 选项A · 写作型

A Letter About Friendship

给一位对你影响深远的人写一封英文信，分享你对"友谊"的新理解。

  • 150-200词
  • 包含至少3个本单元所学表达
  • 手写信件或电子文档

📊 选项B · 图示型

Friendship Infographic

用信息图展示"友谊的科学"——将STEAM视角与英语表达结合。

  • 至少4个学科视角
  • 每个配英文说明20-30词
  • A3海报 / Canva / PPT单页

🎤 选项C · 视频/演讲型

My Friendship Speech

录制一段英文演讲，主题自选。

  • 2-3分钟
  • 有开场白、主体论点(≥2)、结束语
  • 配合肢体语言或辅助道具

A级 · 挑战项 开放层：创造性表达

任务：围绕 "Friendship + STEAM" 主题，创作一份完全由你设计的原创作品。

方向参考：

  • 🎬 2分钟英文"Friendship"主题短片（动画/实拍/混剪）
  • 📐 设计"友谊指数"测量工具（问卷 + 数据分析图表，英文呈现）
  • 🎵 创作英文短诗/说唱，配背景音乐录制
  • 🖼️ 英文漫画（4-8格），讲述一个友谊故事
  • 🎮 英文互动游戏（如友谊知识问答卡）

✨ 额外加分项：在作品中展示从STEAM融合中学到的跨学科洞察

📐 评价标准（Rubric）

维度A级 超越 (9-10分)B级 达到 (7-8分)C级 接近 (5-6分)D级 需改进 (0-4分)
语言质量英文准确自然，词汇丰富，句式多样少量错误，使用了核心词汇较多错误，能传达基本意思错误严重，难以理解
内容深度独特见解，体现STEAM跨学科思考内容完整，有清晰观点内容基本完整但缺乏个人思考不完整或偏离主题
结构逻辑结构清晰，层层递进结构完整，衔接自然有基本结构但衔接不畅结构混乱
创意与形式形式服务内容，设计精美有创意形式恰当，完成度好形式基本恰当但平淡形式不当或完成度低
反思与成长清晰表达跨学科认知提到学习收获简单提及不够具体未提及反思
