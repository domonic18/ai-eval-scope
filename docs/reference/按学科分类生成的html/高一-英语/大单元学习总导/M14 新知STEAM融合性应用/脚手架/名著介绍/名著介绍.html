📖 中国著作：《论语》（选读"交友"篇）

📚 哲学经典
🎯 核心问题：君子之交与现代"人脉"有何本质区别？
目录结构：学而篇"有朋自远方来" → 里仁篇"德不孤，必有邻" → 颜渊篇"子贡问友" → 季氏篇"益者三友，损者三友"
核心观点：孔子将友谊建立在"德"与"仁"之上，以道相交、以义相合。益友三标准：友直（正直）、友谅（诚信）、友多闻（博学）。
关键片段：子贡问友，孔子答"忠告而善道之，不可则止，毋自辱焉"——以诚相劝、保留边界，与现代心理学中的"健康边界"不谋而合。
适读人群：对中国传统文化感兴趣、希望从哲学高度理解友谊本质的同学
获取路径：杨伯峻《论语译注》（中华书局）| 线上：ctext.org/analects

🌟 外国经典：《小王子》 — Antoine de Saint-Exupéry

🌍 世界文学
🎯 核心问题："驯服"（tame）如何定义一段真正的友谊？
目录结构：第1-9章 离开B-612星球 → 第10-15章 遇见国王、商人、点灯人 → 第16-20章 遇见狐狸 → 第21-27章 告别与玫瑰
核心观点：友谊需要的不是短暂的热闹，而是"驯服"（建立联系）后的责任。"正是你为玫瑰花费的时间，才使你的玫瑰变得如此重要。"
关键片段：狐狸对小王子说："请你驯服我吧！"——这一请求揭示友谊的本质：耐心、仪式感、独一无二的关系。狐狸的"秘密"——"真正重要的东西用眼睛是看不见的，要用心去感受。"
适读人群：英语初学者可读中英对照版；适合所有年龄层，尤其适合反思"拥有 vs 关系"的区别
获取路径：中英对照版（上海译文出版社）| 英文原版 Project Gutenberg

🔍 跨学科/当代作品：《被讨厌的勇气》 — 岸见一郎 / 古贺史健

🧠 心理学
🎯 核心问题：友谊是否需要"被所有人喜欢"？
目录结构：第一夜 否定因果论 → 第二夜 一切烦恼源于人际关系 → 第三夜 课题分离 → 第四夜 共同体感觉 → 第五夜 活在当下
核心观点：阿德勒认为"一切烦恼皆源于人际关系，但一切幸福也源于人际关系"。友谊的核心是"课题分离"——他人的评价是别人的课题，你只需坚持自己的善与诚。被讨厌不是目的，而是自由的代价。
关键片段："我可以把马牵到水边，但不能强迫它喝水。"朋友的情绪、选择、评价都是对方的课题；你只需管好自己的课题：真诚、尊重、有边界。
适读人群：在人际关系中感到焦虑、讨好型倾向明显的高中及以上学生
获取路径：《被讨厌的勇气》（机械工业出版社）| 得到App/微信读书有声版
