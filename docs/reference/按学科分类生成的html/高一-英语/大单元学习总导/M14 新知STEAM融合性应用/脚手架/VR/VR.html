🎯 观察任务

戴上 VR 设备后，仔细观察你所处的场景。注意以下细节：

  • 场景识别 — 你身处什么地方？英美高中校园？国际夏令营？跨文化笔友见面会？
  • 人物关系 — 场景中的人物之间是什么关系？他们正在做什么？
  • 语言表达 — 记录至少 3 个与"友谊"相关的英语表达
  • 非语言信号 — 表情、手势、距离——文化差异如何影响友谊的建立？
    • 👁️
    "A good observer sees what others overlook."
    戴上 VR，像探索者一样观察每个细节

    🕹️ 操作任务

    在 VR 场景中主动参与互动，完成以下操作：

    1. 自我介绍：用英语向虚拟人物介绍自己，包含姓名、兴趣和交友邀请
    2. 情境选择：从以下三个 VR 场景中任选一个：
    🍔
    场景 A 美国高中 cafeteria
    与陌生同学拼桌聊天，聊出共同话题
    🏕️
    场景 B 国际志愿者营地
    与不同国家的队友合作完成任务
    📹
    场景 C 跨文化笔友视频见面
    用英语介绍自己文化，询问对方习俗

    3. 回应与追问：回应对方提问并追问一个问题，保持对话流畅。

    🆚 替代方案（无 VR 设备时）

    暂时没有 VR 设备？别担心，以下方式能获得接近的体验：

    🎬
    3D 沉浸式视频
    搜索 "360° virtual campus tour" 或 "VR friendship conversation"，配合 cardboard 眼镜即可观看
    🖱️
    交互式视频
    使用 ThingLink 或 H5P Interactive Video 观看带交互热点的视频，点击选择对话选项
    🎭
    角色扮演 + 视频
    两人一组，根据场景卡进行英语角色扮演，录制后回看分析
