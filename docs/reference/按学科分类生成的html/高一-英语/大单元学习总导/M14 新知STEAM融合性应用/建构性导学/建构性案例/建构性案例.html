案例一 · 个人理解型

🖼️ 我的"友谊观"画像

面对新班级，每个人对"朋友"的理解不同。你的友谊观是什么？
绘制个人"友谊观画像"——用关键词、图画或短句，表达你心中"朋友"最重要的3个特质。

原理① 先验知识激活

  1. 回忆你印象最深的一位朋友，写下对方最打动你的一个瞬间
  2. 从那个瞬间提炼出你认为友谊中最重要的东西
  3. 用3个关键词概括你的友谊观
  4. 为每个关键词配一句简短解释（中文或英文均可）
  5. 将成果整理在一张A4纸上，可用图画辅助表达

🧱 句式参考："Friendship is ____ because ____"

成果名称："My Friendship Profile"
格式：图文手抄报（A4纸），含3个关键词及解释
呈现建议：2人一组互相分享，用1分钟向同伴介绍你的友谊观

✅ 自评量规（点击勾选）

我写出了3个与友谊相关的关键词
每个关键词都有简短解释
解释内容来自我的真实经历或感受
我尝试用英文书写了至少部分内容
我能向同伴清楚地说明我的想法
案例二 · 生活应用型

💌 当朋友需要安慰时

你的好朋友因为考试没考好而非常沮丧。你想安慰他/她，但不知道该说什么才合适。
为这位朋友写一封100词左右的英文安慰信，表达理解、鼓励和支持。

原理② 情境驱动认知

  1. 闭眼想象朋友坐在你面前的表情和状态
  2. 列出你想对他/她说的3件事（可用中文打底稿）
  3. 参考教材Unit 1中关于情感表达的短语
  4. 将这些要点组织成一封英文短信
  5. 读一遍，检查语气是否温暖自然
  6. 抄写到信纸或卡片上
成果名称："A Comfort Letter to My Friend"
格式：80-120词英文信，含称呼、正文、署名
呈现建议：配一张手绘小插图，让信件更有温度

✅ 自评量规

信件包含称呼、正文、署名三部分
正文表达了对朋友处境的理解
使用了至少2个教材中的情感表达短语
信件长度在80-120词之间
整体语气温暖、真诚
案例三 · 校园问题解决型

🤝 班级"交友困难"解决方案

班里有同学反映，开学一个月了还没交到朋友，不知道该怎么主动和别人说话。
小组合作设计一份"破冰指南"——列出3种适合新同学的主动交友方法，并用英文写出对话样例。

原理③ 协作建构 原理④ 多元视角

  1. 小组讨论：新同学难交朋友的原因是什么？
  2. 头脑风暴：每个人提出一种"破冰方法"
  3. 筛选出最可行的3种方法
  4. 为每种方法写一个英文对话示例（4-6句对话）
  5. 将成果整理成一份小册子样式
成果名称："Icebreaker Guide — Make a Friend Today!"
格式：A4纸对折成4页小册子，含封面+3种方法+对话
呈现建议：各组互换阅读，评选"最实用破冰法"

✅ 自评量规

我们列出了3种不同的破冰方法
每种方法都配有一个英文对话示例
对话自然流畅，符合真实交流场景
小组成员每个人都贡献了想法
小册子排版清晰，可读性强
案例四 · 学科表达型

📝 用英语讲述友谊故事

学校英语广播站征集"友谊故事"稿件。你需要写一个200词左右的真实小故事。
用英文叙述一个真实的友谊故事，包含"起因—经过—结果/感悟"结构。

原理① 先验知识激活 原理⑤ 表达驱动建构

  1. 回忆一件你和朋友之间印象最深的事
  2. 用中文给这件事列一个时间线（开头→发展→高潮→结尾）
  3. 将时间线翻译成英文关键词
  4. 用英文连词成句，写出完整故事
  5. 加一句结尾句：这件事让你对友谊有了什么新认识？
  6. 读给同桌听，根据反馈修改不通顺的地方

🧱 连接词参考：first, then, later, finally, because, so, but, however

成果名称："A Friendship Story I'll Never Forget"
格式：180-220词英文叙事文，含标题和3段落
呈现建议：用手机录音朗读，投稿到班级英语广播站展示墙

✅ 自评量规

故事有明确的"起因—经过—结果"结构
字数在180-220词之间
使用了至少3个叙事连接词
故事真实，来自个人经历
结尾有一句关于友谊的感悟
案例五 · 跨学科整合型

🌍 "Friendship Around the World" 跨文化对比

不同国家的人对友谊的理解和表达方式可能不一样。例如在日本朋友很少直接说"我爱你"；在巴西见面会拥抱。
选择一个国家，研究该国交友文化，制作一张A3英文对比海报，展示与中国的异同。

原理④ 多元视角 原理⑥ 反思性建构

  1. 从推荐列表中选择一个国家（日本/巴西/德国/肯尼亚/印度）
  2. 收集3条该国友谊文化信息
  3. 对照中国文化，找出相同点和不同点各2条
  4. 用英文整理成对比表格
  5. 将表格和插图排布到A3海报上
  6. 在海报底部写一句个人反思："我发现友谊____"
成果名称："Friendship Around the World — A Comparison"
格式：A3海报，含对比表格、插图、个人反思句
呈现建议：张贴在教室文化墙，举行小型"文化博览会"

✅ 自评量规

我选择了1个国家并收集了至少3条友谊文化信息
海报包含相同点和不同点各2条
所有文字使用英文书写
海报设计美观，有图画辅助
我写了一句个人反思，表达了对友谊的新认识
案例六 · 社会/未来情境型

🤖 "AI可以成为朋友吗？"未来情境辩论

AI聊天机器人越来越像真人了。有人觉得AI可以当朋友，也有人觉得AI没有真实情感，不能算朋友。
选择正方或反方，写一份3分钟英文辩论稿（约250词），阐明你的立场和理由。

原理⑤ 表达驱动建构 原理⑦ 元认知监控

  1. 阅读教师提供的2篇短观点文（正方&反方各一篇）
  2. 确定自己的立场，用思维导图列出3条理由
  3. 为每条理由找一个例子支撑
  4. 按"观点—理由1—理由2—理由3—总结"结构写稿
  5. 用手机计时器试读，控制在3分钟内
  6. 修改超长或卡顿的句子

🧱 句型参考：I believe... / My first reason is... / For example... / In conclusion...

成果名称："Can AI Be a Friend? — My Debate Speech"
格式：200-280词英文讲稿，含标题和5段落
呈现建议：在班级辩论会上发表，录制视频上传班级群

✅ 自评量规

我明确选择了正方或反方立场
我列出了3条支持立场的理由
每条理由都有例子或证据支持
讲稿字数在200-280词之间
试读时间在2分30秒到3分30秒之间
案例七 · 综合创新型

🏫 创建班级"Friendship Wall"综合项目

班级需要一个让大家互相了解、增进友谊的公共空间——线下墙或线上平台。
以小组为单位，设计并制作一个班级"Friendship Wall"，包含照片、友谊故事、鼓励语、趣味互动等。

原理③ 协作建构 原理⑥ 反思性建构 原理⑦ 元认知监控

  1. 小组讨论：友谊墙放在哪里？包含哪些板块？（照片墙、暖心留言、每周友谊挑战等）
  2. 分配角色：设计师、文案员、技术员（拍照/排版）、监督员
  3. 每个成员独立完成自己负责的部分
  4. 组内互审：检查英文拼写、排版美观度
  5. 组装拼接，正式上墙
  6. 全班巡展：每个小组派代表用英文介绍本组亮点
成果名称："Our Class Friendship Wall"
格式：实体海报墙或Canva在线页面，≥4个板块
呈现建议：邀请隔壁班同学参观，用英文为访客讲解

✅ 自评量规

友谊墙包含至少4个功能板块
每个板块都有英文内容
组内每个成员都有明确分工和贡献
整体设计美观协调，没有拼写错误
我能用英文为本组作品做2分钟介绍
