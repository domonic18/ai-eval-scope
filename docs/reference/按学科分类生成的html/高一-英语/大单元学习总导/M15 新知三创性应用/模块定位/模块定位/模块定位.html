M15 新知三创性应用

模块定位 · 大单元 Unit 1 Friendship · 必修1

📖 本模块学习什么

本模块学习的是将你对 Unit 1 Friendship 的理解转化为创造性应用成果。不是再学一遍"友谊是什么"，而是请你运用已经学到的语言知识和主题理解，主动建构出属于你自己的友谊主题表达作品。

具体来说，你要学习：

  • 如何从情境、问题、经验、证据和表达中建构新知识
  • 如何系统分析友谊相关的语篇和情境
  • 如何运用建构性原理设计自己的表达方案
  • 如何评价和优化自己的学习成果

💡 本模块为什么重要

  • 因为学习不只是"理解"：把你学会的英语知识真正"用起来"，才是语言学习的终极目标
  • 因为创造力是未来核心竞争力：无论未来做什么行业，能够创造性地表达自己的思想都是核心能力
  • 因为友谊值得深入思考：友谊是人生最重要的关系之一，用英语思考友谊、表达友谊，会让你的英语学习更有深度和温度

✨ 核心提示：本模块是你的"成果生成"环节——所有学习的最终呈现！

🔗 本模块与大单元主题的关系

大单元 Unit 1 Friendship 的主题是"友谊"——理解友谊、分析友谊、表达友谊。前序模块帮助你完成了"理解"和"分析"的部分，现在你要进入"创造性地表达"阶段。

理解 → 分析 → 创造性地表达 → 成果呈现

本模块是你所有学习的最终呈现——从输入到输出，从理解到创造。

🔄 本模块与前后模块的衔接

  • 前序模块（M14 及之前）：完成了对友谊主题的解析性理解——你学会了如何分析关于友谊的语篇、句子和词汇使用
  • 本模块（M15）：在理解基础上，进入创造性应用——建构自己的友谊主题成果
  • 后续模块：本模块产出的成果将成为后续模块进一步学习和评价的基础

模块衔接示意图：

M14（解析性理解）→ M15（创造性应用） → 后续模块（评价与拓展）

本模块处于"从输入到输出"的关键转折点。
