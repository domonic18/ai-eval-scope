建构性七阶适配案例

七个从易到难的建构性案例，覆盖个人理解、生活应用、跨学科整合等多元场景

个人理解型

我的友谊价值观卡片

情境：你收到学校英语角的任务：用3句话向外国朋友解释"友谊对你意味着什么"。如何用简短英语表达友谊价值观？

建构任务：制作一张 Friendship Values Card（友谊价值观卡片）

情境锚定原理语言赋能原理

1反思个人友谊经历
2提炼核心价值词
3组织英语表达
4设计卡片版式

成果表达：一张包含 1 句定义 + 2 句个人感悟的英语卡片

表达了个人真实的友谊观
用到了本单元 3 个以上词汇
语言简洁有力
生活应用型

给室友的沟通信

情境：你和室友在作息时间上有冲突，想用英语写一封表达理解并提出解决方案的信。

建构任务：写一封 Roommate Communication Letter

问题识别原理经验链接原理

1明确冲突点
2理解对方角度
3表达自己的感受
4提出折中方案

成果表达：英文书信（150 词以上）

开头表达了理解
具体描述了问题
提出了可操作的方案
校园问题解决型

跨文化交友指南

情境：学校有来自不同国家的交换生，他们反映很难和中国学生交朋友。

建构任务：制作一份 Cross-cultural Friendship Guide（跨文化交友指南）

经验链接原理循环优化原理

1调研交友困难
2分析文化差异
3总结实用建议
4设计图文版式

成果表达：图文并茂的迷你指南（可手绘/数字化）

包含至少 3 条文化差异提示
每条提示有具体例子
语言中性、尊重
学科表达型

友谊主题英语微演讲

情境：班级要举办英语主题演讲，题目是"What Makes a True Friend"。

建构任务：撰写并录制一段 3 分钟的英语演讲稿

情境锚定原理语言赋能原理循环优化原理

1确定演讲主线
2收集个人例子
3组织演讲稿结构
4练习语调表情

成果表达：3 分钟英语演讲视频/音频

有明确的观点
包含个人真实经历
语速适中、发音清晰
跨学科整合型

友谊的"科学 + 人文"双重解读

情境：心理学研究发现高质量友谊能延长寿命、降低压力。文学作品中也有大量歌颂友谊的篇章。

建构任务：制作一份 "Friendship: A Dual Perspective" 信息图

问题识别原理经验链接原理

1收集科学研究数据
2选取文学名言/段落
3对比科学和人文视角
4设计信息图布局

成果表达：A4 大小的信息海报

包含至少 1 个科学事实
包含 1 段文学引用
双重视角对比清晰
社会/未来情境型

线上友谊与线下友谊

情境：你的表弟沉迷社交网络，认为在网上有 200 个"朋友"，但在学校却很孤独。

建构任务：设计一个 Digital vs. Real Friendship Comparison Framework 对比框架

情境锚定原理循环优化原理

1整理线上友谊特点
2整理线下友谊特点
3设计对比维度
4形成个人结论

成果表达：双栏对比分析表 + 个人结论

包含 3 个以上对比维度
每个维度有具体例证
结论体现了你的判断
综合创新型

班级友谊文化建设方案

情境：班主任请你为班级设计一个"友谊建设计划"，用英语展示，促进班级凝聚力。

建构任务：撰写一份 Class Friendship Building Plan 方案书

综合运用前 6 条原理

1调研班级现状
2设计 3 个友谊建设活动
3规划时间表和资源
4制作方案展板

成果表达：英语方案书（含活动设计、时间表、预期效果）

活动设计具体可执行
方案包含时间和资源规划
体现了对班级现状的理解
