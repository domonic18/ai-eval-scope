评价重点：词汇准确性、语法正确性

B 层 · 分析聚焦 — 适合想提升分析能力的你

评价重点：分析深度、证据使用

C 层 · 创造聚焦 — 适合想挑战表达创意的你

评价重点：创意性、语言自然度、场景真实感
