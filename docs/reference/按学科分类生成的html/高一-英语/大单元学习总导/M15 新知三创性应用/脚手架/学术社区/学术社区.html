讨论议题（结构化）

1 你的文化中，"好朋友"的标准是什么？ 与其他同学的文化背景有什么不同？对比一下，你会发现友谊的文化差异远比你想象的大。
2 如果你和好朋友发生了矛盾，你通常会怎么处理？ 你的方式有效吗？有没有更好的处理方法值得尝试？
3 社交媒体上的"朋友"算真正的朋友吗？ 为什么？请用英语阐述你的立场，并至少引用一个例子支持你的观点。

角色分工（小组合作参考）

提出讨论话题，控制时间进度
📝 记录员
用英语记录组员的核心观点
🎤 发言人
代表小组向全班/社区分享成果
❓ 质疑者
提出不同角度的反问，促进深度讨论
🏠 家校连接 · 访谈任务 ▼

🗣️ 家长/抚养人访谈任务

询问一位你信任的长辈："你年轻时最重要的朋友是谁？为什么？"

用英语记录访谈内容，并在班级共享文档中分享。通过长辈的故事，你可能发现跨越时代的友谊真谛。

📎 反馈机制：将访谈记录上传至班级共享文档，互相阅读并给出至少一条有意义的评论。

📨 国际笔友任务

与一位英语为母语的笔友交换一封关于"友谊"主题的邮件，对比你们的文化差异。用英语完成并保存邮件副本作为学习成果。

📤 输出机制

  • 将讨论和访谈成果整理成 英语短文，发布在班级博客/学习平台上
  • 每人至少 阅读和评论 2位同学的分享
  • 期末汇总形成 "友谊主题班级文集" 电子书

💡 从身边的对话到跨文化的交流，每一步都是对友谊理解的拓展
