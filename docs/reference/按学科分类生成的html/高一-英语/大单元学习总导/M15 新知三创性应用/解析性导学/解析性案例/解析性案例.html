1生活情境——课间的小摩擦
#先观察不评判#人际冲突
📋 案例材料
你和同桌在课间因为一件小事发生了口角，你认为是ta不小心碰倒了你的水杯却不道歉，ta认为你反应过度。事后两人都不说话。
🎯 解析任务
用"先观察不评判"原理，描述这个事件的事实层面，区分客观事实与主观判断。
🔗 解析路径
记录客观事实 → 区分主观感受 → 识别各自诉求
📊 分析记录表
填写区你的记录
🔹 事实水杯被碰倒 → 对方未道歉 → 双方情绪升级 → 冷战
🔹 感受你觉得被冒犯；对方觉得被误解
🔹 期望你期望一句道歉；对方期望多些包容
💡 参考：你描述的"事实"中是否混入了判断词？试试去掉"过度""过分"这类词

2校园情境——小组作业的分工困难
#问题归类法#团队协作
📋 案例材料
英语老师布置了一个小组项目"介绍不同国家的友谊文化"。你们组三个人：小A英语好但不爱发言，小B积极但总是跑题，小C很认真但速度慢。
🎯 解析任务
用问题归类法，判断这个团队问题属于什么类型。
🔗 解析路径
识别任务要求 → 列出成员特点 → 判断匹配/不匹配点
📊 成员匹配分析
成员优势挑战匹配度
小A英语好不爱发言⭐ 适合写稿
小B积极容易跑题⭐ 适合展示
小C认真速度慢⭐ 适合查资料
💡 这是分工问题？沟通问题？还是能力配比问题？——用问题归类法判断

3社区情境——公共空间冲突
#三层面分析法#社区矛盾
📋 案例材料
你家小区有一块公共绿地，年轻人想在那里打篮球，老年人想在那里散步聊天，双方发生了争执。
🎯 解析任务
用三层面分析法分析这个社区矛盾。
🔗 解析路径
事实层（发生了什么） → 感受层（各自感受） → 期望层（各自期望）
📊 三方 × 三层面矩阵
事实层感受层期望层
🧑 年轻人绿地空置时想打球觉得没有活动空间希望增设运动设施
👴 老年人平时在绿地散步聊天担心安全和安静被打破希望保留休闲空间
🏢 社区管理者两方诉求冲突协调难度大找到两全方案
💡 每组人的"事实""感受""期望"可能完全不同——三层面帮你理清

4学科核心情境——友谊观对比
#差异定位法#英语语篇
📋 案例材料
比较两篇关于友谊的英语短文。第一篇讨论"友谊需要付出努力"，第二篇强调"真正的友谊是自然而然的"。
🎯 解析任务
用差异定位法，对比两篇文章的核心观点差异。
🔗 解析路径
确定对比维度 → 提取各自观点 → 定位差异点 → 分析差异原因
📊 对比分析表
对比维度文章一文章二
核心观点友谊需要付出努力真正友谊是自然的
友谊的性质需要经营维护水到渠成
对冲突的看法冲突需主动解决真正的朋友经得起考验
文化背景偏西方主动交友观偏东方自然缘分的观念
💡 差异背后可能反映了不同的文化背景或作者经历

5跨学科情境——"朋友数量与幸福感"的数据解读
#追溯溯源法#数据分析
📋 案例材料
你看到一组研究数据：每周见面3次以上的朋友数量与自评幸福感正相关，但朋友总数超过15人后幸福感不再上升。
🎯 解析任务
用追溯溯源法分析——为什么"更多朋友"不等于"更幸福"？
🔗 解析路径
看清数据事实 → 分析变量关系 → 追溯可能机制 → 提出解释
📊 分析记录表
数据现象可能原因证据支持其他解释
3次+/周与幸福感正相关深度互动增进情感纽带多项心理学研究可能忙碌者本身更积极
15人后幸福感不再上升社交精力有限，广度≠深度邓巴数理论可能泛泛之交消耗精力
💡 质量可能比数量更重要——"深度关系"和"浅层关系"的差异在哪？

6社会情境——社交媒体时代的"朋友"定义
#问题归类法#差异定位法
📋 案例材料
社交媒体上，人们把关注对象也叫"朋友"。一段网络辩论：A方认为网络好友是真正的友谊，B方认为只有线下互动才算真朋友。
🎯 解析任务
用问题归类法 + 差异定位法，分析两方的核心分歧。
🔗 解析路径
归类双方观点类型 → 找到分歧根源 → 判断各自逻辑是否自洽
📊 观点分析表
观点类型定义分歧论据质量
A方：网络好友=真友谊友谊基于情感交流而非物理接触有真实情感交流案例支撑
B方：只有线下才是真朋友友谊需要共同经历和身体在场传统友谊理论支持
💡 分歧核心可能是对"友谊"的定义不同——先厘清定义，再判断论据

7综合迁移情境——你的友谊决策
#综合运用#价值冲突
📋 案例材料
你最好的朋友做了一件你觉得不对的事（抄袭作业），你要不要告诉老师？告诉了友谊可能破裂；不告诉对不起自己的原则。
🎯 解析任务
综合运用多种原理，系统分析这个决策困境。
🔗 解析路径
观察事实 → 归类问题类型 → 三层面分析 → 差异定位（你的原则 vs 友谊）
📊 自由分析框架
分析维度你的思考
🔹 事实层朋友抄袭作业；你发现了；需要考虑是否告知老师
🔹 归类这是一个"忠诚 vs 诚信"的价值冲突问题
🔹 各方的期望你：想坚持诚实原则；朋友：希望得到包庇
🔹 差异点你的诚信原则与对朋友的忠诚之间的矛盾
💡 问题本质上是一个价值冲突——不同选择各有什么后果？
