（一）从示例到原理

为什么要从案例中提炼原理？因为一个个案是"孤例"，原理才是可迁移的"思维框架"。好比学会了做一道菜，更重要的是掌握"炒、煎、炖"的烹饪原理，以后遇到任何食材都能应对。

下面这5条原理，就是从李华的邮件解析中提炼出来的核心方法。

（二）解析性原理提炼

原理1：情感信号识别原理
通过标注情感词汇（upset, grateful, lonely等）判断作者态度。
在本示例中的体现：圈出邮件中所有情感词汇作为分析起点。
如何使用：标注所有情感词 → 按正面/负面分类 → 分析情感强度 → 判断态度。
可迁移场景：分析新闻评论、读后感、社交媒体帖子等。
使用提醒：注意文化差异导致的情感表达方式不同。
原理2：语篇结构拆解原理
将语篇按功能拆分为"叙述—表达—诉求"等模块。
在本示例中的体现：将李华的邮件拆分为事件叙述、情感表达、求助诉求三部分。
如何使用：通读全文 → 判断各段落功能 → 标注结构标记词 → 绘制结构图。
可迁移场景：分析英语作文、演讲稿、信件等。
使用提醒：有些语篇结构不是线性的，可能有穿插。
原理3：上下文推演原理
不孤立看一句话，而是放在上下文中理解含义。
在本示例中的体现：某个看似抱怨的句子在上下文中可能是求助信号。
如何使用：找到关键句 → 读前后文 → 推断真实意图 → 验证推断。
可迁移场景：阅读理解推断题、听力理解态度判断。
使用提醒：避免过度推断，要有文本证据支撑。
原理4：证据—结论匹配原理
判断一个结论是否有足够证据支撑。
在本示例中的体现：分析"李华认为新室友不好"这个结论是否有足够证据。
如何使用：找出结论 → 列出所有证据 → 判断证据充分性 → 评估结论可信度。
可迁移场景：议论文分析、辩论逻辑判断。
使用提醒：分清客观事实和主观感受。
原理5：跨文化视角原理
注意中西方表达方式的差异。
在本示例中的体现：英语中表达情感的方式和中文有何不同。
如何使用：对比同一情境的中英文表达 → 分析差异原因 → 调整理解方式。
可迁移场景：跨文化交际、中英互译。
使用提醒：不要用自己的文化标准评判另一种文化的表达。
