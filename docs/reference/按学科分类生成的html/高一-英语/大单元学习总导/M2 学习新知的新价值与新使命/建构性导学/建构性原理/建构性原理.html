🔧 建构性原理提炼
1 情境激活原理

核心思想：从真实的情感体验出发，而不是从语法规则出发学习英语。

来源：在建构性示例中，学生从"收到外国笔友邮件"的真实情境切入，先感受再分析。

适用场景：任何需要建立"情感连接"的英语学习场景——角色扮演、书信写作、观点表达。

注意事项：情境要真实可信，不能虚假造作。让学生觉得"这真的会发生在我身上"。

2 框架建构原理

核心思想：先搭骨架（结构），再填血肉（内容）。

来源：语篇解析五步法提供了一个分析框架，学生按框架逐层深入。

适用场景：写作教学（先给结构模板再自由创作）、阅读理解（先把握结构再分析细节）。

注意事项：框架不能成为束缚——要在学生掌握后鼓励突破框架、创新表达。

3 证据驱动原理

核心思想：用具体事例支撑抽象观点，避免空洞。

来源：在解析语篇时，所有对作者立场和意图的判断都必须回到文本中找语言证据。

适用场景：议论文写作、语篇分析、演讲表达——任何需要"有理有据"的英语输出。

注意事项：证据要有代表性而不只是"凑数"；要学会区分"直接证据"和"间接暗示"。

4 循环迭代原理

核心思想：写作不是一次成型，而是修改出来的。

来源：建构过程中强调反馈修改环节——同伴交换解析笔记、讨论差异、完善理解。

适用场景：英语写作（初稿→同伴反馈→修改→定稿）、口语表达（练习→录音→回听→改进）。

注意事项：迭代要有方向——每次修改都聚焦一个具体维度（如词汇多样性、逻辑连贯性）。

5 读者意识原理

核心思想：始终想着"我的表达是为谁而写的"。

来源：从"给转学生写欢迎信"的案例中提炼——写信时要考虑对方的身份、感受和期待。

适用场景：应用文写作（邮件、书信、邀请函）、海报设计、公开演讲。

注意事项：不要过度揣测读者心理而失去自己的表达风格——在"读者友好"和"个性表达"之间找平衡。

🏋️ 建构性原理迁移应用

📝 练习1：介绍你的友谊圈

情境：向寄宿家庭用英语介绍你的日常友谊文化

建议原理：情境激活 + 读者意识

脚手架：先列出3个你与朋友相处的典型场景，每个场景用2-3句英语描述

🌏 练习2：中西方友谊观短评

情境：比较中西友谊观差异，写一段80-100词英语短评

建议原理：框架建构 + 证据驱动

脚手架：先用表格列出3个维度（交友方式、表达亲密的方式、冲突处理），再逐项展开
