🧬 案例一 · 个人理解型

"My Friendship DNA" — 友谊双螺旋模型

个人理解情境

📖 真实建构情境

你在整理房间时翻到了一本小学毕业同学录，上面写着当年的好朋友给你的留言："We will be best friends forever!" 但你现在和这位朋友已经几乎不联系了。你决定不从网上搜答案，而是从自己的真实经历出发，建构一个属于自己的"友谊 DNA"模型——把塑造友谊的"基因要素"找出来。

🎯 建构任务

建构一张 "My Friendship DNA 双螺旋结构图"，其中每条横档（rung）代表你心目中友谊的一个核心构成要素，左右两条链分别代表"给予（What I give）"和"获得（What I get）"，并用英语写一段100词以上的"Friendship DNA 说明书"。

📐 建构步骤

  1. 发散：在左侧列出你从友谊中"获得"的所有东西（如 emotional support, fun, advice），右侧列出你"给予"的所有东西（如 loyalty, time, honest feedback）。
  2. 配对：从左右各选一个匹配度最高的要素，形成一组横档（如"Shared laughter ↔ Comfort in silence"）。
  3. 排序：将配好的横档按重要性从高到低排列。
  4. 可视化：画出双螺旋结构，每条横档标注英文要素对。
  5. 撰写：用英语写一段说明书，解释为什么你选择了这些要素。

🧰 脚手架

"The most fundamental gene in my friendship DNA is ___ because ___."
"On the giving strand, I place ___, which matches ___ on the receiving strand."
"Without ___, the whole structure would fall apart."

✅ 自评量规

  • 我是否画出了双螺旋结构，且清晰地标注了至少4组要素对？
  • 每组要素对是否体现了"给予"和"获得"的对应关系？
  • 我的说明书是否达到了100词以上，且包含了主题句和总结句？
  • 我是否至少使用了4个本单元的核心词汇（如 loyalty, mutual, bond, companionship）？
  • 我的模型是否真实反映了我个人的友谊经历？
✅ 案例二 · 生活应用型

"The Friendship Health Checklist" — 友谊健康自检清单

生活应用情境

📖 真实建构情境

你发现身边有同学因为一段"有毒友谊"（toxic friendship）而长期不开心。你决定做一件更有用的事：设计一份"友谊健康自检清单"（Friendship Health Checklist），让你的同学可以对照清单自行判断自己的友谊是否健康。

🎯 建构任务

设计一份包含6-8个检查项的"Friendship Health Checklist"（中英双语），每个检查项包含：一个健康信号（Green flag）和一个警示信号（Red flag），并配上一段150词的"使用指南"（英文）。

📐 建构步骤

  1. 提取维度：回顾你的 Friendship DNA 模型，提取3-4个关键维度。
  2. 设计信号：为每个维度设计 Green flag 和 Red flag。
  3. 编排格式：用表格呈现。
  4. 写使用指南：用英语说明目的和使用建议。
  5. 测试迭代：用清单评估自己的1-2段友谊。

📋 清单模板

Dimension | Green Flag | Red Flag Communication | "TA listens when I need to talk"|"TA only talks about themselves" Respect | "TA values my opinions" | "TA puts me down" Support | "TA shows up when I need help" | "TA only reaches out when they need something" Growth | "We help each other improve" | "We bring out the worst in each other"

✅ 自评量规

  • 我是否设计了至少6个检查项，覆盖了友谊的不同维度？
  • 每个维度是否同时包含了 Green flag 和 Red flag？
  • 使用指南是否达到150词以上？
  • 信号是否具体、可观察，而非模糊描述？
🎒 案例三 · 校园问题解决型

"The New Kid's Friendship Starter Kit" — 转学生友谊启动包

校园问题解决情境

📖 真实建构情境

学校下个月将迎来一批转学生。很多转学生最焦虑的不是学习，而是"交不到朋友"。你决定为转学生设计一份实用的"Friendship Starter Kit"，帮助他们在新学校快速、健康地建立友谊。

🎯 建构任务

设计一份双面 A4 折页式的"Friendship Starter Kit"，包含：一张"破冰对话指南"、一张"友谊养成路线图"（从陌生人到朋友的4阶段路径），以及一段英语鼓励信。

Conversation starters 范例：

"What's your favorite class so far?"
"I noticed you're really good at ___. How did you learn?"
"Want to grab lunch together?"
"Did you understand the homework? I'm a bit confused."

避免的"雷区句子"：过于私人的问题、负面评价其他同学、抱怨学校。

友谊养成4阶段：

Stranger（陌生人）→ Acquaintance（认识）→ Friend（朋友）→ Close Friend（挚友）

每阶段配2-3条行动建议。用箭头串联各阶段，标注预期时间。

📊 案例四 · 学科表达型

"Friendship Across Genres" — 体裁对比海报

学科核心情境

📖 真实建构情境

教材 Unit 1 中包含了不同类型的文本：日记（Anne's diary）、建议信（letter of advice）、诗歌（friendship poems）、对话（daily conversations）。同样是谈论友谊，不同体裁的表达方式差别很大。你决定建构一个"同主题跨体裁对比框架"。

🎯 建构任务

制作一张 "Friendship Across Genres" 对比海报，选取3种不同体裁的文本片段，从"语言风格""情感强度""叙事视角""核心信息"4个维度进行对比分析，并撰写一段120词的总结短文。

📋 对比框架

维度 | 日记 (Diary) | 建议信 (Letter) | 对话 (Dialogue) Language Style | | | Emotional Intensity | | | Perspective | | | Core Message | | |
"While the diary uses ___ to express ___, the letter adopts a more ___ tone."
"This difference suggests that ___."
🧠 案例五 · 跨学科整合型

"The Psychology of Friendship Infographic" — 友谊心理学信息图

跨学科整合情境

📖 真实建构情境

你在心理课上学习了马斯洛需求层次理论（Maslow's Hierarchy of Needs），其中"爱与归属感"位于第三层。你决定将心理学的需求层次理论与你在本模块建构的友谊要素结合起来，设计一张"友谊心理学信息图"。

🎯 建构任务

设计一张 A4 信息图，呈现你建构的"友谊需求层次金字塔"——从底层到顶层依次是：

🏆 Growth（成长感）
激励与共同进步

👏 Esteem（尊重感）
认可与欣赏

🤝 Belonging（归属感）
陪伴与接纳

🛡 Safety（安全感）
信任与诚实

"At the most basic level, friendship provides ___."
"Once ___ is established, friendship can address the need for ___."
📬 案例六 · 社会/未来情境型

"My Friendship Philosophy for 2035" — 友谊哲学宣言

社会与未来情境

📖 真实建构情境

一家教育科技公司发起了一个"写给未来自己的信"活动：请你站在2035年的时间点（你大约25岁时），回答"十年后，你认为什么才是真正重要的友谊品质？" 你决定先建构一份"2035 Friendship Philosophy"宣言，再用信来表达。

🎯 建构任务

完成两张A4纸的成果：

纸1："2035 Friendship Philosophy"宣言海报（包含5条核心信念 + 每条阐释）

纸2：一封300词的英语信"Dear 18-year-old me"，用未来视角讲述你的友谊哲学领悟。

💡 思考引导

想象10年后的你——经历了大学、职场、可能去了不同城市——你的友谊观会发生什么变化？

哪些今天你"觉得重要"的品质，10年后可能不那么重要？反之亦然？

10年后的你可能会更看重内在品质（integrity, empathy）还是外在品质（fun, social status）？

"I believe that ___."
"After ten years, I've learned that ___."
📁 案例七 · 综合创新型

"My Complete Friendship Portfolio" — 我的友谊作品集

综合创新情境

📖 真实建构情境

本模块的学习接近尾声了。你将前6个案例的成果整合成一个统一的作品——一本属于你自己的 Friendship Portfolio，并从中提炼出一个贯穿始终的、高度凝练的个人友谊定义。

🎯 建构任务

完成一本"Friendship Portfolio"（A4纸对折成小册子，4-6页）：

  • 封面：自创英文书名 + 副标题
  • 第1-3页：前6个案例中最优秀的3个成果的精简版
  • 第4页：200词以上的"Integrated Definition of Friendship"——融合所有建构成果的终极定义
  • 封底：一句原创英文"Friendship Quote"
  • 附录：150词"建构过程回顾"，说明定义如何一步步进化

📋 作品集结构

封面：[自创书名] + [你的名字] P1：Case 成果精简版（自选1） P2：Case 成果精简版（自选2） P3：Case 成果精简版（自选3） P4：Integrated Definition of Friendship（200词+） 封底：[个人原创 Quote] 附录：Construction Journey Reflection（150词）
"Through this module, I have come to understand friendship not as ___, but as ___."
"Drawing from my six case studies, I define friendship as ___ because ___."
📈 总体递进逻辑
个人 → 生活 → 校园 → 学科 → 跨学科 → 社会 → 综合

案例一从最私人的经历出发 → 案例六加入时间维度展望未来 → 案例七完成对前面所有成果的整合与升华，形成最终的、个人化的 Friendship 定义与作品集。每个案例都要求你主动建构具体成果，而非被动回答问题。
