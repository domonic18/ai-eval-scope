📚 推荐学习方式

🔍 方法一：对比分析法
找一段中文和一段英文的友谊描述文本，对比用词和视角差异。英语更强调 action 还是 emotion？这会帮你突破中式英语思维定式。
🗺️ 方法二：概念建模法
拿一张白纸，把 Friendship 写在中间，向外辐射所有相关要素——trust, honesty, fun, support... 然后连线、分组、分层。画完之后你会惊讶于自己的理解有多丰富。
🔄 方法三：定义迭代法
先写初版定义，然后问自己三个问题：太窄了吗？太宽了吗？能用英语表达吗？迭代三次以上，你的理解会有质的提升。
📤 方法四：案例迁移法
学完一个示例后，找一篇新文本（英文歌词、电影台词、博客）用刚学的方法分析。迁移是学习发生的标志。
💡 关键提醒：这一模块不是靠死记硬背能完成的。尝试以上方法，事半功倍。

👤 值得了解的关键人物

🏛️ 亚里士多德

古希腊哲学家，提出三种友谊：功利之友、欢愉之友、德性之友。他的分类至今是西方友谊研究的理论基石。

🧮 罗宾·邓巴

提出"邓巴数字"：稳定社交关系上限约150人，真正亲密朋友不超过5人。他用数学解释了友谊的生物学上限。

🧬 海伦·费舍尔

研究发现友谊和爱情共享同一套神经化学机制，证明了"情感连接"有生物学基础。

📖 更多深度阅读

🌟《小王子》中的友谊观

"Tame me... and we shall need each other." 狐狸教小王子的"建立联系"正是友谊建构过程的经典诠释。

📊 Facebook 友谊推荐算法

"你可能认识的人"背后是一套 friendship prediction 模型——友谊在数据科学中是可以量化和预测的。

💡 前沿追问

以下问题没有标准答案。选 2-3 个写下你的思考：

🤖 AI 能和人类建立"友谊"吗？
如果 ChatGPT 越来越"懂你"，你对它的信任感和对你最好朋友的信任感有本质区别吗？如果 AI 主动说"I miss you"，这算友谊吗？
🌐 网络友谊是"真正的友谊"吗？
你有一个从未见过面的网友，但互相信任、互相支持。这符合你建构的 friendship 定义吗？
⏳ 友谊有"有效期"吗？
曾经无话不谈的朋友，现在一年不联系——他们还算是朋友吗？你如何定义 ex-friend 和 old friend 的区别？
🌍 跨文化友谊中最大的障碍是什么？
是语言本身？是对"朋友"这个词的不同理解？还是对"信任"的不同定义？用 qualities / behaviors / emotions / rules 框架分析。
🗣️ 英语水平会影响国际友谊吗？
一个用简单英语但真诚表达的人，和一个用复杂词汇但保持距离的人，谁更容易交到朋友？
