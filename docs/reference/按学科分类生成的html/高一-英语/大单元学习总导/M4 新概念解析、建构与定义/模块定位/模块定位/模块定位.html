📖 本模块学习什么

围绕 Unit 1 Friendship 的核心语言素材，带你完成三项关键任务：

🔍 解析
从课文中提取核心词汇与表达
→
🧩 建构
将零散词句重组为概念网络
→
✍️ 定义
用英语写出属于你自己的定义

简单说：先读透 → 再串起 → 最后用自己的话讲出来。这是从"输入"到"内化"再到"输出"的完整闭环。

📌 点击查看三项任务详解

解析 理解 confide in, on purpose, get along with, trust, loneliness 在真实语境中的情感色彩和语用功能。

建构 按语义关联重组（信任链、冲突链、情感表达链），形成你专属的"Friendship 概念地图"。

定义 不是抄词典定义，而是用新学的语言资源，写出属于你自己的句子和段落。

❓ 本模块为什么重要

你可能发现：高一英语的难点往往不在语法本身，而在"学了一堆词句，却不知道怎么用"。M4 就是要解决这个痛点：

🧠
告别"学完就忘"

记住的不是孤立单词，而是语义网络——看见 upset 就能联想到 ignore、calm down、share feelings 这一整条链。

✏️
为写作打地基

M4 是第一次要求你"收网"——把散落的语言点重新组织成你自己的表达系统。

🎯
训练抽象思维

给抽象概念（友谊、信任、孤独）下定义，是语言学习的"高阶动作"。从能读懂到能定义，英语水平会上一个台阶。

🔄 与前后模块的衔接

M1 激活的感性认知 → 在 M4 转化为理性建构；M2 理解的文本内容 → 在这里成为概念解析的材料；M3 聚焦的语言点 → 在这里被纳入语义网络。

M4 是整个单元的"承重墙"——它的建构质量有多高，M5 的产出就有多顺畅。
  • M1 话题激活 → 打开了 Friendship 话题的大门，激活了已有生活经验
  • M2 文本研读 → 深入理解 Anne's Best Friend 的故事与情感
  • M3 语言聚焦 → 梳理了单元核心语法和句型
  • M4 概念熔炉 ← 你在这里
  • M5 综合运用与产出 → 直接要求你用 M4 建构的概念网络与个人定义完成写作

✅ 预期学习成果

完成本模块后，请你回头检查是否做到以下三点：

  • 能解析：面对 Friendship 话题的陌生文本，能自主提取核心概念词，分析其情感色彩与语境含义，而非仅靠词典中文释义。
  • 能建构：能用思维导图、概念图或其他可视化方式，将 15-20 个核心词汇按语义关联归类，并解释你的分类逻辑。
  • 能定义：能用 3-5 句英文写一段"我眼中的友谊"的个性化定义，包含至少 5 个本单元新学词汇，逻辑连贯、情感真实。

💡 记住：这一模块没有"标准答案"。你的概念网络和你的友谊定义，都是属于你自己的——老师更在意的是你的思考过程，而非结果的整齐划一。
