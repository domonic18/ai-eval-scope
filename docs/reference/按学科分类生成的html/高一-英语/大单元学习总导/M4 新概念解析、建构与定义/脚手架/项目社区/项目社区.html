🎯 项目选择建议

📘 基础一般 · 时间有限
推荐 A1 或 B1（短周期），单任务、低门槛，快速建立成就感。
📗 基础良好 · 愿意合作
推荐 A2 或 B2（中周期），小组合作，产出有展示价值。
📕 能力强 · 追求深度
推荐 A3 或 C3（长周期），研究性/行动性项目，要求综合能力。

💡 核心目标

  • 概念迁移：将 M4 建构的 Friendship 模型应用于真实情境
  • 语言综合运用：听说读写全流程使用英语
  • 深度学习能力：问题界定、信息搜集、数据分析、成果呈现
  • 社会情感学习：反思友谊的现实意义，培养同理心与公民意识

A1 友谊语录解码 · Friendship Quotes Deck

搜集 10-15 条中英文友谊经典语录，分类并撰写英文评论，完成一套语录卡牌。

  1. 搜集（30分钟）：从教材语篇、经典文献或在线资源搜集 10-15 条英文友谊语录，附中文翻译。
  2. 分类（20分钟）：按 Trust / Support / Communication / Forgiveness / Shared Joy 分类，每类 2-3 条。
  3. 评论（30分钟）：为每条语录写 1-2 句英文个人评论。
  4. 呈现（40分钟）：制作实体卡牌或数字版（Canva/Google Slides）。

✅ 评价标准

B1 友谊主题海报 · Friendship Poster

设计一张英文海报，以视觉化方式呈现自己对 Friendship 的定义和理解。

  1. 构思（20分钟）：回顾 M4 建构的 Friendship 模型，选择 1-2 个核心要素。
  2. 设计（60分钟）：设计海报布局——标题、核心定义句、3-5 个要素配图标。
  3. 制作（60分钟）：使用 Canva/PPT/手绘完成，所有文字为英文。
  4. 展示：拍照上传班级论坛或张贴在"Friendship Wall"上。

✅ 评价标准

C1 友谊日行动计划 · Friendship Day Plan

策划一个班级层面的 Friendship Day 活动方案，用英文撰写完整计划书。

  1. 调研（30分钟）：采访 3-5 位同学，了解他们认为最能增进友谊的活动。
  2. 策划（40分钟）：设计 45 分钟活动方案，含名称、流程（3-4 环节）、材料、分工。
  3. 撰写（45分钟）：英文计划书（200-300 词），含 Background, Objectives, Activity Design 等。
  4. 展示（3分钟）：英文展示方案，全班投票选出最佳方案并执行。

A2 校园友谊微调查 · Friendship Survey

开展关于"中学生友谊观"的小规模问卷调查，收集 30-50 份问卷，用英文撰写调查报告。

  1. 问卷设计（1-2天）：8-10 题英文问卷（5-6 选择题 + 2-3 开放题）
  2. 数据收集（3-6天）：发放问卷，目标 30-50 份有效问卷
  3. 数据分析（7-8天）：计算百分比，提取开放题关键词并归类
  4. 报告撰写（9-10天）：英文调查报告（400-500 词），含 Introduction, Methodology, Findings 等，≥ 3 个图表
  5. 成果发布（11-14天）：在班级或英语角展示

B2 友谊之声广播剧 · Friendship Podcast

编写并录制一段 5-8 分钟的英文广播剧，包含旁白和角色对话。

  1. 剧本创作（1-4天）：小组合作编写英文剧本（800-1000 词）
  2. 排练（5-7天）：分配角色，注意语音语调和情感表达
  3. 录制制作（8-10天）：使用手机/Audacity 录制，添加背景音乐和音效
  4. 上线（11-14天）：上传至班级平台，写英文作品简介

C2 跨班笔友计划 · Pen Pal Project

与另一个班级建立笔友关系，完成 4 轮英文书信往来 + 1 篇反思报告。

  1. 配对（1-2天）：学生一对一配对，交换电子邮箱
  2. 第一封信（第3天）：自我介绍信（80-120 词）
  3. 第二封信（第5天）：讨论"What makes a good friend"（100-150 词）
  4. 第三封信（第7天）：讨论友谊中的"两难情境"（100-150 词）
  5. 第四封信（第9天）：笔友友谊总结信（120-180 词）
  6. 反思报告（10-14天）：英文反思报告（200-300 词）

A3 友谊话语语料库研究 · Friendship Discourse Corpus

建立一个小型"Friendship Discourse Corpus"，收集 20-30 篇友谊主题文本，进行语言特征分析。

  1. 语料搜集（第1周）：20-30 篇英文友谊主题文本（歌曲歌词、电影台词、演讲稿等）
  2. 关键词提取（第2周）：使用 AntConc 提取频率最高的 20 个 friendship 相关关键词
  3. 话语模式分析（第3周）：分析高频搭配、隐喻用法、情感词汇极性
  4. 报告撰写（第4周）：英文研究报告（800-1000 词），含 Background, Findings, Discussion 等
  5. 成果展示：A1 海报 + 5 分钟英文演讲

B3 友谊主题英文杂志 · Friendship Spectrum

小组创办一本英文电子杂志，包含 6-10 篇原创文章，涵盖多种体裁。

  1. 编辑部组建（第1周）：分配角色（主编/栏目编辑/美术编辑/校对），确定栏目设置
  2. 内容创作（第2-3周）：Feature Story / Interview / Fiction / Poetry / Art Gallery / Reader's Voice
  3. 编辑排版（第3周）：使用 Canva / Issuu 统一排版
  4. 出版推广（第4周）：发布电子杂志，举办"杂志发布派对"

C3 校园友善大使 · Kindness Initiative

设计并实施一个校园友善倡议行动，完成方案、实施、评估全流程。

  1. 问题诊断与方案设计（第1周）：调查校园不友善现象，设计 Kindness Initiative 方案
  2. 行动实施（第2-3周）：执行 4-6 项具体行动（Kindness Challenge / Friendship Bench 等）
  3. 效果评估（第4周）：前后对比调查，收集反馈，撰写评估报告（500-700 词）
  4. 成果展示：展示板 + 5-8 分钟英文展示 + 可复制工具包
