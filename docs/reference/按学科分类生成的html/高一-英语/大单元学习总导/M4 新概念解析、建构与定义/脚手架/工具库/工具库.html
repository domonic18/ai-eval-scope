基础 基础工具层 · 入门级

适合全体学生，用于建立友谊相关的语义网络和概念边界

🧠 概念图 Concept Map

可视化 · 词汇网络

将"友谊"核心词汇以节点-连线方式可视化，激活已有词汇网络。

步骤：中心写 "Friendship" → 一级分支：trust, loyalty, communication, shared interests, support → 每个分支写出 ≥2 个关联词或例句
示例： Friendship → Support → "encourage each other" / "be there in difficult times"

⭕ Venn 图 Venn Diagram

对比 · 概念辨析

对比 "Friend" 与 "Acquaintance" 等概念，在对比中精准定义"朋友"。

步骤：左侧写 Friend 独有属性 → 右侧写 Acquaintance 独有 → 中间交集写共有属性
示例：交集 = "know each other's name"；Friend 独有 = "share secrets"

📊 SWOT 分析（简易版）

框架 · 课文分析

从 Strengths / Weaknesses / Opportunities / Threats 四维度分析友谊关系。

步骤：选取教材人物（如 Anne 与日记）→ 四象限填英文短句 → 小组讨论
适用：课文深度分析的框架支架

进阶 进阶工具层 · 中级

适合大部分学生，深化语言产出与同伴协作

🤖 AI 对话助手 Chatbot Scaffolding

AI 辅助 · 语言产出

利用 AI 工具充当"英语思维伙伴"，进行半结构化问答，降低写作焦虑。

提示词示例："Help me define 'a true friend' in three sentences." → AI 给出示范 + 提问 → 学生模仿写出自己的定义 → 对比修改
进阶提示："Give me 5 adjectives to describe a loyal friend, with example sentences."

📝 同伴互评清单 Peer Review Checklist

互评 · 批判性思维

为互评"友谊定义段落"提供结构化标准。

示例条目：
□ 定义包含 ≥2 个具体例证
□ 正确使用 ≥3 个单元词汇
□ 有清晰的主题句
□ 逻辑连贯，有过渡词
流程：交换草稿 → 逐项勾选写评语 → 修改提交

☁️ 词云工具 Word Cloud

集体建构 · 可视化

将全班关于 "What is friendship?" 的回答生成可视化词云。

步骤：每人写 3 个关键词 → 收集输入词云生成器（Mentimeter / WordArt）→ 全班观察高频词，讨论为何某些词出现最多
适用：导入或总结环节

高阶 高阶工具层 · 研究级

适合英语能力较强的学生，培养批判研究与元认知能力

🔗 社会网络分析 Social Network Diagram

文本分析 · 批判

用节点图分析文本中的人际关系网络，从结构层面理解友谊的社会属性。

步骤：选取人物 → 绘制节点，连线表示关系强度（虚线=弱，实线=强）→ 英文标注依据 → 撰写 150-200 词分析短评
示例：Anne 社交圈 — 虚线连 Peter，实线连 Kitty（日记本）；分析：战争环境下真实人际关系的断裂

🔍 语料库检索工具 Corpus Query

语料驱动 · 归纳

通过小型语料库检索 "friend*" 的真实用法，从语料驱动角度建构词汇定义。

步骤：输入 "friend" 或 "friendship" → 观察 concordance lines → 归纳常见搭配（adj.+friend, v.+friendship...）→ 据语据归纳定义
适用：单元末研究性学习

📓 反思日志框架 Reflective Journal

元认知 · 深层学习

三栏式日志，回顾学习过程，评估自己对新概念的建构路径。

三栏结构：
① What I learned（学到的内容）
② How I learned it（学习方法）
③ How my understanding changed（理解如何改变）
每栏 80-100 词，对比 M4 开始/结束时对"friendship"定义的差异
