入门 ★ 案例1：个人理解型 — 交朋友行动计划 ▼

情境

你独自在国外短期交流，感到孤独，需要给自己制定一个"交朋友"计划。

建构任务

用英文写一份"两周交朋友行动计划"。

调用原理

原理1（明确问题）+ 原理2（调用经验）

建构步骤

  1. 写下你的具体困难：什么样的情况让你感到孤独？
  2. 回忆过去你成功交朋友的方式，列出可迁移的经验。
  3. 明确目标：两周内你想认识几位新朋友？通过什么场合？
  4. 制定每日/每周行动项，并标注完成标准。

成果表达

行动清单（Action Checklist），可手写或用表格呈现。

自评量规

#检查项
✓问题描述是否具体？
✓是否借鉴了以往经验？
✓目标是否可量化？
✓行动项是否有明确的时间节点？
✓整份清单是否用英文撰写并清晰可读？
入门 ★ 案例2：生活应用型 — 英文鼓励信 ▼

情境

你的好朋友最近情绪低落，你需要用英语给 ta 写一封鼓励信。

建构任务

写一封英文鼓励信。

调用原理

原理4（表达出来）

建构步骤

  1. 回忆朋友最近的状态，确定你想传达的核心情感。
  2. 选择2-3个具体的共同回忆作为支撑，让信更真诚。
  3. 按书信格式（称呼、正文、祝福、署名）组织内容。
  4. 用英文写出初稿，朗读一遍调整语气。
  5. 修改语法和用词，确保自然流畅。

成果表达

书信格式（Letter Format），含称呼、正文、祝福语和署名。

自评量规

#检查项
✓格式是否完整（称呼、正文、结尾、署名）？
✓情感是否真诚且有具体事例支撑？
✓语气是否得当（鼓励而不说教）？
✓语法拼写是否正确？
✓读完后能否让朋友感受到支持？
进阶 ★★ 案例3：校园问题解决型 — 班级团结活动方案 ▼

情境

班级里出现了"小圈子"现象，部分同学被孤立。

建构任务

设计一份"班级团结活动方案"（英文版）。

调用原理

原理1（明确问题）+ 原理3（收集证据）

建构步骤

  1. 明确问题：小圈子现象的具体表现是什么？被孤立同学的感受？
  2. 收集证据：通过观察或简短访谈了解现状。
  3. 确定方案目标：希望达到怎样的团结效果？
  4. 设计2-3个具体活动（如破冰游戏、小组合作任务）。
  5. 写出英文方案，包含活动名称、目的、流程和预期效果。

成果表达

英文活动方案（Activity Proposal），标题 + 背景 + 目标 + 活动详情 + 预期效果。

自评量规

#检查项
✓问题分析是否基于真实观察？
✓方案目标是否具体可衡量？
✓活动设计是否具有可操作性？
✓英文表达是否清晰、结构完整？
✓方案是否考虑了不同同学的参与意愿？
进阶 ★★ 案例4：学科表达型 — 3分钟英文演讲 ▼

情境

要用英文做一个关于 "What Makes a Good Friend" 的3分钟演讲。

建构任务

写出演讲稿并录制。

调用原理

原理4（表达出来）+ 原理5（反思）

建构步骤

  1. 确定核心论点：你认为"好朋友"最重要的品质是什么？
  2. 搭建结构：引言 — 2-3个论点 — 结尾总结。
  3. 为每个论点配一个具体例子或个人经历。
  4. 写出完整演讲稿，控制字数（约300词）。
  5. 练习朗读、计时、调整节奏后录制视频。
  6. 回看录制，反思语言和肢体表达并改进。

成果表达

演讲稿（Script）+ 录制视频（Video Recording）。

自评量规

#检查项
✓论点是否明确且有个人见解？
✓结构是否完整（引言-正文-结尾）？
✓是否使用了具体例子支撑论点？
✓演讲时长是否控制在3分钟左右？
✓是否完成了反思并记录改进点？

情境

建构任务

调用原理

原理3（收集证据）

建构步骤

✓信息层级是否清晰易懂？
✓英文表达是否准确简洁？
✓视觉设计是否吸引人且信息不冗余？
挑战 ★★★★ 案例6：社会/未来情境型 — 线上友谊英文短评 ▼

情境

随着社交媒体发展，"线上友谊"算不算真正的友谊？写一篇英文短评。

建构任务

写一篇150词的英文观点短文。

调用原理

原理1（明确问题）+ 原理3（收集证据）

建构步骤

  1. 明确你要论证的立场：线上友谊是 / 不是真正的友谊？
  2. 收集2-3个论据支持立场（个人经历、社会现象、研究数据）。
  3. 搭建结构：立场陈述 → 论据1 → 论据2 → 让步/反驳 → 结论。
  4. 用英文写150词短文，用词精准，逻辑连贯。
  5. 检查逻辑链条是否完整，是否有潜在的漏洞。

成果表达

150词英文观点短文（Opinion Piece）。

自评量规

#检查项
✓立场是否明确且一致？
✓论据是否有说服力（有具体支撑）？
✓结构是否清晰（立场-论据-结论）？
✓字数是否控制在150词左右？
✓是否考虑了反方观点（让步/反驳）？
挑战 ★★★★★ 案例7：综合创新型 — Buddy System 英文提案 ▼

情境

为学校设计一个 "Buddy System（伙伴系统）"，帮助新生适应。

建构任务

撰写一份完整的英文提案（含目标、方案、预算、评估）。

调用原理

全部5条原理：原理1（明确问题）+ 原理2（调用经验）+ 原理3（收集证据）+ 原理4（表达出来）+ 原理5（反思）

建构步骤

  1. 明确问题：新生适应的核心困难是什么？调查或假设2-3个痛点。
  2. 调用经验：回想自己或他人适应新环境的有效经验。
  3. 收集证据：借鉴其他学校已有 Buddy System 的做法。
  4. 设计完整方案：目标、匹配机制、活动安排、培训、时间线。
  5. 制定预算和评估方式（如何衡量成功）。
  6. 用英文撰写正式提案，结构完整，数据可靠。

成果表达

英文提案报告（Proposal Report）—— 标题页 + 摘要 + 正文（背景、目标、方案、预算、评估、时间线）。

自评量规

#检查项
✓问题分析是否基于真实需求？
✓方案是否完整（目标+机制+活动+培训）？
✓预算是否合理且有依据？
✓评估方法是否可操作？
✓英文表达是否正式、专业、结构完整？
