📖 名著部分

《尼各马可伦理学》 《人性的弱点》 《朋友之间》

📜 尼各马可伦理学 亚里士多德中国经典

核心问题：什么是"好"的生活？友谊在其中扮演什么角色？

核心观点：最高形式的友谊是基于"美德"的友谊——两个好人因为彼此的优秀品质而成为朋友。

关键章节：第八卷（友谊的种类）→ 第九卷（友谊与幸福）

适读建议：基础 → 导读版 | 进阶 → 原文全本

获取路径：纸质书（商务印书馆版）· 电子书（微信读书）

🤝 人性的弱点 Dale Carnegie外国经典

核心问题：如何赢得朋友并影响他人？

核心观点：真诚的关心和倾听是建立关系的核心。

关键章节："真诚地关心他人"——这是整本书最有力量的一章。

适读建议：基础 → 中文版 | 挑战 → 英文原版

获取路径：纸质书 · 电子书 · 有声书（喜马拉雅）

🌐 朋友之间 Friendship: A History跨学科

核心问题：友谊的概念在历史上是如何变化的？

核心观点：友谊的定义随着社会和科技不断演变——从古希腊到数字时代。

内容脉络：古希腊友谊 → 中世纪友谊 → 现代友谊 → 数字时代友谊

适读建议：进阶（建议前两本读完后挑战）

获取路径：图书馆或线上书店
