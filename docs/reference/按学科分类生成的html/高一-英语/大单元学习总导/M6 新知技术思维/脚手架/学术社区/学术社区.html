🎯 议题 1：线上友谊是真正的友谊吗？

分组 支持方 VS 反对方

引导问题

  • 线上友谊缺少什么？
  • 线上友谊又多了什么？
  • 如果用亚里士多德的"三种友谊"来分析，线上友谊属于哪一种？

🎯 议题 2：如何在新环境中交朋友？

形式 小组分享 + 互相补充

引导问题

  • 如果你到了一个陌生的环境（新学校、国外），你会怎么做？
  • 你的经验中什么方法最有效？
  • Rogers 的"无条件积极关注"能帮你吗？

🎯 议题 3：教材中的友谊 vs 你的经历

形式 比较讨论

引导问题

  • 教材中的友谊故事和你的经历有什么异同？
  • 哪些情节让你产生共鸣？哪些让你觉得不真实？
  • 文化差异如何影响我们对友谊的理解？

👥 角色分工

在小组讨论中，尝试以下角色——每轮交换一次：

📝记录员记录要点和不同观点
🎙️发言人汇报小组观点
❓质疑者提出不同看法
🔗总结者归纳整理成果

📤 输出机制

  • 📋 小组成果 — 每个议题讨论后，形成一份英文讨论纪要
  • 📢 分享展示 — 将成果在班级群或学习平台上展示
  • 🔄 互评反馈 — 对其他小组的成果进行点评

🔗 连接与拓展

🏠 家校支持：采访父母的交友经验，分享他们的故事。把讨论成果带回家与父母交流。
🧑‍🏫 专家链接：推荐观看心理学家的相关讲座视频，拓展对友谊的学术理解。
🏫 机构推荐：了解学校的心理健康辅导中心资源——学会求助也是一种能力。
