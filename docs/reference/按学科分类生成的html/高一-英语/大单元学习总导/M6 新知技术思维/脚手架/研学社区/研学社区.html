🌱 基础层

转学生的故事

小明从农村转到城市学校，一个朋友也没有。

核心问题：他该如何融入新环境？
决策路径：A 等待主动接近 / B 主动参加社团 / C 求助老师
结果：方案 B 最有效——参加篮球队找到共同话题

可迁移启示：共同兴趣爱好是交朋友最自然的方式。

分支任务：如果你是小明，你会选哪种方案？为什么？

📜 进阶层

历史上的"友谊书信"

1920年代，两位从未见面的作家通过书信建立了30年友谊。

核心问题：没有见过面的友谊是真实的吗？
结果：书信集成为研究那个时代的珍贵资料

可迁移启示：真诚沟通可以跨越时空建立深度连接。

分支任务：你觉得今天还有"笔友"这种友谊形式吗？

⚙️ 挑战层

社交平台的友谊算法

某平台调整推荐算法后，用户发现"朋友的动态越来越少"。

核心问题：算法在重新定义"友谊"——谁该出现在你的社交圈？
决策路径：平台选择用商业价值而非用户真实关系来推荐
结果：用户社交圈变窄，平台收入增加

可迁移启示：技术工具可能改变我们对"朋友"的定义，需要保持批判意识。

分支任务：你认为社交平台应该如何设计"友谊"功能？
