📱 案例1：生活情境 — 微信聊天误解

📄 案例材料：一段英文聊天记录：

A: "Hey, wanna hang out this Saturday?"
B: "Sorry, I'm busy."
A: "You're always busy lately..."
B: "What's that supposed to mean?"
A: "Nothing. Forget it."

🎯 解析任务：分析误解产生的原因，并提出改进的沟通方式

🧰 建议原理：原理3：因果链分析

🛤️ 分析路径：

识别情境 → 定位问题 → 拆解对话 → 发现矛盾 → 提出建议

📋 分析记录表

说话人话语可能的意图对方理解误会原因
A"You're always busy lately..."感到被冷落被指责用"always"过度概括
B"What's that supposed to mean?"感到被攻击（仍在防御）语气变得尖锐
A"Nothing. Forget it."回避冲突问题被搁置沟通彻底关闭

💡 参考思路：从用词、语气、文化差异等角度思考

🏫 案例2：校园情境 — 同学争吵

📄 案例材料：两名同学关于小组作业分工的英文对话冲突

🎯 解析任务：分析冲突的本质，评价双方的观点，提出解决方案

🧰 建议原理：原理2：分类思维 原理4：证据说话

🛤️ 分析路径：拆解争论点 → 分类各方的核心诉求 → 找出共同目标 → 基于事实提出方案

💡 思考：双方的诉求分别是什么？有没有双赢方案？

👩‍👧 案例3：家庭情境 — 母女对话

📄 案例材料：一段英文对话——青春期女儿和母亲关于"交朋友"的分歧

🎯 解析任务：分析两代人对友谊的理解差异

🧰 建议原理：原理1：先整体后局部

🛤️ 分析路径：整体把握双方立场 → 分别听清楚双方的核心观点 → 对比差异 → 分析差异背后的观念原因

💡 考虑：母亲的担忧是什么？女儿的需求是什么？能否互相理解？

📖 案例4：学科核心 — Anne的日记

📄 案例材料：Unit 1 Friendship 中的核心课文——Anne Frank 的日记选段

🎯 解析任务：分析 Anne 在特殊时期对友谊的理解和渴望

🧰 建议原理：全部4条原理

🛤️ 分析路径：整体阅读（理1）→ 分类她的情感需求（理2）→ 因果分析处境与渴望（理3）→ 引用原文佐证（理4）

💡 关键：Anne 在困境中对"真正友谊"的理解有什么特别之处？

📊 案例5：跨学科 — 友谊的科学数据

📄 案例材料：英文信息图——"Friendship and Health: The Scientific Evidence"

🎯 解析任务：分析数据之间的关系，评价结论的说服力

🧰 建议原理：原理4：用证据说话

🛤️ 分析路径：理解数据来源 → 分析变量关系 → 评估统计方法 → 判断结论可信度

💡 思考：相关≠因果。数据能证明"友谊带来健康"还是只是"有关联"？

🌐 案例6：社会议题 — 社交媒体与友谊

📄 案例材料：两段英文短评——
观点A：「Social media destroys real friendship. People chat online but never truly connect.」
观点B：「Social media enriches friendship. It helps us stay in touch with friends far away.」

🎯 解析任务：比较分析两种观点，给出你的判断

🧰 建议原理：原理2：分类思维

🛤️ 分析路径：分别理解两种逻辑 → 找出各自的证据类型 → 分析各自的盲点 → 形成自己的立场

💡 可以问：双方说的"friendship"是同一个概念吗？

🧠 案例7：综合迁移 — 复杂友谊关系

📄 案例材料：一段英文故事——三个人之间的复杂友谊关系

🎯 解析任务：综合分析关系网络，解释各方的动机和行为，预测可能的发展

🧰 建议原理：全部4条原理

🛤️ 分析路径：整体把握关系格局（理1）→ 分类各方诉求（理2）→ 分析因果链条（理3）→ 用原文证据说话（理4）

💡 这是对你的综合考验——把4条原理都用上吧！
