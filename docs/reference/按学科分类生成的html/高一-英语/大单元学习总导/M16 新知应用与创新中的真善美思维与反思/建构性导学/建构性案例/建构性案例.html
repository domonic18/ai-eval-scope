🏗️ 建构性七阶适配案例

M16 新知应用与创新中的真善美思维与反思 · 英语 · 必修1 · 7阶递进

运用建构性原理，亲自动手建构不同的友谊主题成果——从个人理解到社会创新，逐步升级

快速定位：
第一阶 个人理解型——"我的友谊定义"
▾
📋 情境

你被邀请为学校英语广播站撰写一段关于 "What friendship means to me" 的短文。

🎯 建构任务

写一篇 150词 的英文短稿，表达你对友谊的个人理解。

🔧 调用原理

#情境卷入法#句型锚定法

📝 建构步骤
  1. 用情境卷入法回想一个让你感受到友谊真谛的瞬间
  2. 用句型锚定法找到表达情感的句式（"Friendship means...", "For me, a true friend is..."）
  3. 写出核心句子作为段落主题句
  4. 围绕主题句展开，加入具体细节或小故事
  5. 调整语气，确保适合广播朗读——口语化、有节奏
📊 自评量规
标准自评
内容表达了个人对友谊的真实理解✓ / ✗
使用了至少2个友谊相关句型✓ / ✗
结构清晰（开头—主体—结尾）✓ / ✗
语言适合广播朗读✓ / ✗
字数控制在140~160词✓ / ✗
🏆 成果形式

英文广播稿短稿（可录制成音频分享）

⭐ 难度：★☆☆☆☆
第二阶 生活应用型——"朋友冲突解决指南"
▾
📋 情境

你发现好朋友最近和你疏远了，你想用英语写一封邮件主动沟通。

🎯 建构任务

写一封英语邮件，真诚表达你的感受并邀请对方坦诚交流。

🔧 调用原理

#读者意识法#结构先行法

📝 建构步骤
  1. 用结构先行法确定邮件框架：寒暄 → 表达关心 → 陈述感受 → 邀请沟通
  2. 用读者意识法考虑收件人的感受——语气要温和，避免指责
  3. 写出初稿，注意使用"I feel..."而非"You always..."句式
  4. 从读者角度重读一遍，调整措辞
💡 脚手架提示

可用句式：

  • "I\'ve noticed that we haven\'t been as close lately..."
  • "I really value our friendship, and I miss how things used to be."
  • "Would you be open to having a honest conversation?"
⭐ 难度：★★☆☆☆
第三阶 校园问题解决型——"友谊支持小组方案"
▾
📋 情境

学校里有同学因为社交困难感到孤独，你所在的英语社打算发起一个 Peer Friendship Support 项目。

🎯 建构任务

设计一个英语宣传海报（含标题、标语、活动介绍）。

🔧 调用原理

#读者意识法#结构先行法

📝 建构步骤
  1. 确定海报的受众（感到孤独的同学）——用读者意识法思考他们需要什么
  2. 设计吸引眼球的标题（如 "You Are Not Alone — Join Peer Friendship Support"）
  3. 写出标语：一句温暖有力的话
  4. 列出活动内容（2~3项），语言简洁有感染力
🏆 成果形式

英语海报（可用 A4/A3 纸设计，或使用数字工具制作）

⭐ 难度：★★★☆☆
第四阶 学科表达型——"人物故事写作"
▾
📋 情境

教材中提到了 Anne 和她的日记——关于在困境中的友谊。请你以 "I want to thank a friend" 为题写作。

🎯 建构任务

写一篇 200词 左右的英语短文，讲述一个真实或虚构的友谊故事。

🔧 调用原理

#情境卷入法#反思迭代法

📝 建构步骤
  1. 选定一个让你感动的友谊瞬间（真实的或想象的）
  2. 用情境卷入法让自己进入故事氛围
  3. 按叙事结构写：背景 → 冲突 → 转折 → 感悟
  4. 初稿完成后用反思迭代法从"真善美"三角度修改
⭐ 难度：★★★☆☆
第五阶 跨学科整合型——"友谊心理学英文报告"
▾
📋 情境

结合心理学中关于"维持友谊的五个关键因素"的研究，做一份英语简报。

🎯 建构任务

阅读一段简单的英语心理学材料，提取要点，写成一份 100词摘要 + 个人评论。

🔧 调用原理

#结构先行法#反思迭代法

📝 建构步骤
  1. 阅读材料，标记5个关键因素
  2. 用结构先行法组织摘要：总起句 → 5个因素 → 简要总结
  3. 写出个人评论：哪个因素最让你惊讶？为什么？
  4. 回看检查：摘要是否准确？评论是否有自己的思考？
💡 脚手架提示

关键因素举例：Positivity（积极互动）, Support（支持）, Openness（坦诚）, Shared Interests（共同兴趣）, Conflict Resolution（冲突解决）

⭐ 难度：★★★★☆
第六阶 社会/未来情境型——"跨文化友谊指南"
▾
📋 情境

你有一个来自英语国家的笔友，你们在交流中发现了文化差异对友谊的影响。

🎯 建构任务

写一份 "A Guide to Cross-cultural Friendship" 英语指南（5条建议）。

🔧 调用原理

#情境卷入法#读者意识法#反思迭代法

📝 建构步骤
  1. 回想你与笔友交流中遇到的文化差异
  2. 思考不同文化对友谊的理解有何不同（隐私观念、表达方式等）
  3. 用读者意识法：如果你的笔友读到这份指南，会觉得被理解吗？
  4. 写出5条建议，每一条从"问题→建议→例子"三部分构成
  5. 用反思迭代法从跨文化视角回看修改
⭐ 难度：★★★★☆
第七阶 综合创新型——"友谊主题创意作品"
▾
📋 情境

学校英语文化节征集友谊主题创意作品（诗歌、短剧、微电影剧本均可）。

🎯 建构任务

自由选择一种体裁，创作一件完整的英语原创作品。

🔧 调用原理

#全系列原理综合运用

📝 建构步骤
  1. 选体裁：诗歌/短剧/微电影剧本——选你最擅长的
  2. 定主题：围绕友谊的一个具体侧面（如宽恕、陪伴、成长）
  3. 用情境卷入法：让自己进入创作情境，感受要表达的情感
  4. 用结构先行法：搭建作品框架（诗歌：意象递进；短剧：起承转合）
  5. 用句型锚定法：找到适合该体裁的英语表达方式
  6. 用读者意识法：考虑评委和观众的感受
  7. 反复迭代：用反思迭代法修改至少2轮
🏆 成果形式

完整的英语原创作品（可提交参评学校英语文化节）

⭐ 难度：★★★★★
7阶递进 · 从个人理解到综合创新 · 运用建构性原理，产出真实成果
