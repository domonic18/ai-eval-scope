🧠 思维导图模板（Friendship Web）

用途：梳理关于友谊的想法和词汇，为写作做准备

使用步骤：

  1. 中心写 "Friendship"
  2. 第一圈分支：Qualities Activities Challenges Feelings
  3. 每个分支再扩展 3—5 个英语词汇或短语
示例：Qualities → helpful, honest, reliable, caring, supportive

✉️ 写作脚手架模板（书信结构）

用途：写英文友谊信件时确保结构完整

书信模板 开头问候：Dear [Name], / Hi [Name], 正文： 第一段 → 寒暄 + 写作目的 第二段 → 主要内容和感受 第三段 → 期望和邀请回复 结尾：Best wishes, / Yours sincerely, / Love, + 签名

🤖 AI 协作写作助手

用途：辅助英语写作，润色表达

  1. 用中文写出你的想法
  2. 请 AI 帮你翻译成英语初稿
  3. 自己修改初稿，加入个人风格
  4. 请 AI 检查语法和表达
示例提示词：
"Please help me check the grammar and suggest better ways to express my feelings about friendship in this paragraph."

推荐工具：ChatGPT / Kimi / DeepSeek（用英语提问效果更好）
