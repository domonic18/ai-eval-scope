M16 新知应用与创新中的真善美思维与反思 · Unit 1 Friendship（人教版 必修1）

📖 基础层：再现与理解

任务A：句型仿写

  1. 用 I think friendship is… because… 句式写 3 个句子
  2. 用 A true friend should be… 句式写 3 个描述短语

✅ 评价标准：词汇准确、语法正确、每个句子完整通顺

示例：
I think friendship is a warm light because it makes me feel less alone.
I think friendship is a strong bridge because it connects different hearts.
A true friend should be honest and trustworthy.
A true friend should be supportive and understanding.
