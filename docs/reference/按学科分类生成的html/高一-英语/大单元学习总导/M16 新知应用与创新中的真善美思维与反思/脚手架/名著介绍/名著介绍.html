📖 名著推荐

① 中国著作：《论语》中关于"友"的篇章（中英双语版）

📌 适读：基础层级

核心问题：孔子如何教导人们交友的原则？

核心观点："益者三友，损者三友。友直，友谅，友多闻，益矣。"

关键导读：对比中英文对"友谊"定义的差异，尝试用英语解释"益友"的含义。

📍 获取：程颐英译 The Analects

② 外国经典：Charlotte's Web by E. B. White

📌 适读：基础层级（蓝思 680L）

核心问题：一只蜘蛛和一头猪之间能建立真正的友谊吗？

核心观点：用最简单的语言讲了最深刻的友谊——关于承诺、牺牲和陪伴。

关键导读：Charlotte 对小猪 Wilbur 说 "I'll be your friend" 的那一段，是英语儿童文学中最动人的友谊告白。

📍 获取：原版纸质书 / 在线免费电子书

"Why did you do all this for me?" asked Wilbur. "I don't deserve it. I've never done anything for you."
"You have been my friend," replied Charlotte. "That in itself is a tremendous thing."

✨ 翻译挑战：将上面这段话翻译成中文，体会 friendship 的语言力量。

③ 跨学科/当代作品：The Friendship Factor by Alan Loy McGinnis

📌 适读：进阶 / 挑战层级

核心问题：友谊可以被"经营"和"培养"吗？

核心观点：从心理学角度分析了友谊的 25 条原则。

关键导读：关于"如何做一个好的倾听者"的章节特别有实操价值。

📍 获取：中英双语版 / 有声书
