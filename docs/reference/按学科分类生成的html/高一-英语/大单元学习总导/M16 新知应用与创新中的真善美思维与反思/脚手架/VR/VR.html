🌍 观察任务

请通过VR场景或替代素材，"走进"以下友谊相关场景并观察：

🏫校园社交场景
观察不同文化背景下青少年如何初次见面、互相介绍、发出交友邀请。注意英语国家学生打招呼时的肢体语言和用语差异。
✉️跨国笔友交流场景
观察英语国家学生之间的非正式对话细节——他们聊什么话题？如何表达兴趣和共鸣？如何自然地切换话题？
🤝冲突调解场景
观察朋友之间发生误会时的肢体语言和表情变化。注意他们用了哪些英语表达来道歉、解释和和解。

🎮 操作任务

✏️ 任务一：自我介绍与交友对话

在VR场景中用英语完成一次完整的自我介绍和交友对话，包含：

  • 问候（Hi / Hello / Hey）
  • 自我介绍（I'm... / My name is...）
  • 发出邀请（Would you like to... / Do you want to...）
  • 结束对话（Nice meeting you / See you around）

✏️ 任务二：捕捉表达时刻

在场景中留意至少3个你能用英语描述或回应的时刻，记录下来。
