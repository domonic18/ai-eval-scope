名言是"思想触发器"，名著是"系统思想载体"。以下名著从不同维度探讨友谊，帮助你构建认知深度与思想格局。

💫 名言部分（思想触发器）

⚡ 冲突型
"A friend to all is a friend to none."
— Aristotle

你同意吗？ 朋友多 ≠ 友谊深。这对你有什么启示？

💡 启发型
"The only way to have a friend is to be one."
— Ralph Waldo Emerson

这句话反过来成立吗？ 你想拥有什么样的朋友，你先成为那样的人。

🔍 批判型
"Social media 'friends' are not the same as real friends."
— Sherry Turkle

你微信/QQ上有多少"好友"？ 其中能称得上真正朋友的又有多少？

🇨🇳 中国著作：《论语》"友"篇节选 ▼
核心问题"益者三友，损者三友"——什么样的朋友值得交？
目录结构学而篇、为政篇、里仁篇中关于"友"的论述
核心观点孔子提出"友直，友谅，友多闻，益矣"——正直的朋友、诚信的朋友、博学的朋友，是有益的。
益者三友，损者三友。友直，友谅，友多闻，益矣。友便辟，友善柔，友便佞，损矣。

🌍 外国经典：《The Four Loves》— C. S. Lewis ▼
核心问题人类四种爱（亲情、友情、爱情、仁爱）分别是什么？友谊在其中扮演什么角色？
目录结构Introduction → Likings → Affection → Friendship → Eros → Charity
核心观点友谊是最不"必要"的爱——即使没有朋友人也能活，但真正的友谊能让生命升华。
"Friendship is the least instinctive, the least organic, of the loves; it has no biological necessity. Yet it is by it that we transcend our own nature."
