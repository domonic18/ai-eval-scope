📝 短周期项目（1-3天）

快速上手，适合零散时间完成

项目A：友谊名言手账 1-3天

搜集关于友谊的英语名言，制作一张"名言手账"页

阶段任务
Day1: 搜索10句名言，选最喜欢的5句
Day2: 每句写英文注释
Day3: 排版设计
成果形式
A4手账页（手写或电子设计）
评价标准：5句名言(5分) + 5句注释(5分) + 排版美观(2分)
🔗 BrainyQuote.com → 搜索 "friendship quotes"
▼ 展开详情

💡 操作建议：可以先在笔记本上草稿，再用Canva或手绘完成最终版。每句名言写下你为什么选它——用英语表达个人感受本身就是练习！

项目B：班级友谊小调查 1-3天

用英语设计一份简短的班级友谊调查问卷，分析结果

阶段任务
Day1: 设计5个英文问题
Day2: 收集10份以上回答
Day3: 写英文总结
成果形式
调查表(英文) + 总结报告(80词+)
评价标准：问题设计合理(4分) + 数据真实(2分) + 总结有洞察(4分)
💡 问题示例："What quality do you value most in a friend?"
▼ 展开详情

📊 分析技巧：把答案分类统计，比如"最看重的品质"中选"honest"的有多少人。在总结中写一个有趣的发现！

📋 中周期项目（1-2周）

深度探索，适合有整块时间的周末

项目C：跨文化友谊对比研究 1-2周

研究中西方友谊文化的异同，制作英语对比报告

阶段任务
第1周：搜集资料（教材Unit 1 + 英文文章）
第2周：整理对比表，写80-100词报告
成果形式
中西友谊对比表 + 英语短文报告
评价标准：≥3个对比维度(6分) + 具体例证(4分) + 语言表达(5分)
🔗 搜索 "friendship in Chinese culture vs Western culture"
▼ 展开详情

🌏 对比维度建议：①交友方式(线上vs线下) ②友谊深度(广度vs深度) ③表达方式(含蓄vs直接)。用教材中的课文作为中文维度的参考！

项目D：一封真实的英文笔友信 1-2周

把你在本模块写的交友信，真正寄出去

阶段任务
第1周：修改完善交友信，请老师/同学校对
第2周：通过国际交流项目或PenPal World寄出
成果形式
最终版英文交友信 + 发送确认
评价标准：格式正确(3分) + 内容丰富(4分) + 语言准确(3分) + 已发出(加分项)
🔗 PenPal World (www.penpalworld.com) 或学校国际项目
▼ 展开详情

✉️ 寄出前检查：①开头问候和自我介绍 ②为什么想交笔友 ③你的兴趣爱好 ④对笔友的期待 ⑤礼貌结尾。让老师帮忙润色一次！

🎯 长周期项目（1月+）

大型挑战，需要协作和持续投入

项目E：班级友谊纪念册 1月+

全班合作制作一本英文版"班级友谊纪念册"

阶段任务
第1周：制定计划分工
第2周：每人写一页"最难忘的同学"
第3周：编辑排版
第4周：打印或线上发布
成果形式
≥20页英文班级友谊纪念册(PDF或打印版)
评价标准：全员参与(10分) + 英语质量(10分) + 排版美观(5分) + 创意(5分)
🔗 Canva.com(排版) / Google Docs(协作)
▼ 展开详情

📖 分工建议：设立编辑组(校对英语)、设计组(排版配图)、素材组(收集照片和故事)。每人写一段100词左右的英文短文，配上照片！

项目F：Friendship Day校园英语活动 1月+

策划并举办一次"Friendship Day"校园英语活动

阶段任务
第1周：写英文策划方案
第2周：制作英文海报和邀请函
第3周：准备活动材料
第4周：举办活动并写总结
成果形式
策划方案 + 宣传材料 + 活动照片 + 总结(全英文)
评价标准：方案可行性(5分) + 宣传效果(5分) + 执行(5分) + 总结反思(5分)
🔗 PosterMyWall.com(海报) / 学校广播站和公告栏
▼ 展开详情

🎉 活动点子：友谊手环交换、英文友谊卡制作、"Find Someone Who"英语互动游戏、友谊主题英语角。别忘了拍照记录！
