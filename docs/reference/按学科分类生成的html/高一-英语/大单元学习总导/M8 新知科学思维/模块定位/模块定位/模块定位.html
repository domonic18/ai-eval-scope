📖 本模块学习什么

本模块名为"新知科学思维"，核心不是学习新的英语词汇或语法，而是学习一套科学的思维方法。具体来说，你要学习两种思维：

🔍 解析性思维

遇到一篇英语材料（如关于友谊的文章、信件），如何"拆解"它——提取关键信息、分析结构关系、理解作者意图、形成自己的判断。

🏗️ 建构性思维

面对一个英语表达任务（如写交友信、做自我介绍），如何"建造"它——从零开始策划、组织内容、选择语言、修改完善。

这两种思维合在一起，就是"新知科学思维"——你获取新知识、产出新成果的底层能力。

💡 本模块为什么重要

在之前的模块中，你积累了大量与友谊相关的英语词汇、句式和阅读经验。但零散的知识就像一堆砖头，而"新知科学思维"就是告诉你怎么用这些砖头盖出房子。
