📝 Friendship Quotes Collection

⏱ 1—3 天
🎯
搜集 10 条关于友谊的英文名言，为每条名言配上你的中文理解和英文评论。
  • 1搜索和筛选名言（线上或书籍）
  • 2为每条名言写作英文评论（30—50 词）
  • 3整理成一份精美的 PDF 文档或卡片集
📦 成果形式
10 张"名言卡片"
📊 评价标准
来源可靠 + 个人理解 + 格式统一
🔗 资源： BrainyQuote.com · Goodreads.com

📘 "Advice for New Students" Multimodal Guide

⏱ 1—2 周
🎯
制作一份面向新生的"如何在学校交朋友"多模态指南（英文）。
  • 1调研：采访 3—5 位同学关于"在学校交朋友的建议"
  • 2整理：把建议分类整理成 3—5 个板块
  • 3制作：用英文写成指南，并配上图片或插图
  • 4展示：在班级中展示并收集反馈
📦 成果形式
英文指南（可打印/数字版）
📊 评价标准
内容实用 + 英文准确 + 排版清晰 + 有调研基础
🔗 资源： Canva（设计工具） · ChatGPT（语法校对）

📖 Class Friendship Journal

⏱ 1 月+
🎯
全班合作撰写一份"班级友谊日志"（英文），记录一学期中班级发生的友谊故事和感悟。
  • 📋策划期 — 确定日志格式、轮值制度和内容规范（第 1 周）
  • ✍️执行期 — 每周 2—3 位同学轮值撰写英文日志（第 2—4 周）
  • 📚汇编期 — 全班投票选出最精彩的 10 篇，汇编成电子杂志（第 5 周）
  • 🎉发布期 — 在班级/学校平台发布，邀请读者留言（第 6 周）
📦 成果形式
电子杂志（PDF / ePub）
📊 评价标准
具体事件 + 反思 + 语言通顺 + 风格统一
🔗 资源： Canva（排版设计） · Flipsnack（电子杂志）
