思维导图 Mind Map
整理思路、建立知识点之间的联系
  1. 中心主题画在纸中央
  2. 向四周画出主干分支
  3. 每根分支延伸出次级分支
  4. 用关键词和图标标注
示例：以"Friendship"为中心，分出 Types — Qualities — Challenges — Benefits 四个主干

KWL 表格
明确已知、想知和学到的内容
  1. 画三列表格：K | W | L
  2. K 栏：写出你已经知道的内容
  3. W 栏：写出你想知道的问题
  4. L 栏：学完后写下你新学到的
适用：单元学习开始和结束时使用，对比前后认知变化
单元学习 · 项目式学习

AI 辅助学习
获取即时反馈、进行口语练习、校对语法
  1. 明确你想要解决的问题
  2. 用英文输入你的问题或文本
  3. 分析 AI 的回复，判断是否采纳
  4. 追问或调整，直到满意
