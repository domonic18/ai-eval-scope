🤝 学术社区脚手架

学习共同体：在讨论中建构、在连接中成长。

👥 同伴连接

▶

议题一：友谊是选择还是偶然？

你同意 "Friendship is a choice, not a chance" 这句话吗？用你自己的交友经历说明。

📋 分工建议：2人正方 — 2人反方 — 1人记录总结

▶

议题二：交友的经验与挑战

在你的经验中，最容易交到朋友的方式是什么？最困难的情况是什么？

📋 分工建议：每人分享 2 分钟，共同归纳出 Top 3 方法

🏡 家校连接

👨‍👩‍👧 家长支持 👩‍🏫 教师反馈
家长支持：邀请家长分享他们的"青少年时期交友经验"——家长也曾是青少年，他们的故事可能比你想象的更精彩。可以在家长会上或通过录制视频的方式分享。
教师反馈：教师对学生的建构成果进行书面反馈。使用 "三明治反馈法"—— 肯定优点 → 提出改进建议 → 鼓励继续努力。

🌍 社会连接

🎓 专家邀请 🏛 机构访问
专家邀请：联系学校心理教师或当地大学心理学系的研究生，做一次关于"青少年友谊心理学"的微讲座（20-30 分钟）。提前准备 3-5 个问题，让讲座更有针对性。
机构访问：参观当地社区中心的青少年社交服务项目，了解社会层面如何支持青少年建立健康的社交关系。

📤 输出机制

1
小组展示 — 各小组用英文展示讨论成果（3 分钟口语汇报）。注意语音语调、肢体语言和眼神交流。
2
海报张贴 — 将讨论成果做成英文海报，在教室展示区张贴。使用 Canva 等工具制作会更美观。
3
线上分享 — 将成果上传至班级学习平台，获得同伴互评。互评时注意：赞美一个优点 + 提出一个改进建议。
🌱 每一次真诚的讨论，都是一次同伴学习的机会。今天你在小组中扮演了什么角色？
