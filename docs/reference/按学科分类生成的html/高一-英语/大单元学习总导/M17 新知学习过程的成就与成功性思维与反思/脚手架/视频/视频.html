🎬 视频脚手架

围绕"友谊"主题的三层视频学习任务——从基础理解到批判思辨，循序渐进提升英语能力。

🌱 基础层·解释型 入门 ▼
▶ Friendship 101（YouTube / Bilibili）
📌 观看任务
看完视频后，用英文列出视频中提到的"5 种交朋友的方法"。
❓ 关键问题
  • 视频中认为交朋友的第一条秘诀是什么？
  • 你同意吗？为什么？
⏸
暂停思考点 02:30 — 视频说"smile"是交朋友的第一步时，想一想你自己在认识新朋友时的第一个表情是什么？
✏️ 输出任务
写一段 50 词的英文反思："I ___ agree/disagree with the video because..."
🔍 进阶层·分析型 进阶 ▼
▶ TED Talk "The Secret to Making Friends as an Adult"
📌 观看任务
分析演讲者的论证结构：problem → cause → solution / claim → evidence → conclusion？
❓ 关键问题
  • 演讲者提出的核心观点是什么？
  • 她用了什么证据来支撑？
  • 这个证据可靠吗？为什么？
⏸
暂停思考点 06:15 — 演讲者讲到自己搬新城市无人认识的经历。你有类似的经历吗？是什么？
✏️ 输出任务
用英文写一段分析报告（80—100 词），分析演讲的论证结构。
⚡ 挑战层·辩论型 挑战 ▼
▶ "Is Social Media Ruining Friendship?" BBC Debate
📌 观看任务
记录正方和反方各 3 个核心论点。
❓ 关键问题
  • 正方的核心逻辑是什么？
  • 反方最有力的反驳是什么？
  • 你站在哪一边？为什么？
⏸
暂停思考 12:30 — 在正方说完、反方回答之前，先记录自己的观点，再听反方回答。
✏️ 输出任务
写一段辩论感言（英文，80 词以上）："After watching the debate, I think..."
