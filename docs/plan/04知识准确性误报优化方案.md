# 知识准确性评估器 misconception 误报优化方案

> 状态：短期 S1 已实施（S2/S3 评估后撤销，见 §3）
> 范围：`commonsense.info_accuracy` 评估器的 misconception 检测精度
> 目标：**不漏报真错误 + 不误报正常文本**

## 1. 问题

用户课件评估报 25 处"错误"，实际仅 **1 处算术错误为真**（`12×2+7+3+5+18=100` 应为 57），其余 **24 处全是 misconception 模式匹配误报**：
- "是……也"（古汉语判断句）→ 匹配正常古汉语教学
- "不是……而是……"（并列选择）→ 匹配正常表述
- "理想/信念"、"价值观/信念"、"归纳/演绎" → 匹配正常概念辨析教学

## 2. 误报根因

| # | 根因 | 严重度 | 数据实证 |
|---|------|--------|----------|
| 1 | **CMMLU 选择题干扰项被用作全文子串匹配 pattern** | 致命 | 5958 条 pattern 中 98% 来自 CMMLU 错误选项标记，裸词化后正常教学文本讨论概念即误报 |
| 2 | **超短 pattern 无锚定** | 致命 | 91 个 ≤8 字符裸词（"道德"/"理想"/"神"/"政治"）+ `pattern.search(text)` 全文子串搜索，无上下文窗口 |
| 3 | **severity="error" 一票否决** | 高 | LLM 路径 `passed = combined_score >= threshold and len(rule_errors) == 0`，misconception error 直接致 CPR=0 |
| 4 | **全量加载** | 高 | `subjects=None` 时加载全部 9 学科 5958 条 pattern 无差别匹配 |

**核心矛盾**：CMMLU pattern 的原意是选择题中的"错误选项标记"（如"理想"=将理想误作信念的答案），但被降维为"文档中出现该词就报错"。正常教学文本讨论这些概念是教学本身的内容，而非"常识错误"。

## 3. 优化方案（分 3 期）

### 短期：立即止血（改评估器逻辑，不动 knowledge 数据）

**S1. misconception finding 降级为 warning（核心止血）**

`_check_misconceptions` 内将 effective severity 统一为 `"warning"`，原始 severity 存 `rule_severity` 字段供报告展示。

- misconception 不再进入 `rule_errors`/`errors` → 消除 LLM 路径一票否决
- 误报下降 ~90%
- 漏报风险极低（warning 仍记录 + 送 LLM prompt 的 warnings 变量，LLM 基于上下文二次判断）

**S2. 裸词 pattern 加上下文错误信号校验 — ⚠️ 评估后撤销**

原型：裸词 pattern 需命中位置附近含"错误信号"（不是/混淆/等同…）才报。评估发现会**漏报真错误**（如"祖冲之是唐朝"无否定词但属事实错误），与"不漏报"冲突，故撤销。

**S3. 未指定 subject 时仅加载 `_defaults.yaml` — ⚠️ 评估后撤销**

原型：默认仅加载 _defaults 6 条高质量 pattern。评估发现会**漏报学科特定错误**（如"祖冲之"在 history.yaml 而非 _defaults），与"不漏报"冲突，故撤销。

> **S1 已满足核心目标**：misconception 降 warning → 不致 FAIL（CPR 不再归零）+ 仍检出（不漏报）。误报以 warning 展示但不影响评估结论。减少误报 warning 的进一步优化由中期 M1-M3（pattern 质量分级）承接，可在不漏报的前提下逐步降低 warning 噪音。

### 中期：pattern 质量治理（2-4 周）

| 改动 | 内容 |
|------|------|
| M1 | 加载时 pattern 质量分级（high=带正则锚定+长度≥12 / medium / low / unreliable=裸词≤4），打 `quality` 标签 |
| M2 | 配置 `misconception_error_quality_floor`，仅 ≥floor 的 pattern 可产出 error |
| M3 | CLI 审查脚本，按 source 溯源统计质量分布，输出待修复 pattern 建议（如"理想" → "理想.*(?:等同\|等于\|就是).*信念"） |

### 长期：LLM 二次确认（1-2 月）

| 改动 | 内容 |
|------|------|
| L1 | misconception 命中（含匹配片段+上下文）作为 LLM 验证候选 |
| L2 | LLM 判断 confirmed/rejected/uncertain，仅 confirmed 计 error |
| L3 | LLM 不可用时 misconception 一律降 warning |

## 4. 预期效果

| 阶段 | 误报下降 | 漏报风险 | 工期 |
|------|----------|----------|------|
| 短期 S1+S2+S3 | ~95%（消除一票否决 + 裸词误报 + 跨学科误报） | 极低（warning 仍记录+送 LLM） | 1.5 天 |
| 中期 M1-M3 | 再降 80%（仅高质量 pattern 参与 error） | 可控（high 级覆盖可验证错误） | 2-4 周 |
| 长期 L1-L3 | 趋近零（LLM 语义区分讨论 vs 错误） | ~5%（LLM 判断） | 1-2 月 |
