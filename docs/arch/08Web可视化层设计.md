# Web еҸҜи§ҶеҢ–еұӮи®ҫи®Ў

> жң¬ж–ҮжЎЈиҜҰз»Ҷйҳҗиҝ° Web еҸҜи§ҶеҢ–еұӮзҡ„и®ҫи®ЎпјҢеҢ…жӢ¬жҠҖжңҜйҖүеһӢгҖҒеүҚеҗҺз«Ҝжһ¶жһ„гҖҒйЎөйқўи®ҫи®ЎгҖҒAPI е®ҡд№үгҖҒж•°жҚ®жЁЎеһӢдёҺи…ҫи®Ҝдә‘еҮҪж•°йғЁзҪІж–№жЎҲгҖӮеұһдәҺ [01 ж•ҙдҪ“жһ¶жһ„и®ҫи®Ў](./01ж•ҙдҪ“жһ¶жһ„и®ҫи®Ў.md) В§дәҢдёӯ"Web еҸҜи§ҶеҢ–еұӮ"зҡ„иҜҰз»Ҷеұ•ејҖгҖӮж•°жҚ®з®ЎзҗҶдёҺй…ҚзҪ®и§„иҢғеҸӮи§Ғ [06 ж•°жҚ®з®ЎзҗҶдёҺй…ҚзҪ®и§„иҢғ](./06ж•°жҚ®з®ЎзҗҶдёҺй…ҚзҪ®и§„иҢғ.md)гҖӮ

---

## дёҖгҖҒи®ҫи®Ўзӣ®ж ҮдёҺиҢғеӣҙ

### 1.1 и®ҫи®Ўзӣ®ж Ү

| зӣ®ж Ү | иҜҙжҳҺ |
|------|------|
| **йЎ№зӣ®зә§зңӢжқҝ** | д»ҘйЎ№зӣ®дёәз»ҙеәҰиҒҡеҗҲиҜ„дј°иҝҗиЎҢпјҢеұ•зӨәжңҖж–°зҠ¶жҖҒдёҺеҝ«йҖҹз»ҹи®Ў |
| **и¶ӢеҠҝеҸҜиҝҪжәҜ** | еұ•зӨә DR / CPR / Reward зӯүж ёеҝғжҢҮж ҮйҡҸж—¶й—ҙзҡ„еҸҳеҢ–жӣІзәҝ |
| **жҠҘе‘ҠиҜҰжғ…** | ж”ҜжҢҒжҹҘзңӢеҚ•ж¬ЎиҜ„дј°иҝҗиЎҢзҡ„иҜҰз»ҶжҠҘе‘ҠдёҺд»»еҠЎзә§з»“жһң |
| **йӣ¶й…ҚзҪ®еҗҜеҠЁ** | йҖҡиҝҮ CLI е‘Ҫд»ӨдёҖй”®еҗҜеҠЁпјҢж— йңҖйўқеӨ–й…ҚзҪ® |
| **дә‘з«ҜеҸҜйғЁзҪІ** | йҖӮй…Қи…ҫи®Ҝдә‘еҮҪж•°йғЁзҪІпјҢж”ҜжҢҒеңЁзәҝи®ҝй—® |

### 1.2 иҢғеӣҙ

- **еҢ…еҗ«**пјҡйЎ№зӣ®зңӢжқҝгҖҒи¶ӢеҠҝеӣҫгҖҒиҝҗиЎҢиҜҰжғ…гҖҒд»»еҠЎиҜҰжғ…гҖҒLLM жәҜжәҗеұ•зӨә
- **дёҚеҢ…еҗ«**пјҡи§„еҲҷзј–иҫ‘з•ҢйқўпјҲP2пјүгҖҒдәәе·Ҙд»ІиЈҒз•ҢйқўпјҲP2пјүгҖҒз”ЁжҲ·и®ӨиҜҒпјҲиҝңжңҹпјү

## дәҢгҖҒжҠҖжңҜйҖүеһӢ

### 2.1 жҠҖжңҜж Ҳ

| еұӮзә§ | жҠҖжңҜ | иҜҙжҳҺ |
|------|------|------|
| **еүҚз«Ҝ** | React 18+ | з»„д»¶еҢ– UIпјҢзӨҫеҢәз”ҹжҖҒдё°еҜҢ |
| **UI з»„д»¶еә“** | Ant Design | дјҒдёҡзә§ UI з»„д»¶пјҢиЎЁж ј/еӣҫиЎЁ/иЎЁеҚ•ж”ҜжҢҒеҘҪ |
| **еӣҫиЎЁ** | ECharts | и¶ӢеҠҝжҠҳзәҝеӣҫгҖҒжҹұзҠ¶еӣҫгҖҒд»ӘиЎЁзӣҳ |
| **HTTP е®ўжҲ·з«Ҝ** | Axios | еүҚз«Ҝ API иҜ·жұӮ |
| **еҗҺз«Ҝ** | Express (Node.js) | иҪ»йҮҸгҖҒйҖӮеҗҲ API жңҚеҠЎ |
| **ж•°жҚ®иҜ»еҸ–** | зӣҙжҺҘиҜ»еҸ– workspace JSON ж–Үд»¶ | MVP йҳ¶ж®өж— йңҖж•°жҚ®еә“ |
| **жһ„е»ә** | Vite | еүҚз«Ҝеҝ«йҖҹжһ„е»ә |
| **йғЁзҪІ** | и…ҫи®Ҝдә‘еҮҪж•° (SCF) | Serverless йғЁзҪІпјҢжҢүйңҖи°ғз”Ё |

### 2.2 йҖүеһӢзҗҶз”ұ

- **React + Express**пјҡз”ЁжҲ·жҳҺзЎ®иҰҒжұӮпјҢеӣўйҳҹжңүйЎ№зӣ®з»ҸйӘҢпјҢйҖӮй…Қи…ҫи®Ҝдә‘еҮҪж•°йғЁзҪІ
- **ж–Үд»¶иҜ»еҸ–иҖҢйқһж•°жҚ®еә“**пјҡдёҺзҺ°жңү workspace ж–Үд»¶зі»з»ҹжһ¶жһ„дёҖиҮҙпјҢMVP йҳ¶ж®өйӣ¶йўқеӨ–дҫқиө–
- **ECharts**пјҡдёӯж–Үж–ҮжЎЈе®Ңе–„пјҢеӣҫиЎЁзұ»еһӢдё°еҜҢпјҢж”ҜжҢҒи¶ӢеҠҝеӣҫ/йӣ·иҫҫеӣҫ/д»ӘиЎЁзӣҳзӯү

## дёүгҖҒеүҚз«ҜйЎөйқўи®ҫи®Ў

### 3.1 йЎөйқўжҖ»и§Ҳ

| йЎөйқў | и·Ҝз”ұ | еҠҹиғҪ | ж•°жҚ®жәҗ |
|------|------|------|--------|
| йЎ№зӣ®еҲ—иЎЁ | `/` | еұ•зӨәжүҖжңүйЎ№зӣ®гҖҒжңҖж–°иҝҗиЎҢзҠ¶жҖҒгҖҒеҝ«йҖҹз»ҹи®ЎеҚЎзүҮ | `projects.json` + `runs_index.json` |
| йЎ№зӣ®иҜҰжғ… | `/project/{id}` | и¶ӢеҠҝеӣҫиЎЁгҖҒиҝҗиЎҢеҺҶеҸІеҲ—иЎЁгҖҒйҳҲеҖјзәҝ | `runs_index.json` жҢү project_id иҝҮж»Ө |
| иҝҗиЎҢиҜҰжғ… | `/run/{id}` | жұҮжҖ»еҚЎзүҮгҖҒз»ҙеәҰеҲҶи§ЈгҖҒд»»еҠЎзә§з»“жһңиЎЁ | `reports/summary.json` + `results/` |
| д»»еҠЎиҜҰжғ… | `/run/{id}/task/{task_id}` | зәҰжқҹйҖҗйЎ№з»“жһңгҖҒLLM Judge жәҜжәҗгҖҒиҫ“еҮәйў„и§Ҳ | `results/{task_id}/rule_results.json` + `scores.json` |
| зӣ®еҪ•жЁЎејҸд»»еҠЎиҜҰжғ… | `/run/{id}/task/{task_id}` (еҗҢдёҠ) | зӣ®еҪ•ж ‘еҸҜи§ҶеҢ–гҖҒжЁЎеқ—зә§зҠ¶жҖҒгҖҒж–Үд»¶зә§зғӯеҠӣеӣҫ | `rule_results.json` + `output/_manifest.json` |

> зӣ®еҪ•жЁЎејҸд»»еҠЎиҜҰжғ…дёҺжҷ®йҖҡд»»еҠЎиҜҰжғ…е…ұдә«еҗҢдёҖи·Ҝз”ұпјҢеүҚз«Ҝж №жҚ® `rule_results.json` дёӯжҳҜеҗҰеӯҳеңЁ `module_results` еӯ—ж®өиҮӘеҠЁеҲҮжҚўеұ•зӨәжЁЎејҸгҖӮ

## е…ӯгҖҒж•°жҚ®зҙўеј•дёҺеӯҳеӮЁ

### 6.1 зҙўеј•ж–Үд»¶

зҙўеј•ж–Үд»¶еӯҳеӮЁеңЁ `workspace/index/` зӣ®еҪ•дёӢпјҢз”ұеҗҺз«Ҝ indexer жңҚеҠЎз»ҙжҠӨпјҡ

| ж–Үд»¶ | еҶ…е®№ | жӣҙж–°ж—¶жңә |
|------|------|----------|
| `projects.json` | йЎ№зӣ®жіЁеҶҢеҲ—иЎЁ | `project.yaml` еҸҳжӣҙж—¶ / йҰ–ж¬ЎиҝҗиЎҢж—¶ |
| `runs_index.json` | жүҒе№іеҢ–иҝҗиЎҢж‘ҳиҰҒ | жҜҸж¬Ў eval иҝҗиЎҢе®ҢжҲҗ / жүӢеҠЁ `agent-eval index` |

### 6.2 зҙўеј•жһ„е»әжөҒзЁӢ

```
agent-eval eval е®ҢжҲҗ
       вҶ“
иҜ»еҸ– reports/summary.json
       вҶ“
жҸҗеҸ– DR/CPR/Reward зӯүжҢҮж Ү
       вҶ“
е…іиҒ” project_idпјҲжқҘиҮӘ --project еҸӮж•°жҲ–й»ҳи®Ө "default"пјү
       вҶ“
иҝҪеҠ еҲ° runs_index.json
       вҶ“
жӣҙж–° projects.json зҡ„ latest_run_id е’Ң run_count
```

### 6.3 е…ЁйҮҸйҮҚе»ә

еҪ“зҙўеј•ж–Үд»¶дёўеӨұжҲ–жҚҹеқҸж—¶пјҢйҖҡиҝҮ `agent-eval index` е…ЁйҮҸйҮҚе»әпјҡ

1. жү«жҸҸ `workspace/runs/` дёӢжүҖжңүиҝҗиЎҢзӣ®еҪ•
2. иҜ»еҸ–жҜҸдёӘиҝҗиЎҢзҡ„ `reports/summary.json` е’Ң `run_manifest.json`
3. иҜ»еҸ– `assets/projects/` дёӢжүҖжңү `project.yaml` жһ„е»әйЎ№зӣ®еҲ—иЎЁ
4. е°ҶиҝҗиЎҢж‘ҳиҰҒдёҺйЎ№зӣ®е…іиҒ”пјҲйҖҡиҝҮ run_manifest дёӯзҡ„ project_idпјү
5. еҶҷе…Ҙ `projects.json` е’Ң `runs_index.json`

```javascript
// web/backend/scf_bootstrap.js
const express = require('express');
const path = require('path');
const app = express();

// йқҷжҖҒиө„жәҗ
app.use(express.static(path.join(__dirname, 'public')));

// API и·Ҝз”ұ
const projectsRouter = require('./routes/projects');
const runsRouter = require('./routes/runs');
const reportsRouter = require('./routes/reports');

app.use('/api/projects', projectsRouter);
app.use('/api/runs', runsRouter);
app.use('/api/reports', reportsRouter);

// SPA еӣһйҖҖ
app.get('*', (req, res) => {
  res.sendFile(path.join(__dirname, 'public', 'index.html'));
});

// SCF е…ҘеҸЈ
const serverless = require('serverless-http');
module.exports.main_handler = serverless(app);
```
