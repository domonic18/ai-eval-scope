# 执行引擎设计

> 本文档详细阐述执行引擎的设计，包括 SUT 驱动、结果采集、过程指标统计以及执行侧的数据模型。属于 [01 整体架构设计](./01整体架构设计.md) §二中"执行引擎"的详细展开。

---

## 一、职责边界

执行引擎负责**驱动被测 Agent 运行并采集结果**，不涉及任何评估逻辑：

| 组件 | 职责 |
|------|------|
| **ExecutionEngine** | 加载任务集、调度 SUTDriver、归档输出、统计过程指标 |
| **SUTDriver** | 与被测 Agent 通信（HTTP / CLI / 扩展） |
| **ResultCollector** | 收集输出文件、记录执行轨迹 |
| **MetricRecorder** | 统计耗时、步数、重试次数等过程指标 |
| **TaskSetBuilder** | 任务集构建工具（从零创建 / LLM 半自动生成） |

```python
class Task(BaseModel):
    id: str                              # 任务唯一标识
```

### 2.4 SUTResponse — 被测 Agent 响应

```python
class SUTResponse(BaseModel):
    success: bool
    output_files: list[str] = []         # 输出文件路径
```

```python
class ExecutionTrace(BaseModel):

    error: str | None = None
```

## 三、ExecutionEngine

### 3.1 核心类

```python
class ExecutionEngine:
    def __init__(self, driver: SUTDriver, output_dir: Path):
        self.driver = driver
        self.output_dir = output_dir

    def run(self, task_set: TaskSet) -> list[ExecutionPackage]:
        """批量执行任务集，返回执行包列表。"""
        return [self.run_single(task) for task in task_set.tasks]

    def run_single(self, task: Task) -> ExecutionPackage:
        """执行单个任务，归档输出到 workspace。"""
        trace = ExecutionTrace(started_at=datetime.now().isoformat())
        try:
            response = self.driver.invoke(task)
            trace.finished_at = datetime.now().isoformat()
            trace.response = response.raw_response
            trace.error = response.error
            ...
        except Exception as e:
            trace.error = str(e)
            ...
```

## 四、SUTDriver — 被测 Agent 驱动

### 4.1 抽象接口

```python
class SUTDriver(ABC):
    @abstractmethod
    def invoke(self, task: Task) -> SUTResponse: ...
```

### 4.2 HTTPDriver（默认）

适配 LangGraph / Hermes 等 HTTP 服务。

```python
class HTTPDriver(SUTDriver):
    def __init__(self, base_url: str, headers: dict = None,
                 request_template: dict = None, response_mapping: dict = None,
                 timeout: float = 120, retries: int = 2):
        self.base_url = base_url
        self.headers = headers or {}
        self.request_template = request_template or {}
        self.response_mapping = response_mapping or {}
        self.timeout = timeout
        self.retries = retries

    def invoke(self, task: Task) -> SUTResponse:
        # 1. 渲染请求模板（Jinja2 变量替换）
        body = self._render_template(self.request_template.get("body", {}), task)
        # 2. 发送 HTTP 请求（带超时与重试）
        response = self._send_request(body)
        # 3. 提取输出文件与元数据
        return self._parse_response(response)
```

**配置项**：

| 配置 | 说明 | 示例 |
|------|------|------|
| `base_url` | 被测 Agent 服务地址 | `http://localhost:8000` |
| `headers` | 自定义请求头（鉴权等） | `Authorization: Bearer xxx` |
| `request_template` | 请求体模板（Jinja2） | `{subject: "{task.input.subject}"}` |
| `response_mapping` | 响应字段映射 | `output_files_field: "files"` |
| `timeout` | 超时时间（秒） | `120` |
| `retries` | 重试次数 | `2` |
